# 亀井亜紀子（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年5月14日 |
| 年齢 | 60歳 |
| 選挙区 | 島根1区（松江市・安来市・雲南市・仁多郡・飯石郡・隠岐郡） |
| 当選回数 | 衆議院3期、参議院1期 |
| 主な役職 | 立憲民主党島根県連代表、元参議院議員 |

## 経歴
- 東京都目黒区出身、本籍は島根県津和野町
- 旧津和野藩主・亀井家の血筋、岩倉具視の来孫
- 父は元衆議院議員・亀井久興
- 1988年、学習院大学法学部政治学科卒業
- 1992年、カナダ・カールトン大学マスコミュニケーション学科卒業
- 長野オリンピック、日韓ワールドカップ等で通訳として活躍
- 亀井久興議員の政策秘書
- 2007年、参議院選挙で島根県選挙区から初当選（国民新党）
- 2017年、第48回衆院選島根1区で比例復活当選（立憲民主党）
- 2024年4月、衆院島根1区補選で当選
- 2024年10月、第50回衆院選島根1区で当選（3期目）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 参議院・衆議院で地方創生、農林水産、経済産業などの委員会に所属。「自民党王国」島根での野党議席獲得という明確な課題を設定し実現。

### ② 解決設計力: 4/5
**評価根拠**: 学習院大・カールトン大卒の学歴、通訳・秘書経験から政策立案能力は高い。衆参両院での経験を持ち、国政に精通。

### ③ 実行力: 5/5
**評価根拠**: 「自民党王国」島根で2024年補選・本選と連勝。自民党の牙城を崩した初の野党議員として歴史的勝利を達成。選挙での実行力は抜群。

### ④ 説明責任: 4/5
**評価根拠**: 地元密着型の活動で有権者への説明を継続。「島根を変える、日本が変わる」というメッセージを発信。県連代表として党の方針も説明。

### ⑤ 一貫性: 3/5
**評価根拠**: 国民新党→みんなの党→みどりの風→立憲民主党と政党を移籍。反自民・改革志向という軸はあるものの、所属政党の変遷がある。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。名門亀井家の出身だが、地道な地元活動で支持を獲得。清廉な政治姿勢。

## 強み・弱み

### 強み
- **選挙強さ**: 自民王国で野党初の小選挙区当選
- **家系**: 亀井家・岩倉家の血筋で地元の信頼
- **経験**: 衆参両院での国会経験
- **語学力**: 通訳経験から国際感覚
- **地元密着**: 県連代表として地域活動

### 弱み（注意点）
- **政党移籍**: 複数政党を経験
- **都会育ち**: 東京出身で地元密着度への懸念
- **野党**: 政策実現の機会が限定的
- **後継問題**: 亀井家の政治的継承への依存
- **党内基盤**: 立憲民主党内での影響力
