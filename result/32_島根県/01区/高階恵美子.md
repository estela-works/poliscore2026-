# 高階恵美子（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1963年12月21日 |
| 年齢 | 62歳 |
| 選挙区 | 島根1区（松江市・安来市・雲南市・仁多郡・飯石郡・隠岐郡） |
| 当選回数 | 衆議院1期（比例中国）、参議院2期 |
| 主な役職 | 元厚生労働副大臣、元参議院議員、看護師 |

## 経歴
- 宮城県仙台市出身
- 看護師・保健師資格取得
- 東北大学大学院医学系研究科博士課程修了
- 日本看護協会政策秘書室長
- 2010年、参議院選挙比例区で初当選
- 厚生労働大臣政務官（第2次安倍内閣）
- 厚生労働副大臣（第4次安倍第1次改造内閣）
- 2016年、参議院選挙比例区で再選
- 2021年、第49回衆院選比例中国ブロックで当選
- 2024年9月、島根1区支部長就任
- 2024年10月、第50回衆院選島根1区で落選、比例復活もならず

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 看護師・保健師出身で医療・福祉政策に精通。厚労副大臣として社会保障、医療政策の課題に取り組む。コロナ対応など国家的課題にも関与。

### ② 解決設計力: 4/5
**評価根拠**: 東北大大学院博士課程修了の学識。日本看護協会政策秘書室長として政策立案経験。厚労省との連携で政策設計能力は高い。

### ③ 実行力: 2/5
**評価根拠**: 参議院比例、衆議院比例では当選実績があるが、小選挙区での勝利経験がない。2024年島根1区では「知名度ゼロからのスタート」で敗北。比例復活もならず。

### ④ 説明責任: 3/5
**評価根拠**: 衆参約15年のキャリアで「即戦力」をアピール。ただし島根での活動期間が短く、地元への説明・浸透が不十分との指摘。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党で一貫して活動。看護・医療政策という専門分野を軸に活動。比例から小選挙区への転身は党の要請による。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 個人的なスキャンダルは報じられていないが、自民党の裏金問題の逆風を受けて敗北。党への逆風で保守票をまとめきれず。

## 強み・弱み

### 強み
- **専門性**: 看護師・保健師資格、医学博士
- **政府経験**: 厚労副大臣、政務官経験
- **国会経験**: 衆参約15年のキャリア
- **政策通**: 医療・福祉政策に精通
- **即戦力**: 党県連からの期待

### 弱み（注意点）
- **知名度不足**: 島根での知名度がゼロからのスタート
- **地元基盤**: 宮城県出身で島根との縁が薄い
- **選挙弱さ**: 小選挙区での当選経験なし
- **党の逆風**: 裏金問題の影響
- **落選**: 2024年小選挙区・比例とも議席なし
