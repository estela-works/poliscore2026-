# 下野幸助（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年11月7日 |
| 年齢 | 47歳 |
| 選挙区 | 三重県2区 |
| 当選回数 | 衆議院1回 |
| 主な役職 | 元三重県議会議員（4期）、立憲民主党三重県第2区総支部長 |

## 経歴
- 三重県鈴鹿市生まれ
- 鈴鹿市立白子小学校、鈴鹿市立鼓ヶ浦中学校卒業
- 国立鈴鹿工業高等専門学校 電気工学科卒業
- 国立豊橋技術科学大学 電気・電子工学課程卒業
- カナダに短期留学
- 内閣情報調査室勤務（情報収集衛星開発計画、北朝鮮ミサイル対応等）
- 2008年 慶應義塾大学大学院経営管理研究科修了（MBA取得）
- 2008年〜2010年 アイシン精機勤務
- 2011年 三重県議会議員初当選（鈴鹿市選挙区）
- 2011年〜2024年 三重県議会議員4期
- 2024年10月 衆院選三重2区で初当選（中川正春の後継）
- 名前の由来：松下幸之助から「幸」と「助」

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 内閣情報調査室での安全保障業務経験から、国の課題を構造的に把握する能力がある。北朝鮮ミサイル対応の実務経験は貴重。「特定の企業や一部の人だけが潤う現状」への問題意識を明確に持ち、物価高対策、中小企業支援、子育て政策など具体的な課題を設定。

### ② 解決設計力: 3/5
**評価根拠**: 県議4期の経験で地方行政の仕組みを熟知。MBAホルダーとして経営的視点も持つ。ただし国政レベルでの政策設計実績はこれから。内閣情報調査室での技術的バックグラウンドは政策立案に活かせる可能性がある。

### ③ 実行力: 4/5
**評価根拠**: 三重県議4期（13年半）を務め上げた実績。2024年衆院選では自民党現職を破り初当選。内閣情報調査室→民間企業→県議→国会議員と着実にキャリアを積み上げてきた。組織での実務経験と政治活動を両立させた実行力がある。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 県議として13年半の説明責任を果たしてきた実績。「自由と多様性を尊重し、支え合い、人間が基軸となる共生社会」という理念を明確に発信。政策活動費の廃止に賛成と明確に回答するなど、政治資金の透明化への姿勢は明確。

### ⑤ 一貫性: 4/5
**評価根拠**: 民主党系（民主党→民進党→立憲民主党→中道改革連合）を一貫して支持。岡田克也に感化されて政治家を目指し、その路線を維持。中川正春の後継として三重2区を継承。政党移動はあるが、いずれも政党再編によるもので、思想的な転向はない。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。政策活動費廃止に明確に賛成。内閣情報調査室という機密性の高い組織での勤務経験があり、情報管理への倫理観は高いと推測される。松下幸之助の名を冠する人物として、清廉な政治への意識がうかがえる。

## 強み・弱み

### 強み
- **安全保障の専門知識**: 内閣情報調査室で情報収集衛星開発、北朝鮮ミサイル対応に従事
- **高い学歴・資格**: 高専→技科大→慶應MBAという技術と経営の両立
- **県議4期の実績**: 13年半の地方議員経験、地域事情への深い理解
- **若さと経験のバランス**: 47歳で国政経験を積み始める適齢期

### 弱み（注意点）
- **国政経験の浅さ**: 衆院議員1期目、国政での実績はこれから
- **知名度不足**: 全国的な認知度は低い
- **中川正春の後継**: 前任者の地盤を継承した形であり、独自色の確立が課題
- **選挙区の激戦**: 川崎秀人との接戦が続く選挙区

## 主な政策主張
- 物価高対策
- 中小企業支援
- 子ども子育て政策
- 政策活動費の廃止
- 自由と多様性を尊重する共生社会の実現
