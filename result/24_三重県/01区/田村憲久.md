# 田村憲久（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年12月15日 |
| 年齢 | 59歳 |
| 選挙区 | 三重県1区 |
| 当選回数 | 衆議院10回 |
| 主な役職 | 元厚生労働大臣（2回）、元働き方改革担当大臣、自民党政務調査会長代行、社会保障を守る会代表 |

## 経歴
- 三重県松阪市出身
- 三重県立松阪高等学校卒業
- 千葉大学法政経学部経済学科卒業
- 1987年日本土建入社
- 1994年伯父・田村元（元衆議院議長）の秘書
- 1996年衆議院議員初当選
- 文部科学大臣政務官、厚生労働大臣政務官
- 総務副大臣
- 2012年-2014年厚生労働大臣（第2次安倍内閣）
- 2020年-2021年厚生労働大臣（菅内閣）、働き方改革担当大臣
- 衆議院厚生労働委員長
- 裁判官訴追委員会委員長（2回）
- 2023年宏池会（岸田派）副会長就任
- 2024年衆院選で10選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 厚生労働大臣として社会保障政策の課題に取り組む。新型コロナ対策、働き方改革など重要課題を担当。「社会保障を守る会」代表として社会保障制度の持続可能性を訴える。

### ② 解決設計力: 4/5
**評価根拠**: 千葉大学法政経学部卒。厚生労働大臣を2回務め、政策立案の実績が豊富。新型コロナワクチン接種体制の構築など大規模政策を実行。

### ③ 実行力: 4/5
**評価根拠**: 10期連続当選という安定した選挙力。厚生労働大臣として新型コロナ対策を指揮。働き方改革担当大臣として法案成立に尽力。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 厚労大臣として記者会見で政策を説明。テレビ討論にも積極的に出演。ただしコロナ対策への批判も受けた。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党で一貫して活動。社会保障分野の専門家として継続的に政策活動。2023年に宏池会入会も政策路線は継続。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 大きなスキャンダルは確認されていない。統一教会との関係についてはアンケートで「一切関係ない」と回答。ただし世界日報社の出版記念会には出席した記録あり。

## 強み・弱み

### 強み
- **専門性**: 厚生労働大臣2回、社会保障政策の専門家
- **安定した選挙力**: 10期連続当選
- **政策実績**: 新型コロナ対策、働き方改革
- **党内地位**: 政調会長代行、宏池会副会長
- **家系**: 伯父は田村元・元衆議院議長

### 弱み（注意点）
- **コロナ対策批判**: 厚労大臣時代の対応への批判も
- **派閥政治**: 宏池会所属
- **世界日報社関連**: 出版記念会に出席
