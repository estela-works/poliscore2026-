# 津島淳（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年10月18日 |
| 年齢 | 58歳 |
| 選挙区 | 青森1区 |
| 当選回数 | 衆議院5回 |
| 主な役職 | 内閣府副大臣、法務副大臣、衆議院財務金融委員長、自民党青森県連会長 |

## 経歴
- フランス・パリ生まれ（父の在フランス日本国大使館出向中）
- 祖父は作家・太宰治（津島修治）、父は元自民党衆議院議員・津島雄二
- 千代田区立番町小学校、麹町中学校、学習院高等科を経て学習院大学文学部国文学科卒業
- 1991年、株式会社関電工入社（14年間勤務）
- 2005年、父・津島雄二の秘書に
- 2009年、衆院選青森1区に無所属で出馬、落選
- 2012年、自民党公認で青森1区から初当選
- 2015年、国土交通大臣政務官兼内閣府大臣政務官
- 2021年、法務副大臣（第1次岸田内閣）
- 2024年、衆院選青森1区で5選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
法務副大臣として成年年齢引下げ、人権擁護、出入国管理など多岐にわたる課題に取り組んだ。地域公共交通再構築議連の事務局長として、地方の交通問題という構造的課題を把握し政策提言を行っている。縄文遺跡群世界遺産登録推進議連の幹事長など、地域振興の観点からも課題を設定できている。ただし、国政全体を変えるような独自の課題提起は限定的。

### ② 解決設計力: 4/5
**評価根拠**:
国土交通大臣政務官として交通政策に関わり、法務副大臣として法制度の設計に携わった実績がある。地域公共交通の再構築に向けた具体的な政策提言を国土交通大臣に提出するなど、制度・予算を含めた設計能力を示している。ただし、画期的な制度設計というよりは既存の枠組み内での改善が中心。

### ③ 実行力: 4/5
**評価根拠**:
衆議院5期当選の実績があり、法務副大臣、国土交通大臣政務官など複数の政府ポストで実務を遂行。2024年衆院選では自民党への逆風の中でも小選挙区で勝利。党内での地位を活かし、議員連盟活動を通じて政策を推進している。ただし、個人として突破した大きな政策実績は限定的。

### ④ 説明責任: 4/5
**評価根拠**:
2024年衆院選では「政治とカネ」問題への批判に対し、「自民党を根本から変える」「政治改革を断行する」と明確に説明。旧統一教会との関係については「会合出席以降、関わりは一切ない」「不適切な団体との付き合いには気を付けたい」と説明責任を果たしている。法務副大臣として「共感と寛容の共生社会づくり」というビジョンを発信。

### ⑤ 一貫性: 4/5
**評価根拠**:
2012年初当選以来、自民党一筋で政党を変えていない。父の地盤を継承し青森1区に一貫して専念。法務・国土交通分野で継続的に活動。2024年総裁選では石破茂を支持するなど、党内での立場も明確。世襲批判を受けながらも長期間にわたり地域の代表として活動を継続している。

### ⑥ 倫理観: 4/5
**評価根拠**:
裏金問題への関与は報じられていない。旧統一教会関連では2017年の「日韓トンネル推進青森県民会議」出席が指摘されているが、本人が認め説明し、以後の関与はないと明言。政治資金面での重大な問題は確認されていない。権力の私物化や差別的言動も見られない。

## 強み・弱み

### 強み
- **安定した政策実行力**: 法務副大臣・国交政務官など政府ポストで着実に実務を遂行
- **地域密着**: 青森1区で5期連続当選、地元の交通・観光・産業振興に継続的に取り組む
- **党内基盤**: 自民党青森県連会長として県内での調整力を持つ

### 弱み（注意点）
- **独自色の不足**: 父・津島雄二の後継として、独自の政策ビジョンや突破力には限界がある
- **世襲批判**: 太宰治を祖父に持つ名門出身だが、その恵まれた環境が庶民感覚との乖離を生む可能性
