# 升田世喜男（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1957年6月5日 |
| 年齢 | 67歳 |
| 選挙区 | 青森1区（比例東北ブロック復活） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 立憲民主党青森県連代表、グリーンハート取締役会長 |

## 経歴
- 青森県小泊村（現・中泊町）生まれ
- 青森県立木造高等学校卒業
- 拓殖大学入学後中退
- 1987年、小泊村議会議員初当選（3期）
- 1999年、青森県議会議員初当選（2期）
- 2003年、県議会農林水産常任委員長
- 2005年、衆院選青森1区に無所属で出馬、落選
- 2009年、衆院選青森1区に無所属で出馬、落選
- 2020年、立憲民主党青森1区支部長就任
- 2021年、衆院選で小選挙区落選、比例も次点で落選
- 2024年、衆院選で小選挙区落選、比例東北で復活当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
「東京一極集中から地方分散へ」「中央集中型国家から地方分権型国家へ」という構造的な課題設定ができている。農林水産業を「市場原理だけに任せず社会共通資本とする」という視点は、地方経済の本質的問題を捉えている。自民党一強政治の問題点として「裏金問題」に至った構造を指摘し、与野党伯仲の必要性を主張。ただし、課題設定は野党議員として一般的な範囲内。

### ② 解決設計力: 4/5
**評価根拠**:
農業戸別所得補償制度（15,000円〜20,000円）の復活・拡充、地方企業への法人税軽減、本社移転企業への法人税免除（10年間）など、具体的な数値目標を含む政策設計を提示。教育無償化も「学校給食→高校→大学」と段階的なロードマップを示している。ただし、財源確保策や実現可能性についての詳細な説明は不足。

### ③ 実行力: 2/5
**評価根拠**:
2005年、2009年、2021年と衆院選に何度も落選し、国政での実績を積む機会が限られていた。2024年にようやく比例復活で衆議院議員となったが、これまでの国会議員としての実行実績は1期のみ。地方議員時代（村議3期、県議2期）の実績はあるが、国政レベルでの政策実現力は未証明。

### ④ 説明責任: 4/5
**評価根拠**:
公式サイトで政策を詳細に説明し、「自民党一強政治が傲りを作って裏金問題につながった」と明確に問題点を指摘。地方出身者として津軽弁で国会発言するなど、親しみやすいコミュニケーションスタイルを持つ。政策変更の説明も行っており、説明責任への姿勢は評価できる。

### ⑤ 一貫性: 4/5
**評価根拠**:
村議から県議、そして国政へと一貫して「地方」の視点を持ち続けている。農林水産業支援、地方分権という軸は長期間維持。政党所属については無所属から立憲民主党へ移ったが、政策の方向性は変わっていない。繰り返しの落選にも関わらず同じ選挙区で挑戦を続ける粘り強さがある。

### ⑥ 倫理観: 4/5
**評価根拠**:
政治資金問題やスキャンダルは確認されていない。裏金問題を批判する立場から、自らも透明性を重視していると推測される。地方議員時代から長年にわたり政治活動を行っているが、重大な倫理的問題は報じられていない。兄がビーインググループ社長だが、利益相反等の問題は指摘されていない。

## 強み・弱み

### 強み
- **地方への深い理解**: 小泊村議、県議を経験し、農林水産業や地方経済の実情を熟知
- **粘り強さ**: 複数回の落選にも関わらず挑戦を継続、2024年についに国政復帰
- **具体的な政策**: 戸別所得補償など数値目標を含む現実的な政策提案

### 弱み（注意点）
- **国政実績の不足**: 衆議院議員としての経験が浅く、法案成立等の具体的成果がまだ少ない
- **高齢**: 67歳で今後の活動期間に限界がある可能性
- **小選挙区での弱さ**: 2024年も小選挙区では敗北し比例復活、地元での浸透度に課題
