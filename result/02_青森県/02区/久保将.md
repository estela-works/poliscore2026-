# 久保将（日本共産党）

## 基本情報
| 項目 | 内容 |

| 年齢 | 71歳 |
| 選挙区 | 青森2区 |
| 当選回数 | 衆議院0回（八戸市議1期） |
| 主な役職 | 日本共産党三八地区委員長、元八戸市議会議員 |

## 経歴
- 青森県八戸市出身
- 1973年〜2016年：八戸市役所に勤務（43年間）
- 2019年：八戸市議会議員選挙に立候補、初当選（1期）
- 2022年：日本共産党県委員、三八地区委員長に就任
- 2024年10月：第50回衆議院議員総選挙に青森2区から出馬、落選
- 2026年：第51回衆議院議員総選挙に青森2区から再出馬予定

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 共産党の方針に沿い、「防衛費増など戦争の準備が進んでいる」状況への問題意識を示し、反戦・平和を軸に課題を設定している。「自民党政治を終わりにしてほしい」という国民の声を代弁する姿勢は見られるが、独自の構造的分析や本質的な問題提起は確認できない。党の基本方針を代弁する段階にとどまっている。

### ② 解決設計力: 2/5
**評価根拠**: 減税政策を掲げ、青森2区の地域課題（三沢米軍基地、十和田の農業など）への言及はあるものの、具体的な政策パッケージや制度設計の提案は確認できない。共産党の全国的な政策を地域で訴える形であり、独自の解決策設計は見られない。市役所での長い勤務経験は行政実務の知識につながるが、政策設計力としては未発揮。

### ③ 実行力: 2/5
**評価根拠**: 八戸市議を1期務めた実績があり、完全な新人ではない。しかし、国政での実績はなく、2024年衆院選で落選。市議としての具体的な政策実現実績も限定的。野党・共産党所属のため、政策を法案として成立させる機会は構造的に少ないが、その中でも目立った実績は確認できない。新人に近い評価とする。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2024年衆院選では「短期間の選挙戦だったので、党の政策や自身の公約が有権者に伝わりきらなかったかもしれない」と敗戦の弁で述べており、発信力・説明力の課題を自覚している。「もう少し時間があれば票を伸ばせた」との反省は率直だが、メディア対応力や反対意見への説明力を示す場面は確認できていない。

### ⑤ 一貫性: 4/5
**評価根拠**: 共産党員として長年活動し、三八地区委員長として党組織を率いている。反戦・平和、減税という党の基本方針を一貫して主張。2024年落選後も同じ主張で再出馬を決意しており、政治姿勢にブレは見られない。政党移籍や政策の大幅変更はなく、一貫性は評価できる。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は報じられていない。43年間の市役所勤務を全うし、その後市議として活動する誠実な経歴。共産党員として権力監視の姿勢を持つ。権力行使の経験が限られるため高評価の根拠は限定的だが、現時点で懸念材料はない。

## 強み・弱み

### 強み
- **地元での長い活動**: 八戸市役所43年、市議1期と地元での活動歴が長い
- **行政実務の知識**: 市役所勤務で培った行政の実務的知見
- **一貫した姿勢**: 共産党員として反戦・平和を長年主張
- **地域課題への理解**: 三沢基地、十和田農業など選挙区の特性を把握

### 弱み（注意点）
