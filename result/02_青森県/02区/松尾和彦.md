# 松尾和彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1963年5月9日 |
| 年齢 | 62歳 |
| 選挙区 | 青森2区 |
| 当選回数 | 衆議院0回（県議4期、町長2期） |
| 主な役職 | 元三戸町長、元青森県議会議員 |

## 経歴
- 青森県三戸郡出身
- 青森県立三戸高等学校卒業
- 1986年：青森大学経営学部卒業
- 参議院議員・松尾官平（4期）の義理の息子
- 1995年：義父の参議院副議長就任に伴い、副議長秘書官に就任
- 1999年：青森県議会議員選挙（三戸郡選挙区）に無所属で出馬、落選
- 2003年：青森県議会議員選挙で初当選（1期目）
- 2007年：民主党公認で県議選再選（2期目）
- 2011年：県議3選
- 2015年：県議4選
- 2016年12月：三戸町長選挙で現職を54票差で破り初当選
- 2020年：三戸町長再選（2期目）
- 2024年10月：衆議院議員総選挙に青森2区から無所属で出馬、落選
- 2025年9月：立憲民主党から次期衆院選の公認内定

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「次世代が主導する地方創生」の必要性を強く訴え、人口減少や産業の衰退といった課題を「未来への可能性の種」として捉える視点を持つ。県議4期・町長2期の経験から、地方自治体が直面する構造的問題を深く理解している。「自民党一強多弱を変えなければならない」という国政の構造的課題への問題提起も行っている。地方の視点から国政の課題を語れる点は評価できる。

### ② 解決設計力: 4/5
**評価根拠**: 5つの政策提案（教育の地域格差是正、社会保障制度改革、予防医療強化、医療のIT化、ジェネリック医薬品促進など）を具体的に提示。世界中の大学を対象とした給付型奨学金、大学までの完全無償化支援など、制度・財源を意識した提案ができている。町長として2期8年間、実際に政策を設計・実行した経験は、机上の空論ではない設計力を示す。

### ③ 実行力: 4/5
**評価根拠**: 県議4期16年、町長2期8年と、地方政治で24年以上の実績がある。2016年町長選では現職を僅差で破り、2020年には再選。地方自治体のトップとして予算編成、政策実行を経験しており、限られた条件下での実行力は実証済み。ただし、国政での実績はなく、2024年衆院選では落選している。地方での実行力と国政での実行力は異なる可能性がある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2024年衆院選では「私の不徳の致すところ」と敗戦を認めたが、無所属での出馬となった経緯や政策の詳細説明について、十分な発信ができていなかった可能性がある。町長・県議としての説明責任は果たしてきたと思われるが、国政レベルでの発信力、メディア対応力は未知数。全国的な知名度も低く、有権者への浸透に課題がある。

### ⑤ 一貫性: 4/5
**評価根拠**: 1999年から25年以上、三戸郡を基盤に政治活動を継続。民主党系として県議を務め、町長として地元に根ざした活動を行ってきた。「次世代への投資」「地方創生」という軸は一貫している。2024年は無所属で出馬したが、2025年に立憲民主党の公認を得ており、政党の軸は民主党系で概ね一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 県議4期、町長2期を通じて、スキャンダルや政治資金問題は報じられていない。参院議員の義父を持つ「地盤」はあるが、自ら選挙を勝ち抜いてきた実績もある。町長として行政のトップを務め、透明性を持った行政運営を行ってきたと推測される。権力の私物化などの懸念材料はない。

## 強み・弱み

### 強み
- **豊富な地方政治経験**: 県議4期・町長2期と、地方自治で24年以上の実績
- **行政トップの経験**: 町長として予算編成、政策決定を経験
- **地元での強い基盤**: 三戸郡を中心に長年活動、地域からの信頼
- **具体的な政策提案**: 教育・社会保障など具体的な制度設計を提示

### 弱み（注意点）
- **国政経験なし**: 衆議院での活動は未知数、2024年選挙で落選
- **発信力の弱さ**: 全国的な知名度が低く、メディア対応力に課題
- **選挙区の広さ**: 三戸町以外の地域（八戸市など）での浸透が課題
- **世襲的イメージ**: 義父が参院議員であり、世襲批判を受けるリスク
