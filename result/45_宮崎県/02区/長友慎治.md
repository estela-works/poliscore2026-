# 長友慎治（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年6月22日 |
| 年齢 | 48歳 |
| 選挙区 | 宮崎県2区 |
| 当選回数 | 衆議院2回（比例復活） |
| 主な役職 | 国民民主党政調副会長、宮崎・熊本両県連代表 |

## 経歴
- 宮崎県宮崎市出身
- 宮崎県立宮崎南高等学校卒業
- 早稲田大学法学部卒業（2002年）
- 大学卒業後、フリーランス記者・ライターとして活動
- コンテンツ制作会社（株）「囲炉裏」代表取締役（2009年設立）
- （株）博報堂ケトル入社、PRコンサル業務担当（2012年）
- 日向市産業支援センター「ひむか-Biz」センター長（2016年〜2020年）
- 「宮崎ひなた食べる通信」創刊（2019年）
- NPO法人「フードバンク日向」理事長
- 2020年衆議院選挙出馬を決意しセンター長辞任
- 2021年衆院選初当選（比例九州ブロック）
- 2024年衆院選2選（比例復活）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 地方創生、中小企業支援、農林水産業振興、子どもの貧困、フードロスなど現場視点での課題設定ができる。ひむか-Bizセンター長として中小企業の課題に向き合い、フードバンク活動を通じて食料問題にも取り組む。国会では103万円の壁、ガソリン減税、物流2024年問題など具体的な政策課題を提起。

### ② 解決設計力: 4/5
**評価根拠**: 早稲田大学法学部卒、博報堂ケトルでのPRコンサル経験を持ち、問題解決のプロフェッショナル。ひむか-Bizでは中小企業支援の実績を上げた。国会では農業法人育成、米粉利用促進、物流効率化の補助金拡充など具体的な政策提案を行っている。

### ③ 実行力: 3/5
**評価根拠**: 2期連続当選を果たしているが、いずれも比例復活で小選挙区では勝利していない。2024年衆院選でも江藤拓に敗北。ただし、ひむか-Bizセンター長やフードバンク理事長として地域での実績は上げている。国民民主党政調副会長として政策立案に関与。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 記者・ライター出身、博報堂でのPR経験から高いコミュニケーション能力を持つ。「宮崎ひなた食べる通信」創刊など情報発信にも積極的。国会質問でも論理的に政策を提案している。SNSでの発信も活発。

### ⑤ 一貫性: 4/5
**評価根拠**: 地方創生、中小企業支援、農業振興という軸で一貫して活動。国民民主党で継続的に活動し、宮崎・熊本両県連代表を務める。政党移籍や方針転換はない。民間経験を活かした政策提言を続けている。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。フードバンク活動など社会貢献にも取り組む。クリーンなイメージを維持している。野党議員として権力に対する批判的視点も持つ。

## 強み・弱み

### 強み
- **民間経験**: 博報堂ケトル、起業経験など豊富なビジネス経験
- **地方創生の実績**: ひむか-Bizセンター長として中小企業支援
- **社会貢献活動**: フードバンク日向理事長
- **政策立案力**: 具体的な政策提案ができる

### 弱み（注意点）
- **小選挙区での勝利なし**: 2回とも比例復活
- **知名度**: 全国的な認知度は限定的
- **野党**: 政権運営の経験がない
- **対立候補の強さ**: 江藤拓が強固な地盤を持つ
