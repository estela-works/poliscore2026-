# 古川禎久（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年8月3日（60歳） |
| 年齢 | 60歳 |
| 選挙区 | 宮崎3区 |
| 当選回数 | 衆議院8期 |
| 主な役職 | 自由民主党幹事長代理、元法務大臣、元財務副大臣 |

## 経歴
- 宮崎県日南市出身
- 東京大学法学部卒業
- 建設省（現国土交通省）入省
- 1996年、第41回衆議院選挙に新進党公認で宮崎3区から出馬、落選
- 2000年、第42回衆議院選挙に無所属で出馬、落選
- 2003年、第43回衆議院選挙に無所属で初当選、追加公認で自民党入党
- 2005年、郵政民営化法案に反対票を投じ自民党を離党、無所属で2選
- 2006年、自民党に復党
- 2011年、衆議院財務金融委員会委員長
- 2013-2014年、財務副大臣（第2次安倍内閣）
- 2021年10月、法務大臣就任（第1次・第2次岸田内閣）
- 2022年8月、法務大臣退任
- 2024年1月、平成研究会を退会（派閥解消を主張）
- 2025年10月、自由民主党幹事長代理就任

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 法務大臣として「世界一安全な国・日本」の実現、外国人材の円滑な受入れと在留管理の徹底、司法制度改革などを課題として掲げた。差別や虐待のない社会の実現、犯罪被害者支援、再犯防止など幅広い課題に取り組んだ。ただし、独自の課題設定というよりは内閣の方針に沿った取り組みが中心。

### ② 解決設計力: 3/5
**評価根拠**: 財務副大臣、法務大臣など要職を歴任し政策実務経験は豊富。入管法改正案について「様々な方策を組み合わせ、パッケージで送還忌避・長期収容問題の解決を図る」と説明するなど、制度設計への理解はある。ただし、独自の立法や政策設計の実績は確認できない。

### ③ 実行力: 4/5
**評価根拠**: 衆議院8期の実績、法務大臣・財務副大臣など閣僚経験。2005年に郵政民営化反対で造反し離党後も無所属で当選、その後自民党に復党し要職を歴任するなど政治的実行力がある。ウクライナ避難民受入れでは「法務省が日本政府のエンジンとして取り組んでいる」と積極的に推進。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 法務大臣として定例記者会見を行い、政策について説明。ただし、旧統一教会問題に関する記者質問には「捜査機関の活動内容に関わる事柄なのでお答えは差し控える」と形式的な対応。派閥解消を主張して平成研究会を退会したが、その詳細な説明は限定的。

### ⑤ 一貫性: 3/5
**評価根拠**: 2005年に郵政民営化反対で造反し離党するなど、信念を貫く姿勢を見せた。外国人参政権反対、人権擁護法案反対、首相の靖国参拝支持など保守的な政策は一貫。一方で、離党後に自民党に復党しており、政党への帰属は変遷がある。派閥の裏金問題を受けて派閥解消を主張した姿勢は評価できる。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 政治資金パーティー収入の裏金問題を受け、全派閥の解消を主張し平成研究会を退会するなど、自浄作用を示す行動をとった。重大なスキャンダルや問題行動は確認されていない。ただし、旧統一教会問題への対応では踏み込んだ対応を避けた。死刑制度について「国民世論の多数が支持している」として維持を支持。

## 強み・弱み

### 強み
- **豊富な閣僚経験**: 法務大臣、財務副大臣など要職を歴任
- **選挙での強さ**: 衆議院8期連続当選、無所属でも勝利
- **行政経験**: 建設省（国交省）出身で官僚経験あり
- **派閥問題への姿勢**: 裏金問題を受け派閥解消を主張

### 弱み（注意点）
- **独自政策の不明確さ**: 独自の立法や政策提言の実績が見えにくい
- **説明責任への姿勢**: 記者質問に対して形式的な回答が多い
- **政党帰属の変遷**: 離党・復党の経歴がある
