# 福田昭夫（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1948年4月17日 |
| 年齢 | 76歳 |
| 選挙区 | 栃木県2区（鹿沼市、日光市、さくら市、宇都宮市の一部、塩谷郡） |
| 当選回数 | 衆議院7回 |
| 主な役職 | 総務大臣政務官（野田内閣）、栃木県知事（第15代）、今市市長（第9代）、立憲民主党栃木県連代表 |

## 経歴
- 栃木県河内郡豊岡村（現・日光市）生まれ
- 栃木県立今市高等学校卒業
- 東北大学教育学部教育学科卒業（在学中は学生運動に参加）
- 1971年、今市市役所入庁
- 1991年、今市市財政課長を最後に退職
- 1991年、今市市長選に当選（第9代市長）
- 2000年、栃木県知事選に当選（第15代知事、1期）
- 2005年、衆院選栃木2区から出馬、比例復活で初当選
- 2009年、栃木2区で再選（森山眞弓の後継を破る）
- 2011年、野田内閣で総務大臣政務官就任
- 2012年、消費増税法案に反対し政務官辞任
- 2024年10月、第50回衆院選で7選達成（68,377票）

## 評価詳細

### ① 課題設定能力（4/5）
消費税問題について「直間比率の見直しを大義名分に消費税をつくったが、行き過ぎた直間比率の見直しで不公平税制をつくり、経済成長を阻害し、超格差社会をつくった」と構造的な問題を指摘。輸出企業への消費税還付（戻し税）問題など、一般的に見過ごされがちな論点を追及している。県知事・市長経験を持ち、地方自治の視点からの課題設定能力も高い。

### ② 解決設計力（4/5）
消費税減税の代替財源として「大企業の法人税と富裕層の所得税（金融所得を含む）に担税力に応じて負担を求める」という具体的な制度設計案を提示。「不公平税制の抜本改革で消費減税の財源をつくる会」を党内有志で設立し、法律案をまとめるなど、実際の立法作業に取り組んでいる。県知事時代の行政経験を活かした制度設計能力を持つ。

### ③ 実行力（4/5）
今市市長1期、栃木県知事1期、衆議院議員7期と、行政・立法両面で長期にわたる活動実績がある。総務大臣政務官として行政実務も経験。消費増税法案に反対して政務官を辞任するなど、信念に基づいた行動も取っている。ただし、消費税減税という自身の最重要課題は実現に至っていない。

### ④ 説明責任（4/5）
国会で消費税問題を継続的に追及し、複雑な税制の問題点をわかりやすく説明しようとしている。「財務省のウソ」「政府は戻し税を隠し通している」など、強い言葉で問題提起を行い、有権者への説明努力を続けている。不都合な質問に対しても論理的に反論する姿勢が見られる。

### ⑤ 一貫性（4/5）
消費税増税反対の姿勢は2012年の政務官辞任以来、一貫している。政党については民主党→民進党→立憲民主党と変遷しているが、これは党の分裂・合流に伴うものであり、本人の意思による離党ではない。消費税減税、地方分権という政策軸は長期間維持されている。

### ⑥ 倫理観（4/5）
知事・市長・国会議員を通じて、大きなスキャンダルや不正行為は報じられていない。2012年には自身の信念（消費増税反対）のために政務官ポストを辞任するなど、権力よりも政策を優先する姿勢を示した。政治資金に関する問題も確認されていない。

## 強み・弱み

### 強み
- **豊富な行政経験**：市長・知事・国会議員と多層的な行政経験を持つ
- **政策の専門性**：税制、特に消費税問題について深い知見と継続的な取り組み
- **信念に基づく行動**：消費増税反対で政務官辞任など、信念を貫く姿勢
- **地元密着**：栃木県での長年の政治活動により、強固な地盤を構築

### 弱み
- **年齢**：76歳と高齢であり、次世代への継承が課題
- **実現困難な政策**：消費税減税は党内でも主流ではなく、実現の見通しが立たない
- **党内影響力**：有志の会を立ち上げるも、党の方針を変えるには至っていない
