# 藤岡隆雄（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年3月28日 |
| 年齢 | 47歳 |
| 選挙区 | 栃木県4区（小山市、下野市の一部、下都賀郡、真岡市、芳賀郡） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 立憲民主党副幹事長、財務金融委員会理事、北朝鮮拉致問題特別委員会理事 |

## 経歴
- 愛知県出身、現在は小山市在住
- 1999年、大阪大学基礎工学部卒業
- 大阪大学大学院在学中に国家公務員I種試験（経済職）合格
- 2001年、金融庁入庁（約9年間勤務）
- 2008年、総務企画局企業開示課課長補佐
- 2010年、衆議院議員政策担当秘書（渡辺喜美代表）
- 2012年、衆院選栃木4区初出馬（次点、得票率22.8%）
- 2014年、2度目の挑戦（次点、得票率32.2%）
- 2017年、3度目の挑戦（次点、得票率37.4%）
- 2018年、立憲民主党栃木4区支部長就任
- 2021年、4度目の挑戦で初当選（比例復活、惜敗率95.7%）
- 2024年10月、5度目の挑戦で小選挙区初当選（96,573票、得票率53.4%）

## 評価詳細

### ① 課題設定能力（4/5）
金融庁で金融商品取引法を担当した経験を活かし、金融政策や中小企業支援の課題を構造的に理解している。コロナ禍での中小企業の過剰債務問題、インボイス制度の問題点、ウクライナ避難者の法的地位など、多角的な課題設定能力を示している。NPO法人から「三ツ星議員」として表彰されるなど、課題設定・政策立案能力は評価されている。

### ② 解決設計力（4/5）
金融庁での制度設計経験を活かし、具体的な法案作成に取り組んでいる。議員立法を5本提出（中小事業者のコロナ債務減免法案、特定医薬品特措法案、インボイス制度廃止法案、ウクライナ避難者特例法案、観光産業事業継続支援金支給法案）しており、立法能力の高さを示している。財務金融委員会理事として政策の制度設計に深く関与。

### ③ 実行力（4/5）
4度の落選を経て5度目で小選挙区初当選を果たした粘り強さは特筆に値する。当選後は国会質疑に少なくとも68回立ち、議員立法5本を提出するなど、精力的に活動。田原総一朗氏のNPO法人から2年連続で「三ツ星議員」表彰を受けている。2024年選挙では得票率53.4%で佐藤勉元総務会長を破る実績。

### ④ 説明責任（4/5）
国会での質疑を積極的に行い、政策課題を有権者に説明しようとする姿勢が見られる。財務金融分野の複雑な問題についても、わかりやすく説明する努力をしている。地元でのPTA活動、ライオンズクラブ会長など地域活動を通じて、有権者との対話も継続している。

### ⑤ 一貫性（4/5）
みんなの党から民主党、そして立憲民主党へと政党は変遷しているが、これは党の解散・合流に伴うものであり、自らの意思による離党ではない。2012年から一貫して栃木4区で活動を続け、4度の落選にもめげず挑戦を継続。金融政策、中小企業支援という政策軸も維持されている。

### ⑥ 倫理観（4/5）
スキャンダルや不正行為は報じられておらず、清廉な政治活動を続けている。地域でのPTA活動、子ども会育成会会長など、私生活でも社会貢献活動に取り組んでいる。政治資金に関する問題も確認されていない。権力を求めるというより、政策実現のために活動する姿勢が見られる。

## 強み・弱み

### 強み
- **専門性**：金融庁出身で金融政策に精通、議員立法能力が高い
- **粘り強さ**：4度落選しても諦めず、5度目で小選挙区当選を果たした
- **評価**：「三ツ星議員」2年連続受賞という客観的評価
- **地域活動**：PTA、ライオンズクラブなど地域に根付いた活動

### 弱み
- **党内基盤**：立憲民主党内での影響力は副幹事長とはいえ限定的
- **全国的知名度**：地元では強いが、全国的な知名度は低い
- **政策実現**：議員立法は提出したが、成立には至っていない
