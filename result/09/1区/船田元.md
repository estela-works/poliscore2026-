# 船田元（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1953年11月22日 |
| 年齢 | 71歳 |
| 選挙区 | 栃木県1区（宇都宮市の一部、河内郡） |
| 当選回数 | 衆議院14回 |
| 主な役職 | 経済企画庁長官、衆議院憲法審査会筆頭幹事、自民党憲法改正推進本部長、自民党総務会長代理 |

## 経歴
- 宇都宮市生まれ、宇都宮大学附属小中学校から栃木県立宇都宮高等学校
- 慶應義塾大学経済学部卒業
- 1979年、祖父・船田中の地盤を引き継ぎ、25歳で史上最年少当選（第35回衆院選）
- 1992年、経済企画庁長官就任（39歳、戦後最年少の閣僚）
- 1993年、宮澤内閣不信任案に賛成し閣僚辞任、新生党を結党
- 1996年、新進党離党後、無所属で当選
- 1997年、自民党に復党
- 2000年、民主党・水島広子に敗北し落選
- 2003年、国政復帰（以降連続当選）
- 2024年10月、第50回衆院選で14選達成

## 評価詳細

### ① 課題設定能力（4/5）
憲法改正推進本部長として、憲法改正に関する論点を体系的に整理し、党内議論をリードしてきた。科学技術政策にも精通し、すばる望遠鏡の建設を推進するなど、長期的視点での課題設定能力を示している。経済企画庁長官として経済政策の課題を把握してきた経験も豊富。ただし、近年は憲法改正以外の分野での新たな課題提起は限定的。

### ② 解決設計力（4/5）
憲法改正に関しては、自衛隊明記、緊急事態条項など具体的な条文案を含む設計を行ってきた。経済企画庁長官時代には経済政策の制度設計に携わった。科学技術分野でも、すばる望遠鏡や次世代望遠鏡TMT計画の推進において具体的な予算・制度設計を主導。長年の経験に基づく制度設計能力は高い。

### ③ 実行力（4/5）
当選14回という長期にわたる議員活動の中で、経済企画庁長官、衆議院文教委員会委員長、消費者問題特別委員会委員長など要職を歴任し、それぞれの任務を遂行してきた。すばる望遠鏡の建設は実際に完成させた。ただし、憲法改正という自身の最重要課題については、長年の取り組みにもかかわらず実現に至っていない。

### ④ 説明責任（2/5）
2023年の障害児通所施設「ピルエット」における補助金不正受給問題について、約640万円の返還を求められる事態となったが、自身の関与や責任について十分な説明がなされていない。2024年の裏金問題では茂木派を離脱したが、離脱理由の説明は「派閥解消を主張」という表面的なもの。2025年には「総裁選やり直し」発言で公明票依存を露呈し、批判を受けた。

### ⑤ 一貫性（2/5）
1993年に自民党を離党し新生党結党、その後新進党を経て1996年に無所属、1997年に自民党復党という政党遍歴がある。離党時は「改革」を掲げたが、復党後は党主流派に復帰。1996年の「政界失楽園」スキャンダル後の転身も含め、政治的立場の変遷が多い。憲法改正への姿勢は一貫しているものの、それ以外の分野では時流に応じた立場変更が見られる。

### ⑥ 倫理観（2/5）
2023年、理事長を務める障害児通所施設で補助金不正受給が発覚（約640万円返還）。1996年の「政界失楽園」スキャンダル（妻子ある身での不倫報道）は当時大きな社会問題となった。2024年には政治資金問題を受けて茂木派を離脱したが、自身の透明性確保への取り組みは不十分。長年の政治活動の中で、ガバナンス面での問題が複数回発生している点は減点要因。

## 強み・弱み

### 強み
- **豊富な経験と人脈**：当選14回、閣僚経験を持ち、党内外に幅広い人脈を有する
- **憲法・科学技術への専門性**：憲法改正論議や科学技術政策で一貫した取り組み
- **政治的生存能力**：スキャンダルや落選を乗り越え、長期にわたり議員活動を継続

### 弱み
- **説明責任の不足**：補助金問題、裏金問題など、問題発生時の説明が不十分
- **過去のスキャンダル**：「政界失楽園」のイメージが払拭できていない
- **政党遍歴**：離党・復党の経歴から「節操がない」との批判
