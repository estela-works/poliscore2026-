# 簗和生（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年4月22日 |
| 年齢 | 45歳 |
| 選挙区 | 栃木県3区（大田原市、矢板市、那須塩原市、那須郡） |
| 当選回数 | 衆議院5回 |
| 主な役職 | 衆議院安全保障委員長、文部科学副大臣、国土交通大臣政務官/内閣府大臣政務官 |

## 経歴
- 東京都小金井市生まれ、現住所は栃木県大田原市
- 桐朋高等学校卒業
- 2003年、慶應義塾大学商学部卒業
- 2006年、東京大学大学院経済学研究科修士課程修了
- 自民党衆議院議員岡部英明秘書、日本経済研究所研究員を経る
- 2012年、衆院選栃木3区で比例復活により初当選（渡辺喜美に敗北）
- 2014年、栃木3区で再選
- 2017年、栃木3区で3選
- 2021年、栃木3区で4選
- 2024年10月、栃木3区で5選（45,546票、178票差の接戦）

## 評価詳細

### ① 課題設定能力（4/5）
東京大学大学院で経済学を学び、経済政策に関する専門性を持つ。衆議院安全保障委員長として安全保障政策の課題を把握し、文部科学副大臣として教育政策にも携わった。農林部会長、畜産・酪農対策委員長として農業政策の課題設定にも関与。北朝鮮拉致問題対策本部幹事として外交・人権問題にも取り組んでいる。

### ② 解決設計力（4/5）
経済学修士として政策分析能力を持ち、日本経済研究所での研究員経験もある。副大臣・政務官として制度設計の実務に携わった経験がある。党内の政策部会で農林水産、安全保障など複数分野の政策立案に関与している。ただし、独自の政策提言よりも、党の方針に沿った制度設計が中心。

### ③ 実行力（4/5）
5期連続当選を果たし、副大臣・政務官・委員長と着実にキャリアを積み上げている。国土交通大臣政務官、文部科学副大臣として行政実務を経験。2024年の選挙では裏金問題による逆風の中、178票差という超接戦を制して当選を果たした。党内での役職を着実にこなしている。

### ④ 説明責任（1/5）
裏金問題について「自身に中抜きや私的使用は一切ない」と主張するが、5年間で1746万円もの不記載があった事実への説明は不十分。「当初から全額を自身の政治団体に寄付」と述べるが、なぜ政治資金収支報告書に記載しなかったのかの説明がない。国民の疑念に対して納得のいく説明ができておらず、説明責任を果たしているとは言い難い。

### ⑤ 一貫性（4/5）
2012年の初出馬以来、一貫して自民党所属で栃木3区から活動を継続。政策的にも保守的なスタンスを維持している。党の方針に沿った活動を続けており、政治的な立場のブレは見られない。安倍派（清和政策研究会）所属として一貫した活動を行ってきた。

### ⑥ 倫理観（1/5）
政治資金パーティー収入の裏金問題で5年間に計1746万円の不記載が発覚。これは公認を受けた議員の中で2番目に高額。2024年4月、自民党から「党の役職停止6カ月」の処分を受け、比例重複も認められなかった。東京地検特捜部から起訴猶予処分となったが、不正は事実。検察審査会への告発も行われた。政治資金の透明性という点で重大な問題があり、政治倫理に反する行為があったと評価せざるを得ない。

## 強み・弱み

### 強み
- **専門性と経歴**：東大大学院修了、副大臣・政務官経験など高い政策能力
- **当選実績**：5期連続当選、接戦を制する選挙力
- **若さ**：45歳と比較的若く、今後のキャリアに期待

### 弱み
- **裏金問題**：1746万円の不記載は深刻な問題であり、有権者の信頼を損ねている
- **説明責任の欠如**：問題発覚後の説明が不十分
- **薄氷の当選**：2024年選挙は178票差であり、次回選挙への不安要素
