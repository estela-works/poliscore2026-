# 菊田真紀子（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年10月24日 |
| 年齢 | 56歳 |
| 選挙区 | 新潟2区 |
| 当選回数 | 衆議院8回 |
| 主な役職 | 立憲民主党常任幹事会議長（第5代）、元外務大臣政務官 |

## 経歴
- 群馬県藤岡市に生まれ、新潟県加茂市で育つ
- 新潟県立加茂高等学校卒業（高校時代はバレー部主将）
- 1988年〜1990年 中国黒竜江大学に留学
- 1993年〜 中国語会話教室を経営
- 1995年 加茂市議会議員選挙で全国最年少（25歳）トップ当選
- 1999年 加茂市議会議員選挙で2期連続トップ当選
- 2003年 衆議院議員選挙で初当選（新潟4区・民主党）
- 外務大臣政務官、民主党幹事長代行、民進党財務局長等を歴任
- 2024年9月 立憲民主党常任幹事会議長に就任
- 2024年10月 衆院選で8期目当選

## 評価詳細
### ① 課題設定能力: 4/5
**評価根拠**: 原発再稼働反対を明確に掲げ、「地震や大雪の時に事故が発生したら避難計画が非現実的」と地域の実情に即した問題提起ができている。国立大学法人法改正案では「大学の自治・学問の自由に対する不当な介入」について質疑し、日本学術会議の会員任命拒否問題を取り上げるなど、政策課題の発掘・設定に長けている。野党連携による政権交代を目指す「有志の会」を立ち上げ、党所属衆院議員の過半数の賛同を得たことも、課題設定力の高さを示している。

### ② 解決設計力: 3/5
**評価根拠**: 野党候補一本化という政権交代への具体的手法を提案し、53人の賛同を集めた点は評価できる。外務政務官経験もあり、政策立案の実務経験はある。ただし、原発問題など複雑な政策課題に対する代替案や段階的解決策の提示は限定的。反対姿勢は明確だが、具体的なエネルギー政策の設計までは見えにくい。

### ③ 実行力: 4/5
**評価根拠**: 25歳での市議トップ当選以来、衆議院8期連続当選という実績は突出している。接戦区である新潟2区で自民党候補を何度も破っており、選挙実行力は非常に高い。外務政務官として政府内での実務経験もある。野党内での組織づくり（有志の会）も成功させている。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 記者会見で無所属出馬の意向を表明したり、野党連携の有志の会設立時に記者会見を開くなど、メディア対応は行っている。国会質問では森会長の女性蔑視発言への対応などを追及。一方で、希望の党への対応など転換点での説明が十分だったとは言い難い面もある。

### ⑤ 一貫性: 2/5
**評価根拠**: 政党の変遷が多い点が課題。自由党→民主党→無所属→立憲民主党と移籍を重ねている。2017年の希望の党騒動では、当初小池百合子氏を「原発ゼロを掲げ、自民党と決別する覚悟を示した腹の据わった政治家」と高く評価しながら、最終的には無所属出馬を選択した。原発反対という政策軸は一貫しているが、所属政党・政治勢力への姿勢には変動が見られる。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特筆すべきスキャンダルは確認されていないが、東日本大震災時のエステ通いが批判された過去がある。2009年の事業仕分けでの漢方薬保険適用問題でも批判を受けた。一方、政治資金や不正に関する重大な問題は報じられていない。長年の政治活動において大きな倫理的問題は起こしていないが、過去の行動への批判は完全には払拭されていない。

## 強み・弱み
### 強み
- **選挙に強い**: 8期連続当選、接戦区での勝利経験が豊富
- **中国語・外交経験**: 中国留学経験と外務政務官経験を活かした国際的視野
- **組織力**: 野党内で有志の会を組織するなどリーダーシップがある

### 弱み（注意点）
- **政党遍歴**: 自由党→民主党→無所属→立憲民主党と所属が変遷
- **過去の批判**: 震災時のエステ通い問題が時折蒸し返される
- **代替案の弱さ**: 反対姿勢は明確だが、具体的解決策の提示が弱い場合がある
