# 梅谷守（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1973年12月9日 |
| 年齢 | 52歳 |
| 選挙区 | 新潟5区 |
| 当選回数 | 衆議院2回（県議2回） |
| 主な役職 | 元新潟県議会議員 |

## 経歴
- 東京都文京区出身、早稲田大学教育学部卒業（1997年）
- 繊維商社「蝶理株式会社」に入社後、約10ヶ月で退社
- 2001年、政策秘書試験に合格。細野豪志、黒岩宇洋議員の政策秘書を務める
- 2007年、新潟県議会議員選挙に初当選（2期）
- 2014年、衆院選に民主党公認で出馬（落選）
- 2017年、無所属で出馬（僅差で落選）
- 2021年、立憲民主党公認で130票差の接戦を制し初当選
- 2024年10月、新潟5区で2期目の当選（114,429票）

## 評価詳細
### ① 課題設定能力: 4/5
**評価根拠**: 7年間の浪人生活を経て地元を丹念に回り、「雪が溶けると春になる」という雪国の感覚と都会の格差を政治の原点として掲げる。豪雪・渇水対策、医師不足、メタンハイドレート商用化など、地域特有の課題を的確に把握している。

### ② 解決設計力: 4/5
**評価根拠**: 政策秘書の経験を活かし、地元の声を具体的な政策や法案に練り上げる能力を持つ。コロナ融資の減免策など、現実的な解決策を提示してきた実績がある。

### ③ 実行力: 4/5
**評価根拠**: 2025年4月には衆議院本会議でAI法案に対する質疑を行うなど、国会活動は活発。県議時代から積み上げた地道な活動が2期連続当選につながっている。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 日本酒配布問題発覚時、記者団への説明は「軽率だった」と陳謝するのみで、過去の同様事案についての質問には回答を拒否。Facebookでの謝罪のみで詳細な説明を避けた姿勢は、説明責任を果たしているとは言い難い。

### ⑤ 一貫性: 4/5
**評価根拠**: 7年間の浪人生活を耐え、一貫して同一選挙区から出馬し続けた粘り強さがある。政党は民主党から立憲民主党、中道改革連合と変遷しているが、政策の基本姿勢は維持している。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 2024年1月、上越市内の町内会行事で複数の有権者に日本酒を配布。公職選挙法違反の疑いで告発され、立憲民主党から党員資格停止処分を受けた。「会合の対価」との弁明は苦しく、参加費不要の行事でも配布していた事実が報じられている。新潟県政界の「慣習」との指摘もあるが、現職国会議員としての倫理意識の欠如は深刻。

## 強み・弱み
### 強み
- **地域密着型の活動**: 7年の浪人生活で培った地元とのパイプと、豪雪・医師不足など地域課題への深い理解
- **政策立案能力**: 政策秘書経験を活かした法案・政策への練り上げ力
- **粘り強さ**: 複数回の落選を乗り越え、130票差の勝利を掴んだ精神力

### 弱み（注意点）
- **公選法違反問題**: 日本酒配布問題は有権者への買収行為と見なされかねず、政治倫理上の重大な汚点
- **説明不足**: 問題発覚時の対応が消極的で、有権者への丁寧な説明を欠いた
- **処分歴**: 党員資格停止・党役職停止の処分歴は今後の政治活動に影響する可能性
