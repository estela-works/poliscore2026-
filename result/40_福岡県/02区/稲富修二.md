# 稲富修二（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年8月26日 |
| 年齢 | 56歳 |
| 選挙区 | 福岡2区（福岡市中央区・南区・城南区） |
| 当選回数 | 衆議院4期 |
| 主な役職 | 立憲民主党副幹事長 |

## 経歴
- 福岡県大野城市出身
- 白陵中学校・高等学校卒業
- 東京大学法学部卒業（1995年）
- 丸紅入社（人事部）、翌年退社
- 松下政経塾入塾（1996年）、卒塾（2000年）
- コロンビア大学国際公共政策大学院修了（2002年）
- 2007年福岡県知事選出馬、落選
- 2009年衆院選（福岡2区）で初当選、山崎拓を破る
- 2012年、2014年衆院選で落選
- 2017年衆院選で比例復活当選（2期目）
- 2021年衆院選で比例復活当選（3期目）
- 2024年衆院選で小選挙区当選（4期目）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 松下政経塾出身、コロンビア大学大学院で国際公共政策を学んだ経験から、政策課題への理解は深い。経済政策、社会保障など幅広い課題に取り組む。

### ② 解決設計力: 4/5
**評価根拠**: 東大法学部卒、コロンビア大学大学院修了と高い学歴・専門性を持つ。松下政経塾での研修経験もあり、政策立案能力は高い。国会での質疑も的確。

### ③ 実行力: 4/5
**評価根拠**: 2024年衆院選で鬼木誠との5度目の対決を初めて小選挙区で制し、15年ぶりの選挙区勝利。4期目の当選を果たし、党副幹事長として党運営にも関与。粘り強さが実を結んだ。

### ④ 説明責任: 4/5
**評価根拠**: 地元での活動報告を継続的に実施。国会質疑でも明確な論理で政策を説明。丸紅、松下政経塾出身として説明能力は高い。

### ⑤ 一貫性: 4/5
**評価根拠**: 民主党→立憲民主党（中道改革連合）と中道リベラルの立場を一貫して維持。福岡2区での活動を15年以上継続し、ついに選挙区での勝利を果たした。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。政治資金についても透明性を維持。野党議員として権力監視の姿勢を堅持。

## 強み・弱み

### 強み
- **高い学歴・専門性**: 東大、コロンビア大学院、松下政経塾
- **粘り強さ**: 15年かけて選挙区勝利を果たした
- **政策立案能力**: 国際公共政策の専門知識
- **党内での地位**: 副幹事長として党運営に関与
- **民間経験**: 丸紅での勤務経験

### 弱み（注意点）
- **野党の限界**: 与党経験がなく政策実現に制約
- **比例復活が多い**: 小選挙区勝利は2009年、2024年のみ
- **党勢の課題**: 立憲民主党の支持率
- **選挙区の接戦**: 鬼木誠との競争が続く
- **知名度**: 全国的な知名度は限定的
