# 黒石裕子（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1991年11月15日 |
| 年齢 | 32歳 |
| 選挙区 | 福岡2区（福岡市中央区・南区・城南区） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党党員、宅地建物取引士 |

## 経歴
- 1989年福岡市生まれ（※一部資料では1991年生まれ）
- 福岡市在住
- 公立高校中退、通信制高校卒業
- 長崎大学卒業
- シングルマザーとして子育てと仕事を両立
- 飲食・美容業界を経て不動産業に転身
- 宅地建物取引士資格取得
- 2024年参政党政治塾受講
- 2024年衆院選（福岡2区）参政党公認で出馬、落選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 不動産業での経験から「暮らしの矛盾」に直面し政治を志した。シングルマザーとしての経験から子育て支援への関心はあるが、政策としての具体的な課題設定は参政党の方針に沿ったもの。

### ② 解決設計力: 2/5
**評価根拠**: 政策立案の実績はなく、政治経験もない新人候補。宅地建物取引士の資格は持つが、国政レベルの政策設計能力は未知数。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で約16,689票（7%）を獲得したが落選。福岡2区は現職2名の激戦区であり、新人の当選は困難だった。党勢拡大途上の参政党として、組織力も限定的。

### ④ 説明責任: 2/5
**評価根拠**: 参政党を通じた活動報告はあるが、個人としての詳細な政策説明は限定的。選挙戦での露出も主要候補と比べると少なかった。

### ⑤ 一貫性: 3/5
**評価根拠**: 「このままでは未来を託せない」という危機感から政治を志し、参政党から出馬。生活者としての視点での政治参入は一貫している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。シングルマザーとして苦労しながらキャリアを築いてきた経歴。

## 強み・弱み

### 強み
- **生活者視点**: シングルマザーとしての実体験
- **若さ**: 32歳で今後の成長可能性
- **不動産知識**: 宅地建物取引士資格
- **多様な職歴**: 飲食・美容・不動産と幅広い経験
- **地元密着**: 福岡市出身・在住

### 弱み（注意点）
- **当選経験なし**: 政治経験ゼロでの出馬
- **知名度不足**: 地元での認知度が低い
- **組織力の限界**: 参政党の組織基盤は発展途上
- **対立候補の壁**: 現職2名がいる激戦区
- **政策の具体性**: 独自政策が見えにくい
