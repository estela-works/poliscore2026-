# 木下敏之（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年2月12日 |
| 年齢 | 65歳 |
| 選挙区 | 福岡県2区 |
| 当選回数 | 衆議院0回（新人）、佐賀市長1期（1999-2005年） |
| 主な役職 | 元佐賀市長、福岡大学名誉教授 |

## 経歴
- 1960年佐賀市生まれ
- ラ・サール中学校・高等学校卒業
- 東京大学法学部卒業
- 1984年農林水産省入省
- 1991年環境庁水質保全局水質管理課
- 1997年農林水産技術会議総務課総括課長補佐を経て退官
- 1999年佐賀市長選挙で初当選（39歳、県庁所在地市長として最年少）
- 2005年新・佐賀市長選で落選
- 2008年官僚国家日本を変える元官僚の会の幹事
- 2009年事業仕分け第1弾に民間有識者として参加
- 2010年福岡市長選挙に出馬、落選
- 2012年福岡大学経済学部教授
- 2025年福岡大学定年退職、非常勤講師
- YouTubeチャンネル登録者約6.5万人

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 農林水産官僚、佐賀市長、大学教授、事業仕分け参加者として、行政改革や地方自治の課題について構造的な理解を持つ。「官僚国家日本を変える」という問題意識を持ち、YouTube等でも政治・経済問題を発信。専門性に基づいた課題設定能力がある。

### ② 解決設計力: 4/5
**評価根拠**: 佐賀市長として行政運営の実務経験を持ち、事業仕分けにも参加した経験から政策設計能力は高いと評価。農水省での政策立案経験、大学教授として九州経済を研究。理論と実務の両面から政策を設計できる。

### ③ 実行力: 3/5
**評価根拠**: 佐賀市長として1期務めた実績があり、行政のトップとしての実行経験を持つ。ただし、再選に失敗し、福岡市長選でも落選しており、長期的な政策実現には至っていない。改革派市長として知られたが、任期は限定的。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: YouTubeチャンネル（登録者約6.5万人）で政治をわかりやすく発信。地元テレビ局のコメンテーターも務める。大学教授として講義経験も豊富。複雑な政策を一般向けに説明する能力は高い。

### ⑤ 一貫性: 2/5
**評価根拠**: 政党・政治的立場の変遷が多い。無所属→官僚国家日本を変える元官僚の会→参政党と、複数の立場を経ている。行政改革・地方分権という軸は見られるが、最近参政党に参加したことで、保守・右派的な立場への転換が見られ、一貫性に疑問。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 佐賀市長時代や官僚時代に大きなスキャンダルは確認されない。事業仕分けに参加するなど、行政の透明性向上に取り組んだ経歴。権力の私物化などの問題は確認されない。

## 強み・弱み

### 強み
- **豊富な行政経験**: 官僚、市長、大学教授と多様なキャリア
- **発信力**: YouTube等での政策発信能力が高い
- **専門性**: 農業政策、地方行政、経済学の知見を持つ

### 弱み（注意点）
- **選挙での敗北経験**: 佐賀市長再選、福岡市長選と連続して落選
- **立場の変化**: 参政党への参加により、従来のイメージとの乖離が生じる可能性
- **衆議院議員としては未経験**: 国政での実績はない
