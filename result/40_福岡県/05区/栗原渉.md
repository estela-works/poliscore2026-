# 栗原渉（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年9月27日 |
| 年齢 | 59歳 |
| 選挙区 | 福岡県5区 |
| 当選回数 | 衆議院1回、福岡県議3期 |
| 主な役職 | 厚生労働大臣政務官、元福岡県議会議長（第69代） |

## 経歴
- 福岡県甘木市（現・朝倉市）中島田出身
- 甘木市立十文字中学校卒業
- 福岡県立朝倉高等学校卒業
- 東京国際大学教養学部卒業（1990年）
- 太田誠一衆議院議員の書生・秘書（1989年〜2009年、約20年間）
- 2010年福岡県議補選：甘木市選挙区で初当選
- 2011年福岡県議選：朝倉市・朝倉郡選挙区で当選
- 農林水産常任委員会委員長（2013-2015年）
- 自民党福岡県議団政策審議会長（2017-2018年）
- 議会運営委員会委員長（2018-2019年）
- 第69代福岡県議会議長（2019-2020年）
- 2021年県議辞職（衆院選出馬断念）
- 2024年衆院選初当選（福岡5区）
- 厚生労働大臣政務官（高市内閣）
- 福岡県砂防協会会長
- 福岡県樹苗農業協同組合組合長
- 福岡県遺族連合会副会長・青年部活動推進部長

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 農林水産常任委員会委員長を務め、農業政策に精通。福岡県議会議長として県政全般の課題に携わった。ただし、国政レベルでの独自の構造的課題設定については、まだ十分な実績を確認できない。

### ② 解決設計力: 3/5
**評価根拠**: 東京国際大学教養学部卒業。太田誠一議員の秘書として約20年間政策実務を経験。県議会で議長まで務め、行政運営の経験は豊富。厚労政務官として社会保障政策に関与し始めている。

### ③ 実行力: 4/5
**評価根拠**: 2024年衆院選で、立憲民主党前職の堤かなめを約2万2千票差で破り初当選。2021年には公認を得られず出馬断念したが、再挑戦で勝利。県議会議長まで上り詰めた実績。地元で砂防協会会長、樹苗農協組合長など多くの役職を務める。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。秘書経験が長く、政治家としてのコミュニケーション能力は培われている。ただし、全国的なメディア露出は限定的。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党で一貫して活動。太田誠一議員の秘書から県議、国会議員と一貫したキャリア。農林水産、地方行政という軸で活動を続けている。2010年補選は無所属だったが、翌年から自民党公認。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。遺族連合会の活動など社会貢献にも取り組む。ただし、国政での権力行使の経験はまだ浅く評価は限定的。

## 強み・弱み

### 強み
- **長い秘書経験**: 太田誠一議員の秘書として20年間
- **県議会での実績**: 議長まで務めた経験
- **地元での基盤**: 多くの団体で役職を務める
- **選挙での強さ**: 前職を大差で破って当選

### 弱み（注意点）
- **国政経験の浅さ**: 当選1回で経験が浅い
- **知名度**: 全国的な認知度は限定的
- **2021年の断念**: 公認が得られず出馬断念した過去
- **高市内閣との関係**: 政権の評価に左右される
