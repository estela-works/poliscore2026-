# 村上智信（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年10月9日 |
| 年齢 | 55歳 |
| 選挙区 | 福岡11区（田川市、行橋市、京都郡等） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 日本維新の会福岡11区支部長 |

## 経歴
- 福岡県築上町生まれ（生家は寺「法顕寺」）
- 福岡県立京都高等学校卒業
- 横浜国立大学工学部物質工学科卒業
- 東京大学大学院工学系研究科化学システム工学専攻修了
- 1995年通商産業省（現経済産業省）入省
- 商務情報政策局医療福祉機器産業室長、情報プロジェクト室長歴任
- 2016年経産省退職
- 2017年衆院選（福岡11区）希望の党から出馬、落選
- 2021年衆院選（福岡11区）無所属で出馬、落選
- 2023年日本維新の会福岡11区支部長
- 2024年衆院選で武田良太を破り初当選（約2200票差）
- 九州・沖縄初の維新小選挙区当選者

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 経産省での医療福祉機器、情報政策の経験から産業政策への理解はある。ただし維新の政策に沿った活動が中心で、独自の課題設定は限定的。地元福岡11区の課題への取り組みは選挙を通じて培ってきた。

### ② 解決設計力: 4/5
**評価根拠**: 東大院卒、経産省20年超のキャリアから政策設計能力は高い。医療福祉機器産業室長として実際の政策立案に携わった経験がある。官僚出身らしい実務能力を持つ。

### ③ 実行力: 4/5
**評価根拠**: 2017年、2021年と2度の落選を経て、2024年に元総務大臣の武田良太を約2200票差で破り初当選。諦めずに挑戦し続けた粘り強さは特筆に値する。九州・沖縄初の維新小選挙区当選という歴史的成果。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。維新の会所属議員として政策説明も行っている。特に問題となる説明責任の欠如はない。

### ⑤ 一貫性: 3/5
**評価根拠**: 希望の党→無所属→日本維新の会と政党が変遷。ただし福岡11区からの出馬、武田良太への挑戦という軸は2017年から一貫している。維新入りは政策的に自然な流れとも言える。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。官僚時代、政治家としての活動において倫理的な問題は見られない。

## 強み・弱み

### 強み
- **官僚経験**: 経産省20年超、産業政策の専門家
- **粘り強さ**: 2度の落選を乗り越えての当選
- **歴史的成果**: 九州・沖縄初の維新小選挙区当選
- **高い学歴**: 東大院修了、政策設計能力

### 弱み（注意点）
- **国会経験が浅い**: まだ1期目で実績を積む段階
- **政党の変遷**: 希望→無所属→維新と所属が変わっている
- **地元との関係**: 選挙区での地盤はまだ確立途上
- **次の選挙**: 武田良太が再挑戦する可能性があり厳しい戦い
