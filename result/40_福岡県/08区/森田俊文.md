# 森田俊文（無所属）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1963年8月28日 |
| 年齢 | 61歳 |
| 選挙区 | 福岡8区（直方市・飯塚市・中間市・宮若市・嘉麻市・遠賀郡・鞍手郡・嘉穂郡） |
| 当選回数 | なし（新人） |
| 主な役職 | フリージャーナリスト、元那珂川町議 |

## 経歴
- 上智大学卒業
- フリージャーナリストとして活動
- 2009年、那珂川町議会議員選挙当選（809票）
- 2012年、那珂川町長選挙落選（4,996票）
- 2013年、那珂川町議会議員選挙当選（1,258票）
- 2017年、第48回衆院選比例九州ブロック（希望の党）落選
- 2019年、福岡市議会議員選挙落選（3,253票）
- 2024年、第50回衆院選福岡8区（無所属）34,407票獲得、落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: ジャーナリスト出身で社会問題への視点を持つ。福岡8区に立憲・国民・維新の候補がいない状況で「受け皿になる」と出馬を表明。野党不在の選挙区での選択肢提供を課題として設定。

### ② 解決設計力: 3/5
**評価根拠**: 上智大学卒、ジャーナリストとしての取材経験から問題分析能力はある。ただし国政レベルでの政策立案経験はなく、町議としての実績に留まる。

### ③ 実行力: 2/5
**評価根拠**: 町議2期の経験はあるものの、町長選、衆院選、市議選と複数回の落選経験。2024年衆院選では34,407票を獲得したが、麻生太郎の圧倒的強さの前に敗北。

### ④ 説明責任: 3/5
**評価根拠**: ジャーナリスト出身で情報発信力はある。野党不在の選挙区での選択肢として立候補の理由を説明。ただし政策の具体的な説明は限定的。

### ⑤ 一貫性: 2/5
**評価根拠**: 2017年は希望の党から比例出馬、2024年は無所属と政治スタンスが変遷。町議から町長、衆院、市議と挑戦する選挙も一定せず、政治家としての軸が見えにくい。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。ジャーナリストとして権力監視の視点を持つ。ただし評価するに足る実績が限定的。

## 強み・弱み

### 強み
- **ジャーナリスト経験**: 取材・情報発信のスキル
- **地方議員経験**: 那珂川町議2期の経験
- **挑戦姿勢**: 複数回の選挙に挑戦する粘り強さ
- **得票力**: 34,407票は無所属として一定の評価
- **受け皿**: 野党不在選挙区での選択肢提供

### 弱み（注意点）
- **連敗**: 複数の選挙で落選経験
- **強敵**: 麻生太郎という圧倒的な相手
- **組織力不足**: 無所属で政党支援なし
- **一貫性**: 政党・選挙種別が変遷
- **知名度**: 麻生太郎との知名度格差
