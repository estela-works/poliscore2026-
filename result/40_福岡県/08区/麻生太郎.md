# 麻生太郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1940年9月20日 |
| 年齢 | 84歳 |
| 選挙区 | 福岡8区（直方市・飯塚市・中間市・宮若市・嘉麻市・遠賀郡・鞍手郡・嘉穂郡） |
| 当選回数 | 衆議院15期 |
| 主な役職 | 自民党副総裁、元内閣総理大臣、元副総理兼財務大臣 |

## 経歴
- 福岡県飯塚市生まれ（生後9か月〜5歳はシカゴ在住）
- 学習院初等科・中等科・高等科卒業
- 学習院大学政経学部卒業
- 麻生産業入社、麻生セメント（現・株式会社麻生）社長
- 日本青年会議所会頭
- 1976年、モントリオール五輪クレー射撃日本代表
- 1979年、衆議院議員初当選
- 経済企画庁長官（第53代）
- 総務大臣（第3・4・5代）
- 外務大臣（第137・138代）
- 2008年、第23代自民党総裁・第92代内閣総理大臣
- 2012年〜2021年、副総理兼財務大臣兼金融担当大臣
- 2021年〜、自民党副総裁
- 2024年9月〜、党最高顧問
- 2025年10月、再び自民党副総裁就任
- 吉田茂元首相の孫、麻生財閥の御曹司

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 首相経験者として国家的課題に対応。経済企画庁長官、財務大臣として経済政策、外務大臣として外交課題に取り組む。リーマンショック対応など危機対応の経験も豊富。

### ② 解決設計力: 4/5
**評価根拠**: 学習院大卒、実業界出身で経営感覚を持つ。財務大臣として長期間経済財政政策を担当。ただし首相在任中は支持率低迷で政策実現に課題も。

### ③ 実行力: 5/5
**評価根拠**: 15期連続当選は現役最多クラス。福岡8区では圧倒的強さで他党に比例復活も許さない。首相、副総理、党副総裁と要職を歴任し、党内での影響力は絶大。

### ④ 説明責任: 2/5
**評価根拠**: 数々の失言問題で批判を受ける。2024年1月の上川外相への容姿発言、「ナチスの手口に学べ」発言、「子どもを産まなかったほうが問題」発言など、説明・撤回を繰り返す。

### ⑤ 一貫性: 4/5
**評価根拠**: 1979年から自民党一筋で45年以上活動。保守政治家として一貫した立場。ただし政局に応じて総裁選での支持候補を変えるなど、権力維持のための柔軟さも見せる。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 失言が「即刻退場レベル」と批判されても副総裁に居座る。裏金問題では党政治刷新本部最高顧問を務めるも、自身の派閥問題への責任は曖昧。「時代錯誤」との批判多数。

## 強み・弱み

### 強み
- **経験**: 首相、副総理、外相、財務相など最高レベルの経験
- **党内影響力**: 副総裁として派閥横断的な影響力
- **選挙強さ**: 福岡8区で15連勝の圧倒的地盤
- **国際経験**: 外相経験、麻生財閥の国際ビジネス
- **知名度**: 日本で最も知名度の高い政治家の一人

### 弱み（注意点）
- **失言癖**: 度重なる失言で国内外から批判
- **高齢**: 84歳で戦前生まれ唯一の衆議院議員
- **時代感覚**: ジェンダー意識など「時代錯誤」との批判
- **裏金問題**: 派閥政治の中心人物としての責任
- **説明不足**: 失言への説明・反省が不十分
