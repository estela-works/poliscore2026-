# 佐藤公治（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年7月28日 |
| 年齢 | 66歳 |
| 選挙区 | 広島5区（三原市・尾道市・府中市・三次市・庄原市・世羅郡・神石郡） |
| 当選回数 | 衆議院5期、参議院1期 |
| 主な役職 | 中道改革連合（旧立憲民主党）組織委員長、立憲民主党広島県連代表、結集ひろしま代表 |

## 経歴
- 1959年7月28日、広島県御調郡向島町（現尾道市）生まれ
- 慶應義塾高等学校卒業
- 慶應義塾大学法学部政治学科卒業
- 電通入社
- 父・佐藤守良（元農林水産大臣・国土庁長官等）の秘書
- 国務大臣秘書官（旧国土庁長官、北海道・沖縄開発庁長官）
- 1998年、参議院議員選挙で初当選（自由党）
- 参議院外交防衛委員長
- 2000年、衆議院議員選挙で広島6区から初当選（比例復活）
- 2003年、民由合併により民主党入党
- 2012年、消費増税に反発し民主党離党、国民の生活が第一→日本未来の党→生活の党
- 2024年10月、第50回衆院選で広島5区から5選
- 立憲民主党広島県連代表、政治団体「結集ひろしま」代表

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 外交防衛委員長として安全保障政策に関与。消費増税反対を明確な課題として設定し離党した経験。

### ② 解決設計力: 4/5
**評価根拠**: 慶應義塾大学法学部卒、電通での実務経験。大臣秘書官として政策立案に関与。参院・衆院両方の経験。

### ③ 実行力: 4/5
**評価根拠**: 衆参通算6期の実績。2024年衆院選で自民・共産を破り5選。広島5区での安定した選挙基盤。

### ④ 説明責任: 3/5
**評価根拠**: 県連代表として県内での活動。国会での質疑等は通常レベル。

### ⑤ 一貫性: 2/5
**評価根拠**: 自由党→民主党→国民の生活が第一→日本未来の党→生活の党→立憲民主党と多くの政党を渡り歩く。消費増税反対という姿勢は一貫。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に大きなスキャンダルは報じられていない。世襲候補だが自身の力で選挙区を勝ち取る。

## 強み・弱み

### 強み
- **高学歴**: 慶應義塾大学法学部卒
- **豊富な経験**: 衆参通算6期、大臣秘書官経験
- **地元基盤**: 広島5区で5選の実績
- **外交経験**: 参院外交防衛委員長
- **県連代表**: 広島県内での影響力

### 弱み（注意点）
- **政党遍歴**: 多くの政党を渡り歩いた経歴
- **世襲**: 父が元農林水産大臣
- **野党**: 与党経験が限定的
- **年齢**: 66歳で世代交代論も
- **党再編**: 中道改革連合への移行
