# 山本深（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年頃 |
| 年齢 | 40歳 |
| 選挙区 | 広島5区（三原市・尾道市・府中市・三次市・庄原市・世羅郡・神石郡） |
| 当選回数 | なし（新人） |
| 主な役職 | 自由民主党広島5区総支部長、元財務省官僚 |

## 経歴
- 1985年頃生まれ
- 財務省入省
- 財務省官僚として勤務
- 2024年3月、広島5区公募に応募するも落選
- 2026年1月19日、自民党広島県連が広島5区公認候補として擁立決定
- 財務省に退職届提出

※注: 2024年衆院選（第50回）では広島5区の自民党候補は小島敏文氏であり、山本深氏は2026年衆院選（第51回）の候補者として活動中
