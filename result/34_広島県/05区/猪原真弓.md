# 猪原真弓（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年1月17日 |
| 年齢 | 65歳 |
| 選挙区 | 広島5区（三原市・尾道市・府中市・三次市・庄原市・世羅郡・神石郡） |
| 当選回数 | なし（新人） |
| 主な役職 | 日本共産党県東部地区委員会常任委員、新日本婦人の会県本部委員 |

## 経歴
- 1961年1月17日、広島県尾道市生まれ
- 尾道商業高等学校卒業
- 医薬品卸会社勤務
- 日本共産党県東部地区委員
- 中学校給食をすすめる会会長
- 日本共産党県東部地区委員会常任委員
- 新日本婦人の会県本部委員
- 2024年10月、第50回衆院選で広島5区から出馬（落選）
- 2025年11月、広島県知事選に共産党推薦で無所属出馬予定

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 消費税5%引き下げ・将来ゼロ、インボイス撤廃、保険証継続使用、給食無償化など具体的な課題を設定。

### ② 解決設計力: 2/5
**評価根拠**: 尾道商業高校卒。市民運動での経験はあるが、政策立案の直接経験は限定的。

### ③ 実行力: 1/5
**評価根拠**: 国政選挙での当選経験がなく、政治的実行力は評価できない段階。

### ④ 説明責任: 3/5
**評価根拠**: 選挙戦で「10年ぶりに共産党から小選挙区候補として立候補できた」と経緯を説明。政策を具体的に訴える。

### ⑤ 一貫性: 4/5
**評価根拠**: 共産党地区委員として長年活動を継続。中学校給食推進、消費税反対など一貫した主張。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。市民運動での経験から清廉なイメージ。

## 強み・弱み

### 強み
- **市民運動経験**: 中学校給食をすすめる会会長
- **地元出身**: 尾道市出身で地域に密着
- **一貫した主張**: 消費税反対、給食無償化など
- **党組織**: 共産党の組織的支援
- **女性候補**: 女性の政治参画を体現

### 弱み（注意点）
- **当選経験なし**: 国政・地方議会とも未当選
- **高学歴でない**: 尾道商業高校卒
- **強敵**: 佐藤公治という5期連続当選の現職
- **党勢低下**: 共産党の全国的な支持率低迷
- **知事選出馬**: 衆院選から知事選へ転身で一貫性に疑問も
