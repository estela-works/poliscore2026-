# 後藤茂之（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1955年12月9日 |
| 年齢 | 70歳 |
| 選挙区 | 長野4区（諏訪市・岡谷市・茅野市・伊那市・駒ヶ根市等） |
| 当選回数 | 衆議院8期 |
| 主な役職 | 元厚労大臣、元経済再生担当大臣、自民党政調会長代理 |

## 経歴
- 東京都港区出身
- 東京教育大学附属駒場高等学校卒業
- 東京大学法学部卒業
- 米国ブラウン大学大学院経済学修士
- 1980年大蔵省入省
- 大蔵省主税局企画調整室長で退官（1995年）
- 2000年衆院選で初当選（民主党公認）
- 2003年民主党離党、自民党入党
- 国土交通大臣政務官、法務副大臣
- 厚生労働大臣（岸田内閣）
- 経済再生担当大臣（第2次岸田改造内閣）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 厚労大臣、経済再生担当大臣として社会保障改革、経済政策、コロナ対策など幅広い課題に取り組んだ。大蔵省出身の財政・経済政策への深い理解を持つ。「こども・若者」輝く未来創造本部長として少子化対策にも注力。

### ② 解決設計力: 5/5
**評価根拠**: 東大法学部卒、ブラウン大学大学院で経済学修士取得。大蔵省キャリアとして細川内閣の「国民福祉税」構想を担うなど、政策立案能力は極めて高い。複数の大臣ポストで実務を経験し、政策設計の実績は豊富。

### ③ 実行力: 4/5
**評価根拠**: 8期連続当選で長野4区の地盤は盤石。厚労大臣としてコロナ対策、経済再生担当大臣として「新しい資本主義」政策を推進。閣僚経験を通じて政策実現の手腕を発揮した。

### ④ 説明責任: 4/5
**評価根拠**: 閣僚として国会答弁、記者会見を数多くこなした。経済政策や社会保障について専門的かつ分かりやすい説明能力を持つ。地元での活動報告も継続的に実施。

### ⑤ 一貫性: 2/5
**評価根拠**: 2000年に民主党から初当選後、2003年に自民党に移籍。政党の移籍は政治的立場のブレと評価されうる。ただし自民党移籍後は20年以上一貫して同党で活動している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に重大なスキャンダルは報じられていないが、裏金問題では自民党議員として説明責任が問われた。民主党からの離党・自民党入党の経緯について、政治姿勢への疑問も指摘される。

## 強み・弱み

### 強み
- **閣僚経験**: 厚労大臣、経済再生担当大臣など重要ポスト歴任
- **政策立案能力**: 大蔵省出身、経済学修士の専門性
- **安定した地盤**: 長野4区で8期連続当選
- **党内影響力**: 政調会長代理、税制調査会小委員長
- **国際経験**: 米国留学で経済学を学ぶ

### 弱み（注意点）
- **政党移籍歴**: 民主党→自民党への転身
- **年齢**: 70歳でベテランの域
- **裏金問題の影響**: 自民党として逆風下
- **官僚的イメージ**: 庶民との距離感
- **後継者問題**: 長期在職による世代交代の課題
