# 武田良介（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年8月13日 |
| 年齢 | 46歳 |
| 選挙区 | 長野4区（諏訪市・岡谷市・茅野市・伊那市・駒ヶ根市等） |
| 当選回数 | 参議院1期（2016-2022）、衆議院なし |
| 主な役職 | 元参議院議員、共産党長野県委員 |

## 経歴
- 長野県中野市出身（りんご農家）
- 飯山北高等学校卒業
- 信州大学教育学部卒業
- 在学中に日本共産党入党
- 信州大学学生自治会連合副委員長
- 民青同盟長野県委員長（2004年～）
- 2012年衆院選（長野1区）落選
- 2014年衆院選（長野1区）落選
- 2016年参院選（比例区）初当選
- 2022年参院選（比例区）落選
- 2024年衆院選（長野4区）出馬、得票率39.78%で落選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 参議院議員として若者雇用、大学学費、消費税、農業政策など幅広い課題に取り組んだ。りんご農家出身として農業問題への関心が深く、若者の視点から社会問題を提起。学生時代からの平和運動、学費値下げ運動が原点。

### ② 解決設計力: 3/5
**評価根拠**: 参議院議員1期の経験があり、国会での質疑を通じて政策立案に関与。消費税減税、最低賃金引上げなど具体的な政策を主張。ただし野党議員として政策実現には至っていない部分も多い。

### ③ 実行力: 3/5
**評価根拠**: 2024年衆院選長野4区で得票率39.78%と健闘したが落選。参院1期の経験はあるものの、2022年参院選で落選し国政から離れていた。共産党の組織力を活かした選挙戦を展開したが、与党現職の壁は厚かった。

### ④ 説明責任: 4/5
**評価根拠**: 参議院議員として国会質疑で政策を積極的に説明。地元での活動報告、街頭演説を継続。若者目線での分かりやすい発信を心がけている。公式サイトでの情報発信も充実。

### ⑤ 一貫性: 5/5
**評価根拠**: 学生時代から共産党で一貫して活動。消費税減税、平和、労働者・農家の立場からの主張を20年以上継続。党内でのキャリアを着実に積み、複数回の選挙に挑戦し続けている。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。企業献金を受けない共産党で清廉な政治姿勢を維持。参議院議員時代も権力の抑制的な行使を心がけていた。

## 強み・弱み

### 強み
- **国会経験**: 参議院議員1期の実績
- **一貫性**: 20年以上の共産党での活動
- **地元への理解**: 長野県中野市出身、農家の息子
- **若者視点**: 学費・雇用問題への関心
- **発信力**: 継続的な情報発信

### 弱み（注意点）
- **参院落選歴**: 2022年参院選で落選
- **衆院当選経験なし**: 長野1区で2度落選
- **党勢の限界**: 共産党単独での当選は困難
- **与党経験なし**: 政策実現の実績が乏しい
- **選挙区変更**: 長野1区→4区への移動
