# 小泉龍司（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1952年9月17日 |
| 年齢 | 72歳 |
| 選挙区 | 埼玉11区（秩父市・本庄市・深谷市等） |
| 当選回数 | 衆議院8回 |
| 主な役職 | 元法務大臣、自民党選挙対策副委員長、同国際局長 |

## 経歴
- 埼玉県出身
- 元大蔵官僚（21年勤務）
- 税務署長・銀行局・証券局等で勤務後、大蔵省を退官
- 2000年第42回衆議院選挙で初当選
- 2005年郵政民営化に反対し自民党を離党
- 2009年無所属で当選（平沼グループ・連合埼玉の支援）
- 2012年自民党に復党
- 2014年以降、無所属のまま自民党二階派に参加
- 2023年法務大臣就任（第2次岸田第2次改造内閣）
- 2024年法務大臣退任（在任384日、死刑執行ゼロ）
- 二階派からの政治資金キックバック問題で派閥退会届提出

## 評価詳細

### ① 課題設定能力（4/5）
大蔵官僚として21年の行政経験があり、財政・金融政策の課題を構造的に理解できる。法務大臣として司法行政の課題にも取り組んだ。

### ② 解決設計力（4/5）
大蔵省での実務経験から、予算・制度設計の知見がある。法務大臣として法制度の運用に携わった。政策の実装経験が豊富。

### ③ 実行力（5/5）
8期連続当選、埼玉県で最も自民党が強い選挙区で圧倒的な強さを維持。郵政民営化反対で離党後も復党し、法務大臣まで上り詰めた政治力がある。

### ④ 説明責任（4/5）
国会答弁、記者会見での説明経験が豊富。法務大臣として司法行政について説明責任を果たした。ただし、裏金問題については説明が不十分との見方も。

### ⑤ 一貫性（2/5）
郵政民営化反対で自民党を離党後、復党という経緯がある。政策信念を貫いた面もあるが、最終的に党に戻ったことで一貫性に疑問も。派閥（二階派）への所属・離脱も変遷がある。

### ⑥ 倫理観（2/5）
**裏金問題への関与**：二階派からのキックバックを受けていたことを国会で答弁。法務大臣として検事総長への指揮権を持つ立場で利益相反が懸念され、派閥退会届を提出。法務大臣在任384日間で死刑執行ゼロだったことへの評価は分かれる。

## 強み・弱み
### 強み
- **官僚経験**：大蔵省21年の実務経験
- **閣僚経験**：法務大臣として司法行政を担当
- **選挙の強さ**：埼玉11区で圧倒的な支持基盤

### 弱み
- **裏金問題**：派閥からのキックバック受領
- **政党離脱歴**：郵政民営化反対で離党の過去
- **年齢**：72歳、世代交代の課題
