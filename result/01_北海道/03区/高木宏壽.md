# 高木宏壽（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年4月9日 |
| 年齢 | 64歳 |
| 選挙区 | 北海道3区（札幌市豊平区・白石区・清田区） |
| 当選回数 | 衆議院3回（2024年落選） |
| 主な役職 | 元復興副大臣、元内閣府大臣政務官兼復興大臣政務官、元北海道議会議員（2期）、米国公認会計士 |

## 経歴
- 北海道札幌月寒高等学校卒業
- 防衛大学校人文・社会科学専攻課程中退
- 慶應義塾大学法学部政治学科卒業
- 米国ウエスタンワシントン大学MBA取得
- 1987年：北海道拓殖銀行入行（ロサンゼルス支店等勤務）
- 1995年：北海道警察に幹部警察官として採用（来日外国人犯罪捜査、刑事課長）
- 2001年：朝日監査法人、KPMG（企業不正調査・リスクコンサルティング）
- 2007年：北海道議会議員初当選（札幌市豊平区、2期連続当選）
- 2012年：衆議院議員初当選（北海道3区）
- 2015-2016年：内閣府大臣政務官兼復興大臣政務官（第3次安倍第1次改造内閣）
- 2022-2024年：復興副大臣（第2次岸田第2次改造内閣・第1次石破内閣）
- 2024年10月：衆議院選挙で落選（比例復活もならず）
- 父は北海道議会議員の高木繁光

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「札幌、北海道を世界の金融センターにする」「GX投資による成長戦略」を掲げるが、具体的な課題分析や実現可能性の検討は乏しい。復興副大臣として東北の復興に携わったが、独自の課題設定で注目された記録は確認できない。憲法改正や自衛隊明記など自民党の基本方針に沿った主張が中心。

### ② 解決設計力: 2/5
**評価根拠**: 米国公認会計士、銀行・監査法人・警察という多様なキャリアを持つが、それらの専門性を活かした具体的な政策設計は見えにくい。復興副大臣・政務官として行政経験はあるものの、独自の制度設計で成果を上げた記録は限定的。

### ③ 実行力: 2/5
**評価根拠**: 衆議院議員3期、復興副大臣・政務官を歴任したが、目立った政策実現の記録は確認できない。2024年衆院選では惜敗率82.976%で比例復活もできず落選。党内での影響力も限定的で、政策を推進する力に課題がある。

### ④ 説明責任: 2/5
**評価根拠**: 物価高対策として電気料・ガソリン料補助、生活困窮者給付金、地方創生交付金などを訴えたが、これらは政府の既存政策の延長線上にある。独自の見解や政策の深い説明は限定的。2024年総裁選で小林鷹之を支持したが、その後石破に投票するなど、政治判断の説明も不十分。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党一筋で政治キャリアを築いており、政党間の移籍歴はない。憲法改正賛成の立場を2017年、2021年、2024年と継続して表明。道議から衆議院へという一貫したキャリアパスも評価できる。

### ⑥ 倫理観: 2/5
**評価根拠**: 2020年のIR汚職事件関連で、中国企業「500ドットコム」との関係が報道された。観光会社から200万円の寄付を受けた件について「適法に処理」と説明したが、IR汚職事件の渦中で疑惑が報じられた点は印象を悪くした。重大な不正は認定されていないものの、グレーゾーンの問題が残る。

## 強み・弱み

### 強み
- **多様なキャリア**: 銀行・警察・監査法人という異色の経歴
- **専門資格**: 米国公認会計士、MBAの取得
- **行政経験**: 復興副大臣・政務官として行政に携わった経験
- **党内での立場**: 自民党札幌市支部連合会会長を務める

### 弱み（注意点）
- **落選による影響力低下**: 2024年選挙で落選し、国政への影響力を失った
- **IR汚職事件との関連報道**: 直接の不正は認定されていないが疑惑報道あり
- **政策実現の成果不足**: 3期の議員経験に比して目立った成果が見えにくい
- **世襲のイメージ**: 父が道議で、政治家一家としての批判を受ける可能性
