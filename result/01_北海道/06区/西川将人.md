# 西川将人（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年11月7日 |
| 年齢 | 56歳 |
| 選挙区 | 北海道6区（旭川市、士別市、名寄市、富良野市、上川総合振興局管内） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 元旭川市長（4期）、衆議院農林水産委員、沖縄北方問題特別委員 |

## 経歴
- 北海道旭川市出身
- 旭川東高等学校、北海道大学工学部卒業
- 日本航空に入社、パイロット資格取得（主に欧州路線に乗務）
- 1999年：日本航空退職、政治活動開始
- 2000年：衆院選北海道6区に自由党公認で出馬、落選
- 2001年：参院選北海道選挙区に自由党公認で出馬、落選
- 2006年：旭川市長選に無所属で出馬、初当選（当時38歳、北海道最年少首長）
- 2006〜2021年：旭川市長4期15年
- 2021年8月：市長辞職、衆院選出馬表明
- 2021年10月：衆院選北海道6区で落選（比例復活もならず）
- 2024年10月：衆院選で比例復活当選（1期目）
- 2026年1月：立憲民主党離党、中道改革連合入党

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
旭川市長15年の経験から、地方自治の課題を構造的に理解している。「地域格差の是正」「一人ひとりに寄り添う政治への転換」など、地方の視点からの課題設定ができている。SNSでの誹謗中傷・いじめ対策の法整備の必要性を国政の課題として提起するなど、時代に即した問題意識も持つ。

### ② 解決設計力: 4/5
**評価根拠**:
15年間の市政運営で、実際に行政を動かした経験がある。パイロット出身という異色の経歴ながら、市長として4期連続当選を果たしたのは、政策設計と実行の両面で一定の評価を得た証拠。ただし、旭川いじめ問題の対応では課題が残った。

### ③ 実行力: 4/5
**評価根拠**:
旭川市長として15年間、4期連続で市政を担った実績は高く評価できる。38歳での当選時は非自民系として28年ぶりの勝利であり、組織基盤が弱い中での実行力を示した。ただし、国会議員としてはまだ1期目で、国政での実行力は未知数。

### ④ 説明責任: 2/5
**評価根拠**:
旭川女子中学生いじめ凍死事件（2021年）が市長在任中に発生。事件への対応が問われる中で市長を辞職し衆院選に出馬したことは、「責任放棄」との批判を受けた。事件の解明や被害者遺族への説明が十分だったとは言い難い。国政転身の判断について、有権者への説明が不十分だった面がある。

### ⑤ 一貫性: 2/5
**評価根拠**:
自由党→民主党→無所属（市長）→立憲民主党→中道改革連合と、政党の移動が多い。小沢一郎の政治塾出身で小沢グループに属していたが、2026年1月に立憲を離党し中道改革連合に入党。北海道5区への国替えも発表されており、政治スタンスの一貫性に疑問符がつく。

### ⑥ 倫理観: 2/5
**評価根拠**:
旭川いじめ凍死事件の発生時、行政のトップとして適切な対応ができたかは疑問が残る。事件の責任を十分に果たさないまま国政転身を図った判断は、権力者としての責任の取り方として問題がある。明確な違法行為は確認されていないが、危機管理と説明責任の観点で課題がある。

## 強み・弱み

### 強み
- **首長経験**：旭川市長4期15年の行政経験は大きな強み
- **民間経験**：JALパイロットとしての国際的な視野と民間感覚
- **地域への理解**：長年の市政運営で地域課題を熟知

### 弱み（注意点）
- **いじめ事件への対応**：市長在任中の事件対応は批判を受けており、説明責任に課題
- **政党の移動**：複数回の政党変更は一貫性への疑念を生む
- **国替え**：6区から5区への国替えは地域との関係性に影響
- **責任放棄との批判**：いじめ事件の渦中での国政転身は、責任の取り方として問題視された
