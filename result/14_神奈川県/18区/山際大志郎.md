# 山際大志郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年9月12日 |
| 年齢 | 56歳 |
| 選挙区 | 神奈川18区（川崎市高津区・宮前区の一部） |
| 当選回数 | 衆議院7回 |
| 主な役職 | 元経済再生担当大臣、自民党半導体戦略推進議員連盟会長、麻生派 |

## 経歴
- 東京都出身
- 神奈川県立湘南高等学校卒業
- 山口大学獣医学部卒業
- 東京大学大学院農学生命科学研究科 獣医学博士取得
- 東京大学動物医療センター獣医師
- ペットクリニック開院・経営
- 2002年：神奈川8区補選に出馬、落選
- 2003年：第43回衆院選で神奈川18区から出馬、比例復活で初当選
- 2012年：内閣府大臣政務官（第2次安倍内閣）
- 2014年：経済産業副大臣（第2次安倍改造内閣・第3次安倍内閣）
- 2017年：衆議院内閣委員長
- 2018年：自民党副幹事長
- 2019年：自民党政調会長代理
- 2021年：経済再生担当大臣、新しい資本主義担当大臣等（第1次岸田内閣）
- 2022年10月：統一教会問題で大臣辞任（事実上の更迭）
- 2024年10月：衆院選で小選挙区3位、比例復活で7選
- 2024年11月：自民党半導体戦略推進議員連盟会長就任

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 経済再生担当大臣として「新しい資本主義」「スタートアップ五か年計画」等の政策に関与。半導体戦略議連会長としてラピダス支援を推進。しかし、独自の構造的課題提起は確認できず、岸田政権の方針を実行する役割が中心。

### ② 解決設計力: 3/5
**評価根拠**: 東大獣医学博士として高い知的能力を持ち、経産副大臣・経済再生相として政策立案に関与。半導体政策では「経済合理性と安全保障のバランス」を主張。ただし、大臣在任中に際立った政策成果は確認できない。

### ③ 実行力: 3/5
**評価根拠**: 衆議院当選7回、大臣・副大臣を歴任した実績。しかし2024年衆院選では小選挙区3位に沈み、次点との差はわずか3票で比例復活。統一教会問題で大臣を更迭されており、逆風下での選挙戦を乗り切る力に課題。

### ④ 説明責任・コミュニケーション力: 1/5
**評価根拠**: 2024年衆院選で12日間にわたり報道機関の取材を完全拒否。「野党の話は我々政府は何一つ聞かない」発言（2022年）は民主主義の基本に反するとして厳しく批判された。専門家は「民主主義が崩壊する」と警鐘を鳴らした。

### ⑤ 一貫性: 2/5
**評価根拠**: 統一教会との関係について当初は曖昧にし、後に次々と接点が発覚する形となった。秘書の統一教会信者問題も「確認できず」と答弁後に発覚。説明が一貫せず、情報を小出しにする姿勢が批判された。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 統一教会との深い関係が問題視され、2022年10月に大臣を事実上更迭された。韓鶴子総裁との写真、国際会議出席、秘書が統一教会信者として活動していた問題など、政教分離の観点から深刻な問題。取材拒否も国民の知る権利を軽視する姿勢。

## 強み・弱み

### 強み
- **閣僚経験**：経済再生担当大臣、経産副大臣など要職を歴任
- **専門性**：獣医学博士としての知的基盤、経済政策の知見
- **当選回数**：7期当選のベテラン議員
- **派閥基盤**：麻生派に所属

### 弱み（注意点）
- **統一教会問題**：深い関係が発覚し大臣を更迭、国民の信頼を大きく損なった
- **説明責任の欠如**：取材拒否、野党の話を聞かない発言など民主主義の基本姿勢に問題
- **選挙での弱体化**：2024年衆院選で小選挙区3位、比例復活でギリギリの当選
- **情報開示の問題**：統一教会との関係を小出しに認める姿勢が批判された
