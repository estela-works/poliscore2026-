# 宗野創（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1993年1月5日 |
| 年齢 | 31歳 |
| 選挙区 | 神奈川18区（川崎市中原区・高津区） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 衆議院議員、立憲民主党神奈川県第18区総支部長 |

## 経歴
- 神奈川県横浜市出身
- 桐蔭学園高等学校卒業
- 早稲田大学政治経済学部卒業
- 三井住友銀行入行（中小企業融資、個人資産管理業務を担当）
- 子ども支援団体事務局長
- 2020年：松下政経塾入塾（第41期生）
- 2024年：松下政経塾卒塾
- 2024年1月：立憲民主党神奈川18区総支部長として公認内定
- 2024年10月：第50回衆院選で小選挙区初当選（得票率30.72%）
- 現在、川崎市溝口在住

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 新人議員のため独自の政策課題設定は確認できない。立憲民主党の方針に沿った政策スタンス（防衛費増額反対、辺野古移設反対、核保有反対など）が中心。子ども支援団体での経験から子育て支援への関心はうかがえる。

### ② 解決設計力: 3/5
**評価根拠**: 三井住友銀行での中小企業融資・資産管理業務経験、松下政経塾での4年間の政策研究は評価できる。早稲田大学政治経済学部卒の学歴も政策立案の基盤となる。ただし政治家としての政策設計実績はまだない。

### ③ 実行力: 3/5
**評価根拠**: 2024年衆院選で自民党現職の山際大志郎（元経済再生相）を小選挙区で破り初当選という大きな実績。統一教会問題で逆風を受けた山際に対し、政治の刷新を訴えて支持を集めた。ただし国会での実績はこれから。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 各種アンケートに明確に回答し、政策スタンスを表明。SNSでの情報発信も行っている。新人議員として特段の問題は確認されていないが、国会での説明責任を評価するにはまだ時期尚早。

### ⑤ 一貫性: 3/5
**評価根拠**: 政治活動を開始したばかりで、一貫性を評価する材料が不足。松下政経塾での4年間の政策研究から立憲民主党での活動への流れは一貫している。中道リベラル的な政策スタンスは明確。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。統一教会問題を抱えた山際に対抗し当選したことから、政治倫理への意識は持っていると推測される。ただし政治家としての実績は浅く判断材料が不足。

## 強み・弱み

### 強み
- **若さ**：31歳と若く、将来性がある
- **松下政経塾出身**：政策立案能力の基盤を持つ
- **金融業界経験**：三井住友銀行での実務経験
- **選挙での勝利**：現職大臣経験者を小選挙区で破った実績

### 弱み（注意点）
- **政治経験の浅さ**：国会議員としての実績はこれから
- **知名度不足**：全国的な認知度は低い
- **独自性の欠如**：立憲民主党の方針に沿った政策が中心
- **地盤の弱さ**：横浜出身で川崎市への移住は比較的最近
