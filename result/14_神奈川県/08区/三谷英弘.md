# 三谷英弘（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年6月28日 |
| 年齢 | 49歳 |
| 選挙区 | 神奈川県8区 |
| 当選回数 | 衆議院4回 |
| 主な役職 | 法務副大臣（2025年〜） |

## 経歴
- 栄光学園（中高）卒業
- 東京大学法学部卒業
- ワシントン大学ロースクール修了
- 2001年 司法修習修了、TMI総合法律事務所に弁護士登録
- 2006年 中央大学法学部兼任講師（「エンターテイメントと法」担当）
- 2012年 衆院選で東京5区から出馬、比例復活で初当選（みんなの党）
- 2014年 みんなの党解党後、無所属で落選
- 2017年 自民党公認で神奈川8区から出馬、比例復活
- 2020年 文部科学大臣政務官、東京オリンピック担当
- 2024年 比例復活で4選
- 2025年 高市内閣で法務副大臣就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 弁護士としてエンタメ法務に携わった経験を活かし、チケット高額転売問題を社会課題として定義。共同養育支援議連事務局長として離婚後の親子関係という構造的問題に取り組む。法的視点から社会問題を整理・言語化できる。

### ② 解決設計力: 4/5
**評価根拠**: チケット不正転売禁止法を議連事務局長として設計し成立に導いた実績。法案作成プロセスを理解し、制度設計ができる。民法改正（家族法制見直し）にも10年以上取り組み、具体的な法改正を推進。

### ③ 実行力: 4/5
**評価根拠**: チケット不正転売禁止法を議員立法として成立させた実績。文部科学大臣政務官、法務副大臣と政府ポストを経験し、政策実現の実績がある。ただし小選挙区では4連敗（すべて比例復活）。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 週刊文春の不倫疑惑報道に対しブログで「全くの事実無根」と明確に反論。イラスト無断使用問題は認めて是正対応。一方、選択的夫婦別姓反対の圧力文書問題では批判への十分な説明がなく、対応に濃淡がある。

### ⑤ 一貫性: 2/5
**評価根拠**: みんなの党→無所属→自民党と政党を移動。知財・コンテンツ分野への関心は一貫しているが、政治的立ち位置は変化。選択的夫婦別姓では反対の圧力活動を行う一方、LGBT理解増進には賛成するなど、保守とリベラルの間で揺れがある。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 重大な不祥事はないが、「逆転裁判」イラスト無断使用問題（2024年）は著作権の専門家として看過できないミス。選択的夫婦別姓で地方議会に圧力文書を送付し「地方議会の独立性を脅かす」と批判された点も問題。

## 強み・弱み

### 強み
- **法整備の専門性**：弁護士資格を活かし、チケット転売禁止法など議員立法を成立させる実力
- **コンテンツ産業への知見**：エンタメ法務の経験を政策に活かせる希少な人材

### 弱み（注意点）
- **小選挙区での弱さ**：4回連続で小選挙区敗北、すべて比例復活。地元での支持基盤に課題
- **政党移動歴**：みんなの党から自民党へ。政治的軸の一貫性に疑問符
