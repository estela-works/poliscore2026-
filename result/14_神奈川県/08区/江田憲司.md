# 江田憲司（中道改革連合）

## 基本情報

|------|------|
| 生年月日 | 1956年4月28日 |
| 年齢 | 69歳 |
| 選挙区 | 神奈川県8区 |

| 主な役職 | 元維新の党代表、元立憲民主党代表代行 |

## 経歴
- 東京大学法学部卒業
- 1979年 通商産業省入省
- 1987年 ハーバード大学国際問題研究所特別研究員
- 1996年 橋本内閣で総理大臣秘書官（政務担当）
- 2002年 神奈川8区補選で初当選（無所属）
- 2009年 みんなの党結党、幹事長に就任
- 2013年 みんなの党離党・除名、結いの党を結党し代表
- 2014年 維新の党共同代表
- 2015年 大阪都構想否決で代表辞任
- 2016年 民進党代表代行
- 2024年 衆院選で8選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 元通産官僚・総理秘書官の経験を活かし、天下り問題（押し付け型天下りを初めて政府に認めさせた）、年金記録問題、埋蔵金187兆円など構造的問題を発掘・追及。60本以上の質問主意書で行政のムダを明らかにした。問題発見力は高い。

### ② 解決設計力: 3/5
**評価根拠**: 消費税ゼロ、身を切る改革など方向性は提示するが、具体的な制度設計や財源論は薄い。野党議員として批判・提案はできるが、与党で政策を実装した経験がないため、解決策の精緻さに欠ける面がある。

### ③ 実行力: 2/5
**評価根拠**: 野党議員として追及は活発だが、議員立法の成立実績は乏しい。みんなの党、維新の党、民進党と組織を次々と移動・離脱しており、党内での合意形成や組織的な政策実現力には課題。小選挙区では強いが政策実現力は限定的。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 著書多数（『首相官邸』『財務省のマインドコントロール』等）で政策を論理的に説明。記者会見でも自身の立場を明確に発信し、消費税減税の必要性など具体的データを示して主張できる。野党追及型の発信力は高い。

### ⑤ 一貫性: 2/5
**評価根拠**: 無所属→みんなの党→結いの党→維新の党→民進党→立憲民主党と政党を頻繁に移動。「身を切る改革」「規制改革」「消費税減税」への関心は一貫しているが、政治的立ち位置が安定せず「嫌われ者」との評判も。紆余曲折が多い。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 金銭スキャンダルや不祥事は見当たらない。天下り禁止、企業・団体献金禁止、政治資金の透明化を一貫して主張し、自らも「身を切る改革」を実践する姿勢を示している。権力の私物化への批判姿勢は明確。

## 強み・弱み

### 強み
- **行政への深い知見**：総理秘書官、通産官僚としての経験を活かした鋭い行政監視・追及力
- **問題発掘力**：質問主意書等で埋蔵金や天下りなど構造的問題を国民に広く認知させた

### 弱み（注意点）
- **党内対立が多い**：渡辺喜美との対立、維新での代表辞任など、組織運営で軋轢を生みやすい
- **政党移動の多さ**：5つ以上の政党を渡り歩き、「何がしたいのか」が見えにくくなっている
