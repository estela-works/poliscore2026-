# 太栄志（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年4月27日 |
| 年齢 | 48歳 |
| 選挙区 | 神奈川13区（横浜市瀬谷区・大和市・綾瀬市） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 衆議院外務委員会理事、政治改革特別委員会委員、元立憲民主党政務調査会長補佐 |

## 経歴
- 沖永良部島出身、中央大学法学部政治学科卒、同大学院法学研究科修了（政治学修士）
- 2003年：民主党・長島昭久議員の公設秘書
- ハーバード大学国際問題研究所研究員、CSIS パシフィックフォーラム研究員などを歴任
- 2017年：第48回衆院選で希望の党から出馬、落選
- 2021年：第49回衆院選で立憲民主党から出馬、甘利明（自民党幹事長）に勝利し初当選
- 2024年10月：第50回衆院選で再選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 米シンクタンク研究員の経験を活かし、外交・安全保障問題を構造的に説明できる。拉致問題、避難シェルター整備、島嶼防衛など具体的課題を提起。ただし、独自の視点での新たな問題発見というより既存議論の延長。

### ② 解決設計力: 3/5
**評価根拠**: 「領域警備・海上保安体制強化法案」「子ども総合基本法案」など議員立法を提出。給付付き税額控除、最低賃金1500円など具体的政策を掲げる。財源論や実施体制への言及もあるが、成立には至っていない。

### ③ 実行力: 2/5
**評価根拠**: 当選2回の若手議員で、提出した議員立法は未成立。2021年に甘利明を破った選挙での実行力は評価できるが、政策面での成果は限定的。野党議員としての構造的制約あり。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 毎日街頭に立ち、毎月タウンミーティングを開催。初当選前の4年間で10万軒の戸別訪問を実施。有権者との対話を重視する姿勢が一貫。国会でも外務委員会理事として積極的に質疑。

### ⑤ 一貫性: 4/5
**評価根拠**: 外交・安全保障を軸に一貫した活動。企業・団体献金を一切受け取らず、政治資金パーティーを開催しない姿勢を維持。2024年代表選では野田佳彦を推薦し、中道・現実路線を支持。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 企業・団体献金の拒否、政治資金パーティー不開催というクリーンな政治姿勢。スキャンダルや問題発言は見当たらない。透明性を重視した活動を継続。

## 強み・弱み

### 強み
- **外交・安保の専門性**：米シンクタンクでの研究経験を持つ数少ない議員
- **クリーンな政治姿勢**：企業献金・パーティー不開催で資金面の透明性が高い
- **地道な対話活動**：街頭演説とタウンミーティングの継続で有権者との接点を維持

### 弱み（注意点）
- **実行実績の不足**：議員立法提出はあるが成立実績なし
- **影響力の限界**：党内での役職は中堅レベル、政策を動かす力は限定的
