# 金子洋一（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1962年6月28日 |
| 年齢 | 63歳 |
| 選挙区 | 神奈川県20区 |
| 当選回数 | 参議院2回、衆議院0回 |
| 主な役職 | 元参議院国土交通委員長、日本維新の会神奈川20区支部長 |

## 経歴
- 神奈川県横浜市南区出身
- 聖光学院中学校・高等学校卒業
- 早稲田大学政治経済学部政治学科中退
- 東京大学経済学部経済学科卒業
- 経済企画庁入庁
- 経済協力開発機構（OECD）出向
- 内閣府大臣官房総務課総括課長補佐を最後に退官
- 2003年、2005年衆院選：三重5区から民主党で出馬→落選
- 2009年参院補選：神奈川県選挙区で初当選（民主党）
- 2010年参院選：再選
- 参議院国土交通委員長
- デフレ脱却議員連盟事務局長
- 民主党神奈川県連代表
- 青山学院大学大学院兼任講師、中央大学大学院客員教授
- 2016年参院選：神奈川県選挙区で落選（6位）
- 2023年日本維新の会神奈川20区支部長就任
- 2024年衆院選：神奈川20区から出馬→落選（3位）
- 2026年衆院選出馬予定

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 経済企画庁官僚出身で、デフレ脱却という明確な課題を早期から提起。「日本銀行のあり方を考える議員連盟」発起人、デフレ脱却議員連盟事務局長として、日本経済の構造的課題を分析。アベノミクスの金融緩和政策の理論的基盤の一部を民主党時代から主張していた。

### ② 解決設計力: 4/5
**評価根拠**: 東京大学経済学部卒、経済企画庁でOECD出向経験を持つ経済政策の専門家。インフレターゲット、日銀法改正など具体的な政策提言を行ってきた。著書「日本経済復活のシナリオ」「デフレ脱却戦記」シリーズで政策を体系的に解説。大学院で講師も務める。

### ③ 実行力: 3/5
**評価根拠**: 参議院議員2期当選、国土交通委員長を務めた実績がある。しかし、2016年参院選で落選後、2024年衆院選でも落選。近年は選挙で実績を出せていない。デフレ脱却議連の活動が後のアベノミクスに影響を与えたとされるが、直接的な成果は限定的。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 経済政策について著書を複数出版し、専門的内容をわかりやすく解説。シノドス（SYNODOS）などのウェブメディアでも発信。大学院での講義経験もあり、説明能力は高い。SNSでも積極的に情報発信している。

### ⑤ 一貫性: 3/5
**評価根拠**: デフレ脱却という政策的軸は一貫している。しかし、民主党→民進党→日本維新の会と政党を変更。選挙区も三重5区→神奈川県選挙区→神奈川20区と変遷。政策は一貫しているが、所属政党の変更が多い。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。官僚出身として政策立案に真摯に取り組む姿勢はある。ただし、権力を持っていた期間が限られており、評価は限定的。

## 強み・弱み

### 強み
- **経済政策の専門性**: 経済企画庁官僚出身、OECD経験
- **国会経験**: 参議院2期、国土交通委員長
- **理論的基盤**: デフレ脱却の政策を体系的に主張
- **発信力**: 著書多数、大学院講師

### 弱み（注意点）
- **近年の落選続き**: 2016年以降、選挙で勝てていない
- **政党の変遷**: 民主党→民進党→維新と移籍
- **年齢**: 63歳で若手ではない
- **地盤の弱さ**: 神奈川20区での基盤は弱い
