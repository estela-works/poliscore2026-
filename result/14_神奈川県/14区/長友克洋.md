# 長友克洋（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年12月29日 |
| 年齢 | 55歳 |
| 選挙区 | 神奈川14区（相模原市緑区・中央区、愛川町、清川村） |
| 当選回数 | 衆議院1回（比例復活）、神奈川県議4期、相模原市議2期 |
| 主な役職 | 立憲民主党神奈川14区総支部長 |

## 経歴
- 神奈川県相模原市緑区橋本出身、大沢育ち
- 長友学園橋本幼稚園・大沢幼稚園
- 相模原市立大沢小学校、大沢中学校
- 神奈川県立弥栄西高等学校（現・相模原弥栄高）卒業
- 法政大学法学部卒業
- 在学中から藤井裕久衆議院議員秘書（約9年間、公設秘書まで）
- 1999年相模原市議会議員初当選（2期）
- 2007年神奈川県議会議員初当選（4期）
- 2019年国民民主党から神奈川14区候補に
- 2020年立憲民主党神奈川14区支部長
- 2021年衆院選（神奈川14区）落選
- 2024年衆院選で比例復活当選（初当選）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 市議2期、県議4期の経験から地方課題への理解は深い。藤井裕久議員秘書として国政の課題にも触れてきた。ただし独自の政策課題設定というより、地域代表としての活動が中心。

### ② 解決設計力: 3/5
**評価根拠**: 法政大学法学部卒で法制度への理解がある。市議・県議として6期にわたり地方行政に関与した実績。国政レベルでの政策設計はこれからの課題。

### ③ 実行力: 3/5
**評価根拠**: 2021年落選後も挑戦を続け、2024年に比例復活で初当選を果たした粘り強さ。ただし選挙区では赤間二郎に敗れており、相模原での自民優位は続いている。得票差は約7,000票に縮まった。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。当選後「今回はぎりぎりで滑り込めた」と率直にコメント。立憲民主党県連での活動報告も行われている。

### ⑤ 一貫性: 4/5
**評価根拠**: 藤井裕久秘書から始まり、民主党系の政党で一貫して活動。国民民主→立憲民主→中道改革連合と政党は変遷するが、野党リベラル系という立ち位置は維持。相模原という地盤も一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。市議・県議として長年活動し、地道な政治活動を継続してきた姿勢は評価できる。

## 強み・弱み

### 強み
- **地方議員経験**: 市議2期、県議4期の豊富な経験
- **地元密着**: 相模原市出身・育ちの地縁
- **粘り強さ**: 2度目の挑戦で国政進出を実現
- **秘書経験**: 藤井裕久議員秘書としての国政への理解
- **若さ**: 55歳で今後の活動が期待できる

### 弱み（注意点）
- **選挙区での敗北**: 比例復活であり選挙区勝利ではない
- **国政経験が浅い**: 1期目で実績を積む段階
- **赤間との差**: 現職閣僚との知名度・実績差
- **政党変遷**: 所属政党の変更歴
