# 先沖仁志（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年4月26日 |
| 年齢 | 49歳 |
| 選挙区 | 神奈川14区（相模原市緑区・中央区、愛川町、清川村） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党神奈川14区支部長 |

## 経歴
- 広島県生まれ
- 神奈川県相模原市中央区小山在住
- 県立呉商業高等学校卒業
- 金融業界等勤務
- 2018年から相模原市内でコンビニ経営
- 2022年参院選（石川県選挙区）出馬、落選
- 2023年参政党神奈川14区に擁立発表
- 2024年衆院選出馬、15,090票（7.3%）で落選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「減税と積極財政で日本経済を立て直す」という参政党の政策を掲げる。「地域に恩返しがしたい」「市民の声を届ける」という問題意識はあるが、独自の具体的な課題設定は限定的。

### ② 解決設計力: 2/5
**評価根拠**: コンビニ経営者としての実務経験はあるが、政策立案の実績はない。金融業界での経験から経済への理解はあると思われるが、国政レベルでの政策設計能力は未知数。

### ③ 実行力: 2/5
**評価根拠**: 2022年参院選（石川県）、2024年衆院選（神奈川14区）と2度出馬も当選に至っていない。2024年衆院選では7.3%（15,090票）を獲得したが、上位候補との差は大きい。

### ④ 説明責任: 2/5
**評価根拠**: 選挙戦では「国民の政治参加のみが腐った政治を直せる」と訴えた。参政党を通じた活動報告はあるが、個人としての詳細な政策説明は限定的。

### ⑤ 一貫性: 2/5
**評価根拠**: 2022年参院選は石川県、2024年衆院選は神奈川14区と選挙区が異なる。参政党からの出馬という点では一貫しているが、地域との結びつきという点では疑問が残る。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。コンビニ経営者として地域で事業を営んできた実績がある。

## 強み・弱み

### 強み
- **経営者経験**: コンビニ経営者としての実務
- **地域密着**: 相模原市で商売を続けてきた地縁
- **金融経験**: 金融業界での勤務経験
- **行動力**: 参院選・衆院選と積極的に挑戦
- **参政党の支持層**: 一定の固定支持層

### 弱み（注意点）
- **当選経験なし**: 2度の選挙とも落選
- **選挙区変更**: 石川→神奈川と選挙区が異なる
- **知名度不足**: 主要候補と比べ知名度低い
- **党勢の限界**: 参政党単独での当選は困難
- **得票率低迷**: 7.3%にとどまる
