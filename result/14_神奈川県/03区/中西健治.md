# 中西健治（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年1月4日 |
| 年齢 | 62歳 |
| 選挙区 | 神奈川3区（横浜市鶴見区・神奈川区） |
| 当選回数 | 衆議院2期、参議院2期 |
| 主な役職 | 元財務副大臣、衆院財政金融委員会筆頭理事、自民党財務金融部会長 |

## 経歴
- 1964年1月4日、東京都練馬区生まれ
- 東京大学法学部卒業
- 1988年、JPモルガン証券会社入社
- 2006年、同社取締役副社長就任
- 2009年、JPモルガン退社
- 2010年、参議院議員選挙（神奈川選挙区）でみんなの党公認で初当選
- 2014年、みんなの党政策調査会長
- 2014年11月、みんなの党解党に伴い無所属
- 2016年、参議院議員選挙で再選
- 2021年、第49回衆院選で神奈川3区から初当選（衆議院）
- 2022年、自民党財務金融部会長（新NISA拡充の中心的役割）
- 2024年、第50回衆院選で2期目当選（得票92,125票）
- 麻生派所属

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 財務金融部会長として資産形成・投資促進を課題として設定。新NISAの大幅拡充を主導した実績から、具体的な政策課題設定能力を発揮。

### ② 解決設計力: 5/5
**評価根拠**: 東京大学法学部卒、JPモルガン証券取締役副社長として金融の最前線で活躍。財務副大臣として政策立案に関与。金融制度改革に精通。

### ③ 実行力: 4/5
**評価根拠**: 新NISA拡充を実現。参議院2期・衆議院2期と安定した選挙基盤。2024年衆院選も約92,000票で圧勝。

### ④ 説明責任: 3/5
**評価根拠**: 国会での質疑や財金委員会での活動あり。ただし一般有権者への情報発信は限定的。

### ⑤ 一貫性: 3/5
**評価根拠**: みんなの党から無所属を経て自民党へと政党を移籍。政策面では金融・財政分野に一貫して取り組む。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に大きなスキャンダルは報じられていない。民間出身で政治資金問題との関連は薄い。

## 強み・弱み

### 強み
- **金融専門性**: JPモルガン副社長としての豊富な金融経験
- **政策実績**: 新NISA拡充の中心的役割
- **学歴**: 東京大学法学部卒
- **選挙基盤**: 神奈川3区で圧倒的な強さ（約92,000票）
- **副大臣経験**: 財務副大臣として行政経験

### 弱み（注意点）
- **政党移籍**: みんなの党→無所属→自民党
- **エリート色**: 金融エリートとしてのイメージ
- **地元密着度**: 参議院時代は県全体、衆議院は3区のみ
- **自民党逆風**: 裏金問題等の党全体への批判
