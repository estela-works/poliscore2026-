# 中村武人（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年12月23日 |
| 年齢 | 51歳 |
| 選挙区 | 神奈川3区（横浜市鶴見区・神奈川区） |
| 当選回数 | なし（2024年衆院選落選） |
| 主な役職 | 元神奈川県議会議員（3期）、NPO法人代表理事 |

## 経歴
- 1974年12月23日、神奈川県横浜市生まれ
- 早稲田大学大学院公共経営研究科卒業
- 営業職サラリーマンを経験
- 日本語教師として日本・タイ・ラオスで勤務
- 帰国後、自身も資本参加する会社で営業に従事
- 衆議院議員秘書を経験
- 神奈川県議会議員（3期）
- NPO法人代表理事
- 2024年6月、立憲民主党神奈川県連が神奈川3区公認候補として決定
- 2024年10月、第50回衆院選で神奈川3区から出馬（71,464票獲得・落選）
- 2025年、党再編に伴い中道改革連合に所属

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「隠さない政治」を掲げ、政治資金の透明化や金権政治からの脱却を課題として設定。物価高や経済格差是正も主張。

### ② 解決設計力: 3/5
**評価根拠**: 早稲田大学大学院公共経営研究科卒。県議3期の経験から地方行政に精通。具体的解決策として「インターネット公開、罰則強化、第三者機関監視」を提案。

### ③ 実行力: 2/5
**評価根拠**: 県議3期の実績があるが、国政選挙では落選。2024年衆院選で比例復活もならず。

### ④ 説明責任: 4/5
**評価根拠**: 政治改革の具体的方策を明確に説明。「平成の政治改革では目的が達成できなかった」と課題を率直に指摘。

### ⑤ 一貫性: 3/5
**評価根拠**: 県議から国政挑戦への一貫した政治活動。立憲民主党から中道改革連合への再編を経験。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 「隠さない政治」を掲げ政治資金透明化を主張。裏金問題批判の立場で清廉なイメージ。

## 強み・弱み

### 強み
- **県議経験**: 3期の地方議員経験
- **学歴**: 早稲田大学大学院公共経営研究科
- **国際経験**: 日本語教師として海外勤務
- **政策明確**: 政治改革への具体的提案
- **地元基盤**: 神奈川区選出の県議

### 弱み（注意点）
- **国政未経験**: 衆院選落選、比例復活もならず
- **知名度不足**: 鶴見区での支持拡大が課題
- **強敵**: 中西健治という強力な現職
- **党再編**: 立憲から中道改革連合への移行
- **得票差**: 約2万票差での敗北
