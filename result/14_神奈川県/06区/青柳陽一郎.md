# 青柳陽一郎（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年8月29日 |
| 年齢 | 55歳 |
| 選挙区 | 神奈川6区 |
| 当選回数 | 衆議院5回 |
| 主な役職 | 立憲民主党神奈川県連代表、国会対策委員長代理、衆院政治倫理審査会筆頭幹事 |

## 経歴
- 東京都中野区生まれ、横浜市保土ケ谷区で育つ
- 日本大学法学部政治経済学科卒業
- 早稲田大学大学院公共経営研究科修了
- 松田岩夫衆議院議員秘書（1993-2005年）、大臣秘書官
- 「ベトナムフェスティバル」「チャイナフェスティバル」など国際交流イベント主催
- 2010年NPO「TOKYO TOMORROW」設立
- 2012年衆院選でみんなの党から初当選（比例復活）
- 結いの党→維新の党→立憲民主党と党籍変遷
- 2017年衆院選で初めて選挙区勝利
- 2022年立憲民主党神奈川県連代表に就任
- 2024年衆院選で926票差で選挙区勝利、5選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「失われた30年、急速に進む少子化、アベノミクスの限界、自民党政治腐敗」と問題を明確に設定。政治改革として国会議員定数削減、文通費改革、政策活動費廃止、インターネット投票導入など具体的な課題を提示。森友問題についても「さらに調査や説明をすべき」と追及姿勢を維持。

### ② 解決設計力: 3/5
**評価根拠**: 国会対策委員長代理として国会運営の実務に携わる。政治倫理審査会筆頭幹事として裏金問題の追及にも関与。早稲田大学大学院公共経営研究科修了という学術的バックグラウンドを持つ。ただし、自ら主導した法案や制度改革の実績は確認できない。

### ③ 実行力: 4/5
**評価根拠**: 5期連続当選、2017年以降は3回連続で選挙区勝利。2024年選挙では926票差の接戦を制した。2022年には現職の阿部知子を破り神奈川県連代表に就任。国際交流イベントの主催やNPO設立など、政治以外でも行動力を示してきた。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 日本会議所属について有権者から問われた際、「脱会した」と説明。森友問題では「さらに調査や説明をすべき」と与党に説明責任を求める立場。X（旧Twitter）での積極的な情報発信も行う。ただし、2022年参院選での県連内混乱については組織運営に課題を残した。

### ⑤ 一貫性: 2/5
**評価根拠**: みんなの党→結いの党→維新の党→立憲民主党と複数の党を渡り歩いてきた。いずれも非自民・野党系であり政治姿勢の軸は維持しているとも言えるが、党の変遷は多い。日本会議に所属していた時期があり、その後脱会したという経緯もある。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや個人的な不祥事は確認されていない。5期の議員活動を通じて汚職や問題行動は報道されていない。政治とカネの問題に対しても批判的立場を維持。2022年参院選での県連運営に関する批判はあったが、組織運営上の問題であり個人の倫理問題ではない。

## 強み・弱み

### 強み
- **安定した選挙基盤**: 5期連続当選、選挙区で3連勝
- **国会対策の経験**: 国対委員長代理として国会運営に精通
- **政治改革への取り組み**: 政倫審筆頭幹事として裏金問題追及
- **多様な経験**: 国際交流イベント主催、NPO設立など民間経験

### 弱み（注意点）
- **党籍の変遷**: 4つの政党を渡り歩いた
- **日本会議問題**: 過去に日本会議に所属していた（脱会済み）
- **県連運営の課題**: 2022年参院選での県連内混乱
- **選挙区の僅差**: 2024年選挙は926票差、安定的とは言えない
