# 佐々木奈保美（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年9月28日（56歳） |
| 年齢 | 56歳 |
| 選挙区 | 神奈川17区 |
| 当選回数 | 衆議院1回（比例復活）※市議3期、県議2期 |
| 主な役職 | 中道改革連合神奈川17区総支部長、衆議院文部科学委員、防災対策特別委員会委員 |

## 経歴
- 神奈川県小田原市田島出身
- 小田原市立国府津小学校、国府津中学校卒業
- 神奈川県立西湘高等学校卒業
- 和光大学人文学部文学科卒業、教員免許取得
- 株式会社ステップ勤務
- 夫と鴨宮に学習塾「ササキ塾」を開業
- 2007年4月：小田原市議選に神奈川ネットワーク運動から出馬、初当選
- 市議3期務める
- 2019年4月：神奈川県議選に立憲民主党公認で出馬、初当選
- 県議2期務める
- 2024年8月：立憲民主党が次期衆院選神奈川17区に擁立を発表
- 2024年10月：第50回衆院選で牧島かれんに小選挙区で敗れるも比例復活当選（得票111,061票）
- 中道改革連合に入党、神奈川17区総支部長就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「暮らし、いのち、子どもたちの未来を真ん中に」というスローガンのもと、子どもの人権、教育、食の安全、有機農業、平和など具体的な課題を設定している。市民活動の経験から当事者性のある課題設定ができている。「政治は生活」という地域密着の問題意識を19年間持ち続けている。

### ② 解決設計力: 2/5
**評価根拠**: 和光大学卒、教員免許取得、学習塾経営の経験がある。市議3期・県議2期で地方政治の実務経験はあるが、国政での政策設計経験はこれから。人権、教育、福祉、ジェンダー平等という方向性は示しているが、具体的な制度設計の詳細は確認できない。

### ③ 実行力: 4/5
**評価根拠**: 市議3期、県議2期の当選実績があり、地方政治での実行力は証明されている。2024年衆院選では現職の牧島かれんに迫る111,061票を獲得し比例復活当選。元デジタル大臣相手に小選挙区で善戦したことは評価できる。市民活動の立ち上げ経験も多数ある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 公開情報が限定的で、発信力や説明力を判断する材料に乏しい。市民活動での発信や地域での活動実績は確認されるが、国会での活動は始まったばかりで、国政レベルでの説明責任を果たした実績はまだない。

### ⑤ 一貫性: 4/5
**評価根拠**: 神奈川ネットワーク運動→立憲民主党→中道改革連合と所属政党は変遷しているが、「政治は生活」「子どもの人権」「武器によらない平和」という政策軸は19年間一貫している。地元小田原での活動を継続し、選挙区を変えずに国政に挑戦した姿勢も評価できる。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。市民活動を通じた草の根の政治姿勢を持ち、子どもの人権、有機農業、平和活動など社会課題に取り組んできた。権力志向というより市民目線での政治活動を続けている。

## 強み・弱み

### 強み
- **地方政治経験**: 市議3期、県議2期の豊富な地方議会経験
- **市民活動実績**: 子どもの人権、有機農業など多分野での市民活動
- **地元密着**: 19年間の地域活動で地盤を構築
- **得票力**: 2024年衆院選で111,061票を獲得

### 弱み（注意点）
- **国政経験の浅さ**: 衆議院議員1期目で国政実績はこれから
- **政党の変遷**: 神奈川ネット→立憲→中道改革連合と移籍
- **知名度**: 全国的な知名度は低い
- **政策の具体性**: 方向性は明確だが制度設計の詳細が不明
