# 後藤祐一（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年3月25日 |
| 年齢 | 56歳 |
| 選挙区 | 神奈川16区（厚木市、伊勢原市、海老名市） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 立憲民主党幹事長代理、財務金融委員会委員 |

## 経歴
- 神奈川県相模原市出身
- 東京大学法学部卒業
- 経済産業省入省、課長補佐まで務める
- 2009年：第45回衆議院選挙で初当選（民主党）
- 2017年：民進党を離党、除籍処分後に希望の党へ
- 2018年：国民民主党結党に参加、政調会長代理
- 2020年：立憲民主党に合流
- 2024年10月：6期目当選（100,677票、小選挙区2連勝）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 経産省出身の政策通として、都市農業振興、政治資金改革など具体的な政策課題を設定。政治改革委員会での「公開方法工夫支出」への追及など、政策の矛盾点を指摘する能力がある。

### ② 解決設計力: 4/5
**評価根拠**: 都市農業推進議員連盟会長として都市農業基本法の成立や相続税改正を実現。62本の法案提出実績があり、質問通告のオンライン化など具体的な制度設計にも関与。

### ③ 実行力: 4/5
**評価根拠**: 6期連続当選を果たし、野党の牙城ではなかった神奈川16区で2連続小選挙区勝利。都市農業基本法の成立など立法実績もある。ただし野党議員のため政策実現には限界。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2017年のパワハラ問題では防衛省女性官僚への高圧的対応が問題視された。国会質疑では相手を追い詰めるスタイルが特徴だが、威圧的との批判もある。スキャンダル後の釈明は一応行っている。

### ⑤ 一貫性: 2/5
**評価根拠**: 民主党→民進党（除籍処分）→希望の党→国民民主党→立憲民主党と政党を転々としている。政策的には一貫した面もあるが、民進党からの除籍や党籍の頻繁な変更は一貫性に疑問。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2015年のタクシー泥酔事件では警察沙汰に。2017年のパワハラ問題では「人事評価を下げてやる」等の発言で官僚を威圧。複数の問題行動があり、権力の扱い方に課題。

## 強み・弱み

### 強み
- **政策立案能力**：62本の法案提出、都市農業基本法成立など具体的な政策実績
- **選挙の強さ**：自民党の牙城で6期連続当選、2連続小選挙区勝利
- **国会質疑力**：与党への追及が鋭く、政治改革委員会等で存在感

### 弱み（注意点）
- **倫理面の問題**：パワハラ・泥酔事件など複数のスキャンダル
- **政党遍歴の複雑さ**：5つの政党を渡り歩き、除籍処分も経験
- **威圧的な対応**：官僚や質疑相手への高圧的な態度が批判されることがある
