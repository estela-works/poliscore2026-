# 土屋品子（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1952年2月9日 |
| 年齢 | 74歳 |
| 選挙区 | 埼玉16区（さいたま市岩槻区・春日部市・吉川市・松伏町） |
| 当選回数 | 衆議院9回 |
| 主な役職 | 復興大臣（第16代）、自由民主党総務会副会長 |

## 経歴
- 東京都港区生まれ、埼玉県春日部市在住
- 父は土屋義彦（元参議院議長、元環境庁長官、元埼玉県知事）
- 浦和市立岸中学校卒業
- 共立女子高等学校卒業
- 聖心女子大学文学部歴史社会学科卒業
- 香川栄養専門学校卒業
- 栄養士、調理師、製菓衛生師の資格取得
- 1996年：第41回衆議院議員総選挙で埼玉13区から無所属で初当選
- 2001年：自由民主党に入党
- 2002年：外務大臣政務官（第1次小泉改造内閣）
- 2006年：環境副大臣（第1次安倍内閣）
- 2013年：厚生労働副大臣（第2次安倍内閣）
- 2023年9月：復興大臣就任（第2次岸田第2次改造内閣）、女性初の復興大臣
- 2024年10月：第50回衆議院議員総選挙で9選（66,909票）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 復興大臣として被災地復興、処理水問題などに取り組んだが、独自の問題提起や構造的な課題設定は確認できない。食育調査会長として食育分野に注力してきたが、社会全体に影響を与えるような問題提起の実績は見当たらない。選択的夫婦別姓賛成派として議員連盟の会長代行を務めるなど個別テーマでの活動はあるが、本質的なボトルネックを言語化した実績は乏しい。

### ② 解決設計力: 2/5
**評価根拠**: 副大臣3回、大臣1回を歴任したが、自ら主導した制度設計や議員立法の目立った実績は確認できない。復興大臣として処理水の安全性発信などに取り組んだが、新たな政策を設計・実現したというよりは既存の方針を継続した形。選択的夫婦別姓制度の議連活動も、法案成立には至っていない。9期27年の議員経験に比して、政策設計の成果は限定的。

### ③ 実行力: 4/5
**評価根拠**: 9期連続当選という安定した選挙実績。2024年選挙では66,909票を獲得し当選。外務大臣政務官、環境副大臣、厚生労働副大臣、復興大臣と政府内ポストを歴任し、政府の一員として役割を果たしてきた。女性初の復興大臣として象徴的な意味も持つ。ただし、大臣としての大きな政策実現や改革断行の実績は確認できず、5点には届かない。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 復興大臣として処理水の安全性を「積極発信」する姿勢を示したが、2023年10月の福島原発廃液事故について「報道で知った」と発言し、情報共有の不備が指摘された。国会での委員会答弁では「『等』は挨拶ということでございまして、事情聴取は含まれておりません」といった曖昧な説明もあった。説明責任を果たす姿勢に課題が見られる。

### ⑤ 一貫性: 4/5
**評価根拠**: 1996年の初当選以来、埼玉県（13区→16区）で継続的に活動。2001年に自民党入党後は一貫して同党に所属。食育、女性活躍、復興支援など関心分野は明確で、選択的夫婦別姓賛成など個別の政策スタンスも維持している。初当選時は無所属だったが、その後の政党移籍はなく、活動に大きなブレは見られない。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2023年9月の復興大臣就任直後、日刊ゲンダイが「政治とカネ」問題を報じた。具体的な内容は明確でないが、内閣改造直後に女性閣僚3人（加藤鮎子、小渕優子、土屋品子）のスキャンダルが報じられた一人。また、2000年の選挙で水着姿の政見放送を製作し話題になるなど、政治家としての品位に疑問符がつく行動もあった。父・土屋義彦元知事の後継として「世襲」の側面もある。

## 強み・弱み

### 強み
- **豊富な経験**: 9期27年の議員経験、副大臣3回、大臣1回の実績
- **安定した地盤**: 埼玉県で長期にわたり安定した当選を続ける選挙基盤
- **女性活躍の象徴**: 女性初の復興大臣、選択的夫婦別姓推進派

### 弱み（注意点）
- **政策実績の弱さ**: 9期の経験に比して、主導した政策や法案の成果が乏しい
- **説明責任の課題**: 廃液事故を「報道で知った」発言など、情報共有・説明に課題
- **世襲批判**: 父・土屋義彦の地盤を継承した世襲議員としての批判
