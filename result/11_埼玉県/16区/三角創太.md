# 三角そうた（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1987年12月16日 |
| 年齢 | 37歳 |
| 選挙区 | 埼玉16区（さいたま市岩槻区・春日部市・吉川市・松伏町） |
| 当選回数 | 衆議院1回（比例北関東ブロック） |
| 主な役職 | 衆議院議員、公認会計士、税理士 |

## 経歴
- 埼玉県和光市生まれ、現在は春日部市粕壁在住
- 母子家庭で育ち、アルバイトをしながら学業を継続
- 2003年：開成中学校卒業
- 2006年：開成高等学校卒業
- 2010年：一橋大学社会学部卒業
- 在学中：手塚仁雄衆議院議員の秘書を務める
- 2010年：株式会社三菱東京UFJ銀行入社（大宮支社、京都支社、本社IR室、信託不動産部）
- 2016年3月：民主党が次期衆院選埼玉13区に擁立を発表
- 2017年10月：第48回衆議院議員総選挙で希望の党公認・埼玉11区から出馬、落選
- 2019年：公認会計士試験合格、春日部市に三角公認会計士事務所開設
- 2019年9月：旧立憲民主党埼玉13区支部長就任
- 2021年：税理士登録、春日部市に三角創太税理士事務所開設
- 2021年10月：第49回衆議院議員総選挙で立憲民主党公認・埼玉13区から出馬、落選
- 2024年10月：第50回衆議院議員総選挙で立憲民主党公認・埼玉16区から出馬、比例復活で初当選
- 2025年6月：第3子誕生

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「どんな家庭の子どもも等しく学習の機会が与えられる社会」を目指すという個人的経験に基づく問題意識は明確。高校時代に家庭を支えるために高校を辞めた同級生の存在が政治家を志す原点という話は具体的。しかし、政策課題としての構造的な分析や、独自の視点での問題提起は確認できない。党の政策（政治改革、経済政策など）を代弁する段階にとどまっている。

### ② 解決設計力: 4/5
**評価根拠**: 公認会計士・税理士として財務・税務の専門知識を持ち、三菱東京UFJ銀行での実務経験もある。議員活動では「税のプロフェッショナルとして即戦力」と評され、財政・税制分野での政策設計能力が期待される。証券アナリスト協会検定会員の資格も持ち、経済・金融政策の設計において専門性を発揮できる可能性が高い。ただし、当選1期目で具体的な成果はまだ出ていない。

### ③ 実行力: 2/5
**評価根拠**: 2017年、2021年と2回落選を経験した後、2024年に比例復活で初当選。8年間の浪人生活を経ての当選は粘り強さの表れ。しかし、当選1期目でまだ議員立法や政策実現の具体的実績はない。選挙区では自民党の土屋品子に敗れており、比例復活という形での当選にとどまった。新人議員として今後の実績に期待するが、現時点での評価は低い。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: SNSやウェブサイトでの発信活動は継続しており、一定のコミュニケーション能力はある。公認会計士・税理士として専門的な内容を説明する能力も持っていると推測される。しかし、全国的な知名度は低く、メディアでの発信や国会での質疑などで説明力を示した実績は確認できない。議員1年目であり、今後の活動に期待。

### ⑤ 一貫性: 2/5
**評価根拠**: 政党の移籍が多い点が懸念。2016年民主党、2017年希望の党、2019年以降立憲民主党、そして現在は中道改革連合と、複数の政党を渡り歩いている。これは野党再編の流れに沿ったものではあるが、有権者から見ると「一貫した軸」が見えにくい可能性がある。政策的には「教育格差の是正」「政治改革」などの主張は継続しているが、政党移籍の多さは減点要因。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや政治資金問題の報道は確認できない。「政治とカネ」問題への取り組みを掲げており、政治改革への意識は高い。公認会計士・税理士として財務の透明性を重視する職業倫理も持っていると推測される。議員1年目で権力行使の実績は少ないが、現時点で倫理的な問題は見当たらない。

## 強み・弱み

### 強み
- **専門性**: 公認会計士・税理士・証券アナリストの資格、財務・税務の専門知識
- **実務経験**: 三菱東京UFJ銀行での銀行業務、中小企業支援の経験
- **若さと粘り強さ**: 37歳と若く、8年間の浪人を経て当選した粘り強さ

### 弱み（注意点）
- **政党移籍の多さ**: 民主党→希望の党→立憲民主党→中道改革連合と複数回移籍
- **当選実績の弱さ**: 選挙区では3回連続敗北、比例復活での当選
- **知名度の低さ**: 全国的にはほとんど知られていない
