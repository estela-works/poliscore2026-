# 大島敦（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年12月21日 |
| 年齢 | 67歳 |
| 選挙区 | 埼玉6区（鴻巣市・上尾市・桶川市・北本市） |
| 当選回数 | 衆議院9回 |
| 主な役職 | 元総務副大臣、元内閣府副大臣、立憲民主党埼玉県連代表 |

## 経歴
- 埼玉県北本市生まれ
- 京華高等学校卒業
- 早稲田大学法学部卒業（会社法ゼミ）（1981年）
- 1981年日本鋼管（現JFEスチール）入社、輸出営業担当、ドイツ駐在3年半
- ソニー生命保険で5年間営業職
- 2000年第42回衆議院選挙で初当選（公明党・若松謙維を破る）
- 2009年内閣府副大臣（鳩山・菅内閣）
- 2012年総務副大臣（野田内閣）
- 民進党幹事長、希望の党幹事長兼政策調査会長、立憲民主党組織委員長を歴任

## 評価詳細

### ① 課題設定能力（4/5）
民間企業（鉄鋼メーカー、生命保険）での実務経験から、産業・経済の現場感覚を持つ。内閣府副大臣として公務員制度改革、消費者行政など幅広い分野を担当。構造的な課題把握ができる。

### ② 解決設計力（4/5）
総務副大臣として地方行政、内閣府副大臣として行政改革に携わった経験から、制度設計の知見がある。民間出身の視点を活かした政策提案ができる。

### ③ 実行力（5/5）
9期連続当選という圧倒的な選挙力。埼玉6区では中根一幸との対決で7勝1敗という実績。民進党・希望の党の幹事長を務め、党運営の実務を担った。困難な局面でも結果を出してきた。

### ④ 説明責任（4/5）
民間企業での営業経験、ドイツ駐在経験から、多様な相手への説明能力がある。野党の立場から政府への追及も行ってきた。

### ⑤ 一貫性（4/5）
民主党系政党の中核として一貫して活動。党の離合集散を経験しながらも、中道・改革路線を維持。立憲民主党埼玉県連代表として地域活動も継続。

### ⑥ 倫理観（4/5）
金銭スキャンダルは確認されていない。裏金問題で非公認となった中根一幸との対決で圧勝したことは、有権者からの信頼の表れ。

## 強み・弱み
### 強み
- **民間経験**：鉄鋼メーカー、保険会社での実務経験
- **選挙の強さ**：9期連続当選、地元での盤石な支持基盤
- **党運営経験**：複数の党で幹事長を務めた組織力

### 弱み
- **年齢**：67歳、ベテランとしての存在感はあるが世代交代の課題
- **野党の限界**：長期間野党にあり、政策実現の機会が限られている
