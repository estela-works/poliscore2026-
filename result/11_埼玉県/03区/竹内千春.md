# 竹内千春（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年7月4日 |
| 年齢 | 56歳 |
| 選挙区 | 埼玉3区（比例北関東で復活当選） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 弁護士、元大学助教授 |

## 経歴
- 宮崎県延岡市出身
- 中央大学文学部文学科英米文学専攻卒業
- ジョージア・サウスウエスタン大学社会行政学部修士課程修了
- ジョージア大学教育学部修士課程修了
- 大宮法科大学院大学法務博士（専門職）
- 電通ヤング＆ルビカム株式会社マーケティング局勤務
- 南九州短期大学国際教養学科助教授（異文化コミュニケーション、英語教科教育法等担当）
- 35歳で法科大学院に進学、41歳で弁護士資格取得
- 日本弁護士連合会国際室室長
- 竹内千春法律事務所設立
- 2021年第49回衆議院選挙に千葉7区から立候補（落選）
- 2024年第50回衆議院選挙で埼玉3区から立候補、比例復活で初当選

## 評価詳細

### ① 課題設定能力（4/5）
弁護士として社会問題に向き合ってきた経験から、現場視点での課題設定ができる。セクハラ被害の相談をきっかけに法曹を志した経緯は、問題意識の明確さを示す。国際経験（米国大学院、日弁連国際室長）から、グローバルな視点も持つ。

### ② 解決設計力（4/5）
弁護士として法的な問題解決の専門性がある。大学助教授としての教育経験もあり、複雑な問題を整理して伝える能力がある。ただし、国政での政策設計の実績はこれから。

### ③ 実行力（2/5）
国会議員としては1期目であり、国政での実行実績は限られている。弁護士としての実務経験、法律事務所経営の経験はあるが、政治家としての実行力は今後の評価。2021年千葉7区での落選を経ての初当選であり、粘り強さはある。

### ④ 説明責任（4/5）
弁護士・大学教員としての経験から、論理的な説明能力がある。国際的な場での活動経験もあり、多様な相手への説明に慣れている。

### ⑤ 一貫性（4/5）
「弱者の味方」という姿勢は一貫している。セクハラ被害者の相談から法曹を志し、弁護士として活動してきた経緯に一貫性がある。立憲民主党での活動も継続的。

### ⑥ 倫理観（4/5）
弁護士としての職業倫理を持つ。権力を持たない立場の人々の支援に取り組んできた経歴は、倫理観の高さを示唆する。

## 強み・弱み
### 強み
- **専門性**：弁護士としての法的知識、国際経験
- **キャリアチェンジの経験**：35歳で法科大学院進学という挑戦
- **現場感覚**：弱者支援の実務経験

### 弱み
- **国政経験の浅さ**：1期目で実績はこれから
- **選挙基盤**：埼玉3区は新しい地盤で、比例復活当選
