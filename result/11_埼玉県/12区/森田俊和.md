# 森田俊和（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年9月19日 |
| 年齢 | 50歳 |
| 選挙区 | 埼玉12区（熊谷市・行田市・加須市・羽生市） |
| 当選回数 | 衆議院3回 |
| 主な役職 | 介護事業経営者、元埼玉県議会議員 |

## 経歴
- 埼玉県熊谷市生まれ
- 早稲田大学大学院修了
- 2003年埼玉県議会議員選挙に無所属で立候補（23票差で落選）
- 2005年介護事業を開始、訪問看護ステーション開設
- 2007年県議会議員選挙で初当選（得票数トップ、無所属、当選後自民党会派入り）
- 2011年県議選で自民党公認で再選
- 2012年自民党支部長公募に応募するも落選、離党届→除名処分
- 2017年希望の党公認で初当選（比例復活、野中厚に492票差）
- 2018年国民民主党に参加
- 2020年立憲民主党に参加
- 2021年第49回衆議院選挙で初の小選挙区当選（野中厚を破る）
- 2024年第50回衆議院選挙で3選

## 評価詳細

### ① 課題設定能力（4/5）
介護事業経営者として、高齢化社会の課題を現場で把握している。県議会議員としての経験から、地方の課題も理解。早稲田大学大学院での学びも政策理解に寄与。

### ② 解決設計力（4/5）
介護事業（訪問看護ステーション）を自ら立ち上げた実績があり、制度を活用した事業設計の経験がある。県議としての政策立案経験も持つ。

### ③ 実行力（4/5）
2003年の23票差落選から再起し、県議2期、国会議員3期と着実にキャリアを積み上げた。野中厚との対決で勝利を収め、選挙での実行力を証明。介護事業の立ち上げ・経営も実行力の表れ。

### ④ 説明責任（4/5）
地元熊谷市を拠点とした地道な活動で有権者との対話を重視。介護現場の経験を踏まえた具体的な説明ができる。

### ⑤ 一貫性（2/5）
政党の移籍が多い（自民会派→自民除名→希望→国民→立憲）。自民党の支部長公募に応募しながら、落選後に野党から出馬という経緯は、政策信念よりも当選を優先したとの見方もできる。「地域のため」という軸はあるが説明が必要。

### ⑥ 倫理観（4/5）
金銭スキャンダルは確認されていない。介護事業という社会貢献度の高い事業を経営。自民党から除名されながらも政治活動を継続した粘り強さ。

## 強み・弱み
### 強み
- **現場経験**：介護事業経営者としての実務経験
- **選挙力**：野中厚との対決で2連勝
- **地元密着**：熊谷市を拠点とした活動

### 弱み
- **政党遍歴**：自民→希望→国民→立憲と移籍が多い
- **除名歴**：自民党からの除名処分
- **イデオロギーの不明確さ**：自民会派から野党への転身
