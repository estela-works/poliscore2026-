# 大塚拓（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1973年6月14日 |
| 年齢 | 51歳 |
| 選挙区 | 埼玉9区 |
| 当選回数 | 衆議院5回（2024年落選） |
| 主な役職 | 衆議院安全保障委員長、内閣府副大臣、財務副大臣、自民党国防部会長 |

## 経歴
- 慶應義塾幼稚舎から慶應義塾大学法学部政治学科卒業
- 2005年：ハーバード大学ケネディ行政大学院 公共政策修士号取得（国際貿易・金融専攻）
- 1997年：東京三菱銀行（現・三菱UFJ銀行）入社
- 2003年：同行退職、不動産経営開始
- 2012年：第46回衆議院議員総選挙で初当選（埼玉9区）
- 法務大臣政務官、財務副大臣、内閣府副大臣を歴任
- 2024年10月：第50回衆議院議員総選挙で落選（78,811票）
- 2025年2月：自民党埼玉県第九選挙区支部長に選任
- 妻は参議院議員の丸川珠代

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 安全保障分野において、「世界はすでに『平時』から『戦時』に移行している」「中国による初の軍用機領空侵犯や44年ぶりの大陸間弾道ミサイルの太平洋への発射など、脅威の段階が上がった」と構造的に問題を指摘。経済安全保障推進本部で骨太方針への経済安全保障課題明記を岸田総理に提言するなど、独自の問題提起ができている。

### ② 解決設計力: 4/5
**評価根拠**: ハーバード大学で公共政策修士号を取得し、銀行勤務経験もある。財務副大臣、内閣府副大臣として実務を担当。防衛力強化、日米同盟強靭化、経済安全保障など具体的な政策設計を示している。当選2期目での法務部会長就任は異例の抜擢であり、政策通としての評価がうかがえる。

### ③ 実行力: 4/5
**評価根拠**: 衆議院5期当選、衆議院安全保障委員長、内閣府副大臣、財務副大臣、法務大臣政務官など要職を歴任。自民党国防部会長として安全保障政策の推進に関与。経済安全保障推進本部メンバーとして政策提言を行い、一定の成果を上げている。

### ④ 説明責任: 1/5
**評価根拠**: 裏金問題に関して政治倫理審査会への出席を拒否。当初「2年分528万円」と公表したが、実際は「5年分994万円」と後に判明し、説明と実態に乖離があった。処分後に「強く憤りを覚える」と反発し、会合で「選管と党に言われた通りにやった」「安倍派のせいにされた」と責任転嫁の発言。裏金問題発覚直後の2023年12月にパーティーを開催し、「自分は何も関係していない」と発言。説明責任を果たす姿勢が著しく欠如。

### ⑤ 一貫性: 2/5
**評価根拠**: 安全保障政策では一貫した立場を維持しているが、裏金問題への対応で問題発言が多い。当初の説明と後の事実が異なり、処分への不満を公言するなど、発言に一貫性がない。「お騒がせして申し訳ない」と言いながら「自分は関係ない」と発言するなど、矛盾した姿勢が見られる。

### ⑥ 倫理観: 1/5
**評価根拠**: 政治資金パーティー収入の不記載額は5年間で994万円に上り、党から戒告処分を受けた。裏金問題発覚直後にパーティーを開催するなど、倫理感覚の欠如が顕著。政治倫理審査会への出席を拒否し、説明から逃げた。別途、日本柔道整復師連盟からの50万円献金の不記載も発覚。検察審査会では不起訴相当となったものの、政治家としての説明責任を果たしていない。

## 強み・弱み

### 強み
- **安全保障の専門性**: 衆議院安全保障委員長、国防部会長として安全保障政策に精通
- **高い学歴・実務経験**: ハーバード大学修士号、銀行勤務、複数の副大臣経験
- **政策設計力**: 経済安全保障など、制度・予算を含めた具体的な政策提言ができる

### 弱み
- **説明責任の欠如**: 裏金問題で政治倫理審査会を拒否し、責任転嫁の発言を繰り返す
- **倫理観の問題**: 約1000万円の不記載、問題発覚直後のパーティー開催など
- **選挙基盤の弱体化**: 2024年衆院選で落選、裏金問題で比例重複も認められず
