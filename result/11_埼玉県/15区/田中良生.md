# 田中良生（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1963年11月11日 |
| 年齢 | 61歳 |
| 選挙区 | 埼玉15区（蕨市・戸田市・さいたま市桜区・南区） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 自由民主党筆頭副幹事長、政務調査会副会長 |

## 経歴
- 立教高等学校（現・立教新座高等学校）卒業
- 1986年：立教大学経済学部経済学科卒業
- 1991年：ケイ・アール・ベンチャー起業
- 蕨ケーブルビジョン専務取締役・代表取締役社長を歴任、現取締役会長
- 1997年：蕨青年会議所第25代理事長
- 2001年：日本青年会議所埼玉ブロック協議会会長
- 2005年：第44回衆議院議員総選挙で初当選
- 2013年9月：経済産業大臣政務官（第2次安倍内閣）
- 2015年10月：自民党経済産業部会長
- 2016年8月：国土交通副大臣
- 2018年：内閣府副大臣（女性活躍・男女共同参画担当）
- 2024年10月：第50回衆議院議員総選挙で6選（73,973票）
- 2024年11月：自民党筆頭副幹事長就任

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「凛とした日本を創る」「強い日本で優しい社会を」といった抽象的なスローガンが中心で、独自の問題提起は確認できない。コロナ禍後の経済成長、物価上昇、少子高齢化など一般的な課題には言及しているが、構造的な分析や独自の視点は乏しい。経産部会長として産業政策に関わったが、際立った問題提起の実績は見当たらない。

### ② 解決設計力: 2/5
**評価根拠**: 副大臣・政務官を歴任し政策立案に関与した経験はあるが、自ら主導した制度設計や議員立法の具体的実績は確認できない。内閣府副大臣として女性活躍推進に取り組んだものの、具体的な政策成果は明確でない。財源や実施体制まで踏み込んだ政策提言は見当たらない。

### ③ 実行力: 4/5
**評価根拠**: 経済産業大臣政務官、国土交通副大臣、内閣府副大臣を歴任し、政府内で一定の役割を果たした。初当選以来一貫して無派閥を貫きながら6期連続当選を果たし、2024年選挙では73,973票を獲得。自民党筆頭副幹事長に就任するなど党内での地位を確立している。ベンチャー企業経営やケーブルテレビ会社経営の実績も実行力を裏付ける。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 地元メディアへの取材対応などは行っているが、全国的な発信力や説明力を示す実績は確認できない。副大臣時代の国会答弁でも特筆すべき評価は見当たらない。政策の具体的な説明や批判への対応についての情報が乏しい。

### ⑤ 一貫性: 4/5
**評価根拠**: 初当選以来一貫して無派閥を貫き、特定の派閥に属さず党改革・政治改革を推進する姿勢を維持。2009年の落選を経ても自民党に残り、埼玉15区で継続的に活動。政策面でも経済成長と生活者支援を軸とした主張に大きなブレは見られない。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2018年の日刊ゲンダイ報道によると、内閣府副大臣就任時に「内閣情報調査室の身体検査では"真っ黒"」「過去に不祥事を起こしている」との指摘があった。具体的な内容は明確でないが、政治資金や過去の行動に疑問符がつく報道がある。詳細な説明は確認できず、透明性に課題がある可能性。

## 強み・弱み

### 強み
- **実務経験の豊富さ**: 副大臣3回、政務官1回と政府内での経験が豊富
- **地盤の強さ**: 6期連続当選、無派閥ながら党内で一定の地位を確立
- **経営者としての経験**: ベンチャー企業・ケーブルテレビ会社経営の実績

### 弱み（注意点）
- **政策の具体性不足**: 独自の政策提言や議員立法の実績が乏しい
- **過去の疑惑**: 副大臣就任時の「身体検査で真っ黒」報道、詳細な説明なし
- **発信力の弱さ**: 全国的な知名度や政策発信力に欠ける
