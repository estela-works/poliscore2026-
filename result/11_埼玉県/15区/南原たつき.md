# 南原たつき（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年5月29日 |
| 年齢 | 65歳 |
| 選挙区 | 埼玉15区（蕨市・戸田市・さいたま市桜区・南区） |
| 当選回数 | 新人（国政選挙当選歴なし） |
| 主な役職 | 日本維新の会参議院比例区第1支部支部長、株式会社LUFTホールディングス会長 |

## 経歴
- 岡山県生まれ、1966年に名古屋市へ転居
- 愛知県立松蔭高校卒業
- 愛知工業大学中退
- 2018年：産業能率大学大学院入学（後に中退）
- 1982年：大学在学中に高級外車の並行輸入を開始
- 1983年：「南原商会」として個人商店をスタート
- 1988年：「オートトレーディングルフトジャパン」創業
- 2001年：TVRの輸入販売権取得
- 2002年：ロータス・カーズの輸入販売権取得
- 2002年〜2004年：テレビ番組「マネーの虎」に出演、「冷徹な虎」と呼ばれる
- 2003年：資本金9億9,771万円、売上高94億9,200万円の企業に成長
- 2005年：MGローバー経営破綻により事業崩壊、一文無しに
- 2016年：年商100億円達成（再起）
- 2024年8月：日本維新の会参議院比例区第1支部支部長就任
- 2025年7月：第27回参議院議員通常選挙比例区に出馬、6,309票で落選（13人中13位）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「政治に欠けているのは経営者的な視点」という問題意識は一定の視点を持っているが、政治課題の構造的な分析は見られない。「税金の無駄な使い方を省き、国力を上げる」という主張は表面的で、具体的なボトルネックの特定には至っていない。経営者としての経験に基づく指摘はあるが、政策課題としての掘り下げが不足。

### ② 解決設計力: 2/5
**評価根拠**: 政策として「起業しやすい社会の実現」「投資促進で経済を活性化」「柔軟な働き方改革」「自動運転と農業革新」などを掲げているが、いずれも具体的な制度設計や財源の説明に欠ける。経営者としては事業を設計・実行してきたが、政策設計の経験・実績はなく、評価できる材料が乏しい。

### ③ 実行力: 1/5
**評価根拠**: 国政選挙での当選実績がなく、議員としての実行力を評価する材料がない。2025年参院選では比例区で6,309票、13人中最下位で落選。政治家としての実績はゼロであり、新人として最低評価が妥当。ただし、ビジネスにおいては一度破綻から再起して年商100億円企業を築いた実行力は評価できるが、それは政治家としての実行力とは別の評価軸。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 「マネーの虎」出演経験があり、テレビでの発信力・コミュニケーション能力は一定程度ある。しかし、政治家としての説明責任を問われた場面がないため、政策説明や批判への対応能力は未知数。経営者として顧問業や講演活動を行っており、説明力は一定あると推測されるが、政治の文脈での評価は困難。

### ⑤ 一貫性: 2/5
**評価根拠**: 政治活動の歴史が浅く、一貫性を評価する材料が乏しい。2024年8月に維新から政治参入を表明したばかりで、政策や主張の変遷を追跡できない。ビジネス面では輸入車販売から多角経営へと事業を変化させており、柔軟性はあるが「一貫した軸」があるかは不明確。政治家としての信念や軸が確立されているかは判断できない。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや政治資金問題の報道は確認できない。ビジネスにおいて2005年のMGローバー破綻で一文無しになった際も、違法行為や不正の指摘はなく、正当な経営判断の結果として破綻を受け入れ再起を図った。政治家としての権力行使の実績がないため評価は限定的だが、現時点で倫理的な問題は見当たらない。

## 強み・弱み

### 強み
- **ビジネス経験の豊富さ**: 40年以上の経営経験、破綻からの再起を経験
- **知名度**: 「マネーの虎」出演による一定の認知度
- **経営者視点**: 財務・経営の知識を政策に活かせる可能性

### 弱み（注意点）
- **政治経験ゼロ**: 国政選挙で当選実績なし、議員活動の経験なし
- **選挙での弱さ**: 2025年参院選で比例最下位落選、集票力に課題
- **政策の具体性不足**: 経営者視点の主張はあるが、具体的な制度設計がない
