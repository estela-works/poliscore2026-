# 枝野幸男（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年5月31日 |
| 年齢 | 60歳 |
| 選挙区 | 埼玉5区（さいたま市西区・北区・大宮区・中央区） |
| 当選回数 | 衆議院11回 |
| 主な役職 | 元内閣官房長官、元経済産業大臣、元外務大臣、立憲民主党元代表 |

## 経歴
- 栃木県宇都宮市生まれ
- 東北大学法学部卒業（1987年）
- 1988年司法試験合格、1991年弁護士登録
- 1993年日本新党公認で初当選（旧埼玉5区）
- 民主党政策調査会長、幹事長代理等を歴任
- 2010年内閣府特命担当大臣（行政刷新）
- 2011年内閣官房長官（菅直人内閣）- 東日本大震災・福島原発事故対応
- 2012年経済産業大臣（野田内閣）
- 2017年立憲民主党を設立、代表就任（2021年まで）
- 2024年外務大臣

## 評価詳細

### ① 課題設定能力（5/5）
弁護士出身で法的・論理的な思考力に優れる。東日本大震災・福島原発事故という未曾有の危機に官房長官として対応し、国家的危機の本質を見極める能力を示した。野党党首として与党の問題点を鋭く指摘してきた。

### ② 解決設計力（4/5）
行政刷新担当大臣として「事業仕分け」を主導。経済産業大臣として政策立案に携わった。野党時代も対案を示す姿勢を重視。ただし、野党経験が長く、政権での政策実現の機会は限られてきた。

### ③ 実行力（4/5）
11期連続当選という圧倒的な選挙力。立憲民主党を一から立ち上げ、野党第一党に育てた実績がある。ただし、政権交代の実現には至っておらず、野党としての限界もある。

### ④ 説明責任（5/5）
弁護士としての論理的説明力に加え、官房長官として連日の記者会見をこなした経験。東日本大震災時の「枝野寝ろ」は国民への誠実な情報発信の象徴。不都合な事実も含めて説明する姿勢がある。

### ⑤ 一貫性（4/5）
リベラル・中道左派としての立場は一貫。立憲主義・国民主権を重視する姿勢は変わらない。ただし、民主党→民進党→立憲民主党と党の変遷があり、党組織としての一貫性には課題も。

### ⑥ 倫理観（4/5）
金銭スキャンダルは確認されていない。権力に対する監視、立憲主義の重視という姿勢は倫理観の表れ。ただし、与党経験時の政権運営には批判もある。

## 強み・弱み
### 強み
- **説明力**：弁護士・官房長官としての経験に裏打ちされた論理的説明
- **危機対応経験**：東日本大震災・福島原発事故を官房長官として経験
- **選挙の強さ**：11期連続当選、地元での圧倒的支持

### 弱み
- **政権復帰の壁**：野党第一党の党首を務めたが政権交代には至らず
- **党内求心力**：代表退任後の党内での立ち位置
