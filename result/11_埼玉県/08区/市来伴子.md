# 市来伴子（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年8月2日 |
| 年齢 | 47歳 |
| 選挙区 | 埼玉8区（比例北関東で復活当選） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 元杉並区議会議員 |

## 経歴
- 宮崎県延岡市土々呂町生まれ（実家は教会、父が牧師、母が幼稚園園長）
- 関西学院大学卒業
- 関西学院大学大学院神学研究科博士前期課程修了（中世ドイツ思想・東洋哲学）
- 学生時代から貧困問題をテーマとする国際NGO、ホームレス支援活動
- 衆議院議員秘書、HP制作会社経営
- 2009年第45回衆議院選挙に社民党公認で兵庫8区から立候補（落選）
- 2011年杉並区議会議員選挙で初当選（社民党公認）、2期8年間務める
- 2019年立憲民主党に移籍
- 2019年参議院選挙に山梨県選挙区から野党統一候補として立候補（落選）
- 2021年第49回衆議院選挙に山梨2区から立候補（落選）
- 2024年第50回衆議院選挙で埼玉8区から立候補、比例復活で初当選

## 評価詳細

### ① 課題設定能力（4/5）
学生時代からの貧困問題・ホームレス支援活動を通じて、社会的弱者の課題を現場で把握してきた。大学院で神学・哲学を学び、社会問題への深い洞察がある。「本当に困っている人のための政治」という理念は明確。

### ② 解決設計力（2/5）
杉並区議としての地方行政経験はあるが、国政での政策設計の実績はこれから。NGO活動、市民運動の経験は豊富だが、大規模な政策の制度設計能力は未知数。

### ③ 実行力（2/5）
国会議員としては1期目であり、国政での実行実績はこれから。過去の選挙では複数回落選しており（兵庫8区、山梨参院、山梨2区）、粘り強さはあるが結果を出すまでに時間がかかった。

### ④ 説明責任（4/5）
大学院での研究経験、区議としての議会活動から、論理的な説明能力がある。多様性を尊重する姿勢から、異なる立場の人への説明にも配慮がある。

### ⑤ 一貫性（2/5）
社民党→立憲民主党への移籍、選挙区の変更（兵庫→山梨→埼玉）が多い。「困っている人の味方」という理念は一貫しているが、活動拠点や所属政党の変遷は説明を要する。

### ⑥ 倫理観（4/5）
金銭スキャンダルは確認されていない。弱者支援への継続的な取り組みは倫理観の表れ。牧師の家庭で育った背景も影響しているか。

## 強み・弱み
### 強み
- **現場経験**：貧困支援、ホームレス支援の実践
- **区議経験**：杉並区で8年間の地方議会活動
- **多様な経験**：NGO、企業経営、議員秘書と幅広い職歴

### 弱み
- **選挙区の定着**：兵庫→山梨→埼玉と活動拠点が変遷
- **国政経験**：1期目で実績はこれから
- **落選経験**：複数回の落選を経ての当選
