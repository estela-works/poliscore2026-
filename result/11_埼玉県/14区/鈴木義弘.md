# 鈴木義弘（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1962年11月10日 |
| 年齢 | 63歳 |
| 選挙区 | 埼玉14区（草加市・八潮市・三郷市） |
| 当選回数 | 衆議院4回 |
| 主な役職 | 衆議院決算行政監視委員長、国民民主党幹事長代理、埼玉県連会長 |

## 経歴
- 埼玉県三郷市戸ヶ崎生まれ
- 日本大学第一高等学校卒業
- 日本大学理工学部卒業
- 1985年土屋義彦参議院議員秘書
- 1999年埼玉県議会議員選挙で初当選（4期13年）
- 県議会環境防災農林委員長、文教常任委員長等を歴任
- 自民党埼玉県連青年部長
- 2012年自民党を離党、日本維新の会に移籍
- 2012年第46回衆議院選挙で初当選（比例復活）
- 希望の党解党後、無所属を経て2020年国民民主党入党
- 2021年国民民主党幹事長代理、埼玉県連会長
- 2024年第50回衆議院選挙で初の小選挙区当選（公明党代表・石井啓一を破る）

## 評価詳細

### ① 課題設定能力（4/5）
県議会議員4期13年の経験から、地方行政の課題を熟知している。環境・防災・農林・文教と幅広い分野の委員長経験があり、多角的な課題把握ができる。理工学部出身で技術的な視点も持つ。

### ② 解決設計力（4/5）
県議会での委員長経験から、政策立案・審議の経験が豊富。決算行政監視委員長として行政のチェック機能を担う。具体的な政策設計の経験がある。

### ③ 実行力（5/5）
公明党代表・石井啓一を1万票差で破るという大金星を達成。「アリさんが象を倒した」と自ら語った勝利は、実行力の証明。4期連続当選、初の小選挙区当選と目標達成能力が高い。

### ④ 説明責任（4/5）
県議・国会議員として長年の議会活動経験がある。地元三郷市を拠点とした活動を継続し、有権者との対話を重視。

### ⑤ 一貫性（2/5）
政党の移籍が多い（自民→維新→次世代→改革結集→無所属→国民）。自民党県連青年部長から維新への移籍は大きな路線変更。ただし、「地域のため」という姿勢は一貫しているとの見方も。

### ⑥ 倫理観（4/5）
金銭スキャンダルは確認されていない。公明党代表との対決でクリーンなイメージで勝利。長年の地道な活動が評価された。

## 強み・弱み
### 強み
- **地方議員経験**：県議4期13年の豊富な経験
- **選挙力**：公明党代表を破った実績
- **委員長経験**：決算行政監視委員長として行政チェック

### 弱み
- **政党遍歴**：自民→維新→国民と移籍が多い
- **イデオロギーの不明確さ**：自民から維新、そして国民民主党への変遷
- **地盤の強さ**：比例復活が続いた（2024年まで）
