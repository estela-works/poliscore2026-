# 山登志浩（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年12月30日 |
| 年齢 | 46歳 |
| 選挙区 | 富山県1区 |
| 当選回数 | 衆議院1回（比例復活） |
| 主な役職 | 衆議院内閣委員会委員、立憲民主党政務調査会長補佐 |

## 経歴
- 愛知県江南市出身
- 愛知県立丹羽高等学校卒業
- 立命館大学法学部法学科卒業
- 立命館大学大学院社会学研究科修士課程修了
- 学習塾経営
- 2007年 江南市議会議員初当選（27歳・最年少）
- 2014年 社会民主党常任幹事（広報・青年担当）
- 2014年 衆院選比例東海ブロック出馬（落選）
- 2015年、2019年 江南市議選当選（通算4期）
- 2020年 富山市に移住
- 2021年 立憲民主党に合流
- 2022年 参院選富山選挙区出馬（落選）
- 2024年 衆院選富山1区出馬（比例復活当選）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 学習塾経営の経験から教育・経済格差是正への問題意識を持つ。「子どもも親も大きな夢を描けなくなっている」という現状認識は具体的。ただし国政レベルでの独自の課題設定はこれから。

### ② 解決設計力: 2/5
**評価根拠**: 地方議員として4期の経験はあるが、国政レベルでの政策設計実績は乏しい。党の政策に沿った活動が中心で、独自の解決策設計は確認できない。

### ③ 実行力: 3/5
**評価根拠**: 江南市議4期当選、2024年衆院選で比例復活当選という実績。1000回以上の街頭演説など地道な活動を継続。ただし小選挙区では738票差で敗北。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 「人の顔と名前を覚えるのが得意」という対人能力。街頭演説を重ねてきた発信力。ただし国政での説明責任を果たす場面はまだ限られている。

### ⑤ 一貫性: 3/5
**評価根拠**: 社会民主党→立憲民主党と政党は移動したが、リベラル系という政治的立場は一貫。愛知県から富山県への移住は地縁（祖父母の出身地）を理由としている。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: スキャンダルや問題行為は確認されていない。地方議員として長く活動してきた実績から、権力行使に関する大きな問題は見られない。

## 強み・弱み

### 強み
- **地方議員としての実績**: 江南市議4期当選、地道な議員活動の経験
- **若さと行動力**: 27歳で最年少市議当選、1000回以上の街頭演説
- **対人能力**: 記憶力に自信、有権者との関係構築能力
- **教育への関心**: 学習塾経営経験から教育格差問題に詳しい

### 弱み（注意点）
- **国政経験の浅さ**: 衆院議員1期目、国政での実績はこれから
- **選挙区での敗北**: 小選挙区では738票差で敗れ比例復活
- **落下傘候補**: 愛知県出身で富山との縁は祖父母の出身地のみ
- **知名度不足**: 全国的な認知度は低い
