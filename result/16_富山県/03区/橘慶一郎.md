# 橘慶一郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年1月23日 |
| 年齢 | 65歳 |
| 選挙区 | 富山3区（高岡市・氷見市・砺波市・小矢部市・南砺市・射水市の一部） |
| 当選回数 | 衆議院6期 |
| 主な役職 | 内閣官房副長官（政務担当）、元高岡市長、元復興副大臣、元総務大臣政務官 |

## 経歴
- 1961年1月23日、富山県高岡市生まれ
- 高岡市立守山小学校（現万葉小学校）卒業
- 高岡市立志貴野中学校卒業
- 富山県立高岡高等学校卒業
- 東京大学法学部第2類（公法コース）卒業
- ケンブリッジ大学大学院修了
- 父は元衆議院議員・富山県議の橘康太郎
- 2004年、高岡市長選挙で初当選
- 2005年、市町村合併により新高岡市長選で無投票当選（2期）
- 2009年、衆議院議員選挙で富山3区から初当選
- 総務大臣政務官（第2次安倍内閣）
- 復興副大臣（第3次安倍第2次改造内閣、第4次安倍改造内閣）
- 衆議院文部科学委員長
- 2024年10月1日、内閣官房副長官（政務担当）就任
- 2024年10月27日、第50回衆院選で6選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 能登半島地震からの復興、防災庁設置、物価高対策など具体的課題を設定。復興副大臣経験を活かした災害対策への取り組み。

### ② 解決設計力: 5/5
**評価根拠**: 東京大学法学部卒、ケンブリッジ大学大学院修了という卓越した学歴。高岡市長として行政経験豊富。官房副長官として政策立案の中枢に関与。

### ③ 実行力: 5/5
**評価根拠**: 6期連続当選の安定した選挙基盤。高岡市長2期、復興副大臣、官房副長官と要職を歴任。石破内閣の官房副長官として政権運営に関与。

### ④ 説明責任: 3/5
**評価根拠**: 官房副長官としての情報発信は限定的。国会での答弁等は通常レベル。

### ⑤ 一貫性: 4/5
**評価根拠**: 2009年から衆議院議員として一貫して自民党所属。地元高岡を基盤に活動継続。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に大きなスキャンダルは報じられていない。世襲候補ではあるが、市長経験を経て国政進出。

## 強み・弱み

### 強み
- **高学歴**: 東大法学部・ケンブリッジ大学院
- **官房副長官**: 政権中枢での政策調整力
- **地方行政経験**: 高岡市長2期
- **6期連続当選**: 安定した選挙基盤
- **復興経験**: 復興副大臣として災害対応経験

### 弱み（注意点）
- **世襲**: 父が元衆議院議員
- **自民逆風**: 党全体への批判
- **知名度**: 官房副長官としての全国的認知は限定的
- **野党候補**: 国民民主党等からの挑戦
