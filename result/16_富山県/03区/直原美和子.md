# 直原美和子（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年頃 |
| 年齢 | 41歳 |
| 選挙区 | 富山3区（高岡市・氷見市・砺波市・小矢部市・南砺市・射水市の一部） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党公認候補、ビジネススクール取締役 |

## 経歴
- 1984年頃生まれ
- 岡山県出身
- 化粧品販売員として美容業界に従事
- 2018年、高岡市に移住
- ビジネススクール取締役
- 2026年、参政党から富山3区公認候補として擁立

※注: 2024年衆院選（第50回）では富山3区に参政党候補は出馬しておらず、直原美和子氏は2026年衆院選（第51回）の候補者として活動中

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「日本復活に命懸ける」という決意表明。参政党の政策方針に沿った活動が想定される。

### ② 解決設計力: 2/5
**評価根拠**: 美容業界での経験、ビジネススクール取締役として経営経験あり。ただし政策立案の直接経験はない。

### ③ 実行力: 1/5
**評価根拠**: 選挙での当選経験がなく、政治的実行力は評価できない段階。

### ④ 説明責任: 3/5
**評価根拠**: 事務所開きで「日本復活を命を懸けてやり遂げる」と決意を説明。

### ⑤ 一貫性: 2/5
**評価根拠**: 政治キャリアが始まったばかりで、一貫性の評価は困難。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。

## 強み・弱み

### 強み
- **ビジネス経験**: ビジネススクール取締役として経営経験
- **美容業界経験**: 化粧品販売員として現場経験
- **若さ**: 41歳で今後の成長に期待
- **地元定着**: 2018年から高岡市在住
- **参政党支持層**: 保守政策を求める有権者にアピール

### 弱み（注意点）
- **政治未経験**: 国政・地方議会ともに経験なし
- **他県出身**: 岡山県出身で地元基盤が弱い
- **強敵**: 官房副長官・橘慶一郎という圧倒的な現職
- **知名度不足**: 有権者への浸透がこれから
- **党勢**: 参政党の組織基盤
