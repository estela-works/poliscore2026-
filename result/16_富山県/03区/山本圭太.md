# 山本圭太（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年頃 |
| 年齢 | 41歳 |
| 選挙区 | 富山3区（高岡市・氷見市・砺波市・小矢部市・南砺市・射水市の一部） |
| 当選回数 | なし（新人） |
| 主な役職 | 国民民主党富山3区総支部長、IT企業経営者 |

## 経歴
- 1984年頃生まれ
- 富山県高岡市出身
- 早稲田大学教育学部卒業
- IT企業経営者
- 現在は東京都中央区在住
- 2026年、国民民主党から富山3区公認候補として擁立

※注: 2024年衆院選（第50回）では富山3区の国民民主党候補は大久保光太氏であり、山本圭太氏は2026年衆院選（第51回）の候補者として活動中

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「手取りを増やす」「子どもを増やす」「政治の信頼回復」を課題として設定。日本全体の閉塞感打破、頑張った人が報われる社会を目指すと主張。

### ② 解決設計力: 3/5
**評価根拠**: 早稲田大学教育学部卒。IT企業経営者としてビジネス経験あり。ただし政策立案の直接経験はない。

### ③ 実行力: 1/5
**評価根拠**: 選挙での当選経験がなく、政治的実行力は評価できない段階。

### ④ 説明責任: 3/5
**評価根拠**: 立候補表明で政策を明確に説明。SNSでの情報発信も行う。

### ⑤ 一貫性: 2/5
**評価根拠**: 政治キャリアが始まったばかりで、一貫性の評価は困難。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。経営者としての実績あり。

## 強み・弱み

### 強み
- **IT経験**: IT企業経営者としてデジタル化に精通
- **若さ**: 41歳で今後の成長に期待
- **学歴**: 早稲田大学教育学部卒
- **地元出身**: 高岡市出身で地域への理解
- **国民民主党の追い風**: 「手取りを増やす」政策への支持

### 弱み（注意点）
- **政治未経験**: 国政・地方議会ともに経験なし
- **東京在住**: 現住所が東京で地元密着度に課題
- **強敵**: 官房副長官・橘慶一郎という圧倒的な現職
- **知名度不足**: 有権者への浸透がこれから
- **組織力**: 富山3区での党組織基盤
