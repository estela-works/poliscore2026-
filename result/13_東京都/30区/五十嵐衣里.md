# 五十嵐衣里（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年1月19日（40歳） |
| 年齢 | 40歳 |
| 選挙区 | 東京30区 |
| 当選回数 | 衆議院1回 |
| 主な役職 | 衆議院議員、弁護士（第二東京弁護士会） |

## 経歴
- 愛知県名古屋市出身
- 中学校でいじめにより不登校、高校には進学せず
- フリーターとして飲食店、4トントラック運転手などを経験
- アルバイト先の労働問題をきっかけに法律に関心
- 高卒認定資格取得
- 24歳で静岡大学夜間主コース入学
- 名古屋大学法科大学院（既修者2年コース）入学
- 30歳で司法試験合格
- 2015年：参議院議員小西洋之事務所にて政策担当秘書
- 2019年：法律事務所入所、弁護士登録（第二東京弁護士会）
- 2021年7月：東京都議会議員選挙（武蔵野市選挙区）初当選
- 2024年10月4日：都議辞職
- 2024年10月27日：第50回衆院選で東京30区から初当選（98,146票）
- 2025年10月2日：第一子出産

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 自身のいじめ・不登校経験、フリーター時代の労働問題経験から、社会的弱者の課題を身をもって理解している。弁護士としての実務経験から法的・制度的課題の構造を把握する能力がある。当事者視点からの問題提起ができる強みを持つ。

### ② 解決設計力: 4/5
**評価根拠**: 弁護士資格を持ち、法的・制度的な解決策を設計する専門知識がある。参議院議員の政策秘書として政策立案の経験も積んでいる。ただし国会議員としてはまだ1期目であり、国政での実績は今後の評価。

### ③ 実行力: 4/5
**評価根拠**: 高校に行かなかったところから司法試験合格という困難なキャリアを切り開いた実行力は高く評価できる。都議選で定数1の武蔵野市選挙区で現職を破り初当選、衆院選でも自民党8期の長島昭久を破るなど、選挙での実行力も証明されている。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 弁護士として依頼者への説明経験、都議会での活動経験がある。自身の経歴（不登校、フリーター、司法試験合格）を率直に語ることで有権者との信頼関係を構築している。衆院選での高得票（98,146票）は説明力の表れ。

### ⑤ 一貫性: 4/5
**評価根拠**: 労働問題をきっかけに法律に関心を持ち、弁護士→政策秘書→都議→衆院議員と、一貫して「弱者の声を政治に届ける」という軸を維持。憲法改正反対の立場も2021年から2024年にかけて一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。弁護士として法的倫理に基づいた活動をしてきた経歴がある。自身の困難な経験から社会的弱者への視点を持っており、権力の濫用への警戒心があると推測される。

## 強み・弱み

### 強み
- **当事者経験**: 不登校、フリーター経験から社会的弱者の立場を深く理解
- **専門資格**: 弁護士資格を持ち、法的知識と論理的思考力がある
- **選挙での強さ**: 2度の選挙（都議選・衆院選）で現職・有力候補を破って当選

### 弱み（注意点）
- **国政経験の浅さ**: 衆議院議員としては1期目、今後の実績形成が課題
- **野党議員としての限界**: 政策実現には与党の協力が必要
- **育児との両立**: 2025年に第一子出産、議員活動との両立が課題となる可能性
