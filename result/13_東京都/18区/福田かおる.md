# 福田かおる（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年5月10日 |
| 年齢 | 39歳 |
| 選挙区 | 東京18区（武蔵野市・小金井市・西東京市） |
| 当選回数 | 衆議院1期 |
| 主な役職 | 文部科学大臣政務官、元農林水産省職員 |

## 経歴
- 富山県高岡市生まれ、埼玉県三郷市育ち
- 戸籍名は北口かおる
- 父は教育学者、母は小学校教員
- 渋谷教育学園幕張中学校・高等学校卒業（剣道部）
- 高校在学中に米国公立高校に約1年留学（同時多発テロに遭遇）
- 東京大学法学部卒業（全国大学生環境活動コンテスト学生代表）
- 米国コロンビア大学国際公共政策大学院修士課程修了
- 2008年、農林水産省入省
- 国土交通省観光庁、内閣官房に出向
- JETROバンコク事務所農林水産・食品部長
- 齋藤健法務大臣秘書官（2022年11月〜）
- 2022年、農水省退職
- 自民党東京18区支部長（公募）
- 2024年10月、第50回衆院選東京18区で初当選（約2200票差）
- 2025年、文部科学大臣政務官就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 農水省で地域振興・観光政策、森林間伐促進法令に携わる。JETROバンコクで農林水産・食品の海外展開を担当。国際的な課題設定能力を持つ。

### ② 解決設計力: 5/5
**評価根拠**: 東大法学部、コロンビア大国際公共政策大学院修了という最高水準の学歴。農水省でのキャリア、大臣秘書官経験から政策設計能力は高い。

### ③ 実行力: 4/5
**評価根拠**: 自民党大敗の中、約2200票差で元武蔵野市長を破り初当選。公募で支部長を勝ち取り、1年生議員で政務官に抜擢される実行力。

### ④ 説明責任: 3/5
**評価根拠**: 政務官として政策説明の立場に。ただし1期目で議会活動の実績は限定的。選挙戦での説明活動は評価。

### ⑤ 一貫性: 3/5
**評価根拠**: 官僚から政治家への転身で政治キャリアは浅い。農水・食品分野での専門性は一貫。自民党での活動は始まったばかり。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。自民党裏金問題の逆風の中での当選だが、個人の問題は報じられず。

## 強み・弱み

### 強み
- **学歴**: 東大・コロンビア大院修了
- **官僚経験**: 農水省・内閣官房・JETROでの実務
- **国際経験**: タイ、インド、バングラデシュ等での活動
- **若さ**: 39歳で今後の成長に期待
- **政務官**: 1期目で政務官抜擢

### 弱み（注意点）
- **政治経験不足**: 1期目の新人
- **地元基盤**: 富山・埼玉出身で東京18区との縁が薄い
- **僅差当選**: 約2200票差で薄氷の勝利
- **党の逆風**: 自民党への批判の中での活動
- **世襲批判**: エリート出身への批判可能性
