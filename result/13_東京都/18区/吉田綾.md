# 吉田綾（諸派）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年頃 |
| 年齢 | 40歳 |
| 選挙区 | 東京18区（武蔵野市・小金井市・西東京市） |
| 当選回数 | なし（新人） |
| 主な役職 | 再生の道所属、元JETRO職員 |

## 経歴
- 茨城県出身
- 上智大学外国語学部ロシア語学科卒業
- 大学在学中、ロシア・サンクトペテルブルク国立大学に留学
- 2009年4月、日本貿易振興機構（JETRO）入社
- 外務省在外公館派遣員制度を利用し、在ロシア日本国大使館勤務
- コーチングファーム勤務（組織風土変革、リーダーシップ開発）
- 国際会議運営ベンチャー勤務
- クライス＆カンパニー（人材紹介会社）勤務
- 2025年5月、クライス＆カンパニー退職
- 2025年、石丸伸二代表率いる「再生の道」に参加
- 2025年7月、参議院選挙東京選挙区に出馬（128,746票獲得、落選）
- 東京都西東京市在住

※注: 2024年衆院選（第50回）では出馬しておらず、次期衆院選（2026年）東京18区の候補者として活動予定

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 再生の道の政策として「教育投資」を掲げる。JETRO・外務省での国際経験から、グローバルな視点での課題認識は持つ。ただし具体的な政策課題の設定は詳細不明。

### ② 解決設計力: 3/5
**評価根拠**: 上智大学外国語学部卒、ロシア留学経験という高い学歴。JETRO・在ロシア大使館での国際経験。コーチングファームでの組織変革支援経験。ただし政策立案の直接経験はない。

### ③ 実行力: 2/5
**評価根拠**: 参院選東京選挙区で128,746票を獲得したが落選。再生の道としての都議選は全42人落選。新党での活動に限界あり。衆院選での当選経験はない。

### ④ 説明責任: 2/5
**評価根拠**: 参院選での敗北後「結果は真摯に受け止める」と発言。政策説明の機会はまだ限られている段階。

### ⑤ 一貫性: 2/5
**評価根拠**: 政治キャリアが2025年から始まったばかりで、一貫性の評価は困難。民間企業から政治への転身。再生の道での活動継続が期待される。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。外務省・JETRO等での信頼性ある経歴。新人候補として清新なイメージ。

## 強み・弱み

### 強み
- **国際経験**: ロシア留学、JETRO、在ロシア大使館勤務
- **学歴**: 上智大外国語学部、サンクトペテルブルク大留学
- **語学力**: ロシア語の専門知識
- **民間経験**: コーチング、人材紹介などの実務経験
- **参院選実績**: 128,746票獲得

### 弱み（注意点）
- **政治経験なし**: 議会経験がない
- **新党の限界**: 再生の道の組織基盤
- **参院選落選**: 東京選挙区で敗北
- **強敵**: 福田・松下という現職2名
- **知名度**: 有権者への浸透がこれから
