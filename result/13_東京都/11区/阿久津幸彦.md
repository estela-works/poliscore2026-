# 阿久津幸彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年6月26日 |
| 年齢 | 68歳 |
| 選挙区 | 東京都11区 |
| 当選回数 | 衆議院5回 |
| 主な役職 | 衆議院財務金融委員長、元内閣総理大臣補佐官 |

## 経歴
- 東京都文京区出身
- お茶の水女子大学附属小学校・中学校
- 東京都立北園高等学校卒業
- 米国ジョージ・ワシントン大学・大学院修了（国際政治学）
- 新党さきがけで菅直人の政策調査会室に勤務
- 1996年旧民主党結党に参加
- 2000年衆院選初当選
- 内閣府大臣政務官
- 内閣総理大臣補佐官
- 東日本大震災政府現地対策本部長代行
- 硫黄島遺骨帰還特命チームリーダー
- 2024年衆院選で下村博文に勝利し5期目当選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 官製談合、公共工事の品質確保、踏切道改良など公共事業に関する課題を国会で取り上げた。硫黄島における戦没者遺骨帰還問題にも取り組み、長年放置されてきた課題に光を当てた。核兵器禁止条約への参加を訴えるなど外交・安全保障分野でも課題提起。ただし、大きな政策ビジョンの提示は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 「官製談合防止法」「あっせん利得処罰法」「踏切道改良促進法改正」「公共工事の品質確保法」など複数の議員立法を提出・成立させた実績がある。硫黄島遺骨収集では米国公文書館での調査を主導し、集団埋葬地の発見につなげた。ジョージ・ワシントン大学院での国際政治学の専門性を持つ。

### ③ 実行力: 3/5
**評価根拠**: 5期当選の選挙実行力。2024年衆院選では、9期連続当選の下村博文を破る番狂わせを実現。内閣総理大臣補佐官、東日本大震災政府現地対策本部長代行として実務を担当。硫黄島遺骨収集では実際に米国に渡り成果を上げた。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 政策や活動について公式サイトやSNSで発信。問題となる発言やスキャンダルは確認されていない。ただし、国政における発信力や注目度は高くなく、メディア露出は限定的。

### ⑤ 一貫性: 4/5
**評価根拠**: 1996年の旧民主党結党から参加し、民主党→民進党→立憲民主党と野党の流れの中で一貫して活動。2005年落選後も再起を果たし、同じ選挙区で活動を継続。政党移籍はあるが、政策的軸は一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。官製談合防止やあっせん利得処罰法など、政治の透明性向上に取り組んできた。クリーンなイメージを維持している。

## 強み・弱み

### 強み
- **議員立法の実績**: 複数の法案を成立させた立法能力
- **国際経験**: 米国大学院修了、硫黄島問題での米国交渉
- **クリーンなイメージ**: スキャンダルがない

### 弱み（注意点）
- **知名度不足**: 全国的な知名度は高くない
- **野党議員としての限界**: 政策実現の機会が限られている
