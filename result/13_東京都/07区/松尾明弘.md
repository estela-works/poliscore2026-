# 松尾明弘（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年1月25日 |
| 年齢 | 50歳 |
| 選挙区 | 東京7区（港区・渋谷区） |
| 当選回数 | 衆議院2期 |
| 主な役職 | 財務金融委員会委員、弁護士 |

## 経歴
- 世田谷区生まれ、杉並区で育つ（双子の兄）
- 武蔵中・高卒業
- 東京大学法学部卒業（1998年）
- 日本電信電話（NTT）入社
- 2003年、ITベンチャー社長就任
- 2006年、成蹊大学法科大学院修了、司法試験合格
- 2007年、松尾千代田法律事務所代表弁護士
- 2011年、一橋大学院国際企業戦略修了
- 2016年、民主党・中山義活元議員の後継者に
- 2017年衆院選、東京2区落選
- 2020年衆院選、初当選（1期目）
- 2022年参院選、東京選挙区落選
- 2024年衆院選、東京7区当選（2期目）、丸川珠代を破る

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 弁護士として個人情報保護、ネット上の誹謗中傷対応に取り組む。会計年度任用職員の待遇改善、赤坂プレスセンター問題、給付付き税額控除など具体的課題を設定。

### ② 解決設計力: 4/5
**評価根拠**: 東大法学部卒、弁護士、ITベンチャー社長、一橋MBA修了と多様な経験。法案作成能力と実務経験を持ち、政策設計力は高い。

### ③ 実行力: 4/5
**評価根拠**: 2024年衆院選で丸川珠代元五輪相を破り当選。86,252票獲得で再選を果たす。会計年度任用職員の給与アップを実現。

### ④ 説明責任: 4/5
**評価根拠**: 弁護士として論理的な説明能力を持つ。裏金問題を明確に批判し、「日本の社会を前へ進める」というビジョンを示す。

### ⑤ 一貫性: 4/5
**評価根拠**: 2017年落選後も活動を継続し、2020年・2024年と当選。参院選落選後も衆院選に挑戦する粘り強さ。立憲民主党（現・中道改革連合）で一貫して活動。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。弁護士として法令遵守の姿勢。裏金問題を批判する立場から透明性を重視。

## 強み・弱み

### 強み
- **専門性**: 弁護士資格、IT・経営の経験
- **学歴**: 東大法学部、一橋MBA
- **政策実績**: 会計年度任用職員の待遇改善
- **選挙実績**: 丸川元五輪相を破る
- **若さ**: 50歳で今後の活躍に期待

### 弱み（注意点）
- **知名度**: 全国的な知名度は高くない
- **党基盤**: 中道改革連合の組織力
- **参院選落選**: 2022年参院選で落選経験
- **競争環境**: 激戦区での戦い
- **与党経験なし**: 野党議員としての限界
