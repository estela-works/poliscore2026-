# 石川友梨香（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1996年6月22日 |
| 年齢 | 28歳 |
| 選挙区 | 東京7区（港区・渋谷区） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党候補 |

## 経歴
- 群馬県美容専門学校卒業
- 美容師としての経験
- 参政党に入党
- 2024年衆院選、当初群馬1区予定から東京7区に変更して出馬
- 落選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「減税と積極財政」「国民が何に困っているか」を重視する姿勢を示したが、具体的な政策課題の設定は党の方針に沿ったもの。経済活性化を主張。

### ② 解決設計力: 2/5
**評価根拠**: 美容専門学校卒で政策立案の専門教育は受けていない。国政経験も地方議員経験もなく、政策設計能力は未知数。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で落選。当初予定の群馬1区から東京7区への選挙区変更という不利な条件での戦い。参政党としての組織力にも限界あり。

### ④ 説明責任: 3/5
**評価根拠**: 「裏金問題は選挙の争点ではない」と独自の視点を提示。減税による経済活性化について具体的に説明を試みた。

### ⑤ 一貫性: 3/5
**評価根拠**: 参政党で活動を継続。減税・積極財政という党の方針に沿った主張を展開。政治キャリアが浅いため評価は限定的。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。28歳の若手候補として清新なイメージを持つ。

## 強み・弱み

### 強み
- **若さ**: 28歳で最も若い候補の一人
- **庶民感覚**: 美容師としての経験
- **減税政策**: 経済活性化への明確な主張
- **挑戦者精神**: 激戦区への挑戦
- **女性候補**: 女性の視点

### 弱み（注意点）
- **政治経験なし**: 議員経験ゼロ
- **専門性不足**: 政策の専門教育なし
- **知名度不足**: 有権者への浸透が課題
- **選挙区変更**: 直前の選挙区変更で不利
- **党勢**: 参政党の組織基盤は発展途上
