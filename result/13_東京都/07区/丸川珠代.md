# 丸川珠代（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1971年1月19日 |
| 年齢 | 54歳 |
| 選挙区 | 東京7区（港区・渋谷区） |
| 当選回数 | 参議院3期（衆院選落選） |
| 主な役職 | 元五輪担当大臣、元環境大臣 |

## 経歴
- 兵庫県氷上郡生まれ、神戸市東灘区御影で育つ
- 東京大学卒業
- テレビ朝日入社、アナウンサーとして活躍
- 「ビートたけしのTVタックル」「朝まで生テレビ」担当
- 2007年参院選で初当選
- 環境大臣（第22代、第3次安倍第1次改造内閣）
- 原子力防災担当大臣
- 男女共同参画担当大臣（菅内閣）
- 東京五輪・パラ担当大臣（菅内閣）
- 2024年衆院選、東京7区に鞍替え出馬、落選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 環境大臣として環境政策、五輪担当として大会運営、男女共同参画担当として女性活躍推進など多岐にわたる政策課題を担当。

### ② 解決設計力: 4/5
**評価根拠**: 東大卒、複数の大臣経験から政策立案能力は高い。環境政策、五輪運営など大規模プロジェクトを担当。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で落選。参院から衆院への鞍替えに失敗。裏金問題により比例重複が認められず、公明党推薦も得られなかった。夫の大塚拓も同時落選。

### ④ 説明責任: 2/5
**評価根拠**: 裏金822万円の不記載について十分な説明ができなかった。選挙結果について「私たちの力不足」と述べたが、問題の本質への説明は不十分。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党で参院議員を3期務め、一貫して活動。ただし参院から衆院への鞍替えは政治的判断として評価が分かれる。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 2018-2022年に計822万円を政治資金収支報告書に不記載（安倍派からのキックバック）。党から戒告処分を受け、比例重複立候補も認められなかった。

## 強み・弱み

### 強み
- **閣僚経験**: 環境大臣、五輪担当大臣等
- **知名度**: 元アナウンサーとしての知名度
- **学歴**: 東京大学卒
- **参院実績**: 3期当選の実績
- **政策経験**: 環境、五輪、男女共同参画

### 弱み（注意点）
- **裏金問題**: 822万円の不記載で処分
- **衆院選落選**: 鞍替え失敗
- **信頼失墜**: 有権者の信頼を損なった
- **党内立場**: 裏金議員としての批判
- **比例重複なし**: セーフティネットなしでの戦い
