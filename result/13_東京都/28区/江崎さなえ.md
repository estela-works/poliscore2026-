# 江崎さなえ（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年12月生まれ |
| 年齢 | 39歳 |
| 選挙区 | 東京28区（練馬区東部）→東京都議会練馬区選挙区 |
| 当選回数 | 都議会議員1回 |
| 主な役職 | 東京都議会議員（練馬区）、参政党東京都議会総務会長、建設会社役員 |

## 経歴
- 1985年12月：東京都豊島区生まれ
- 2004年：東横学園高等学校卒業
- 2006年：共立女子短期大学生活科学科卒業
- 2006年4月：三菱食品株式会社入社（物流事務業務）
- 2011年3月：三菱食品株式会社退社
- 官公庁、IT企業で事務職を経験
- 建設会社役員就任
- 2022年5月：参政党入党
- 2022年12月：参政党東京第10支部（豊島区・文京区）支部長
- 2023年5月：支部長退任
- 2023年6月：東京第28区国政改革委員就任
- 2024年10月：第50回衆院選で東京28区から出馬、11,109票で落選
- 2024年12月：東京都政改革委員（練馬区）就任
- 2025年6月：東京都議会議員選挙練馬区選挙区で31,323票を獲得し当選
