# 安藤高夫（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年4月1日 |
| 年齢 | 65歳 |
| 選挙区 | 東京28区（練馬区東部） |
| 当選回数 | 衆議院2回（比例復活含む） |
| 主な役職 | 医師、医療法人永生会理事長、元厚生労働大臣政務官（第2次石破内閣） |

## 経歴
- 東京都渋谷区生まれ
- 1984年：日本大学医学部卒業
- 日本大学付属板橋病院第三内科（消化器内科）入局
- 1989年：父の死去に伴い、医療法人永生会2代目理事長就任
- 2010年：参院選比例代表から民主党公認で出馬、落選
  - 日本医師連盟、日本医療法人連盟、日本病院会政治連盟、全日本病院会政治連盟などの推薦
- 2017年：第48回衆院選で初当選（自民党、東京比例）
- 2021年：第49回衆院選で2選（東京比例）
- 2024年10月：第50回衆院選で東京28区から出馬、336票差で敗れるも比例復活で3選
- 2024年11月：厚生労働大臣政務官就任（第2次石破内閣）
- 2025年9月：高市早苗の総裁選推薦人、高市支持を表明
- 2025年10月：石破内閣総辞職に伴い政務官任を終える

## 現在の役職

- 自民党文部科学部会副部会長
- 自民党内閣第一部会副部会長
- 新型コロナウイルス対策医療系議員団事務局長
- 熱中症対策推進議員連盟事務局長代行
- 外国人観光客に対する医療PT事務局長

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 医師・病院理事長としての経験を活かし、医療・介護政策で明確な課題設定。練馬区の二次救急病院不足、医師偏在是正、医療・介護人材の働き方改革など具体的な課題を提示。コロナ禍では医療系議員団事務局長として対応。

### ② 解決設計力: 4/5
**評価根拠**: 日本大学医学部卒、医療法人永生会理事長として35年以上の医療現場経験。厚生労働大臣政務官として政策立案に関与。医療機関・介護施設への支援、補助金確保などで具体的な制度設計に携わっている。

### ③ 実行力: 3/5
**評価根拠**: 衆議院2期当選、厚生労働大臣政務官を経験。コロナ対策で医療機関への支援を国に働きかけた実績。ただし2024年衆院選では小選挙区で336票差の僅差で敗北し、比例復活での当選となった。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 医療政策について明確に発信し、専門性を活かした国会質問も行っている。地元練馬区の課題についても具体的に言及。ただし政党変遷（民主→自民）についての説明は十分ではない。

### ⑤ 一貫性: 2/5
**評価根拠**: 2010年は民主党から参院選に出馬→その後自民党に移籍という政党変遷がある。医療政策への取り組みは一貫しているが、政治的立場の変化については疑問が残る。高市早苗支持を表明するなど党内で立場が明確。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。医療法人理事長として長年医療に従事してきた経歴。ただし政党変遷があり、政治信条の一貫性には疑問符。

## 強み・弱み

### 強み
- **医療の専門家**：医師、病院理事長としての豊富な経験
- **政務官経験**：厚生労働大臣政務官として政策立案に関与
- **医療団体との連携**：日本医師連盟等から支援を受ける基盤
- **地元課題への取り組み**：練馬区の医療・防災課題に具体的に取り組む

### 弱み（注意点）
- **小選挙区での弱さ**：2024年衆院選で336票差の僅差で敗北
- **政党変遷**：2010年民主党→その後自民党への転換
- **地盤の弱さ**：新設選挙区で地盤構築が課題
- **比例依存**：比例復活での当選が続いている
