# 平沢勝栄（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1945年9月4日（79歳） |
| 選挙区 | 東京17区 |
| 当選回数 | 衆議院10期 |
| 主な役職 | 復興大臣（第11代）、自民党広報本部長、内閣府副大臣 |

## 経歴
- 岐阜県大野郡白川村生まれ
- 東京大学法学部卒業
- 1968年 警察庁入庁
- 在イギリス日本国大使館一等書記官
- デューク大学院修士課程修了
- 内閣官房長官秘書官
- 岡山県警本部長
- 警察庁長官官房審議官
- 1996年 衆院選初当選
- 2020年 復興大臣就任
- 2024年 裏金問題で自民党非公認、無所属で出馬し10選
- ※安倍晋三の家庭教師を約2年間務めた

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 警察官僚として治安・安全保障に精通。復興大臣として東日本大震災・福島原発事故からの復興を担当。拉致問題特別委員長として北朝鮮問題にも取り組んだ。

### ② 解決設計力: 4/5
**評価根拠**: 東大法学部卒、警察庁での豊富な行政経験。デューク大院修了の国際経験。大臣・副大臣として政策立案に関与。

### ③ 実行力: 4/5
**評価根拠**: 10期連続当選の実績。裏金問題で非公認となっても無所属で当選する底力。野党乱立の状況を生き抜く選挙力。

### ④ 説明責任・コミュニケーション力: 1/5
**評価根拠**: 1817万円の政治資金不記載について十分な説明なし。統一教会との関係を問うアンケートで回答拒否。説明責任を果たしていない。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党一筋だったが、裏金問題で非公認。当選後も「党所属国会議員」扱いとされず、立場が不安定。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 2018-2022年で1817万円の政治資金収支報告書不記載（裏金問題）。統一教会施設近くに選挙事務所設置、関係を問うアンケート回答拒否。

## 強み・弱み

### 強み
- **経験豊富**: 衆議院10期、復興大臣
- **警察官僚**: 治安・安全保障の専門家
- **選挙の強さ**: 非公認でも当選
- **安倍人脈**: 安倍晋三の家庭教師

### 弱み（注意点）
- **裏金問題**: 1817万円不記載で非公認
- **統一教会関係**: 関係を問うアンケート回答拒否
- **党内地位**: 「党所属国会議員」扱いされず
- **高齢**: 79歳
- **説明責任**: 不記載問題の説明不十分
