# 若宮健嗣（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年9月2日 |
| 年齢 | 64歳 |
| 選挙区 | 東京都5区（世田谷区） |
| 当選回数 | 元職（衆議院5回） |
| 主な役職 | 元国際博覧会担当大臣、元消費者担当大臣、元外務副大臣、元防衛副大臣 |

## 経歴
- 東京都出身
- 慶應義塾高等学校卒業
- 慶應義塾大学商学部卒業
- セゾングループ堤清二代表秘書（1984-1990年）
- 会社代表（1990-2005年）
- 2005年衆院選で初当選
- 2013年防衛大臣政務官（第2次安倍内閣）
- 2015年防衛副大臣兼内閣府副大臣
- 2017年自民党国防部会長
- 2018年衆議院外務委員長
- 2019年外務副大臣
- 2021年国際博覧会担当大臣、消費者担当大臣（第1次岸田内閣）
- 2022年自民党幹事長代理、経理局長
- 2024年衆院選で落選（比例復活もできず）
- 2024年防衛大臣補佐官就任（2025年退任）
- 旧茂木派所属

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 防衛・外交分野で政務官、副大臣を歴任し、安全保障の課題認識はある。万博担当大臣として国際博覧会の課題にも取り組んだ。ただし、独自の構造的課題設定というよりは与えられた役職での職務遂行が中心。

### ② 解決設計力: 3/5
**評価根拠**: 慶應義塾大学商学部卒業、セゾングループ秘書・会社経営者としての実務経験を持つ。防衛・外交分野での政策経験はあるが、万博担当大臣としては課題山積のまま退任。

### ③ 実行力: 2/5
**評価根拠**: 5期連続当選の実績はあったが、2024年衆院選で手塚仁雄に敗れ、比例復活もできず落選。選挙に弱いと指摘されていた。また、党員獲得ノルマ未達成で罰金200万円を払った過去がある。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 大臣として記者会見などに対応した経験がある。SNS（X）でも情報発信を行っている。ただし国民への発信力は限定的との評価。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党で一貫して活動。防衛・外交という軸で政務官→副大臣→大臣とキャリアを積んだ。政党や政策の大きな変更はない。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 大きなスキャンダルは確認されていない。ただし、党員獲得ノルマ未達成で罰金という組織運営の問題があった。落選後も防衛大臣補佐官として役職を得るなど、権力との距離は近い。

## 強み・弱み

### 強み
- **大臣経験**: 国際博覧会担当大臣、消費者担当大臣を歴任
- **防衛・外交の専門性**: 政務官、副大臣として長年従事
- **実務経験**: セゾングループ秘書、会社経営者としての経験
- **党内人脈**: 幹事長代理、経理局長を歴任

### 弱み（注意点）
- **選挙に弱い**: 2024年衆院選で落選、比例復活もできず
- **党員獲得問題**: ノルマ未達成で罰金200万円の過去
- **万博問題**: 担当大臣として課題が山積
- **元職**: 現在は議席を持っていない
