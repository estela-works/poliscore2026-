# 手塚仁雄（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年9月14日 |
| 年齢 | 59歳 |
| 選挙区 | 東京都5区（世田谷区） |
| 当選回数 | 衆議院6回、東京都議1期 |
| 主な役職 | 衆議院懲罰委員長、元内閣総理大臣補佐官（野田内閣）、元立憲民主党幹事長代行 |

## 経歴
- 東京都目黒区出身
- 目黒区立宮前小学校、世田谷区立八幡中学校卒業
- 早稲田大学本庄高等学院卒業
- 早稲田大学第一文学部卒業
- 中国南京大学短期留学
- 政策集団「自由社会フォーラム」にて政策研究
- 1993年日本新党目黒支部長就任
- 1993年東京都議会議員選挙で初当選（23,702票、最年少26歳）
- 民主党の結党に参画
- 1996年衆院選東京5区で次点
- 2000年衆院選東京5区で当選（比例復活）
- 2003年衆院選東京5区で落選
- 2009年衆院選で小選挙区初当選（149,623票）
- 2011年内閣総理大臣補佐官（野田内閣）
- 2017年立憲民主党に参加
- 立憲民主党幹事長代行
- 2024年衆院選で6期目当選
- 2025年懲罰委員長選任
- 中道改革連合に参加

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 首相補佐官として政策運営・国会対策を担当した経験から、国政の課題認識はある。安保法制への反対など一定の政策的立場を持つ。ただし独自の構造的課題設定というよりは党の方針に沿った活動が中心。

### ② 解決設計力: 3/5
**評価根拠**: 早稲田大学卒業、自由社会フォーラムでの政策研究経験がある。首相補佐官として政策運営に携わった実績もある。ただし、特定分野での専門性は限定的。

### ③ 実行力: 4/5
**評価根拠**: 衆議院6期当選という選挙での強さを示す。都議から国会議員へ、首相補佐官まで務めた。2024年衆院選では若宮健嗣（元大臣）を破って当選。複数回の落選後も再挑戦し当選した粘り強さがある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: ヤフオク問題で「国会の郵便局から出して何が悪いんですか」と反論するなど、批判に対する説明が不十分との指摘。西日本豪雨中のパーティー開催も批判された。

### ⑤ 一貫性: 4/5
**評価根拠**: 日本新党→民主党→立憲民主党→中道改革連合とリベラル系政党で一貫して活動。安保法制反対の立場を維持。政策的な軸は大きくぶれていない。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 首相補佐官在任中にヤフオク出品を行い、議員会館で写真撮影、政策秘書に発送作業をさせるなど公私混同が指摘された。西日本豪雨災害中のパーティー開催も批判された。

## 強み・弱み

### 強み
- **長い政治経験**: 衆議院6期、都議1期、首相補佐官経験
- **選挙での強さ**: 複数回の落選を乗り越え再当選
- **野田佳彦との関係**: 野田内閣で首相補佐官を務めた
- **地盤の強さ**: 東京5区で安定した支持

### 弱み（注意点）
- **ヤフオク問題**: 首相補佐官中の公私混同が批判された
- **豪雨中パーティー**: 災害中のパーティー開催で批判
- **説明責任の課題**: 批判への対応が不十分
- **野党の限界**: 長年の議員活動にも関わらず政策実現の成果が限定的
