# 松原仁（無所属）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年7月31日 |
| 年齢 | 69歳 |
| 選挙区 | 東京都26区 |
| 当選回数 | 衆議院9回、都議2回 |
| 主な役職 | 元国家公安委員長、元拉致問題担当大臣 |

## 経歴
- 東京都板橋区出身
- 東京都立豊島高等学校卒業
- 早稲田大学商学部卒業
- 松下政経塾第2期生
- 1989年東京都議会議員初当選（2期）
- 2000年衆議院議員初当選
- 国土交通副大臣（野田内閣）
- 国家公安委員長・拉致問題担当大臣（野田改造内閣、2012年）
- 民主党国対委員長
- 衆議院拉致問題等に関する特別委員長
- 2023年立憲民主党離党、無所属に
- 2024年衆院選で9選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 北朝鮮による拉致問題を長年の政治課題として取り組み、拉致問題担当大臣、特別委員長を歴任。消費者問題、治安問題など幅広い課題に取り組んできた。松下政経塾出身で政策立案の訓練を受けている。

### ② 解決設計力: 3/5
**評価根拠**: 国家公安委員長として治安政策を担当。拉致問題では国際的な連携を模索。ただし、拉致問題の根本的解決には至っていない。国土交通副大臣として実務経験もある。具体的な法案成立実績は確認しにくい。

### ③ 実行力: 4/5
**評価根拠**: 9期連続当選の選挙実行力。国家公安委員長として入閣し、実務を担当。立憲民主党を離党しても無所属で当選する強さを見せた。2024年選挙では新設の東京26区で勝利。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: SNS等で積極的に発信している。拉致問題についての啓発活動にも取り組んできた。ただし、離党の経緯や迂回献金疑惑への説明は十分とは言えない面もある。

### ⑤ 一貫性: 2/5
**評価根拠**: 2023年に立憲民主党を離党し無所属に。離党理由は選挙区調整の不満とされ、政策的な理由ではなかった。2025年首班指名では自身に投票するなど、党派を超えた独自行動を取る。拉致問題への取り組みは一貫している。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2025年12月、資金管理団体が政治資金規正法の上限を超える「迂回献金」を受けた疑いが報道された。松原氏側は違法性を否定しているが、政治資金の透明性に疑問が呈されている。

## 強み・弱み

### 強み
- **豊富な経験**: 9期当選、大臣経験者
- **拉致問題への取り組み**: 長年のライフワーク
- **選挙での強さ**: 無所属でも当選する力

### 弱み（注意点）
- **迂回献金疑惑**: 政治資金の透明性に疑問
- **党を離党**: 政策より選挙区の不満で離党
- **高齢**: 69歳と高齢
