# 伊藤達也（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年7月6日 |
| 年齢 | 63歳 |
| 選挙区 | 東京都22区 |
| 当選回数 | 衆議院10回（2024年比例復活） |
| 主な役職 | 元金融担当大臣、自民党幹事長代理 |

## 経歴
- 東京都調布市生まれ
- 調布一小、調布中、慶應義塾高校卒業
- 慶應義塾大学卒業
- 松下政経塾第5期生（高市早苗と同期）
- カリフォルニア州立大学院留学
- 西調布でピザ屋を経営（政治活動費を稼ぐため）
- 1993年 日本新党から衆院選初当選
- 1996年 新進党から再選
- 1998年 自由民主党入党
- 2004年 金融担当大臣（小泉内閣）
- 2023年 自民党幹事長代理
- 2024年 衆院選で小選挙区敗北、比例復活で10選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 金融政策、中小企業政策、IT政策など幅広い分野で課題を把握。e-Japan特命委員会事務局長としてブロードバンド網構築に貢献。自身のピザ屋経営経験から中小企業政策への問題意識も持つ。

### ② 解決設計力: 4/5
**評価根拠**: 金融担当大臣として金融改革を主導。歳出改革PTの「インナー」として政策原案作成を担当。中小企業・小規模事業者政策調査会長として制度設計に関与。

### ③ 実行力: 4/5
**評価根拠**: 10期当選のベテラン。金融担当大臣、幹事長代理など要職を歴任。ただし2024年衆院選で小選挙区敗北は選挙力の陰り。比例復活での当選にとどまった。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 松下政経塾出身で政策通としての評価は高い。自民大敗後「野党と丁寧に議論しながら進めていかないといけない」と発言するなど、協調姿勢は見せている。

### ⑤ 一貫性: 2/5
**評価根拠**: 日本新党→新進党→自民党と政党を渡り歩いた。派閥も複数経験。政策的には金融・経済分野を一貫して担当しているが、政党移動の多さはマイナス要因。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 大きなスキャンダルは確認されていない。裏金問題への直接の関与も確認されず。ただし自民党議員として、党全体の政治とカネ問題への対応は問われる。

## 強み・弱み

### 強み
- **政策通**: 松下政経塾出身、金融・経済政策に精通
- **実務経験**: 金融担当大臣、幹事長代理など閣僚・党要職を歴任
- **中小企業への理解**: 自身のピザ屋経営経験から現場感覚を持つ
- **10期当選の経験**: 長年の政治キャリアによる人脈と知見

### 弱み（注意点）
- **選挙区での苦戦**: 2024年小選挙区で敗北、支持基盤の弱体化
- **政党遍歴**: 日本新党→新進党→自民党と政党移動が多い
- **党の逆風**: 自民党大敗の中で、党全体の問題も背負う立場
