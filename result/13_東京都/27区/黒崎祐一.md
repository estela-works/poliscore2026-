# 黒崎祐一（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年11月19日 |
| 年齢 | 48歳 |
| 選挙区 | 東京27区（中野区・杉並区東部） |
| 当選回数 | なし（新人） |
| 主な役職 | 自民党東京27区支部長、元港区議（2期） |

## 経歴
- 明治大学付属中野中学校・高等学校卒業
- 全日本ラグビー高校代表選出
- 明治大学政治経済学部政治学科卒業（1999年）
- 全国大学ラグビー選手権大会優勝
- 1999年、日商岩井株式会社入社
- 2003年、メタルワン（日商岩井・三菱商事鉄鋼部門統合）転籍
- 2014年、退職
- 2015年、港区議会議員初当選（1,406票・25位）
- 2019年、港区議2期目当選（2,148票・6位）
- 2022年、明治大学公共政策大学院修了
- 2023年、自民党東京27区支部長就任
- 2024年衆院選、落選（59,952票）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 区議として地方政治の課題に取り組んだ経験。商社マン時代の経済経験から経済政策への理解。国政レベルの課題設定はこれから。

### ② 解決設計力: 3/5
**評価根拠**: 明治大学公共政策大学院修了で政策学を学ぶ。商社での実務経験、区議2期の経験はあるが、国政レベルの政策設計は未経験。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で59,952票を獲得したが落選。長妻昭に大差で敗北。区議から国政への転換に挑戦したが実現できず。

### ④ 説明責任: 4/5
**評価根拠**: noteで選挙活動を詳細に報告。落選後も翌朝から駅頭活動を継続。有権者への説明を丁寧に行う姿勢。

### ⑤ 一貫性: 3/5
**評価根拠**: 港区議から国政挑戦へ。「スピードと実行力」を掲げて活動継続。ただし区議3期目を返上しての国政挑戦という転換あり。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。落選後も真摯に活動を継続する姿勢は好感を持たれる。

## 強み・弱み

### 強み
- **スポーツ経験**: 明治大ラグビー部、全日本高校代表
- **ビジネス経験**: 商社勤務15年の実務経験
- **学歴**: 公共政策大学院で政策学習
- **地方議員経験**: 港区議2期
- **若さ**: 48歳で今後の成長に期待

### 弱み（注意点）
- **国政未経験**: 衆院選落選
- **強敵**: 長妻昭という知名度抜群の対立候補
- **地盤**: 港区議だが選挙区は中野・杉並
- **自民逆風**: 裏金問題での党への批判
- **組織力**: 新設区での支持基盤構築が課題
