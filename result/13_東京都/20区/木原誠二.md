# 木原誠二（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年6月8日 |
| 年齢 | 54歳 |
| 選挙区 | 東京20区（東村山市・清瀬市・東久留米市・東大和市・武蔵村山市） |
| 当選回数 | 衆議院6期 |
| 主な役職 | 自民党選挙対策委員長、元内閣官房副長官 |

## 経歴
- 東京都渋谷区生まれ（生後9か月〜5歳はシカゴ在住）
- 武蔵中学校・高等学校卒業
- 東京大学法学部卒業
- 1993年、大蔵省入省
- 1995-1997年、ロンドン・スクール・オブ・エコノミクス留学
- 1999-2001年、英国大蔵省出向
- 2005年、財務省退職、衆院選初当選
- 外務大臣政務官（第2次安倍内閣）
- 外務副大臣（第3次安倍改造内閣）
- 衆議院内閣委員長
- 2021年、内閣官房副長官（岸田内閣）
- 2023年、官房副長官辞任
- 2024年、自民党選挙対策委員長代行→委員長
- 旧岸田派所属

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 官房副長官として安全保障、経済政策など国家的課題に取り組む。EPA首席交渉官として日本・モンゴルEPA妥結。バングラデシュテロ事件で現地対策本部長。

### ② 解決設計力: 5/5
**評価根拠**: 東大法学部、大蔵省、LSE、英国大蔵省出向と最高水準の教育・実務経験。EPA交渉を妥結させた政策設計能力。官房副長官として政権中枢を担当。

### ③ 実行力: 5/5
**評価根拠**: 6期連続当選で選挙での実行力は抜群。官房副長官、選対委員長と党・政府の要職を歴任。モンゴルから北斗七星勲章授与。

### ④ 説明責任: 3/5
**評価根拠**: 官房副長官として政府方針を説明する立場にあった。ただし週刊誌報道への対応など説明が不十分との批判も。衆院選大敗の責任についても議論あり。

### ⑤ 一貫性: 4/5
**評価根拠**: 2005年から自民党一筋で活動。旧岸田派所属だが、総裁選決選で高市支持と報じられるなど、党内での立場は流動的な面も。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 週刊誌による妻の元夫に関する報道で批判を受ける。衆院選大敗の選対委員長代行としての責任。党内での権力闘争に関与との見方も。

## 強み・弱み

### 強み
- **官僚経験**: 大蔵省・英国大蔵省の実務経験
- **外交**: EPA交渉、外務副大臣経験
- **党内地位**: 選挙対策委員長
- **知性**: 東大法学部、LSE修了
- **国際経験**: 英国大蔵省出向、シカゴ在住経験

### 弱み（注意点）
- **週刊誌報道**: 家族に関する報道で批判
- **衆院選大敗**: 選対責任者としての責任
- **派閥問題**: 旧岸田派所属での立場
- **説明不足**: 疑惑への説明が不十分との批判
- **権力集中**: 官邸中枢への権力集中批判
