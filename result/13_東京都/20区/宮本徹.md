# 宮本徹（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1972年1月22日 |
| 年齢 | 52歳 |
| 選挙区 | 東京20区（東村山市・清瀬市・東久留米市・東大和市・武蔵村山市） |
| 当選回数 | 衆議院3期（元職） |
| 主な役職 | 党中央委員、元衆議院財務金融委員 |

## 経歴
- 兵庫県三木市生まれ、明石市で育つ
- 東京大学文科一類入学、教育学部卒業
- 学生自治会委員長、中央委員会議長
- 日本共産党武蔵野三鷹地区委員長
- 東京都委員会副委員長
- 党中央委員、政策委員会副責任者
- 2005年衆院選、東京18区落選（菅直人に敗北）
- 2012年衆院選、比例単独落選
- 2014年衆院選、比例東京で初当選
- 2017年・2021年衆院選、東京20区次点、比例復活当選
- 2024年衆院選、東京20区次点、比例落選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 財務金融、厚生労働、予算各委員会で活動。「現場の声で政治を動かす」姿勢で社会保障、税制など具体的な課題を設定。

### ② 解決設計力: 4/5
**評価根拠**: 東大教育学部卒で学識あり。衆議院議員3期の経験から政策立案能力は高い。2022年通常国会・臨時国会で質問時間が衆院トップ。

### ③ 実行力: 2/5
**評価根拠**: 東京20区では3回連続で次点に終わり、2024年は比例復活もできず落選。国会質問では活躍も、選挙では木原に勝てない状況が続く。

### ④ 説明責任: 5/5
**評価根拠**: 2022年は衆院で最も多く質問した議員。財務金融、厚生労働など専門分野で政府を追及。「現場の声」を国会に届ける姿勢は高く評価できる。

### ⑤ 一貫性: 5/5
**評価根拠**: 共産党で一貫して活動。学生時代から社会運動に関与。労働者・市民の立場からの主張を20年以上継続。選択的夫婦別姓推進など政策も一貫。

### ⑥ 倫理観・権力の扱い方: 5/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。企業献金を受けない共産党で清廉な政治姿勢。自身も旧姓を通称使用し、不利益を訴える。

## 強み・弱み

### 強み
- **質問力**: 衆院トップの質問時間
- **学識**: 東大教育学部卒
- **政策知識**: 財務金融、厚労分野に精通
- **一貫性**: 20年以上の政治活動
- **清廉さ**: スキャンダルなし

### 弱み（注意点）
- **選挙弱さ**: 東京20区で3連敗
- **2024年落選**: 比例復活もできず
- **党勢の限界**: 共産党の議席減
- **強敵**: 木原誠二という大物が相手
- **野党**: 政策実現の機会が限定的
