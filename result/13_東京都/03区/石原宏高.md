# 石原宏高（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年6月19日（60歳） |
| 年齢 | 60歳 |
| 選挙区 | 東京3区 |
| 当選回数 | 衆議院6回 |
| 主な役職 | 環境大臣（第35代）、内閣府特命担当大臣（原子力防災） |

## 経歴
- 神奈川県逗子市出身
- 慶應義塾幼稚舎→普通部→高等学校→大学経済学部卒業
- 1990年：日本興業銀行（現・みずほ銀行）入行
- 1997年：ニューヨーク支店勤務
- 2002年：退職、政治活動開始
- 2005年：第44回衆院選で初当選
- 2013年：外務大臣政務官（第2次安倍内閣）
- 2016年：内閣府副大臣
- 2019年：環境副大臣兼内閣府副大臣
- 2023年：内閣総理大臣補佐官（核軍縮・不拡散問題担当）
- 2025年10月：環境大臣に就任（高市早苗内閣）

※父は石原慎太郎（元東京都知事）、長兄は石原伸晃（元衆議院議員）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 環境大臣として気候変動対策、クマの獣害対策、太陽光パネル開発抑制などを掲げるが、いずれも既存の政策課題の域を出ない。本人独自の視点による問題提起は確認できず、政府・党の方針に沿った対応が中心。

### ② 解決設計力: 2/5
**評価根拠**: 議員立法の提出実績なし。政務官・副大臣・大臣として政府方針を実行する立場だが、本人主導の政策設計は見られない。ハンターの自治体雇用など具体策を示す場面もあるが、制度設計レベルには至っていない。

### ③ 実行力: 3/5
**評価根拠**: 政務官→副大臣→補佐官→大臣と政府役職を順調に歴任。ただし選挙では松原仁との激戦が続き、2021年は比例復活で辛うじて当選。政策実現の具体的成果は限定的だが、政府内でのポジション確保は評価できる。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: UE社コンサルティング料問題で朝日新聞の取材に「お答えできない」と説明を拒否した過去あり。大臣就任会見では一定の説明を行っているが、不都合な事実への説明姿勢に課題。

### ⑤ 一貫性: 3/5
**評価根拠**: 東京3区で20年以上活動を継続。環境副大臣から環境大臣へと環境政策への関与は一貫している。ただし政策面での独自の軸は見えにくい。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: UE社コンサルティング料問題で、妻が役員を務める会社が落選中に毎月100万円、計1800万円を受領。説明を拒否したことはグレーな印象を与える。名門政治家一族出身という背景も含め、利益相反への意識に疑問。

## 強み・弱み

### 強み
- **政府での経験**: 政務官、副大臣、補佐官、大臣と政府役職を幅広く経験
- **政治家一族のネットワーク**: 父・石原慎太郎、兄・石原伸晃などとの人脈

### 弱み（注意点）
- **コンサルティング料問題**: 妻の会社への1800万円支払いについて説明拒否、透明性に欠ける
- **選挙基盤の弱さ**: 松原仁との激戦が続き、2021年は比例復活。6期目でも安定した地盤とは言えない
- **独自性の欠如**: 議員立法なし、独自の政策提案が見えにくい
