# 伊藤俊輔（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年8月5日 |
| 年齢 | 45歳 |
| 選挙区 | 東京都23区 |
| 当選回数 | 衆議院3回 |
| 主な役職 | 立憲民主党代表補佐、党政調副会長、都連選対委員長 |

## 経歴
- 東京都町田市生まれ
- 父は国土庁長官を務めた元衆議院議員・伊藤公介
- 桐蔭学園小学校・中学校・高校
- 北京大学留学
- 中央大学経済学部国際経済学科卒業
- 航空貨物会社を起業、経営
- 2012年 維新政治塾入塾
- 2012年12月 衆院選出馬（日本維新の会、落選）
- 2017年10月 衆院選初当選（希望の党、比例復活）
- 2018年5月 国民民主党に参加
- 2019年1月 国民民主党離党、立憲民主党会派入り
- 2021年10月 衆院選再選
- 2024年10月 衆院選3選（立憲民主党、選挙区当選）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 国際経済学専攻、航空貨物会社経営の経験から経済・物流分野の課題を把握。国土交通委員会理事、偽情報対策チーム事務局次長など幅広い分野に関与。ただし、独自の政策提言は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 党政調副会長として政策立案に関与。防衛部門、国交部門の副部門長を務める。起業経験を活かした経済政策への視点はあるが、具体的な法案成立実績は確認できない。

### ③ 実行力: 4/5
**評価根拠**: 衆院3期当選。2024年選挙では選挙区で11万票超を獲得し、自民新人を比例復活させず勝利。2017年は比例復活、2021年も比例復活だったが、3期目で選挙区当選を果たした。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: SNS等での発信は継続的に行っている。ただし、国民民主党離党時には党から「党の結束を乱す行為」として辞任勧告を受けており、党内コミュニケーションには課題があった。

### ⑤ 一貫性: 2/5
**評価根拠**: 日本維新の会→希望の党→国民民主党→立憲民主党と複数回の政党移動。国民民主党離党時は党から批判を受けた。政策的な軸の一貫性には疑問が残る。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 大きなスキャンダルは確認されていない。ただし、父の地盤を継ぐ世襲的要素がある。国民民主党離党時の対応は党内で問題視された。

## 強み・弱み

### 強み
- **経営経験**: 航空貨物会社起業・経営
- **国際経験**: 北京大学留学、国際経済学専攻
- **選挙力**: 2024年選挙区勝利、3期連続当選
- **党内地位**: 代表補佐、政調副会長、都連選対委員長
- **若さ**: 45歳、現役世代

### 弱み（注意点）
- **政党転々**: 維新→希望→国民→立憲と4党を渡り歩く
- **離党時の問題**: 国民民主党から辞任勧告を受けた
- **世襲**: 父が元国土庁長官
- **比例復活が続いた**: 3期目でようやく選挙区当選
