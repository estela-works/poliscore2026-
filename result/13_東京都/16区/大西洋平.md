# 大西洋平（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1978年1月12日 |
| 年齢 | 46歳 |
| 選挙区 | 東京16区（江戸川区の大部分） |
| 当選回数 | 衆議院1回（江戸川区議3回） |
| 主な役職 | 外務大臣政務官、元江戸川区議会議員、大西英男の次男 |

## 経歴
- 東京都江戸川区出身
- 江戸川区立西小松川小学校卒業
- 江戸川区立松江第二中学校卒業
- 私立修徳高等学校卒業（ラグビー部、花園予選東京都大会ベスト8）
- 立正大学卒業（ラグビー部主将、U19日本代表候補）
- 明治大学専門職大学院ガバナンス研究科（公共政策）修士課程修了
- キヤノン販売会社勤務
- 東急リバブル株式会社勤務（サラリーマン生活計11年）
- 2011年：江戸川区議会議員に初当選（3期連続当選）
- 江戸川区議会議会運営委員長、自民党区議団幹事長を歴任
- 「江戸川区インターネット健全利用促進条例」を議員発議で立案（東京23区初）
- 2021年：東京都議選に出馬、243票差で次点落選
- 大西英男衆議院議員秘書
- 2024年9月：父・大西英男の引退に伴い後継候補に選出
- 2024年10月：第50回衆院選で東京16区から初当選
- 2025年10月：外務大臣政務官就任（高市内閣）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 少子化問題、社会保障、首都直下型地震対策、北朝鮮・中国・台湾有事などの課題を挙げているが、自民党の一般的な政策に沿った主張が中心。独自の構造的課題提起は確認できない。「若さと子育て世代の目線」というコンセプトは明確。

### ② 解決設計力: 3/5
**評価根拠**: 明治大学大学院で公共政策の修士号を取得し、政策立案の基盤を持つ。江戸川区議時代に「インターネット健全利用促進条例」を議員発議で立案し、東京23区初の条例制定に尽力した実績は評価できる。

### ③ 実行力: 3/5
**評価根拠**: 2024年衆院選で小選挙区初当選。江戸川区議3期の実績、外務大臣政務官への就任。選挙期間中は自転車遊説で500km以上を走破する体力と行動力。2021年都議選では243票差で惜敗するも、再挑戦で国政進出を果たした。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 公式サイトやSNSで活動報告を行っているが、国会議員としての説明責任を評価する材料が乏しい。新人議員として国会での発言実績はこれから。自転車遊説など有権者との接点を重視する姿勢は評価できる。

### ⑤ 一貫性: 3/5
**評価根拠**: 江戸川区での政治活動を一貫して継続。自民党での活動も一貫している。ただし国会議員としての実績は浅く、政策スタンスの一貫性を評価するには時期尚早。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。ただし父・大西英男からの世襲という批判がある。11年間のサラリーマン経験を経て政治の道に入った点は、一般社会の感覚を持っていると推測される。

## 強み・弱み

### 強み
- **若さと行動力**：46歳、自転車遊説500km超の体力
- **学歴・経歴**：明治大学大学院公共政策修士、11年間の民間経験
- **地元基盤**：江戸川区議3期の実績、父の地盤を継承
- **条例制定実績**：インターネット健全利用促進条例（23区初）

### 弱み（注意点）
- **世襲批判**：父・大西英男の地盤を引き継いだ「七光」との批判
- **国政経験の浅さ**：衆議院議員1期目、国会での実績はこれから
- **独自性の欠如**：政策が自民党の一般的方針に沿っており独自色が薄い
- **都議選落選**：2021年都議選で243票差の惜敗経験
