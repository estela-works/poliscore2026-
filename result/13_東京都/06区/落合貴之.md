# 落合貴之（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年8月17日 |
| 年齢 | 46歳 |
| 選挙区 | 東京都6区 |
| 当選回数 | 衆議院4回 |
| 主な役職 | 立憲民主党副幹事長、インボイス問題検討議員連盟幹事長、若者政策推進議員連盟副会長 |

## 経歴
- 東京都世田谷区生まれ
- 世田谷区立駒沢小学校、駒沢中学校卒業
- 國學院高等学校卒業
- 慶應義塾大学経済学部経済学科卒業
- 三井住友銀行入行
- 江田憲司衆議院議員、松田公太参議院議員の公設第一秘書
- 2012年衆院選みんなの党公認で出馬、落選
- 2013年特定秘密保護法賛成に反発しみんなの党離党
- 2014年結いの党、維新の党に参加
- 2014年衆院選比例復活で初当選
- 2017年希望の党への合流に反対し立憲民主党参加
- 2017年衆院選で越智隆雄を1978票差で破り小選挙区初当選
- 2021年・2024年当選で4期目

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「保守本流の政治」を提唱。インボイス問題、若者政策を重視。「民政立国論」で尾崎行雄記念財団ブックオブザイヤー大賞受賞。

### ② 解決設計力: 4/5
**評価根拠**: 慶應義塾大学経済学部卒、三井住友銀行での金融実務経験。議員秘書として政策立案を経験。著書を通じた政策提言。

### ③ 実行力: 4/5
**評価根拠**: 4期連続当選。2017年以降は小選挙区で連続当選。元厚労副大臣・越智隆雄を破る。2024年は他候補に差をつけて当選。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 著書「君も政治家になろう」「民政立国論」で政策を発信。「進次郎以外の若手政治家10人」に選出される発信力。

### ⑤ 一貫性: 4/5
**評価根拠**: 特定秘密保護法賛成に反発して離党、希望の党合流拒否など、信念に基づく行動。党は移動したが「リベラル保守」の軸は一貫。

### ⑥ 倫理観・権力の扱い方: 5/5
**評価根拠**: スキャンダルは確認されていない。統一教会との関係も確認されていない。党の方針に反する場合は離党も辞さない姿勢。

## 強み・弱み

### 強み
- **若手論客**: 著書が受賞するなど政策提言力
- **金融経験**: 三井住友銀行での経験
- **選挙力**: 4期連続当選、小選挙区3連勝
- **地元出身**: 世田谷区生まれ育ち
- **クリーン**: スキャンダルなし
- **信念**: 党の方針に反して離党した経験

### 弱み（注意点）
- **党の変遷**: みんなの党→結いの党→維新→立憲と移動
- **知名度**: 全国的な知名度は限定的
