# 畦元将吾（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年4月30日 |
| 年齢 | 67歳 |
| 選挙区 | 東京都6区 |
| 当選回数 | 元職（衆議院2期） |
| 主な役職 | 元厚生労働大臣政務官、元自民党副幹事長、株式会社AZE創業者 |

## 経歴
- 広島市出身
- 安田学園安田小学校、あきる野市立秋多中学校、山陽学園山陽高等学校卒業
- 1980年国際医学総合技術学院（現・岐阜医療科学大学）診療放射線技師科卒業
- 2024年岐阜医療科学大学大学院保健医療学研究科修士課程修了
- 1980年社会保険広島市民病院放射線科勤務
- 1985年横河メディカルシステム（現・GEヘルスケア・ジャパン）入社
- 1999年有限会社オフィス・アゼモト設立（後に株式会社AZE）、代表取締役社長
- 2019年繰り上げ当選で衆議院議員に
- 2021年衆院選比例中国ブロックで再選
- 厚生労働大臣政務官（第2次岸田改造内閣）
- 2024年衆院選比例東京ブロック23位で落選
- 2025年参院選比例区で落選
- 2026年衆院選東京6区から出馬

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 医療分野での専門性を活かした政策。憲法改正、自衛隊の位置づけを政策として掲げる。

### ② 解決設計力: 4/5
**評価根拠**: 診療放射線技師の専門資格、GEヘルスケア・ジャパンでの勤務経験。医療系企業AZEを創業し経営。大学院修士号取得。

### ③ 実行力: 3/5
**評価根拠**: 2期当選も、いずれも比例。2024年衆院選、2025年参院選と連続落選。厚労政務官を経験。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 医療政策について専門家として発信。経営者としての経験を活かした政策説明。

### ⑤ 一貫性: 3/5
**評価根拠**: 旧岸田派所属。選挙区を比例中国→比例東京→東京6区と移動。医療分野への関心は一貫。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。医療経営者から政界入りしたクリーンなイメージ。
