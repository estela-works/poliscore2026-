# 小田原潔（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年5月23日 |
| 年齢 | 61歳 |
| 選挙区 | 東京21区（立川市、日野市、国立市等） |
| 当選回数 | 衆議院4回（現在落選中） |
| 主な役職 | 元外務副大臣、元外務大臣政務官 |

## 経歴
- 大分県出身
- 東京都立富士高等学校卒業
- 東京大学経済学部卒業（1987年）
- 富士銀行入行
- 1996年メリルリンチ証券
- 2000年ゴールドマン・サックス証券
- 2004年ドイツ証券
- 2007年モルガン・スタンレー証券
- 2010年参院選（大分県選挙区）落選
- 2012年衆院選（東京21区）初当選
- 外務大臣政務官（第3次安倍第2次改造内閣）
- 外務副大臣（第1次岸田内閣）
- 2024年裏金問題で自民党非公認、無所属出馬で落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 外務副大臣として外交政策に関与。外資系金融機関での豊富なキャリアから経済・金融政策への理解は深い。ただし独自の課題設定より党の方針に沿った活動が中心であった。

### ② 解決設計力: 4/5
**評価根拠**: 東大経済学部卒、外資系金融機関（ゴールドマン・サックス、モルガン・スタンレー等）での実務経験から政策立案能力は高い。外務副大臣として外交政策に関与した実績もある。

### ③ 実行力: 2/5
**評価根拠**: 4期連続当選を果たしたが、2024年衆院選で裏金問題により自民党非公認となり落選。外務副大臣まで務めたキャリアが一挙に崩壊した。選挙での敗北は大きなマイナス評価となる。

### ④ 説明責任: 1/5
**評価根拠**: 裏金問題について5年間で1240万円の不記載が判明したが、十分な説明がなされていない。非公認で出馬しながら選挙公報に「自民党」と明記したことも批判された。「嵐からのスタート」と述べたが有権者への説明責任を果たしていない。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党所属で活動を続けてきたが、裏金問題で非公認となり無所属出馬。選挙公報に「自民党」と記載するなど党との関係が不明確。旧安倍派所属だったことも立場の不安定さを示す。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 政治資金パーティー収入の5年間で1240万円を政治資金収支報告書に不記載。半年間の党役職停止処分を受け、最終的に非公認となった。政治資金の透明性を著しく欠いた行為は倫理観に重大な疑義を生じさせる。

## 強み・弱み

### 強み
- **高い専門性**: 外資系金融機関での豊富な経験
- **外交経験**: 外務副大臣・政務官としての実績
- **学歴**: 東大経済学部卒の知識
- **地元での活動**: 東京21区で4期の実績

### 弱み（注意点）
- **裏金問題**: 1240万円不記載で非公認処分
- **選挙敗北**: 2024年衆院選で落選
- **説明責任の欠如**: 問題への説明が不十分
- **信頼失墜**: 有権者の信頼を大きく損ねた
- **党との関係**: 非公認でありながら自民党を名乗る矛盾
