# 安田伸（参政党）

## 基本情報
| 項目 | 内容 |

| 年齢 | 53歳 |
| 選挙区 | 東京10区（豊島区・文京区） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党東京第10支部長、国政改革委員 |

## 経歴
- 豊島区生まれ豊島区育ち
- 1989年：東京農業大学第一高校卒業
- 1993年：東京農業大学農学部農学科卒業
- 名古屋商科大学大学院（会計ファイナンス履修）
- 1994年〜：株式会社オンワード樫山経理部
- その後、10社ほどの企業でコーポレートファイナンス業務に従事
- 現在：ブルーベリー農園の開園準備中、自営業
- 双子の父
- 2024年10月：衆院選東京10区に参政党公認で出馬（15,500票で落選）
