# 大津綾香（みんなでつくる党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1992年11月18日（31歳） |
| 選挙区 | 東京都9区（練馬区西部） |
| 当選回数 | なし（新人） |
| 主な役職 | みんなでつくる党党首（第2代） |

## 経歴
- 神奈川県横須賀市出身
- 幼少期に子役として活動
- 2002-2005年 NHK『週刊こどもニュース』出演（池上彰の娘役）
- 中学生で芸能界引退
- アメリカ留学
- 日本大学芸術学部デザイン学科中退
- 建築デザイナーとして建築事務所勤務
- 2023年3月 政治家女子48党党首就任（立花孝志の後任）
- 2023年4月 神奈川県知事選落選（4人中3位）
- 2023年4月 目黒区議選落選（57人中48位）
- 2024年 立花孝志との法廷闘争で勝訴
- 2024年10月 衆院選東京9区から出馬（落選）

## 評価詳細

### ① 課題設定能力: 1/5
**評価根拠**: 情報不足。党首として活動しているが、具体的な政策課題の設定についての情報が乏しい。立花孝志との党内抗争に多くのエネルギーを費やしてきた印象。

### ② 解決設計力: 1/5
**評価根拠**: 情報不足。建築デザイナーとしての経験はあるが、政策設計に関与した実績は確認できない。党運営についても立花孝志との対立が続いた。

### ③ 実行力: 1/5
**評価根拠**: 神奈川県知事選、目黒区議選、衆院選と全て落選。選挙に勝ち抜く力が示せていない。党内では代表者地位をめぐる訴訟で勝訴したが、党勢拡大には至っていない。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 『週刊こどもニュース』出演経験からメディア対応力はある。立花孝志との対立をめぐりメディアに登場することも多いが、政策に関する発信は限定的。

### ⑤ 一貫性: 1/5
**評価根拠**: 芸能界→建築→政治と転身が多い。党首就任から短期間で複数の選挙に出馬し、選挙区も神奈川県知事→目黒区議→東京9区と変わっている。政治的軸が不明確。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: スキャンダルの報道はない。立花孝志との党内抗争で代表者地位を守り抜いた。ただし、党運営の混乱は有権者にとってマイナス要素。

## 強み・弱み

### 強み
- **若さ**: 31歳の若手政治家
- **メディア経験**: 子役時代のテレビ出演経験
- **粘り強さ**: 立花孝志との法廷闘争で勝訴
- **党首の地位**: みんなでつくる党のトップ

### 弱み（注意点）
- **連続落選**: 県知事選、区議選、衆院選と全敗
- **党内混乱**: 立花孝志との対立で党のイメージ悪化
- **政策の不明確さ**: 具体的な政策ビジョンが見えにくい
- **党勢の弱さ**: みんなでつくる党は政党要件を失う見込み
- **経験不足**: 政治経験が浅く、当選経験なし
