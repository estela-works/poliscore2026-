# 薬師寺みちよ（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年5月3日 |
| 年齢 | 61歳 |
| 選挙区 | 東京都9区 |
| 当選回数 | 参議院1回、衆議院0回 |
| 主な役職 | 元参議院議員、みんなの党総務委員長、デフリンピック運営委員会副委員長 |

## 経歴
- 福岡県久留米市出身、父は久留米大学学長を務めた医師
- 1989年東京女子医科大学医学部卒業、医師免許取得
- 1996年医学博士号取得
- 2005年九州大学大学院医療経営・管理学修士号取得
- 2003〜2010年内閣官房構造改革特別区域推進本部評議員
- 2013年みんなの党から参院選出馬、初当選（愛知県選挙区）
- 2019年自民党入党、参議院任期満了で退任
- 2024年衆院選で自民党比例九州ブロックから出馬、落選
- 2025年国民民主党から参院選出馬、落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 医師・医学博士としての専門知識を活かし、ゲノム医療、難聴者支援、産業保健など具体的な課題を設定。4年かけてゲノム医療推進法案をまとめるなど、専門分野では構造的な課題理解を示す。ただし、他分野での独自の問題提起は限定的。

### ② 解決設計力: 3/5
**評価根拠**: ゲノム医療推進法案を立案し、電話リレーサービスの公共インフラ整備に関与。制度設計への関与実績はあるが、法案成立までは至らず「次の国会へバトン」を渡す形となった。設計力はあるが完遂に課題。

### ③ 実行力: 2/5
**評価根拠**: 参議院議員6年間で307回、95時間16分の国会質問実績があるが、自身が主導して成立させた議員立法は限定的。衆議院議員としての実績はなく、2024年衆院選・2025年参院選ともに落選。発言は多いが実現に至った政策は少ない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 予算委員会で手話による国会質疑を初めて行い、記者会見に手話通訳を付けるなど、多様な聴衆への配慮を示す。ただし、政党移動（みんなの党→自民党→国民民主党）についての説明は十分とは言えない。

### ⑤ 一貫性: 2/5
**評価根拠**: みんなの党→無所属→自民党→国民民主党と複数の政党を渡り歩いている。医療・福祉政策への取り組み自体は一貫しているものの、政党選択の軸が不明確。「麻生太郎氏に近い」「リエゾン役」との報道もあり、立ち位置への疑問が呈されている。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 明確なスキャンダルや不祥事は確認されない。現職医師として睡眠専門クリニックで診療を続け、産業医も務めるなど、政治活動と専門職を両立。手話質疑などマイノリティへの配慮姿勢も評価できる。

## 強み・弱み

### 強み
- **医療専門家としての知見**: 医師・医学博士として医療政策に深い理解を持つ
- **障がい者支援への取り組み**: 手話質疑の実施やデフリンピック招致活動など一貫した姿勢

### 弱み（注意点）
- **政党渡り歩きによる信頼性の問題**: 10年で4つの政党を経験し、政治的立場が不明確
- **実行力の不足**: 質問回数は多いが成立法案は限定的
