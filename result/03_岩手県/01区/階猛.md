# 階猛（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年10月7日 |
| 年齢 | 58歳 |
| 選挙区 | 岩手1区 |
| 当選回数 | 衆議院7回 |
| 主な役職 | 立憲民主党ネクスト財務・金融大臣、元総務大臣政務官 |

## 経歴
- 1966年：岩手県盛岡市生まれ
- 岩手県立盛岡第一高等学校卒業（野球部）
- 1987年：東京大学文科一類入学（野球部で神宮球場で活躍）
- 1991年：東京大学法学部卒業、日本長期信用銀行入行
- 在籍中に司法試験合格
- 新生銀行で社内弁護士として金融法務全般を担当
- 2007年1月：みずほ証券に転職
- 2007年7月：衆院補選（岩手1区）で初当選
- 2009年8月：総選挙で再選
- 2009-2010年：総務大臣政務官（政策評価、行政監視、公務員制度改革担当）
- 2010年11月：事業仕分け第3弾に仕分け人として参加
- 2011年9月：民主党政策調査会副会長、決算行政監視部門座長
- 民主党役員室長、岩手県連代表、副幹事長を歴任
- 民進党政務調査会長
- 2022年9月：立憲民主党「次の内閣」ネクスト財務・金融大臣就任
- 2024年10月：第50回衆院選で7期目当選

## 評価詳細

### 1. 課題設定能力: 4/5
**評価根拠**:
銀行員・弁護士の経歴を活かし、金融・財政政策の課題を構造的に理解している。「ネクスト財務金融相」として金融所得課税の強化やエッセンシャルワーカーの待遇改善など、格差是正の観点から課題設定を行っている。NPO法人万年野党から「三ツ星議員」を6回受賞しており、国会での質問力・問題提起力は高く評価されている。

### 2. 解決設計力: 4/5
**評価根拠**:
法曹資格を持ち、金融実務と法務の両方を経験しているため、制度設計への理解が深い。総務大臣政務官として政策評価・行政監視・公務員制度改革に取り組み、事業仕分けにも参加。政調会長経験もあり、政策パッケージの設計能力は実証されている。ただし野党のため大規模な政策実現の機会は限られている。

### 3. 実行力: 4/5
**評価根拠**:
当選7回の実績があり、政務官として実際に行政運営に携わった経験を持つ。民主党政権時代に政策評価や行政監視の分野で成果を上げた。事業仕分けでは仕分け人として無駄削減に貢献。ただし野党期間が長く、近年は法案を通す機会が限られている点は考慮が必要。選挙区では7連勝と強固な地盤を築いている。

### 4. 説明責任: 4/5
**評価根拠**:
弁護士資格を持ち論理的な説明能力が高い。国会質問では「三ツ星議員」として6回表彰されており、質疑の質の高さが認められている。政治不信に対しては「政権交代で信頼を取り戻す」と訴え、政策的な対案を示す姿勢を維持。支持者以外への説明努力も見られる。

### 5. 一貫性: 2/5
**評価根拠**:
政党の変遷が多い。民主党→民進党→立憲民主党と渡り歩いている。これ自体は党の再編に伴うものだが、2020年には立憲民主党岩手県連から政治資金の移動をめぐって民事訴訟を起こされた経緯がある（2022年に県連側敗訴）。政策的な軸は金融・財政分野で一貫しているが、党内・県連との関係で摩擦が生じた点はマイナス。

### 6. 倫理観: 4/5
**評価根拠**:
重大なスキャンダルは確認されていない。政治資金をめぐる県連との訴訟は裁判で「適正」と認定された。弁護士として法令遵守の意識は高いと推察される。行政監視・政策評価を担当した経験から、権力のチェック機能への意識も高い。

## 強み・弱み

### 強み
- **専門性の高さ**：弁護士資格と金融実務経験を兼ね備え、政策の設計・評価能力が高い
- **国会活動の質**：三ツ星議員6回受賞は国会議員として突出した実績
- **安定した選挙基盤**：岩手1区で7連勝、地元での支持は盤石

### 弱み（注意点）
- **党内・県連との摩擦**：政治資金訴訟の経緯は信頼関係に課題があることを示唆
- **野党の限界**：優れた政策能力を持ちながら、実現の機会が限られている
- **存在感**：党内での発言力・メディア露出は限定的
