# 藤原崇（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1983年8月2日 |
| 年齢 | 41歳 |
| 選挙区 | 岩手3区 |
| 当選回数 | 衆議院4回（2024年落選） |
| 主な役職 | 元財務大臣政務官、元自民党青年局長、元自民党岩手県連会長 |

## 経歴
- 1983年：岩手県和賀郡西和賀町生まれ
- 岩手県立黒沢尻北高等学校卒業
- 2006年：静岡大学人文学部法学科卒業
- 2009年：明治学院大学大学院法務職研究科（法科大学院）修了
- 2010年12月：新第63期司法修習生修了、弁護士資格取得
- 参議院議員丸山和也の公設第1秘書
- 2012年12月：第46回衆院選岩手4区で初当選（小沢一郎に敗北も比例復活）
- 2014年12月：第47回衆院選で再選（比例復活）
- 2017年10月：第48回衆院選岩手3区で3選（比例復活）
- 内閣府大臣政務官兼復興大臣政務官（第4次安倍第2次改造内閣）
- 財務大臣政務官（第2次岸田内閣）
- 自由民主党青年局長（第54代）
- 自由民主党岩手県支部連合会長
- 2021年10月：第49回衆院選で小沢一郎に初勝利（選挙区当選）
- 2024年3月：青年局長辞任（過激ダンスショー問題）
- 2024年8月：自民党岩手県連会長辞任
- 2024年10月：第50回衆院選で落選（比例重複認められず）

## 評価詳細

### 1. 課題設定能力: 4/5
**評価根拠**:
弁護士出身で法務分野に精通し、法務部会長代理として政策立案に関与。衆議院予算委員会・内閣委員会・法務委員会の理事を務め、幅広い政策分野で課題認識を深めてきた。復興大臣政務官として東日本大震災からの復興課題にも取り組んだ。若手議員ながら政策通として知られ、構造的な問題理解力は高い。

### 2. 解決設計力: 4/5
**評価根拠**:
法科大学院修了・弁護士資格を持ち、法制度設計への理解が深い。財務大臣政務官として財政・税制の実務を経験し、復興大臣政務官として被災地復興の政策実行に携わった。党青年局長として若手政治家の育成にも取り組み、組織運営能力も示した。複数の政務官経験は政策設計力の証左。

### 3. 実行力: 4/5
**評価根拠**:
当選4回で財務・復興の政務官を歴任。2021年には小沢一郎という強敵に選挙区で初勝利し、岩手3区で自民党の議席を守った。党青年局長、県連会長と党の要職も務め、実行力は実証されている。ただし2024年は裏金問題と過激ダンス問題で比例重複を認められず落選。不祥事対応で実行力を発揮できなかった。

### 4. 説明責任: 2/5
**評価根拠**:
2024年3月の過激ダンスショー問題で「私の記憶では触っていない」と曖昧な説明に終始。裏金14万円のキックバック問題でも、十分な説明がなされたとは言えない。県連会長として広瀬めぐみ議員の不祥事対応に追われ、「擁立した県連にも責任がある」と辞任したが、自身の不祥事と合わせて説明責任を十分果たしたとは言い難い。

### 5. 一貫性: 4/5
**評価根拠**:
2012年の初出馬以来、一貫して自民党所属で岩手の選挙区から出馬。小沢一郎という強敵に4回挑み続け、2021年についに勝利を収めた粘り強さは評価できる。政策的にも保守本流の路線から外れていない。ただし不祥事による役職辞任が相次ぎ、政治家としての一貫した姿勢を示しきれなかった面がある。

### 6. 倫理観: 1/5
**評価根拠**:
複数の倫理的問題が発覚。(1) 2018-2022年の5年間で安倍派から14万円の裏金キックバックを受領。(2) 2024年3月、自民党青年局近畿ブロック会議後の懇親会で過激なダンスショー問題が発覚し、青年局長を引責辞任。(3) 自身が擁立に関与した広瀬めぐみ参院議員の公設秘書給与詐取問題。裏金問題により比例重複が認められず落選した。権力の座にいながら自制できなかった典型例。

## 強み・弱み

### 強み
- **法務の専門性**：弁護士資格を持つ政策通
- **政務官経験**：財務・復興の政務官を歴任し実務能力が高い
- **小沢一郎に勝った実績**：2021年の選挙区勝利は自民党岩手にとって歴史的快挙

### 弱み（注意点）
- **倫理問題の連発**：裏金、過激ダンス、広瀬議員問題と不祥事が続出
- **説明不足**：問題発覚時の説明が不十分で信頼回復に失敗
- **比例復活不可**：裏金問題で比例重複を認められず、政治生命の危機
