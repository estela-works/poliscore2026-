# 中谷元（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1957年10月14日 |
| 年齢 | 68歳 |
| 選挙区 | 高知県1区 |
| 当選回数 | 衆議院12回 |
| 主な役職 | 防衛大臣、自民党高知県連会長 |

## 経歴
- 高知市生まれ、土佐中・土佐高卒業
- 防衛大学校本科（理工学専攻）卒業
- 陸上自衛隊入隊、レンジャー教官を歴任（2等陸尉で退官）
- 1990年衆院選で初当選
- 2001年防衛庁長官（歴代最年少、自衛官出身初）
- 2014年防衛大臣兼安保法制担当大臣
- 2021年内閣総理大臣補佐官（国際人権問題担当）
- 2024年防衛大臣（石破内閣）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 安全保障分野で30年以上にわたり課題を設定し続けてきた。自衛隊出身者として現場を熟知し、日本の安全保障環境の変化に対応した政策提言を行っている。日米同盟強化、統合作戦司令部新設など、具体的な課題を構造的に把握。ただし専門分野以外での独自の問題提起は限定的。

### ② 解決設計力: 4/5
**評価根拠**: 安全保障関連法の設計に安保法制担当大臣として中心的に関与。自民党安全保障調査会長、安保法制整備推進本部長として党内での政策設計を主導。制度・法案レベルでの設計力は高い。ただし法案の憲法適合性について「憲法を法案に適応させる」と発言し批判を受けるなど、理論的整理に課題。

### ③ 実行力: 5/5
**評価根拠**: 防衛庁長官・防衛大臣を3度務め、安保法制を成立させた実績は特筆に値する。12期連続当選の政治的実行力、大臣として政策を実現する力は極めて高い。2025年には統合作戦司令部新設を主導。困難な政策を党内・国会で通す力を持つ。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 安保法制審議で答弁が不安定と批判を受け、「上から目線」の発言で謝罪に追い込まれた。2025年にはT4練習機墜落事故で記者会見やり直しという失態。専門的知識はあるが、国民への説明力・コミュニケーション力に課題がある。退任時に号泣するなど感情的な場面も。

### ⑤ 一貫性: 5/5
**評価根拠**: 1990年の初当選以来、安全保障・防衛政策を一貫して専門分野として活動。自衛隊出身者として防衛への思いは揺るがず、30年以上同じ軸で政治活動を継続。自民党総裁選で石破茂を支持するなど、政策的な一貫性も高い。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 重大なスキャンダルは確認されていない。裏金問題では党執行部を「説明責任が欠けている」と批判する姿勢を示した。ただし旧統一教会系メディアとの関係が指摘されている点はマイナス。防衛省不祥事では大臣として謝罪。

## 強み・弱み

### 強み
- **防衛の専門家**: 自衛隊出身、レンジャー資格を持つ本物の防衛専門家
- **政策実現力**: 安保法制成立など大きな政策を実現する力

### 弱み（注意点）
- **説明力の弱さ**: 国会答弁が不安定、国民への説明が苦手
- **強面イメージ**: 上から目線と批判されることがある
