# 大内久美子（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1949年8月14日 |
| 年齢 | 75歳 |
| 選挙区 | 茨城1区 |
| 当選回数 | 衆議院0回（水戸市議5期、茨城県議5期） |
| 主な役職 | 日本共産党茨城県副委員長、茨城革新懇世話人 |

## 経歴
- 福島県須賀川市出身、水戸市堀町在住
- 茨城大学養護教員養成所卒業
- 県内初の産休明け無認可保育所に勤務
- 1975年4月：25歳で水戸市議会議員に初当選（以降5期20年）
- 1995年1月：茨城県議会議員に当選（以降5期20年）
- 地方議員として計40年間活動
- 2019年7月：参議院選挙茨城県選挙区に出馬（129,151票獲得、落選）
- 2024年10月：衆議院選挙茨城3区に出馬（落選）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「ムダな公共事業をやめ、福祉や教育の充実」「原発ゼロ」「ジェンダー平等」など共産党の基本政策に沿った課題提起をしている。40年の地方議員経験から地域の実情には詳しいが、独自の構造的課題設定というより、党の方針を地域で展開する形式。課題の指摘は的確だが、優先順位や因果関係の深い分析は確認できない。

### ② 解決設計力: 2/5
**評価根拠**: 県議会で「凛とした態度、迫力ある質問で議会を圧倒」と評されるが、野党議員として批判・提言が中心。与党として政策を設計・実現した経験はない。福祉・教育の充実を訴えるが、財源や制度設計の具体的提案は党の方針に依存している。

### ③ 実行力: 2/5
**評価根拠**: 水戸市議5期・県議5期という長い議員経験はあるが、一貫して野党の立場。「他党議員も認めざるをえない提案で、諸問題を次つぎと解決」との評価があるが、大規模な政策実現の主導経験はない。地方議会での質疑・追及による行政チェック機能は果たしてきた。国政では参院選・衆院選いずれも落選しており、国会での実績はない。

### ④ 説明責任: 4/5
**評価根拠**: 40年の議員活動で培った説明力・発信力がある。県議会では「迫力ある質問」で知られ、論理的な追及ができる。保育所勤務から政治を志した原点を一貫して語り、自身の活動理由を明確に説明している。「しんぶん赤旗」等を通じた情報発信も継続的。

### ⑤ 一貫性: 5/5
**評価根拠**: 25歳で共産党から水戸市議に立候補して以来、50年間一貫して日本共産党で活動。政党移籍は一切なく、「福祉・教育の充実」「平和・人権」という主張も変わっていない。働く女性を支援する社会の必要性という原点から、ブレることなく活動を継続している点は高く評価できる。

### ⑥ 倫理観: 4/5
**評価根拠**: 40年の政治活動でスキャンダルや政治資金問題は報じられていない。「利権うずまく県政」への批判者として、クリーンな政治姿勢を維持してきた。権力からの自立を重視する共産党の方針に沿い、特定利益への便宜供与とは無縁の活動。

## 強み・弱み

### 強み
- **40年の議員経験**: 水戸市議5期・県議5期という圧倒的な地方議会経験
- **一貫した主張**: 50年間ブレない政治姿勢と共産党への忠誠
- **福祉・教育への知見**: 保育所勤務経験を原点に、福祉政策に詳しい
- **地元での知名度**: 水戸市を中心に長年の活動で一定の支持基盤

### 弱み
- **国政経験なし**: 参院選・衆院選で複数回落選、国会議員としての実績がない
- **高齢**: 75歳という年齢は新たな挑戦への体力面で課題
- **野党一筋**: 与党経験がなく、政策を実現に導いた経験に乏しい
- **共産党の支持率**: 茨城県では共産党の組織力が限定的
