# 牧野俊一（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年8月13日 |
| 年齢 | 40歳 |
| 選挙区 | 鹿児島県1区 |
| 当選回数 | 新人（国政当選なし、2025年参院選落選） |
| 主な役職 | 医師、予備自衛官、参政党鹿児島県国政改革委員 |

## 経歴
- 京都府出身
- 2013年 北海道大学医学部卒業
- 2014年 御嶽山噴火時に長野県DMATとして出動
- 2016年 災害・山岳医療に関心を持ち予備自衛官に
- 約6年前に鹿児島に移住
- 2019年～ 鹿児島大学大学院にて腫瘍溶解性ウイルス（がん治療ウイルス）を研究
- 曽於市末吉の救急クリニックで勤務
- 株式会社LIVE CRAFT代表取締役（不動産を活用した街づくり事業）
- アドベンチャーレーサーとしても活動（ニセコのレースに毎年出場）
- 2022年 参政党に入党
- 2025年7月 参院選鹿児島選挙区に出馬（約17万票獲得、落選）
- 2026年1月 衆院選鹿児島1区に参政党公認で出馬

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
「長期の経済停滞と新型コロナ禍後のインフレで国民は疲弊」「インフラ整備の先送りこそ未来世代へのツケ」という課題認識は明確。南米旅行をきっかけに「国の豊かさとは何か」を探求し、マクロ経済を独学で学んだ経緯も示している。医療現場・離島医療・災害医療の経験から、現場の課題を肌で理解している点も評価できる。

### ② 解決設計力: 3/5
**評価根拠**:
「消費税ゼロ、幅広い減税、積極財政の両立」という方向性を提示。短期的には国債、長期的にはインフラ投資で成長を促進するというロジックを示している。ただし、実現可能性の詳細な検討や、医療政策など専門領域での具体的な制度設計は公開情報から確認できない。三橋貴明氏、松田学氏、安藤裕氏らの理論に依拠しており、独自の設計は限定的。

### ③ 実行力: 2/5
**評価根拠**:
政治家としての実績はゼロ（2025年参院選落選）。ただし、医師としてDMAT出動、予備自衛官、大学院での研究、会社経営など、医療・ビジネス領域では実行力を発揮している。2025年参院選で約17万票を獲得したことは一定の組織動員力を示すが、当選には至らず。政策実現の実績は未知数。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**:
記者会見での政策説明、SNSでの情報発信に取り組んでいる。街頭演説で「支援の輪の広がりを実感した」との報告あり。敗戦後も「力不足で申し訳ない」と率直にコメント。ただし、参政党全体の反ワクチン・陰謀論的主張との整合性説明は確認できない。

### ⑤ 一貫性: 4/5
**評価根拠**:
2022年参政党入党以来、減税・積極財政路線を一貫して主張。2025年参院選落選後も「政治に関わり続けていく」と表明し、半年後の衆院選に再挑戦。医療・災害対応・研究という軸も一貫している。短期間での再挑戦は執念を示すが、選挙区の変更（参院選・全県区→衆院選・1区）はあり。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**:
個人としてのスキャンダルは確認されていない。医師としての職業倫理、予備自衛官としての公共への貢献は評価できる。ただし、参政党全体として代表・神谷宗幣氏の秘書急死問題・パワハラ疑惑、外国人への排外主義的発言、反ワクチン的主張への批判が存在する。所属政党の姿勢との関係をどう考えるかは有権者の判断材料となる。

## 強み・弱み

### 強み
- **多様な専門性**: 医師・研究者・予備自衛官・経営者と幅広い経験
- **現場経験**: DMAT出動、離島医療、救急医療の実践経験
- **明確な政策主張**: 消費税ゼロ・減税・積極財政という明快な旗印
- **継続的挑戦**: 参院選落選後も半年で再挑戦する粘り強さ
- **比較的若い世代**: 40歳で医師・研究者としての専門性

### 弱み（注意点）
- **当選実績なし**: 2025年参院選で落選、議会経験ゼロ
- **所属政党の問題**: 参政党の反ワクチン主張・排外主義的傾向への批判
- **政策の独自性**: 既存論者（三橋貴明氏等）の理論に依拠
- **地元との結びつき**: 鹿児島移住6年で、地元基盤はまだ発展途上
