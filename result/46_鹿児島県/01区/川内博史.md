# 川内博史（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年11月2日 |
| 年齢 | 64歳 |
| 選挙区 | 鹿児島県1区 |
| 当選回数 | 衆議院8回 |
| 主な役職 | 元衆議院国土交通委員長、元衆議院文部科学委員長、元衆議院科学技術・イノベーション推進特別委員長、元旧立憲民主党常任幹事会議長 |

## 経歴
- 鹿児島県鹿児島市原良町出身
- ラ・サール中学校・高等学校卒業
- 早稲田大学政治経済学部経済学科卒業（雄弁会所属）
- 大和銀行入行
- 1988年株式会社大月ホテル取締役
- 1993年第40回衆院選に日本新党から出馬、落選
- 1996年民主党から初当選（比例復活）
- 衆議院沖縄及び北方問題特別委員長
- 衆議院国土交通委員長
- 衆議院文部科学委員長
- 「ガソリン値下げ隊」隊長
- 2021年衆院選で落選
- 2024年4月繰り上げ当選
- 2024年10月衆院選で宮路拓馬を破り8期目当選
- 2026年1月「中道改革連合」に合流

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 国土交通委員長、文部科学委員長として交通・教育政策の課題に取り組んだ。「ガソリン値下げ隊」隊長として物価問題にも対応。長年の国会活動で幅広い政策課題に関与。

### ② 解決設計力: 4/5
**評価根拠**: 早稲田大学政治経済学部卒、銀行員経験。複数の委員長ポストを歴任し、政策立案に携わった実績がある。

### ③ 実行力: 4/5
**評価根拠**: 8期当選という実績。2021年落選後も2024年に繰り上げ当選を経て衆院選で勝利、返り咲きを果たした。委員長として法案審議を主導。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 雄弁会出身で弁論に長ける。SNSでの情報発信も活発。政策論議に積極的に参加。

### ⑤ 一貫性: 3/5
**評価根拠**: 日本新党→新党さきがけ→民主党→民進党→立憲民主党→中道改革連合と政党変遷が多い。ただし中道リベラル路線は一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。野党議員として権力監視の姿勢を維持。クリーンな印象を維持。

## 強み・弱み

### 強み
- **豊富な経験**: 8期当選、複数の委員長歴任
- **選挙力**: 落選後の返り咲き
- **専門性**: 交通・教育政策
- **クリーン**: スキャンダルなし
- **学歴**: ラサール、早稲田政経

### 弱み（注意点）
- **政党変遷**: 所属政党が何度も変わっている
- **高齢化**: 64歳
- **落選経験**: 2021年に落選
