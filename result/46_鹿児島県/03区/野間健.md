# 野間健（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年10月8日 |
| 年齢 | 66歳 |
| 選挙区 | 鹿児島3区（日置市、薩摩川内市、いちき串木野市等） |
| 当選回数 | 衆議院4回 |
| 主な役職 | 立憲民主党国対副委員長、元国民新党政務調査会長 |

## 経歴
- 東京都世田谷区生まれ、鹿児島県日置市出身
- 慶應義塾大学法学部法律学科卒業（弁論部幹事長）
- 総合商社兼松江商入社
- 1991年松下政経塾入塾（12期生）
- 金融担当大臣秘書官
- 国民新党政務調査会長（第7代）兼国会対策委員長
- 2012年衆院選で鹿児島3区から初当選（国民新党）
- 2014年無所属で再選、2021年・2024年立憲民主党で当選
- 2023年小沢一郎中心の議員グループ「一清会」旗揚げに参加

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 農林水産業振興、食料自給率向上（7割目標）、郵政株売却阻止、水道民営化反対など地域密着の課題を設定。松下政経塾出身らしく政策志向はあるが、国政レベルで突出した独自の課題設定は見られない。

### ② 解決設計力: 3/5
**評価根拠**: 松下政経塾12期生として政策立案の訓練を受けており、国民新党時代は政調会長として政策設計に関与。ただし野党議員として自らの政策を法案化・実現させた顕著な実績は確認できない。

### ③ 実行力: 4/5
**評価根拠**: 2012年初当選以来4期連続当選。特に2024年選挙では開票直後に当選確実が出る圧勝で、現職農水大臣の小里泰弘を比例復活も許さず破った。選挙区での強固な地盤構築は高く評価できる。国対副委員長として党務も担う。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトで政策や活動を発信しているが、詳細な国会活動報告は限定的。国会質問や質問主意書の提出実績はあるものの、特筆すべき説明責任の実践は確認できない。

### ⑤ 一貫性: 2/5
**評価根拠**: 国民新党→無所属→民進党→希望の党→国民民主党→立憲民主党と所属政党が頻繁に変遷。2023年には小沢一郎の「一清会」にも参加。政策軸は農林水産・地域振興で一定だが、政党遍歴の多さは一貫性に疑問符。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。4期の議員活動において倫理的な問題は見られず、クリーンな政治家像を維持している。

## 強み・弱み

### 強み
- **選挙の強さ**: 4期連続当選、2024年は現職農水大臣を圧倒的大差で破る
- **政策立案基盤**: 松下政経塾出身、国民新党政調会長経験
- **地域密着**: 農林水産業振興など地域課題への継続的取り組み

### 弱み（注意点）
- **政党遍歴**: 所属政党が多数変遷しており、有権者から見た安定感に欠ける
- **野党歴が長い**: 与党経験が短く、政策実現の実績は限定的
- **全国的知名度**: 地元では強いが全国的な存在感は薄い
