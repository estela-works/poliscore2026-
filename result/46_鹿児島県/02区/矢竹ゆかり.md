# 矢竹ゆかり（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 不明 |
| 年齢 | 61歳 |
| 選挙区 | 鹿児島2区 |
| 当選回数 | なし（新人） |
| 主な役職 | 生命保険代理店経営、自営業 |

## 経歴
- 大分県中津市在住
- 福岡県立理容美容高等専門学校中退
- 出身の福岡県で家業を手伝う
- 生命保険代理店を経営
- 当初は大分3区からの出馬を予定
- 2024年衆院選で鹿児島2区に選挙区変更して出馬（7,982票、4.5%で落選）

## 強み・弱み

- **民間経験**: 生命保険代理店経営など実業経験がある

### 弱み（注意点）
