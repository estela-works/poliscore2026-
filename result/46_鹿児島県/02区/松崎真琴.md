# 松崎真琴（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年4月16日 |
| 年齢 | 66歳 |
| 選挙区 | 鹿児島2区 |
| 当選回数 | 衆議院なし（新人）、鹿児島県議4期 |
| 主な役職 | 日本共産党中央委員、元党鹿児島県書記長 |

## 経歴
- 鹿児島大学卒業
- 小学校教員を経て政治の道へ
- 2003年から鹿児島県議会議員を4期務める（〜2019年）
- 日本共産党鹿児島県書記長を歴任
- 日本共産党中央委員に選出
- 2019年参院選比例代表に出馬
- 2024年衆院選鹿児島2区に出馬（12,255票、6.9%で落選）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 自民党の裏金問題、統一教会との癒着、奄美での日米共同訓練による「戦争する国づくり」など、共産党の視点からの課題認識は明確。憲法改悪反対、原発ゼロという軸も持つ。ただし党の方針に沿った課題設定が中心で独自性は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 批判・反対の表明が中心で、問題をどう解決するかの具体的設計は見えにくい。共産党の政策に沿った主張はあるが、独自の制度設計や法案提案の実績は確認できない。

### ③ 実行力: 2/5
**評価根拠**: 県議4期16年の経験があり地方政治での実績はある。しかし国政では2019年参院選、2024年衆院選と落選が続き、国政での実行力を発揮する機会がない。2024年衆院選では6.9%の得票に留まった。

### ④ 説明責任: 3/5
**評価根拠**: 自身の立場や政策を明確に発信している。2025年参院選では鹿児島選挙区出馬を断念し立憲候補支援に回る決断をしたが、「自民党政治を終わらせるための一本化」と理由を説明。野党共闘への姿勢は明確。

### ⑤ 一貫性: 4/5
**評価根拠**: 日本共産党一筋で活動を継続。憲法改悪反対、原発ゼロ、平和という政策軸は長年にわたり一貫している。教員から政治家への転身も「子どもたちの未来を守る」という軸の延長線上にあると説明できる。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑はない。企業献金を受けない共産党の方針に沿い、クリーンな政治活動を継続。県議時代も含めて倫理的な問題は見られない。

## 強み・弱み

### 強み
- **県議4期の経験**: 16年間の地方議員経験で政治活動の基盤がある
- **政策の一貫性**: 憲法・平和・原発ゼロという明確な軸を長年維持
- **党組織の支援**: 中央委員として党内での地位も確立

### 弱み（注意点）
- **国政での実績なし**: 衆院選・参院選と落選が続き国政進出できていない
- **得票力の限界**: 2024年衆院選で6.9%の得票は組織票の限界を示す
- **解決策の具体性不足**: 批判・反対は明確だが、どう解決するかの具体案が弱い
