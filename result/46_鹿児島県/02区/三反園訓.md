# 三反園訓（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年2月13日 |
| 年齢 | 67歳 |
| 選挙区 | 鹿児島2区 |
| 当選回数 | 衆議院2回 |
| 主な役職 | 財務大臣政務官（高市内閣）、元鹿児島県知事 |

## 経歴
- 鹿児島県指宿市出身
- 早稲田大学教育学部卒業（1980年）
- テレビ朝日入社、政治記者として官邸・自民党等を担当
- 「ニュースステーション」政治担当キャスター
- 2016年鹿児島県知事選で初当選（脱原発を掲げる）
- 2020年知事選で落選
- 2021年衆院選で鹿児島2区から無所属で初当選
- 2024年衆院選で再選（保守分裂を制す）
- 2025年1月自民党入党、10月財務大臣政務官就任

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: テレビ朝日の政治記者・キャスターとして長年政治を取材した経験があり、政策への理解はある。ただし知事選で掲げた脱原発という課題設定を当選後に撤回しており、本当に課題として認識していたのか疑問が残る。

### ② 解決設計力: 2/5
**評価根拠**: 知事選で「川内原発を廃炉にする方向で取り組む」と公約したが、当選後に具体的な解決策を示せず撤回。専門委員会を設置したが「原発の稼働・停止を判断しない」機関に留まり、公約実現の設計ができていなかった。

### ③ 実行力: 3/5
**評価根拠**: 知事選での現職破り、衆院選での保守分裂を制しての当選、財務政務官就任など、政治的な目標達成の実行力はある。ただし知事時代の政策面では公約を実現できず、「実行」よりも「当選」に力点が置かれている印象。

### ④ 説明責任: 2/5
**評価根拠**: 報道機関に「BPOに訴えてもいいんだぞ」と圧力とも取れる発言。首長への選挙協力依頼問題では当初否定後に認めて釈明。コロナ緊急事態宣言下の事務所開きでも「配慮が足りなかった」と陳謝するなど、後手の対応が目立つ。

### ⑤ 一貫性: 1/5
**評価根拠**: 2016年知事選で反原発派と政策合意を交わし「川内原発を廃炉にする方向で取り組む」と公約。当選後は合意を反故にし、原発再稼働を容認。毎日新聞から「看過できない変節ぶり」と批判された。無所属→自民党入党も政治的変節。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 週刊文春が選挙違反疑惑を報道。首長に地元要望事業を挙げて選挙協力を依頼した問題も発覚。報道機関への圧力的発言、緊急事態宣言下での不適切な事務所開きなど、権力者としての自覚に欠ける行動が複数見られる。

## 強み・弱み

### 強み
- **メディア経験**: テレビ朝日での長年の政治取材・キャスター経験で発信力がある
- **選挙の強さ**: 知事選、衆院選と保守分裂選挙でも勝ち抜く選挙戦術

### 弱み（注意点）
- **公約の一貫性欠如**: 脱原発公約を撤回した「変節」は信頼性を大きく損なう
- **複数の問題行動**: 選挙違反疑惑、報道機関への圧力、首長への選挙協力依頼など問題が多い
