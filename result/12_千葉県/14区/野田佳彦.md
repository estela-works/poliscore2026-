# 野田佳彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1957年5月20日 |
| 年齢 | 68歳 |
| 選挙区 | 千葉14区（船橋市・鎌ケ谷市等） |
| 当選回数 | 衆議院10回 |
| 主な役職 | 中道改革連合共同代表（初代） |

## 経歴
- 千葉県船橋市生まれ
- 1980年：早稲田大学政経学部卒業
- 1980年：松下政経塾入塾（1期生）
- 千葉県議会議員（2期）
- 1993年：衆議院議員初当選
- 1996年：落選
- 2000年：復帰
- 2009年：財務副大臣（鳩山内閣）
- 2010年：財務大臣（第14代、菅内閣）
- 2011年9月：内閣総理大臣（第95代）就任
- 2012年12月：衆院選敗北、首相退任
- 2017年：民進党幹事長
- 2024年9月：立憲民主党代表（第3代）
- 2026年1月：立憲民主党離党、中道改革連合共同代表就任

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 首相として財政再建と社会保障の一体改革という国家的課題を明確に設定。少子高齢化社会における持続可能な財政の必要性を訴え続けてきた。中道改革連合では「暮らしをど真ん中に」を掲げ政策軸を明示。

### ② 解決設計力: 4/5
**評価根拠**: 社会保障と税の一体改革で消費税増税の法整備を実現（三党合意）。年金・医療・介護の制度改革を設計。ただし民主党政権の公約「消費税増税しない」との齟齬から党分裂を招いた。

### ③ 実行力: 4/5
**評価根拠**: 不人気を承知で消費税増税法案を成立させた決断力。衆院解散を自ら決断し政権を手放した。10期連続当選の選挙の強さ。ただし民主党政権崩壊を防げなかった。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 「どじょう演説」で知られる独特の話術。財政問題の深刻さを国民に説明し続けた。ただし消費増税について党内の説得に失敗し分裂を招いた。

### ⑤ 一貫性: 3/5
**評価根拠**: 財政規律重視の姿勢は首相時代から一貫。一方で立憲民主党代表から離党し中道改革連合を結成という大きな転換を行った。政党の離合集散に関与してきた面も。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 大きなスキャンダルなく清廉なイメージを維持。解散総選挙で自ら身を引いた潔さ。ただし公約にない消費増税を実施したことへの批判はある。

## 強み・弱み

### 強み
- **首相経験**：行政の最高責任者として国政運営を経験
- **政策実現力**：消費税増税という困難な政策を実現
- **選挙の強さ**：10期連続当選、野党で唯一の元首相
- **松下政経塾出身**：政策立案能力と理念

### 弱み（注意点）
- **消費税増税のイメージ**：増税を推進した政治家としての評価
- **党分裂の責任**：民主党分裂を招いた経緯
- **政党移動**：立憲から離党し新党結成という選択
- **政権実績の評価**：民主党政権への否定的評価が残る
