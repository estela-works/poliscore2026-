# 白石ちよ（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年2月 |
| 年齢 | 49歳 |
| 選挙区 | 千葉2区（千葉市花見川区・八千代市） |
| 当選回数 | 新人（国政未経験） |
| 主な役職 | 日本共産党千葉県准委員、党千葉県中部地区副委員長 |

## 経歴
- シングルマザーとしてダブルワークをしながら子育て
- 日本共産党千葉県中部地区副委員長
- 2023年千葉県議会議員選挙に立候補（落選）
- 2024年10月第50回衆議院議員総選挙に千葉2区から立候補（48,500票、落選）
- 2025年7月第27回参議院議員通常選挙に千葉選挙区から立候補（135,751票、落選）

## 評価詳細

### 1. 課題設定能力（2/5）
「軍事費より暮らしの予算を」「物価高対策」「消費税減税」など、課題認識は党の方針に沿った一般的なもの。シングルマザーとしてのダブルワーク経験から生活困窮の実態は把握しているが、構造的な原因分析や独自の視点での課題設定は確認できない。党のスローガンの域を出ていない。

### 2. 解決設計力（2/5）
「消費税減税の財源は大企業や富裕層の応分の負担で」「賃上げ・学費軽減・医療介護の充実」といった方向性は示すが、具体的な制度設計や実現プロセスの詳細は不明。農林水産予算の増額や価格保障・所得補償など政策項目はあるが、優先順位やトレードオフへの言及がない。

### 3. 実行力（1/5）
2023年県議選、2024年衆院選、2025年参院選と連続して落選しており、国政・地方議会での実行実績はない。新人候補であり、提案や主張の段階にとどまっている。選挙活動での積極的な街頭演説は行っているが、政策を実現する立場に至っていない。

### 4. 説明責任（2/5）
SNS（X、YouTube、Instagram、TikTok等）で積極的に情報発信を行っている。参院選後も「政策への共感が広がった」と振り返り、「政策実現のために全力を尽くす」と決意を述べている。ただし、国会質疑や記者会見での説明責任を果たした実績はなく、批判的質問への対応は未知数。

### 5. 一貫性（4/5）
日本共産党に一貫して所属し、政党の渡り歩きはない。「軍事費増は絶対に許さない」「命と暮らし優先」という主張を県議選から参院選まで継続している。党の方針に沿った活動を続けており、政策的なブレは見られない。複数回の落選にも関わらず主張を変えない姿勢は評価できる。

### 6. 倫理観（4/5）
政治資金問題やスキャンダルは確認されていない。シングルマザーとしてダブルワークをしながら子育てした経験から、生活者目線での政治姿勢を持つ。難民・移民の人権尊重、気候変動対策の重要性など、人権・環境問題にも積極的に取り組む姿勢を示している。権力の私物化などの問題は見られない。

## 強み・弱み
### 強み
- **一貫した政治姿勢**：共産党一筋で政策的なブレがなく、主張が明確
- **当事者としての視点**：シングルマザー・ダブルワーク経験による生活者目線
- **積極的な発信**：SNSを活用した幅広い情報発信と選挙活動

### 弱み
- **実行実績の欠如**：3回連続落選で議員経験がなく、政策実現能力は未知数
- **政策の具体性不足**：党の方針を超えた独自の政策設計が見えない
- **得票力の限界**：衆院選48,500票、参院選135,751票と一定の支持はあるが当選ラインに届かない
