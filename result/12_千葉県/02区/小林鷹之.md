# 小林鷹之（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年11月29日 |
| 年齢 | 50歳 |
| 選挙区 | 千葉2区（千葉市花見川区・八千代市） |
| 当選回数 | 衆議院5回 |
| 主な役職 | 自民党政務調査会長（第64代）、元経済安全保障担当大臣（初代） |

## 経歴
- 千葉県市川市出身
- 開成中学校・高等学校卒業
- 東京大学法学部卒業（1999年）
- 1999年大蔵省入省
- 2003年ハーバード大学ケネディ行政大学院修士号取得
- 2007-2010年在米日本大使館勤務（一等書記官）
- 2010年財務省退職、自民党千葉2区支部長就任
- 2012年衆議院議員初当選
- 2016-2017年防衛大臣政務官
- 2021-2022年経済安全保障担当大臣（初代）
- 2024年自民党総裁選出馬（9人中5位）
- 2025年自民党総裁選出馬

## 評価詳細

### 1. 課題設定能力（4/5）
経済安全保障という新しい政策領域を日本で制度化した功績は大きい。2020年に自民党新国際秩序創造戦略本部の事務局長として、経済と安全保障の融合が必要な時代背景を構造的に説明し、技術流出防止やサプライチェーンリスクなど本質的なボトルネックを言語化できている。

### 2. 解決設計力（4/5）
経済安全保障推進法の策定において、特許非公開制度やセキュリティクリアランス制度など、具体的な制度設計を主導。リスク点検の仕組みを導入するなど、運用面まで含めた設計ができている。ただしトレードオフの明示がやや不足。

### 3. 実行力（4/5）
初代経済安全保障担当大臣として、経済安全保障推進法を成立させた実績は評価できる。財務省出身の政策通として、党内調整や法案作成のプロセスを着実に進める能力を持つ。2度の総裁選出馬で知名度向上にも成功。

### 4. 説明責任（2/5）
裏金問題について「検察のような権限を持たない自民党が調査をするのも限界がある」と実態解明に消極姿勢。旧統一教会関連イベント参加の説明も「記憶にない」「軽率だった」と曖昧。政策面では説明力があるが、不都合な事実への対応は定型的で、反対意見や批判的質問への回答が弱い。

### 5. 一貫性（4/5）
財務省退職後から一貫して自民党所属。経済安全保障を軸に継続的に活動し、政策的なブレは少ない。2024年総裁選落選後も党広報本部長を固辞し、独自路線を維持。消費税については「聖域なく検討」と柔軟姿勢を見せるが、説明可能な範囲。

### 6. 倫理観（2/5）
旧統一教会関連団体「ピースロード2021」への参加が確認されており、「教義に共感する発言をした」との証言もある（本人は否定）。裏金問題への対応も消極的で、党内の透明性確保に踏み込まない姿勢は権力側への配慮が見える。明確な違法行為はないが、透明性に欠ける。

## 強み・弱み
### 強み
- **経済安全保障の専門性**：初代担当大臣として制度構築を主導した実績と専門知識
- **政策立案能力**：財務省・ハーバード出身の知見を活かした構造的な政策設計力
- **安定した基盤**：千葉2区で5期連続当選の地盤と若手・中堅からの支持

### 弱み
- **説明責任の弱さ**：政治とカネ、旧統一教会問題への対応が曖昧で不十分
- **総裁選での伸び悩み**：2度の出馬も党員票・世論調査で上位に届かず、発信力に課題
- **透明性への消極姿勢**：裏金問題の実態解明を避け、党内改革への踏み込みが不足
