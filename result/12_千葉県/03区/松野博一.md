# 松野博一（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1962年9月13日 |
| 年齢 | 63歳 |
| 選挙区 | 千葉3区 |
| 当選回数 | 衆議院9回 |
| 主な役職 | 元内閣官房長官、元文部科学大臣 |

## 経歴
- 千葉県木更津市生まれ
- 早稲田大学法学部卒業
- ライオン株式会社勤務を経て松下政経塾入塾
- 1996年、自民党千葉県連の候補者公募に日本初の公募制度で合格
- 2000年、第42回衆議院議員総選挙で初当選
- 厚生労働大臣政務官、自民党千葉県連会長、自民党総務会長代行などを歴任
- 2016年、第3次安倍第2次改造内閣で文部科学大臣（初入閣）
- 2021年、第1次岸田内閣で内閣官房長官に就任
- 2023年12月、裏金問題で官房長官を辞任
- 2024年10月、第50回衆院選で比例非登載ながら小選挙区で9選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 文部科学大臣として学習指導要領改定や給付型奨学金創設など、教育分野の本質的課題を適切に設定。「52年ぶりの教職員待遇改善」提言など、長年放置されてきた問題に取り組む姿勢を評価。雇用政策でもコロナ禍での雇用調整助成金特例措置を提言するなど、課題把握能力は高い。

### ② 解決設計力: 4/5
**評価根拠**: 日本初の返還不要の給付型奨学金制度、高校新科目「公共」の創設、産学官連携の「オープンイノベーション機構」創設など、具体的かつ実現可能な制度設計を行った実績あり。政策立案能力は閣僚経験者として十分に示されている。

### ③ 実行力: 4/5
**評価根拠**: 文科大臣時代に学習指導要領改定を実現、給付型奨学金を創設するなど、構想を形にする力を発揮。官房長官として内閣の要を務め、ワクチン接種推進も担当。9期連続当選という選挙での強さも実行力の証左。

### ④ 説明責任: 1/5
**評価根拠**: 裏金問題発覚後、報道各社からの会見要請を拒否し続けた。国会での追及に対しても「政府の立場としてお答えを差し控える」と繰り返し回答を避けた。政倫審では「経理を知らなかった」と主張し、「全て人のせい」との批判を受けた。官房長官という政府スポークスマンの立場にありながら説明を尽くさなかった姿勢は深刻な問題。

### ⑤ 一貫性: 2/5
**評価根拠**: 安倍派事務総長として派閥運営に関与しながら、裏金問題では「知らなかった」と責任を否定。発言と行動の乖離が顕著。また、官房長官辞任後も議員辞職せず、「政治責任はある」としつつ具体的な責任の取り方を示していない点も一貫性に欠ける。

### ⑥ 倫理観: 1/5
**評価根拠**: 5年間で1051万円の裏金キックバックを受領していた疑惑は、政治資金規正法の趣旨に反する重大な問題。さらに官房長官辞任直前の2週間で官房機密費4660万円を自身に支出していた問題も発覚。使途不明の公金支出は権力の私物化を疑わせる行為であり、倫理観の欠如は明白。

## 強み・弱み

### 強み
- **豊富な政策実績**: 文科大臣・官房長官として教育・雇用分野で具体的成果を残した
- **政策立案能力**: 松下政経塾出身の政策通として、制度設計力に定評がある
- **選挙での強さ**: 裏金問題の逆風下でも小選挙区で勝利する地元での支持基盤

### 弱み
- **深刻な説明責任の欠如**: 裏金問題で国民への説明を避け続けた姿勢は致命的
- **倫理観の問題**: 裏金受領と官房機密費問題は政治家としての信頼を大きく損なった
- **責任転嫁体質**: 派閥幹部でありながら「知らなかった」と責任を回避する姿勢
