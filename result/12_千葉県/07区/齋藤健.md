# 齋藤健（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年6月14日 |
| 年齢 | 65歳 |
| 選挙区 | 千葉7区（野田市・流山市） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 経済産業大臣（第30代）、法務大臣（第107代）、農林水産大臣（第60・61代）、自民党千葉県連会長 |

## 経歴
- 東京都新宿区生まれ
- 東京大学経済学部卒業、ハーバード大学修士号取得
- 通商産業省（現・経済産業省）入省
- 経済産業省電力基盤整備課長
- 埼玉県副知事（経済産業省より出向）
- 2009年：第45回衆議院議員総選挙で比例南関東ブロックから初当選
- 2017年：農林水産大臣就任（60年ぶりの農協改革、70年ぶりの水産・林業改革を断行）
- 2022年：法務大臣就任
- 2023年12月：経済産業大臣就任（裏金問題で更迭された西村康稔の後任）
- 2024年9月：自民党総裁選出馬を断念（推薦人確保できず）
- 2024年10月：第50回衆議院議員総選挙で6選
- 2024年11月：自民党千葉県連会長就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 農水大臣として「人口減少下で農家所得を確保するには輸出拡大と6次産業化が必要」と明確に課題を設定。水田の保水・防災機能に着目し飼料米へのシフトを提唱するなど、本質的な課題把握能力を示した。経産大臣としてもGX推進、産業競争力強化、福島復興など複合的課題を適切に整理している。

### ② 解決設計力: 4/5
**評価根拠**: 60年ぶりの農協改革、50年ぶりの酪農改革、70年ぶりの水産・林業改革を設計・断行した実績は高く評価できる。TPP・日EU経済連携協定では国内農業を守りながら協定締結に導く制度設計を行った。通商官僚時代の日米自動車交渉でも23ヶ月に及ぶ交渉を決裂させなかった。

### ③ 実行力: 4/5
**評価根拠**: 農林水産大臣として歴史的な改革を複数実現。経産大臣としてALPS処理水問題やエネルギー政策に取り組んだ。ただし、2024年総裁選では推薦人20人を確保できず出馬断念となり、党内での求心力に限界も見られた。衆院選では6期連続当選を果たすなど選挙での実行力は高い。

### ④ 説明責任: 4/5
**評価根拠**: 国会答弁では丁寧な説明を心がけ、「国民から信頼される行政の実現」を重視する姿勢を示している。2024年政治改革特別委員会では自民党の裏金問題について「心からおわび申し上げる」と謝罪。前任大臣の問題については「本人が説明すべき」と適切な距離を取りつつ、再発防止への決意を表明した。

### ⑤ 一貫性: 4/5
**評価根拠**: 2018年総裁選で現職閣僚ながら石破茂支持を表明し、圧力に屈せず「クビを切ってくれ」と反論した姿勢は一貫性の表れ。無派閥を貫き、政策本位の姿勢を維持。農業・経済政策についても官僚時代からの専門性を活かした一貫した主張を続けている。

### ⑥ 倫理観: 4/5
**評価根拠**: 無派閥であったことから自民党裏金問題の当事者とはならず、むしろ問題発覚後に経産大臣に抜擢された。政治資金やスキャンダルの報道はない。法務大臣在任中は死刑執行命令を出さなかった点は、価値判断が分かれるが慎重な姿勢の表れとも言える。

## 強み・弱み

### 強み
- **政策実現力**: 農水・経産・法務の3閣僚経験で歴史的改革を複数断行した実績
- **専門性**: 通商官僚出身で経済・通商政策に精通、国際交渉の経験も豊富
- **独立性**: 無派閥を貫き、政策本位で行動する姿勢

### 弱み
- **党内基盤の弱さ**: 総裁選で推薦人を集められず、派閥の支援がない
- **知名度の低さ**: 一般国民への知名度は高くなく、総裁候補としての求心力に課題
