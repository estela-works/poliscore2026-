# 矢崎堅太郎（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1967年9月29日 |
| 年齢 | 57歳 |
| 選挙区 | 千葉5区（市川市・浦安市） |
| 当選回数 | 衆議院1回（千葉県議4期） |
| 主な役職 | 元千葉県議会立憲民主・千葉民主の会会長 |

## 経歴
- 東京都杉並区生まれ、10歳で千葉県浦安市に転居
- 駒澤大学法学部政治学科卒業
- 富士銀行（現みずほ銀行）入行、中小企業向け融資等を担当
- 2007年千葉県議会議員選挙で初当選、4期15年務める
- 2021年・2023年衆院選で次点、2024年10月に初当選
- 2025年より中道改革連合所属

## 評価詳細

**① 課題設定能力（4/5）**
県議4期の経験を通じて、災害対応、中小企業支援、生活者目線の政策を一貫して掲げている。東日本大震災では復旧・復興対策特別委員会で活動し、2019年台風被害でも現場主義で課題を拾い上げた。「ベーシックサービス」の充実や給付型奨学金の拡充など、具体的な課題設定ができている。

**② 解決設計力（4/5）**
銀行員時代の中小企業支援経験を活かし、資金繰りや経済政策に明るい。県議時代には知事への提言活動を積極的に行い、コロナ禍では会派代表として県民の声を集約して政策提言した実績がある。制度設計への理解度は高い。

**③ 実行力（4/5）**
県議4期15年の実績は評価に値する。東日本大震災後の復旧・復興、台風被害対応など、現場に足を運び続ける姿勢が特徴。3度目の挑戦で国政初当選を果たした粘り強さもある。ただし国会議員としてはまだ実績を積む段階。

**④ 説明責任（2/5）**
2024年11月の不倫報道に対し、事実を認め「家族に大変申し訳ない」と謝罪はしたものの、事前に党に自己申告して口頭注意で済ませるなど、対応は最小限。有権者への直接的な説明は不十分であり、「友達としてやっていきたい」と相手女性に語ったことも不誠実さを感じさせる。

**⑤ 一貫性（4/5）**
民主党系政党に所属し続け、災害対応・中小企業支援・生活者重視という政策軸はブレていない。15年間の県議活動と3度の国政挑戦を通じて、政治信条の一貫性は保たれている。2025年の中道改革連合への参加も、政策方向性としては整合的。

**⑥ 倫理観（1/5）**
当選直後に発覚した不倫スキャンダルは深刻。選挙直前に示談書を交わして40万円を支払い、選挙後に「友達として」と関係継続を求めるなど、倫理的判断力に大きな問題がある。「自民党のウミを出し切る」と訴えて当選した議員が、自らの不祥事を起こしたことは有権者への裏切り。

## 強み・弱み
### 強み
- **豊富な地方議会経験**：県議4期15年で培った政策立案能力と地域課題への深い理解
- **粘り強さ**：3度の挑戦で国政入りを果たした執念と継続力
- **災害対応への実績**：東日本大震災、台風被害での現場主義的な活動実績

### 弱み
- **倫理観の欠如**：当選直後の不倫スキャンダルで政治家としての信頼性が大きく毀損
- **説明責任の不足**：不祥事への対応が党内処理に留まり、有権者への説明が不十分
- **国政経験の不足**：当選間もなく、国会での実績はこれから
