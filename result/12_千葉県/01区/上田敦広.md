# 上田敦広（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1983年4月29日 |
| 年齢 | 41歳 |
| 選挙区 | 千葉1区 |
| 当選回数 | 衆議院0回（新人） |
| 主な役職 | 参政党員、介護福祉士、ケアマネジャー |

## 経歴
- 植草学園短期大学卒業
- 介護福祉士資格取得
- 様々な介護施設での勤務を経験
- 現在はケアマネジャー（介護支援専門員）として従事
- 2022年参議院選挙を機に参政党に入党
- 2023年4月、統一地方選挙で千葉市稲毛区から出馬し2814票獲得（落選）
- 2024年10月、第50回衆議院選挙に千葉1区から参政党公認で出馬（落選）

## 評価詳細

### ① 課題設定能力（2/5）
介護現場での実務経験から、高齢者介護や医療・福祉分野の課題認識は一定程度あると推察される。しかし、個人としての政策主張や課題設定についての公開情報が極めて限られている。参政党の政策（子供の教育、食と健康・環境保全、国のまもり）に沿った活動をしているが、独自の課題設定能力を評価する材料が不足している。

### ② 解決設計力（2/5）
新人候補として、具体的な政策提案や法案作成の実績はない。介護現場での経験を政策に反映させる可能性はあるものの、実際にどのような解決策を設計できるかは未知数。参政党の政策に依拠した活動が中心であり、独自の解決設計力を示す機会がなかった。

### ③ 実行力（1/5）
政治家としての実績は皆無。2023年の千葉市議選では2814票を獲得したものの落選、2024年の衆議院選挙でも落選している。介護現場での実務経験はあるが、政治の場での実行力を評価する材料がない。新人候補として低評価とならざるを得ない。

### ④ 説明責任（2/5）
候補者としての情報発信は限定的。公式サイトや詳細な政策説明は見当たらず、有権者への説明責任を果たすための基盤が脆弱。参政党全体としての情報発信に依存している状況であり、個人としての説明責任を評価することが困難。

### ⑤ 一貫性（4/5）
2022年の参政党入党以来、同党の理念に沿った活動を継続している。介護の現場から政治の世界への挑戦という姿勢は一貫しており、地方選挙から国政選挙へとステップアップを目指す姿勢も明確。短期間ではあるが、参政党員としての活動に一貫性がある。

### ⑥ 倫理観（2/5）
個人としてのスキャンダルや不祥事は報じられていない。しかし、所属する参政党では神谷宗幣代表の元公設秘書自殺問題、パワハラ・セクハラ疑惑、議員事務所への監視カメラ設置問題、複数議員の不倫スキャンダルなど、様々な問題が報じられている。党のガバナンスに問題があり、党所属候補としてその影響を受けざるを得ない。また、参政党の「終末期延命治療の医療費全額自己負担化」という主張は、介護士としての立場から見て倫理的に疑問が残る政策である。

## 強み・弱み

### 強み
- **介護現場の実体験**：ケアマネジャーとして第一線で働いており、高齢者福祉の現場を熟知している
- **若さと熱意**：41歳という比較的若い年齢で国政に挑戦する意欲がある
- **継続性**：地方選挙から国政選挙へと継続的に挑戦しており、諦めずに活動を続けている

### 弱み
- **実績不足**：政治家としての実績がなく、能力を評価する客観的材料が乏しい
- **情報発信不足**：個人としての政策主張や活動内容の発信が極めて限定的
- **党の問題**：参政党で相次ぐスキャンダルが党全体のイメージを損なっており、候補者個人にも影響
- **知名度の低さ**：地域での知名度が低く、田嶋要・門山宏哲という知名度のある候補との戦いで不利
