# 茂木敏充（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1955年10月7日 |
| 年齢 | 69歳 |
| 選挙区 | 栃木県5区（足利市、栃木市、佐野市） |
| 当選回数 | 衆議院11回 |
| 主な役職 | 自民党幹事長（第55代）、外務大臣、経済再生担当大臣、経済産業大臣、自民党政調会長、自民党選対委員長 |

## 経歴
- 栃木県足利市出身
- 東京大学経済学部卒業
- ハーバード大学大学院修了（公共政策修士）
- 丸紅、マッキンゼー・アンド・カンパニー勤務
- 読売新聞社記者
- 1993年、衆院選旧栃木2区で初当選
- 以降、栃木5区で9回連続当選（計11期）
- 2003年、小泉内閣で科学技術・IT担当大臣として初入閣
- 2008年、金融担当大臣・行政改革担当大臣
- 2012年、経済産業大臣
- 2017年、経済再生担当大臣
- 2019年、外務大臣
- 2021年、自民党幹事長就任、茂木派（平成研究会）領袖
- 2024年4月、茂木派解散を決定
- 2024年9月、自民党総裁選出馬（9人中6位で敗北）
- 2024年10月、衆院選栃木5区で11選（99,897票、得票率62.8%）

## 評価詳細

### ① 課題設定能力（5/5）
東京大学経済学部、ハーバード大学大学院という学歴、マッキンゼーでのコンサルタント経験を背景に、経済・外交・安全保障など幅広い分野で構造的な課題設定能力を有する。経済再生担当大臣として「アベノミクス」の推進役を担い、外務大臣として日米同盟強化、インド太平洋戦略の推進に携わった。総裁選では「名目GDP700兆円」「年収1割増」など具体的な数値目標を掲げた。トランプ米大統領から「タフネゴシエーター」と評されるほど、国際交渉においても課題を的確に把握する能力がある。

### ② 解決設計力（4/5）
マッキンゼー出身の実務家として、政策の制度設計能力は高い。経済産業大臣として成長戦略、外務大臣として外交戦略の設計に携わった。総裁選では「物価連動型の発注事業」「生活支援特別地方付金」など具体的な制度設計案を提示。ただし、幹事長として裏金問題への対応では、有効な再発防止策を設計できなかったとの批判もある。

### ③ 実行力（5/5）
当選11回、大臣歴任、党三役（幹事長・政調会長・選対委員長）すべてを経験するなど、政治的実行力は極めて高い。外務大臣として日米貿易協定締結、経済産業大臣として産業政策の推進など、具体的な成果を上げている。選挙でも圧倒的強さを誇り、2024年選挙では得票率62.8%で圧勝。党内での調整力、国際交渉力ともに高い評価を受けている。

### ④ 説明責任（2/5）
幹事長として裏金問題の対応にあたったが、「国民に疑念が広がっている」と認めつつも、具体的な説明責任を果たしたとは言い難い。総裁選では「幹事長として政権を支えきれなかったとの指摘があるなら、謙虚に受け止めたい」と述べたが、責任の所在を明確にしなかった。自身の政治資金団体への資金移動問題についても、野党から「脱法行為」と批判されたが、十分な説明がなされていない。

### ⑤ 一貫性（4/5）
1993年から一貫して自民党所属で活動を継続。経済政策、外交政策において自民党の路線を推進してきた。茂木派の領袖となり、2024年に派閥解散を決定したが、これは裏金問題への対応であり、政治的信念の変更ではない。総裁選での敗北後も党内での活動を継続しており、政治家としての軸は維持されている。

### ⑥ 倫理観（2/5）
幹事長として裏金問題を抱える党を統括する立場にあり、問題発覚時の対応には疑問が残る。自身の政治資金管理団体から使途公開基準の緩い団体への資金移動（14年間で4億4590万円）が「脱法行為」と批判された。週刊新潮による「ランチにしか見えない支出」報道など、政治資金の使途についても疑問が呈されている。茂木派自体は裏金問題で立件されていないが、党幹事長として問題対応の責任は免れない。

## 強み・弱み

### 強み
- **高い実務能力**：マッキンゼー出身、複数の大臣経験による政策立案・実行能力
- **国際交渉力**：「タフネゴシエーター」と評される外交力
- **選挙の強さ**：栃木5区で圧倒的な得票率を維持
- **党内での影響力**：幹事長・政調会長・選対委員長すべてを経験

### 弱み
- **人望の問題**：総裁選で9人中6位と低迷、「人望がない」との評も
- **説明責任の欠如**：裏金問題、政治資金問題での説明が不十分
- **党内批判**：裏金議員への処分が甘いとの批判を招いた
