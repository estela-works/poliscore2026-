# 五十嵐清（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年12月14日 |
| 年齢 | 55歳 |
| 選挙区 | 栃木県2区（鹿沼市、日光市、さくら市、宇都宮市の一部、塩谷郡） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 環境大臣政務官（第2次石破内閣）、栃木県議会議長（第105代）、栃木県議会議員（5期） |

## 経歴
- 栃木県小山市出身
- 小山市立小山第一小学校、小山市立小山中学校、栃木県立石橋高等学校卒業
- 1993年、オーストラリア・ボンド大学経営学部経営経済学科卒業
- 約10年間、茂木敏充・佐藤勉・岩崎純三の各議員の秘書を務める
- 2003年、栃木県議会議員選挙（小山市選挙区）で初当選
- 県議会議員を5期連続当選
- 第105代栃木県議会議長、第109代副議長を歴任
- 2021年6月、自民党栃木県連が栃木2区候補予定者に決定
- 2021年10月、衆院選で比例復活により初当選
- 2024年10月、衆院選で比例復活により2選（51,955票、選挙区2位）
- 2024年11月、環境大臣政務官就任

## 評価詳細

### ① 課題設定能力（2/5）
県議会議員として地方自治に関わってきたが、国政レベルでの独自の課題設定は目立たない。農林水産委員会、法務委員会などに所属しているが、特定分野で構造的な問題提起を行った実績は確認できない。環境大臣政務官就任後も、政策面での独自の発信は限定的。党や政府の方針に沿った活動が中心。

### ② 解決設計力（2/5）
県議会議長として議会運営の経験は豊富だが、政策の制度設計能力を示す具体的な実績は乏しい。国会では質問回数も限られており、法案提出や政策立案での主導的役割は確認できない。秘書経験から政策プロセスへの理解はあるものの、自ら設計を行う力は未知数。

### ③ 実行力（4/5）
県議会議員5期、県議会議長という経歴から、地方レベルでの実行力は証明されている。議員秘書として10年間の実務経験も持つ。2回連続で比例復活当選を果たし、政治活動を継続している。環境大臣政務官として行政実務に携わる立場となり、今後の実行力発揮が期待される。

### ④ 説明責任（2/5）
国会での質問・発言は限定的であり、メディア露出も少ない。環境大臣政務官としての発信も始まったばかり。県議会議長時代の議会運営については経験があるものの、政策について有権者に対して積極的に説明する姿勢は目立たない。

### ⑤ 一貫性（4/5）
2003年から一貫して自民党所属で活動を継続。県議から国政への転身も党内の正規ルートを経ており、政治的立場の変遷はない。茂木敏充・佐藤勉という栃木県選出議員の秘書を経て、地元での活動を継続してきた点も一貫している。

### ⑥ 倫理観（4/5）
スキャンダルや不正行為は報じられていない。裏金問題についても「無派閥であり、派閥の問題とは無縁」との立場を表明しており、関与は確認されていない。長年の県議会活動、秘書活動を通じて、大きな問題を起こすことなく政治活動を継続している。

## 強み・弱み

### 強み
- **地方政治の経験**：県議会議員5期、議長経験など豊富な地方行政経験
- **政務官就任**：環境大臣政務官として行政経験を積む機会を得た
- **党内基盤**：茂木敏充、佐藤勉の秘書経験により、党内人脈が豊富
- **クリーンな政治姿勢**：裏金問題とは無縁

### 弱み
- **選挙区での勝利なし**：2回とも比例復活であり、選挙区での勝利経験がない
- **政策発信力**：独自の政策提言や問題提起が少ない
- **知名度**：全国的な知名度は低く、メディア露出も限定的
