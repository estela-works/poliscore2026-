# 佐藤勉（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1952年6月20日 |
| 年齢 | 72歳 |
| 選挙区 | 栃木県4区（小山市、下野市の一部、下都賀郡、真岡市、芳賀郡） |
| 当選回数 | 衆議院10回 |
| 主な役職 | 総務大臣（第11代）、国家公安委員会委員長（第78代）、自民党総務会長（第58代）、自民党国対委員長（第55代）、衆議院議院運営委員長 |

## 経歴
- 栃木県下都賀郡壬生町生まれ
- 栃木県立壬生高等学校卒業
- 日本大学工学部土木工学科卒業
- 1975年、間組入社
- 1987年、栃木県議会議員に初当選（3期）
- 1996年、衆院選栃木4区で初当選
- 2008年、麻生内閣で国家公安委員会委員長・内閣府特命担当大臣として初入閣
- 2009年、鳩山邦夫総務大臣辞任に伴い、総務大臣・内閣府特命担当大臣を兼務
- 2020年、自民党総務会長就任
- 2021年、衆院選栃木4区で9選
- 2024年10月、衆院選で比例復活により10選（75,260票、選挙区2位）
- 2024年11月、自民党経理局長就任

## 評価詳細

### ① 課題設定能力（2/5）
総務大臣、国家公安委員会委員長など要職を歴任したが、政策面での独自の課題設定は目立たない。党務（国対委員長、総務会長）や議会運営（議院運営委員長）での調整役としての活動が中心であり、政策提言型の政治家ではない。2014年の国会では、質問・議員立法・質問主意書のいずれも提出していない議員として名前が挙がっている。

### ② 解決設計力（2/5）
行政経験は豊富だが、政策の制度設計を主導した実績は確認できない。総務大臣としては前任者の辞任に伴う「つなぎ」的な就任であり、独自の政策推進は限定的だった。国対委員長、総務会長として党内調整には長けているが、これは解決設計力とは異なる能力。

### ③ 実行力（4/5）
県議会議員3期、衆議院議員10期と長期にわたる議員活動を継続。総務大臣、国家公安委員会委員長として行政を担当し、自民党総務会長、国対委員長として党運営の要職を務めた。当選10回という実績は政治的な実行力の証。ただし、2024年選挙では選挙区で敗北しており、地元での支持に陰りが見られる。

### ④ 説明責任（2/5）
国会での質問活動が少なく、政策について積極的に発信する姿勢は見られない。裏金問題については「無派閥であり関係ない」と説明しているが、党全体の問題としての説明責任は果たしていない。メディアへの露出も限定的であり、有権者への説明努力は不十分。

### ⑤ 一貫性（4/5）
1996年から一貫して自民党所属で栃木4区から活動を継続。無派閥を貫いており、派閥間の争いには距離を置いている。政治的立場の変遷はなく、党内での立ち位置も安定している。2024年の総裁選では石破茂に投票するなど、自身の判断で行動している。

### ⑥ 倫理観（2/5）
2004年、日歯連闘献金事件で迂回献金の疑惑が報じられ、民主党から政治資金規正法違反容疑で告発された。最終的に不起訴処分となったが、検察審査会から「不起訴不当」議決を受けた経緯がある。献金は返金したと説明しているが、疑惑自体は存在した。また、長期政権の中核を担う立場で、裏金問題が発生した党の責任の一端を担う立場でもある。

## 強み・弱み

### 強み
- **豊富な経験**：当選10回、大臣・党三役経験者としての経験値
- **党内人脈**：長年の活動で築いた党内の幅広い人脈
- **安定性**：無派閥で特定の派閥争いに巻き込まれない立ち位置

### 弱み
- **選挙区敗北**：2024年選挙で選挙区敗北、比例復活
- **政策発信力**：独自の政策提言や発信が少ない
- **過去の疑惑**：日歯連事件の迂回献金疑惑は完全に払拭されていない
