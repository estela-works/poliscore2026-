# 緑川貴士（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1985年1月10日 |
| 年齢 | 40歳 |
| 選挙区 | 秋田2区（能代市、大館市、男鹿市、鹿角市、潟上市、北秋田市など） |
| 当選回数 | 衆議院3回 |
| 主な役職 | 元立憲民主党秋田県連代表、元農林水産委員会理事 |

## 経歴
- 埼玉県飯能市出身
- 早稲田大学社会科学部卒業
- 2009年：秋田朝日放送にアナウンサーとして入社
- 2014年：秋田朝日放送退社
- 2014年：民主党公認で衆院選秋田2区から出馬、落選
- 2017年：希望の党から出馬、比例復活で初当選
- 2018年：国民民主党に参加
- 2020年：立憲民主党に参加、秋田県連代表に就任
- 2021年：衆院選秋田2区で再選
- 2024年：衆院選秋田2区で3選（70,895票で当選）
- 2025年：中道改革連合に参加

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
秋田の地域課題（人口減少、少子高齢化、地場産業の存続、生活インフラの維持、自然災害対応）を具体的に把握し、国政レベルでの解決を訴えている。「自民党政治によって失墜した政治と政治家への信頼を取り戻す」という課題設定は、2024年の政治状況を的確に捉えている。農林水産委員会理事として、地方の一次産業の課題にも取り組んでいる。

### ② 解決設計力: 2/5
**評価根拠**:
「権限と財源を移譲する真の地方分権」「本気の政治改革・政治刷新」を掲げるが、具体的な制度設計や実現へのロードマップは示されていない。野党議員として政策提言は行っているが、制度・予算を含めた設計を主導した実績は確認できない。

### ③ 実行力: 4/5
**評価根拠**:
2017年の初当選以来、100回を超える国会質疑・討論を実施。農林水産委員会理事として実務に携わり、災害ボランティアなど現場活動も継続。2024年衆院選では自民党候補（元大館市長）を約5,700票差で破り、選挙区で当選。野党議員として、地道に影響力を拡大している。

### ④ 説明責任: 4/5
**評価根拠**:
アナウンサー出身で、複雑な政策課題も分かりやすく説明できる。地域を歩き、住民の声を直接聞く活動を7年間継続。SNSでの情報発信も積極的で、政治活動の透明性を確保している。

### ⑤ 一貫性: 2/5
**評価根拠**:
政党遍歴が複雑。民主党→希望の党→国民民主党→立憲民主党と、わずか6年間で4つの政党に所属。いずれも政党再編に伴う移籍ではあるが、自らの判断で所属政党を選択した場面もある。政策的な軸（地方分権、政治改革）は維持しているものの、有権者から見ると一貫性に疑問が残る。

### ⑥ 倫理観: 4/5
**評価根拠**:
政治資金問題やスキャンダルの報道はない。自民党の裏金問題を追及する立場として、自らも政治資金の透明性を訴えている。地域密着型の活動を続け、有権者との直接対話を重視する姿勢は、権力の私物化とは対極にある。

## 強み・弱み

### 強み
- **若さと行動力**: 40歳と若く、100回以上の国会質疑など精力的に活動
- **コミュニケーション力**: アナウンサー出身で説明力が高い
- **選挙区での強さ**: 2024年衆院選で選挙区当選を果たす

### 弱み（注意点）
- **政党遍歴**: 短期間で複数の政党を渡り歩いた経歴
- **野党経験のみ**: 与党として政策を実現した経験がない
- **県外出身**: 埼玉県出身で、秋田との縁は就職がきっかけ
