# 福原淳嗣（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1967年12月18日 |
| 年齢 | 57歳 |
| 選挙区 | 秋田2区（能代市、大館市、男鹿市、鹿角市、潟上市、北秋田市など） |
| 当選回数 | 衆議院1回（比例復活）、大館市長3回、大館市議2回 |
| 主な役職 | 衆議院議員、元大館市長（3期） |

## 経歴
- 秋田県大館市出身
- 秋田県立大館鳳鳴高等学校卒業
- 慶應義塾大学法学部卒業
- 1995年：大館市議会議員に初当選（27歳）
- 1999年：大館市議再選
- 2003年：野呂田芳成衆議院議員の公設第一秘書
- 2008年：野呂田議員の政策担当秘書
- 2009年：金田勝年衆議院議員の政策担当秘書
- 2015年：大館市長に初当選
- 2019年：大館市長再選
- 2023年：大館市長3選
- 2024年8月：大館市長を退職
- 2024年10月：衆院選秋田2区に出馬、比例復活当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
大館市長時代に「秋田県北部エコタウン計画」の企画立案を主導し、地方都市の持続可能性という課題に先駆的に取り組んだ。「国産米の競争力を増すための土地改良」「中山間地の農業と農村を守る」など、農業構造転換の課題を具体的に把握している。地方首長として、地域課題の構造的理解を示している。

### ② 解決設計力: 4/5
**評価根拠**:
大館市長として3期9年間、実際に政策を設計・実行してきた経験がある。エコタウン計画の立案、コロナ禍での財源確保（市長・副市長・教育長の期末手当全額カット）など、制度・予算を含めた設計能力を実証。「生産原価を下げる一丁目一番地が土地改良」という発言は、手段と目的の因果関係を理解した設計思考を示している。

### ③ 実行力: 4/5
**評価根拠**:
大館市長として3期当選し、市政を運営してきた実績がある。2015年の市長選では現職を約1万票差で破るという困難な選挙を突破。2020年のコロナ対応では、市長自らの報酬カットという決断を実行。国政は初挑戦だが、地方首長としての実行力は証明済み。

### ④ 説明責任: 2/5
**評価根拠**:
市長を任期途中で辞職し国政に転身した理由の説明が十分とは言えない。2024年8月末に辞職し、10月の衆院選に出馬するまでの経緯について、有権者への丁寧な説明は確認できない。また、自民党の裏金問題が争点となる中、政治とカネの問題への明確な立場表明も限定的。

### ⑤ 一貫性: 4/5
**評価根拠**:
大館市議→衆議院議員秘書→大館市長→衆議院議員という経歴は、地方政治から国政へという一貫したキャリアパス。自民党との関係も、野呂田芳成、金田勝年両議員の秘書を経て、自民党公認で出馬するなど一貫している。政策面でも、地方創生・農業振興という軸は維持されている。

### ⑥ 倫理観: 4/5
**評価根拠**:
政治資金問題やスキャンダルの報道はない。コロナ禍で自らの報酬をカットし財源に充てる決断は、権力の私物化とは対極の行動。ただし、自民党所属として、党全体の裏金問題についてどのような立場を取るかは今後の課題。

## 強み・弱み

### 強み
- **首長経験**: 大館市長を3期務め、行政運営の経験が豊富
- **政策立案力**: エコタウン計画など、具体的な政策を企画・実行
- **地元密着**: 大館市で生まれ育ち、地域との強い結びつき

### 弱み（注意点）
- **国政経験の浅さ**: 衆議院議員としては1期目、比例復活当選
- **市長辞職の説明不足**: 任期途中での辞職理由が十分に説明されていない
- **選挙区落選**: 2024年衆院選で緑川氏に敗北し、比例復活
