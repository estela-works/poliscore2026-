# 寺田学（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年生まれ |
| 年齢 | 49歳 |
| 選挙区 | 秋田1区（秋田市） |
| 当選回数 | 衆議院7回 |
| 主な役職 | 元内閣総理大臣補佐官（2回）、政治倫理審査会筆頭幹事、財務金融・外務・法務委員会筆頭理事 |

## 経歴
- 秋田県横手市出身
- 横手高校卒業
- 中央大学卒業
- 三菱商事入社、東南アジア諸国を担当
- 2003年：衆議院議員に初当選（27歳、当時最年少）
- 2009年：行政刷新会議「事業仕分け」第1WG主査
- 2011年：菅直人内閣で内閣総理大臣補佐官（原発事故対応担当）
- 2012年：野田佳彦内閣で内閣総理大臣補佐官（社会保障と税の一体改革担当）
- 2024年：政治倫理審査会野党筆頭幹事として自民党裏金問題を追及
- 2025年：政界引退を表明（妻・寺田静参議院議員のサポートのため）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
自民党の裏金問題において、政治倫理審査会の野党筆頭幹事として、問題の本質を「説明責任の欠如」と位置づけ、安倍派・二階派幹部への追及を主導。海外アーティストの興行ビザ取得の課題など、他者が見落としがちな問題も法務省への働きかけにつなげた。ライフワークとする「音楽支援」「教育の多様化」「性犯罪撲滅」も具体的な課題設定がなされている。

### ② 解決設計力: 4/5
**評価根拠**:
内閣総理大臣補佐官として、菅直人内閣では原発事故対応、野田佳彦内閣では社会保障と税の一体改革という大規模政策の設計・実施に関与。2009年の事業仕分けでは第1WG主査として具体的な予算見直しを主導。制度・予算を含めた設計能力を実証している。興行ビザ要件緩和という具体的な政策成果も挙げている。

### ③ 実行力: 4/5
**評価根拠**:
20年間の議員活動で100回以上の国会質疑・討論を実施。内閣総理大臣補佐官を2度務め、政権中枢で実務を担った実績がある。野党の立場でも、政治倫理審査会の公開実現に向けた与党との交渉を粘り強く行い、結果を出した。単なる批判者ではなく、実行者としての実績がある。

### ④ 説明責任: 4/5
**評価根拠**:
自身の政界引退表明において、「妻の政治活動を優先するため」「客観的にどちらが国会議員をやった方がいいか考えた」と率直に説明。政策議論においても、共同親権の国会質疑で自身の子育て経験を基に議論を展開するなど、具体的かつ論理的な説明ができている。

### ⑤ 一貫性: 4/5
**評価根拠**:
民主党→民進党→立憲民主党と政党再編の流れに沿った所属変更はあるが、基本的な政治姿勢（リベラル、行政改革推進、市民生活重視）は一貫。ライフワークとする分野も10年以上継続している。政界引退という大きな決断も、妻のサポートという明確な理由を説明している。

### ⑥ 倫理観: 4/5
**評価根拠**:
20年間の政治活動において、金銭スキャンダルや倫理的問題は報道されていない。自民党の裏金問題を追及する立場として、自らの政治資金にも透明性が求められる中、問題なく活動を継続。政界引退の理由として「カネや女性のスキャンダルがあったら、いますぐ辞めます」と明言するなど、倫理面での自信を示している。

## 強み・弱み

### 強み
- **政権経験**: 内閣総理大臣補佐官を2度務め、政策の設計・実行の経験が豊富
- **追及力と実行力の両立**: 野党として追及するだけでなく、具体的な政策成果も出せる
- **説明力**: 自身の立場や決断を論理的に説明できる

### 弱み（注意点）
- **政界引退**: 2025年に引退を表明しており、次期衆院選には出馬しない
- **与党経験の短さ**: 民主党政権の3年間のみで、その後は野党
- **秋田1区での僅差敗北**: 2024年衆院選では872票差で選挙区落選（比例復活）
