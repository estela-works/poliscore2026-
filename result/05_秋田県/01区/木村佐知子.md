# 木村佐知子（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1986年2月 |
| 年齢 | 39歳 |
| 選挙区 | 秋田1区 |
| 当選回数 | 衆議院0回（台東区議1期） |
| 主な役職 | 弁護士、台東区議会議員 |

## 経歴
- 秋田県秋田市生まれ。医師の父、教師の母のもとで育つ
- 秋田県立秋田高等学校卒業
- 2005年：東京大学文科一類入学
- 2010年：東京大学法学部卒業
- 2012年：中央大学法科大学院修了
- 2013年：司法試験合格（上位約10%）、最高裁判所司法研修所
- 2014年：弁護士登録（東京弁護士会）、さくら共同法律事務所勤務
- 2023年3月：木村佐知子法律事務所を台東区上野で独立開業
- 2023年4月：台東区議選に日本維新の会から出馬、トップ当選
- 2024年7月：日本維新の会を離党
- 2025年：東京大学公共政策大学院入学（社会人学生）
- 2026年1月：国民民主党から衆議院秋田1区への公認内定

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 弁護士として法律の専門知識を持ち、台東区議として子育て支援や所得制限撤廃などの課題に取り組んでいる。しかし、課題設定は既存の政策議論の範囲内にとどまり、独自の構造的問題提起は確認できない。国民民主党の「手取りを増やす」路線に沿った主張が中心で、秋田特有の地域課題への深い分析も不明。

### ② 解決設計力: 2/5
**評価根拠**: 台東区議として、児童手当の所得制限撤廃に向けた上乗せ給付の提言（実現中）、議会委員会のオンライン化推進など、具体的な提案実績がある。ただし、これらは地方議会レベルの施策であり、国政レベルでの制度設計能力は未知数。法律の専門家としての知識は政策立案に活かせる可能性があるが、現時点では限定的な実績にとどまる。

### ③ 実行力: 2/5
**評価根拠**: 台東区議選でトップ当選を果たし、議会活動では委員会オンライン化の推進など一定の成果を上げている。しかし、国政経験はなく、衆議院議員としての実行力は評価できない。地方議員1期目の実績は限定的であり、新人に近い評価とする。法律事務所の独立開業、社会人として東大公共政策大学院への入学など、キャリア面での行動力は認められる。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 弁護士として約12年のキャリアがあり、論理的な説明能力は高いと推測される。自身のウェブサイトやSNSで積極的に情報発信を行い、議会のオンライン配信充実や情報公開の徹底を政策として掲げている点は、透明性への意識の高さを示す。維新離党についても支持者への説明を行っており、説明責任への姿勢は評価できる。

### ⑤ 一貫性: 2/5
**評価根拠**: 2023年に日本維新の会から台東区議に当選したが、わずか1年余りの2024年7月に離党。その後、国民民主党から故郷・秋田への出馬を決意。政党の移籍と選挙区の変更が短期間で行われており、「断腸の思いで区議を辞し」とは述べているものの、有権者から見ると軸がぶれている印象は否めない。政策的な一貫性よりも、機会主義的な動きに見られるリスクがある。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや政治資金問題は報じられていない。弁護士として法令遵守の意識は高いと推測される。情報公開の徹底を政策に掲げ、議会の透明性向上に取り組む姿勢は評価できる。権力の行使経験が限られるため高評価の根拠は限定的だが、現時点で懸念材料はない。

## 強み・弱み

### 強み
- **法律の専門知識**: 弁護士として約12年のキャリア、複数の法律書籍を共同執筆
- **高い学歴と知性**: 東大法学部卒、司法試験上位合格、現在も公共政策大学院で学び続ける向上心
- **若さと発信力**: 39歳と若く、SNSでの積極的な情報発信、子育て中の母親としての視点

### 弱み（注意点）
- **地元との縁の薄さ**: 高校卒業後は東京で活動、秋田との接点が20年以上途絶えている
- **政党・選挙区の変遷**: 維新→国民民主、台東区→秋田1区と短期間での変更は不安定な印象
- **国政経験なし**: 衆議院での活動は未知数、地方議員としての実績も1期途中
- **落下傘候補への批判リスク**: 地元で活動してきた対立候補と比較され不利になる可能性
