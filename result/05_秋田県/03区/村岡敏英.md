# 村岡敏英（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年7月25日（65歳） |
| 年齢 | 65歳 |
| 選挙区 | 秋田3区 |
| 当選回数 | 衆議院3期 |
| 主な役職 | 国民民主党選挙対策委員長、決算行政監視委員長、国民民主党秋田県連代表 |

## 経歴
- 秋田県由利本荘市生まれ
- 日本大学商学部卒業
- 民間会社に5年間勤務
- 1990年、運輸大臣（村岡兼造）政務秘書官
- 1997年、内閣官房長官政務秘書官
- 2005年、衆議院選挙に秋田3区から無所属で出馬、落選
- 2012年、日本維新の会公認で秋田3区から出馬、比例復活で初当選
- 2014年、維新の党公認で出馬、比例復活で再選
- 2017年、希望の党公認で出馬、落選
- 2021年、秋田県知事選挙に出馬、193,538票で惜敗
- 2022年、参議院選挙に出馬、162,889票で惜敗
- 2024年、国民民主党公認で秋田3区から出馬、初の小選挙区当選（父・兼造以来24年ぶり）
- 2025年9月、国民民主党選挙対策委員長就任

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「秋田再起動」をスローガンに地域振興を掲げる。父・兼造の秘書経験から政治の現場を熟知。国民民主党の政策（手取りを増やす、消費税減税など）に沿った課題設定。ただし、独自の課題設定や政策提言は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 政務秘書官として政策立案の現場を経験。衆議院議員3期の経験もあるが、独自の立法実績や政策設計を示す材料は少ない。国民民主党選挙対策委員長として党の選挙戦略を担当。

### ③ 実行力: 3/5
**評価根拠**: 2005年から挑戦を続け、2024年に初めて御法川信英を破り小選挙区当選を果たした粘り強さ。知事選、参院選と敗北を重ねても挑戦を続けた。決算行政監視委員長として国会運営に関与。ただし、具体的な政策実現の実績は確認できない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 著書「スタッフ～秘書が見た日本政治の十六年」で政治の内幕を発信。公式サイトやSNSで活動報告を行っている。国民民主党秋田県連代表として地元との対話を続けている。

### ⑤ 一貫性: 3/5
**評価根拠**: 秋田3区からの国政挑戦を一貫して続けてきた姿勢は評価できる。しかし、所属政党は無所属→日本維新の会→維新の党→希望の党→国民民主党と変遷が多い。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 重大なスキャンダルや問題行動は確認されていない。父・兼造の政治資金規正法違反事件との関連は不明だが、本人についての問題は報じられていない。

## 強み・弱み

### 強み
- **地元での知名度**: 村岡家として秋田での知名度がある
- **粘り強さ**: 何度の落選を経ても挑戦を続けた実績
- **政務秘書経験**: 大臣、官房長官秘書官として政治の現場を熟知
- **党内での地位**: 国民民主党選挙対策委員長

### 弱み（注意点）
- **政党変遷の多さ**: 5つの政党を渡り歩いた経歴
- **独自政策の不明確さ**: 独自の政策提言や立法実績が見えにくい
- **過去の落選歴**: 2017年、2021年知事選、2022年参院選と落選が続いた時期がある
