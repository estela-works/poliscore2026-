# 山田勝彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1979年7月19日 |
| 年齢 | 45歳 |
| 選挙区 | 長崎2区（比例九州） |
| 当選回数 | 3回（2021年比例、2024年補選、2024年比例） |
| 主な役職 | 立憲民主党青年局長兼副幹事長 |

## 経歴
- 長崎県大村市出身、長崎県立大村高等学校卒業
- 法政大学社会学部卒業
- 株式会社プレナス勤務、父・山田正彦（元農水大臣）秘書
- 2010年農林水産省大臣秘書官
- 2013年株式会社やまびこ学苑設立（発達障害児支援）
- 2018年就労支援やまびこ農苑創業（自然栽培・障害者雇用）
- 2018年立憲民主党長崎県連代表就任
- 2022年第208回国会「三ツ星議員」表彰

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 発達障害児支援、食の安全、農業問題など自身の事業経験に基づく具体的課題を提起。子どもの健康被害増加への問題意識は明確。

### ② 解決設計力: 3/5
**評価根拠**: 「やまびこ学苑」「やまびこ農苑」を実際に創業・運営。障害者雇用と自然栽培を組み合わせた事業モデルは具体的な解決設計の実績。

### ③ 実行力: 3/5
**評価根拠**: 起業家として複数事業を成功させた実績。国会では「三ツ星議員」として表彰されるなど活発な活動。ただし選挙区では勝てず比例復活が続く。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: SNS、ブログでの発信活動あり。「もう変えんば！」のスローガンで分かりやすい発信。国会質疑でも活発。

### ⑤ 一貫性: 4/5
**評価根拠**: 発達障害児支援、食の安全、農業問題という軸が一貫。事業活動から政治活動まで同じテーマで継続。父の影響もあるが自身の実体験に基づく。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダル・問題行動なし。障害者福祉、有機農業など社会貢献型の事業に長年従事。

## 強み・弱み

### 強み
- **実業経験**: 発達障害児支援事業の創業者として社会課題に取り組む
- **一貫性**: 子ども・食・農業という軸がブレない

### 弱み（注意点）
- **選挙力**: 選挙区では勝てず比例復活が続く
- **世襲**: 父が元農水大臣という2世議員
