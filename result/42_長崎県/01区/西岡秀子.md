# 西岡秀子（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年3月15日 |
| 年齢 | 61歳 |
| 選挙区 | 長崎県1区 |
| 当選回数 | 衆議院3回 |
| 主な役職 | 国民民主党政調会長代理、党副幹事長、長崎県連代表 |

## 経歴
- 長崎県長崎市生まれ
- 父は西岡武夫（元参議院議長、文部大臣、自民党総務会長）
- 祖父は元長崎県知事、祖母も参議院議員という政治家一家
- 長崎大学教育学部附属小学校・中学校卒業
- 活水高等学校卒業
- 学習院女子短期大学卒業
- 学習院大学法学部卒業
- NEC勤務
- 父・西岡武夫の秘書
- 2016年参院選：長崎県選挙区から民進党公認で出馬→落選
- 2017年衆院選：希望の党から長崎1区で初当選（希望の党新人で小選挙区当選は唯一）
- 2018年国民民主党に参加
- 2020年新・国民民主党に参加（立憲との合流を選ばず）
- 2021年衆院選：長崎1区で再選
- 2024年衆院選：長崎1区で3選
- 長崎県唯一の女性国会議員

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 長崎県唯一の女性国会議員として、男女共同参画推進本部長代理を務め、ジェンダー問題への関心を示している。政治家一家出身で政治的課題への感度はあると推測される。ただし、独自の構造的課題分析については確認できない情報が多い。

### ② 解決設計力: 3/5
**評価根拠**: 学習院大学法学部卒業、NEC勤務経験があり、法律と民間企業の知識を持つ。父の秘書として政策立案の現場を経験。国民民主党政調会長代理として政策立案に関与している。ただし、具体的な政策設計の実績は確認が限定的。

### ③ 実行力: 4/5
**評価根拠**: 2017年に希望の党の新人として小選挙区で唯一当選するという難しい選挙を勝ち抜いた。2021年、2024年と自民党候補に比例復活も許さず3期連続当選。長崎1区で安定した支持基盤を構築している。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。国民民主党の顔として長崎県で活動。ただし、全国的なメディア露出は限定的で、コミュニケーション力を十分評価する材料は不足。

### ⑤ 一貫性: 3/5
**評価根拠**: 希望の党→国民民主党と政党再編の中で変遷があったが、立憲民主党との合流を選ばず新・国民民主党を選択するなど、自身の政治信条に基づいた判断をしている。ただし、政党の変遷自体は複雑。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。政治家一家出身だが、自ら選挙を戦い抜いてきた。クリーンなイメージを維持している。

## 強み・弱み

### 強み
- **選挙での強さ**: 3期連続で自民候補を破る
- **政治家一家の経験**: 父は参議院議長、政治の現場を熟知
- **女性議員**: 長崎県唯一の女性国会議員として独自の視点
- **地盤**: 長崎1区で安定した支持

### 弱み（注意点）
- **野党**: 政権運営の経験がない
- **知名度**: 全国的な認知度は限定的
- **政党の変遷**: 希望の党→国民民主党と複雑な経緯
- **世襲的イメージ**: 政治家一家への批判がある可能性
