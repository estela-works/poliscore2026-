# 佐々木紀（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年10月18日 |
| 年齢 | 51歳 |
| 選挙区 | 石川2区（小松市・加賀市・能美市・白山市・野々市市等） |
| 当選回数 | 衆議院5期 |
| 主な役職 | 国土交通副大臣、自民党石川県連会長、自民党情報調査局長 |

## 経歴
- 石川県能美市出身
- 石川県立小松高等学校卒業
- 東北大学法学部卒業（税法専攻）
- 仙台にて友人と輸入ビジネスを起業
- 2003年、父の経営するビル管理会社入社
- 2010年、小松青年会議所理事長
- 2011年、日本青年会議所石川ブロック協議会会長
- 2012年衆院選で初当選（森喜朗元首相の後継として公募で選出）
- 2019年、国土交通大臣政務官（第4次安倍第2次改造内閣）
- 2024年4月、自民党石川県連会長就任
- 2024年衆院選で5選（得票率78.4%）
- 自民党情報調査局長・選挙対策副委員長
- 2025年10月、国土交通副大臣（高市内閣）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 国交政務官・副大臣として交通インフラ、防災対策に取り組む。能登半島地震の被災地選出として復興課題に直面。党情報調査局長として世論分析も担当。

### ② 解決設計力: 4/5
**評価根拠**: 東北大法学部卒、起業経験あり。国交政務官・副大臣として実務経験豊富。5期の議員経験から政策立案能力は高い。県連会長として組織運営も担う。

### ③ 実行力: 5/5
**評価根拠**: 5期連続当選で石川2区の地盤は盤石。2024年衆院選では得票率78.4%で圧勝。森喜朗元首相の後継として保守基盤を継承。国交副大臣として閣僚級の実行力を発揮。

### ④ 説明責任: 4/5
**評価根拠**: 副大臣として国会での説明責任を果たす立場。公式サイトで活動報告を継続。県連会長として党の方針説明も担う。

### ⑤ 一貫性: 4/5
**評価根拠**: 2012年の初当選以来、自民党で一貫して活動。旧安倍派所属で保守路線を維持。国交畑でキャリアを積み、政務官から副大臣へ順当に昇格。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 旧安倍派所属で裏金問題に関与。裏金問題で役職停止となった宮本周司の後任として県連会長に就任したことで、問題の収拾役を担うことになったが、派閥の一員としての責任は残る。

## 強み・弱み

### 強み
- **閣僚経験**: 国交政務官、国交副大臣
- **圧倒的地盤**: 得票率78%超の支持
- **党の要職**: 県連会長、情報調査局長、選対副委員長
- **森元首相の後継**: 強固な保守基盤
- **地元との結びつき**: 松井秀喜と同級生という話題性

### 弱み（注意点）
- **裏金問題**: 旧安倍派所属による責任
- **派閥政治**: 清和会の負の遺産
- **世襲的構図**: 森元首相の後継という批判も
- **野党の批判**: 自民批判票の結集点に
- **能登半島地震**: 復興の遅れへの責任追及
