# 小森卓郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年5月21日 |
| 年齢 | 55歳 |
| 選挙区 | 石川県1区 |
| 当選回数 | 衆議院2回 |
| 主な役職 | 経済産業大臣政務官兼復興大臣政務官兼内閣府大臣政務官、元総務大臣政務官、元金融庁総合政策課長 |

## 経歴
- 神奈川県横浜市出身
- 東京大学法学部卒業（弁論部）
- 1993年大蔵省（現財務省）入省
- 1997年プリンストン大学大学院修了（ベン・バーナンキに師事）
- 2002-2005年IMFエコノミスト（日本経済担当）
- 2009-2011年官房副長官秘書官（松井孝治・福山哲郎両副長官）
- 2011-2014年石川県出向（企画振興部長・総務部長）
- 北陸新幹線金沢開業準備、IRいしかわ鉄道設立に携わる
- 2014年北村茂男衆議院議員の長女と結婚
- 2021年金融庁総合政策課長を辞し政治家へ
- 2021年初当選（馳浩の後継として石川1区）
- 2023年総務大臣政務官
- 2024年再選
- **2024年能登半島地震：政府現地非常災害対策本部副本部長として指揮**
- 2025年経済産業大臣政務官（高市内閣）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 財務省・金融庁での経験から経済・金融政策の課題認識は深い。石川県での地方行政経験から地方創生の課題も把握。能登半島地震では即座に被災地支援の課題に取り組んだ。

### ② 解決設計力: 5/5
**評価根拠**: 東大法学部、プリンストン大学院（バーナンキに師事）という高い学歴。IMFエコノミスト、金融庁総合政策課長として政策立案の実績。北陸新幹線開業準備やIRいしかわ鉄道設立など具体的な成果を出した。

### ③ 実行力: 4/5
**評価根拠**: 2期連続当選。能登半島地震では200名超の組織を立ち上げ、前線の支援を指揮。総務大臣政務官、経済産業大臣政務官と要職を歴任。財務官僚としての実務能力は高い。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 弁論部出身でコミュニケーション能力は高い。国会答弁も適切にこなす。能登半島地震では県庁との連携で情報発信。SNSでの発信も積極的。

### ⑤ 一貫性: 4/5
**評価根拠**: 財務官僚から政治家へという転身だが、経済・地方創生という軸は一貫。石川県での活動を継続し、馳浩の後継として地盤を引き継ぐ。自民党で一貫して活動。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 統一教会問題・裏金問題との関連は確認されていない。能登半島地震では私心なく被災地支援に当たった（家族も被災）。スキャンダルは確認されず。

## 強み・弱み

### 強み
- **高い専門性**: 財務省・IMF・金融庁での経験
- **災害対応実績**: 能登半島地震での現地対策本部副本部長
- **政策立案能力**: 北陸新幹線開業準備など具体的成果
- **クリーン**: 裏金・統一教会問題と無関係
- **国際経験**: IMF、プリンストン大学

### 弱み（注意点）
- **落下傘候補**: 横浜出身で石川県との縁は出向から
- **知名度**: 馳浩に比べて全国的知名度は低い
- **2期目**: まだ経験は浅い
