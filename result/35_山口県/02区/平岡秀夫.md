# 平岡秀夫（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1954年1月14日 |
| 年齢 | 70歳 |
| 選挙区 | 山口2区（岩国市、下松市、光市、柳井市、周南市など） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 立憲民主党山口県連代表、元法務大臣（第87代）、元総務副大臣、元内閣府副大臣 |

## 経歴
- 1954年1月14日：山口県岩国市生まれ
- 山口県立岩国高等学校卒業
- 東京大学法学部第1類（私法コース）卒業
- 1975年10月：司法試験合格（在学中）
- 大蔵省入省（理財局配属）
- 主計局調査課調査第二係長心得
- 酒田税務署長
- 東海財務局理財部長
- 東京国税局課税第二部長
- 在インド日本大使館一等書記官
- 内閣法制局参事官
- 1998年：大蔵省退官
- 弁護士登録、岩国市で平岡秀夫法律事務所開設
- 2000年：第42回衆院選で初当選（山口2区、比例復活）
- 2010年9月：内閣府副大臣（菅直人内閣）
- 2010年9月：総務副大臣（菅直人第1次改造内閣）
- 2011年9月：法務大臣就任（野田内閣、第87代）
- 2012年1月：法務大臣退任
- 2012年12月：第46回衆院選で落選
- 2024年5月：山口2区の党公認候補予定者に
- 2024年10月：第50回衆院選で比例復活当選（6期目）、惜敗率98.36%

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 安全保障政策での外交努力重視、原発ゼロの推進など、明確な政策課題を設定。法務大臣経験から司法・人権分野での課題認識も持つ。ただし12年間の政界離脱期間があり、最新の課題への対応力は不明。

### ② 解決設計力: 4/5
**評価根拠**: 東京大学法学部卒、司法試験合格（在学中）、大蔵官僚、弁護士という経歴。法務大臣、総務副大臣、内閣府副大臣を歴任し、政策設計の実務経験は豊富。内閣法制局参事官としての経験もある。

### ③ 実行力: 3/5
**評価根拠**: 衆院議員6期当選、法務大臣など閣僚経験。ただし2012年から2024年まで12年間は政界を離れていた。2024年衆院選では岸信千世に1,724票差で敗れ比例復活。選挙での実行力に課題。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 国会法務委員会で他党議員の質疑中に「奇声を発した」と批判された。「あんまり気にしないでくださいよ」と開き直りの発言。国会の品位を損なう行為として厳しい批判を受けている。

### ⑤ 一貫性: 3/5
**評価根拠**: 原発ゼロ、外交重視の安全保障政策など、政策的には一貫している。民主党→立憲民主党と政党は維持。ただし12年間の政界離脱があり、継続性に課題。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 法務大臣時代、詐欺で有罪判決を受けた元秘書を大臣秘書官に起用し問題に。就任直後に「大臣規範」で自粛が明記されている大規模パーティーを開催。国会での奇声問題で品位を疑われている。

## 強み・弱み

### 強み
- **豊富な経歴**：大蔵官僚、弁護士、法務大臣
- **高い学歴**：東大法学部卒、司法試験在学中合格
- **閣僚経験**：法務大臣、総務副大臣、内閣府副大臣
- **地元出身**：岩国市生まれ育ち
- **接戦能力**：岸信千世に1,724票差まで迫った

### 弱み（注意点）
- **12年間の政界離脱**：2012年から2024年まで不在
- **国会での奇声問題**：品位を損なう行為と批判
- **法務大臣時代の問題**：有罪経歴の秘書起用、パーティー問題
- **年齢**：70歳と高齢
- **比例復活**：小選挙区では敗北
