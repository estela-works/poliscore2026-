# 上野賢一郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年8月3日 |
| 年齢 | 59歳 |
| 選挙区 | 滋賀県2区 |
| 当選回数 | 衆議院6回 |
| 主な役職 | 厚生労働大臣（第30代）、元財務副大臣、元国土交通大臣政務官 |

## 経歴
- 滋賀県長浜市出身
- 滋賀県立虎姫高等学校卒業
- 京都大学法学部卒業（1990年）
- 自治省入省
- 岩手県庁勤務
- 総務省自治税務局都道府県税課長補佐を最後に退官（2003年）
- 立命館大学大学院非常勤講師、龍谷大学非常勤講師
- 2003年衆院選：滋賀1区から出馬→落選
- 2004年参院選：滋賀県選挙区から出馬→落選
- 2005年衆院選：滋賀2区から出馬→初当選
- 2010年滋賀県知事選：出馬→落選（嘉田由紀子に敗北）
- 2012年衆院選：滋賀2区で国政復帰
- 国土交通大臣政務官（第2次安倍改造内閣・第3次安倍内閣）
- 財務副大臣（第3次安倍第3次改造内閣〜第4次安倍改造内閣）
- 衆議院内閣委員長
- 自民党滋賀県連会長
- 2024年衆院選6選
- 2025年10月厚生労働大臣就任（高市内閣、初入閣）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 自治官僚出身で地方行政の課題に精通。財務副大臣として財政問題、厚労大臣として社会保障問題に取り組んでいる。ただし、独自の構造的課題設定よりも、政府方針に沿った対応が中心。

### ② 解決設計力: 4/5
**評価根拠**: 京都大学法学部卒、自治省出身で政策立案の専門性が高い。財務副大臣として3年間勤務し、税財政政策に精通。厚労大臣として賃上げに向けた労働市場改革、社会保障の負担と給付のあり方について「国民会議」設置を提案するなど具体的な政策を推進。

### ③ 実行力: 4/5
**評価根拠**: 6期連続当選。2010年の知事選落選を経ても国政に復帰し、着実にキャリアを積んで厚労大臣まで上り詰めた。財務副大臣として長期間務めた実績。滋賀2区で安定した支持基盤を構築している。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 政治資金の使途について疑問が提起されている。資金管理団体が赤坂のスナックに計31万円超、女性歌手のファンクラブに約3万円を支出していたことが報じられた。政治資金の使途について説明責任が問われる。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党で一貫して活動しているが、2010年に知事選出馬のため一度離党し、その後復党。財政・税制という軸では一貫しているが、政治的な判断には柔軟性（日和見的な面）も見られる。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 政治資金のスナック支出、ファンクラブ支出が報じられ、「炎上」と報道された。政治資金の使途について市民感覚との乖離が指摘される。公私混同の疑念を招いている。

## 強み・弱み

### 強み
- **閣僚経験**: 厚労大臣として政策を主導
- **官僚出身**: 自治省出身で政策立案能力が高い
- **選挙の強さ**: 6期連続当選
- **財政の専門性**: 財務副大臣を3年間務めた

### 弱み（注意点）
- **政治資金問題**: スナック、ファンクラブへの支出が批判
- **過去の落選**: 知事選で嘉田由紀子に敗北
- **説明責任**: 政治資金の使途について説明が不十分
- **高市内閣との関係**: 高市政権の評価に左右される
