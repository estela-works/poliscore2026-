# 平尾道雄（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1950年12月24日 |
| 年齢 | 73歳 |
| 選挙区 | 滋賀県2区 |
| 当選回数 | 衆議院0回、米原市長4期 |
| 主な役職 | 前米原市長 |

## 経歴
- 滋賀県坂田郡伊吹村高番（現・米原市）出身
- 滋賀県立米原高等学校卒業
- 東洋大学社会学部卒業
- 米原町役場に奉職（1973年）
- 山東町・伊吹町・米原町合併協議会事務局長（2003年）
- 2005年米原市長選：初当選
- 2009年米原市長選：落選
- 2013年米原市長選：返り咲き当選
- 2017年米原市長選：3回目当選
- 2021年米原市長選：無投票で4回目当選
- 2024年9月市長辞職願提出
- 2024年衆院選：立憲民主党から滋賀2区で出馬→落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 米原市長として4期にわたり地方行政を担当し、地域の課題に精通。市町村合併後の自治体運営という困難な課題に取り組んだ。ただし、国政レベルでの課題設定は未経験で、独自の構造的分析は確認できない。

### ② 解決設計力: 3/5
**評価根拠**: 東洋大学社会学部卒業。米原町役場での長年の行政経験、合併協議会事務局長として市町村合併をまとめた実績がある。市長として行政運営の経験は豊富だが、国政レベルでの政策設計経験はない。

### ③ 実行力: 3/5
**評価根拠**: 米原市長を4期務めた実績。2009年に落選しながらも2013年に返り咲くなど粘り強さがある。ただし、2024年衆院選では落選。国政での実行力は証明されていない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 市長として市民に対する説明責任を果たしてきたと推測される。後援会を通じた活動も行っている。ただし、全国的なメディア露出は限定的で、国政レベルでの説明力は未知数。

### ⑤ 一貫性: 3/5
**評価根拠**: 立憲民主党（現・中道改革連合）から出馬し、地元密着型の政治姿勢は一貫。ただし、市長から国会議員への転身という大きな方向転換があり、政治的な軸が国政でどう発揮されるかは未知数。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。米原町役場から市長まで地元で長年活動し、クリーンなイメージを維持。市長4期という長期政権も市民からの信頼の証。

## 強み・弱み

### 強み
- **行政経験**: 米原市長4期の豊富な経験
- **地元密着**: 米原市で長年活動、地元の事情に精通
- **市町村合併の実績**: 合併協議会事務局長として難題をまとめた
- **クリーンなイメージ**: スキャンダルがない

### 弱み（注意点）
- **国政経験なし**: 衆院選で当選経験がない
- **高齢**: 73歳で体力面の懸念
- **2024年衆院選敗北**: 上野賢一郎に敗れた
- **知名度**: 県外での認知度は低い
