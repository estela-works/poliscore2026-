# 大岡敏孝（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1972年4月16日 |
| 年齢 | 52歳 |
| 選挙区 | 滋賀県1区 |
| 当選回数 | 衆議院5回 |
| 主な役職 | 元環境副大臣 |

## 経歴
- 滋賀県甲賀郡（現・甲賀市）土山町出身
- ラ・サール中学校・高等学校卒業
- 早稲田大学政治経済学部経済学科卒業
- 1995年スズキ入社（国内・海外四輪営業部）
- 1998年スズキ退社
- 1999年浜松市議会議員選挙で初当選
- 2003年浜松市長選挙に立候補、落選
- 2005年中小企業診断士取得
- 2007年静岡県議会議員選挙で初当選（浜松市中区）
- 2011年静岡県議再選
- 2012年静岡県議辞職、衆院選滋賀1区から立候補し初当選（民主党重鎮・川端達夫を破る）
- 環境副大臣（岸田内閣）
- 2024年衆院選で小選挙区敗北、比例復活で5期目

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 環境政策に取り組み、副大臣として環境行政に関与。地方議会経験を活かした政策立案。

### ② 解決設計力: 4/5
**評価根拠**: 早稲田大学政経学部卒、スズキでの民間経験、中小企業診断士資格。地方議会から国政まで幅広い経験。

### ③ 実行力: 4/5
**評価根拠**: 5期連続当選。2012年に民主党の重鎮・川端達夫を破る金星。環境副大臣として行政経験。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 政治資金問題で複数の疑惑。雇用調整助成金問題では当初正当性を主張後、批判を受けて陳謝・返還に転じた。

### ⑤ 一貫性: 3/5
**評価根拠**: 浜松市議・静岡県議から滋賀1区へと地盤を変更。政策面では環境問題に継続的に取り組む。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: **企業献金上限超過（2012年、1000万円受領で上限750万円を超過）**。**政治資金パーティーで自治連合会長ら招待し無料で酒食提供、公選法違反で告発**。**雇用調整助成金受給問題（2020年、後に返還）**。**旧統一教会関連団体（世界平和女性連合）に懇親会費15,000円支出**。

## 強み・弱み

### 強み
- **経験**: 5期当選、環境副大臣経験
- **学歴・資格**: 早稲田大学卒、中小企業診断士
- **民間経験**: スズキでの勤務経験
- **実績**: 川端達夫を破った選挙力
- **地方政治経験**: 市議・県議を経験

### 弱み（注意点）
- **政治資金問題**: 企業献金上限超過、パーティー問題
- **雇調金問題**: コロナ禍で助成金受給し批判
- **統一教会関連**: 世界平和女性連合との関係
- **2024年敗北**: 小選挙区で敗れ比例復活
- **地盤変更**: 浜松から滋賀への移動
