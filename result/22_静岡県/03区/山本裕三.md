# 山本裕三（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1982年12月3日 |
| 年齢 | 42歳 |
| 選挙区 | 静岡県3区 |
| 当選回数 | 新人（国政）、掛川市議3期 |
| 主な役職 | 自民党静岡県第三選挙区支部長、元掛川市議会議長 |

## 経歴
- 静岡県掛川市出身
- 静岡県立掛川西高等学校卒業
- 立教大学経済学部卒業
- 株式会社リクルートHRM（現リクルートホールディングス）入社
- 2009年英国リーズ大学語学留学
- 2011年家業の酒屋・株式会社松風屋入社
- 2013年掛川市議会議員に当時最年少で初当選
- 掛川市議3期当選
- 2023年掛川市議会議長に歴代最年少（40歳）で就任
- 2024年8月自民党静岡3区支部長に選任
- 2024年8月市議会議員辞職
- 2024年衆院選で落選
- 自民党静岡県連青年局次長

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 地域インフラ整備（リニア、国道）、農業振興（抹茶、世界農業遺産）、多文化共生などの政策を掲げる。政治の透明性向上を訴える。

### ② 解決設計力: 3/5
**評価根拠**: 立教大学経済学部卒、リクルートでの営業経験。市議会議長として議会運営の経験。国政レベルの政策立案経験はこれから。

### ③ 実行力: 3/5
**評価根拠**: 市議3期当選、最年少で議長就任。2024年衆院選では落選も、支部長として留任し再挑戦。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: リクルートでの営業経験。市議として地域の声を聴く活動。敗戦後も「ここからがスタート」と前向きな姿勢。

### ⑤ 一貫性: 4/5
**評価根拠**: 掛川市で一貫して活動。市議→国政挑戦という段階的なキャリア。自民党で継続的に活動。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。「政治とカネ」問題の透明性向上を訴える姿勢。

## 強み・弱み

### 強み
- **地方議会経験**: 市議3期、最年少議長
- **民間経験**: リクルート勤務、家業の酒屋経営
- **若さ**: 42歳
- **地元出身**: 掛川市出身で地域密着
- **党内地位**: 県連青年局次長として活動
- **クリーン**: スキャンダルなし

### 弱み（注意点）
- **国政経験なし**: 2024年落選
- **知名度**: 小山展弘に比べ低い
- **前任者の問題**: 前支部長・宮澤博行の不祥事辞職の影響
