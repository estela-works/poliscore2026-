# 小山展弘（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年12月26日 |
| 年齢 | 50歳 |
| 選挙区 | 静岡県3区 |
| 当選回数 | 衆議院4回 |
| 主な役職 | 農林水産委員会委員、協同組合振興研究議員連盟事務局長、超党派石橋湛山研究会事務局長 |

## 経歴
- 静岡県小笠郡大須賀町（現・掛川市）出身
- 静岡県立磐田南高等学校卒業
- 早稲田大学政治経済学部卒業
- 早稲田大学大学院政治学研究科修士課程修了（政治学修士）
- 農林中央金庫勤務（ストラクチャードファイナンス、森林関係融資等に従事）
- 松下政経塾入塾
- 2009年衆院選静岡3区から出馬、柳澤伯夫（元厚労大臣）を破り初当選
- 2012年衆院選落選
- 2021年衆院選で野党統一候補として当選（川勝平太知事の支援）
- 2024年衆院選で4期目当選

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 協同組合振興という明確な政策テーマを10年以上一貫して追求。「労働者協同組合法」「協同組合憲章」「協同組合基本法」の成立を訴える。農業・食糧安全保障にも注力。

### ② 解決設計力: 4/5
**評価根拠**: 早稲田大学大学院修士（政治学）。農林中央金庫での金融実務経験。松下政経塾での政策研究。協同組合振興研究議員連盟を立ち上げ、国会決議素案を作成。

### ③ 実行力: 4/5
**評価根拠**: 4期当選。2009年に元厚労大臣・柳澤伯夫を破る。落選後も復活し当選を続ける。協同組合振興の議員連盟事務局長として活動。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 協同組合、農業政策について一貫した発信。農協法改正反対、種子法廃止反対で国会討論を行う。政策の説明が明確。

### ⑤ 一貫性: 5/5
**評価根拠**: 協同組合振興を10年以上追求。農林中央金庫→政治家という一貫したキャリア。農業政策・協同組合政策で軸がぶれない。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。協同組合の「新しい公共」としての価値を重視する姿勢。

## 強み・弱み

### 強み
- **専門性**: 農林中央金庫での金融経験、協同組合政策の専門家
- **学歴**: 政治学修士、松下政経塾出身
- **一貫性**: 協同組合振興という明確な政策軸
- **地元出身**: 掛川市出身で地域に根差す
- **選挙実績**: 柳澤伯夫、宮澤博行に勝利
- **クリーン**: スキャンダルなし

### 弱み（注意点）
- **落選経験**: 2012年落選で空白期間あり
- **党の変遷**: 民主党→民進党→国民民主党→立憲民主党
