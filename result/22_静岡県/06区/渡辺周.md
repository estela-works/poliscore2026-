# 渡辺周（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年12月11日 |
| 年齢 | 64歳 |
| 選挙区 | 静岡6区 |

| 主な役職 | 衆院政治改革特別委員長、党政治改革本部長、元防衛副大臣、元総務副大臣 |

## 経歴
- 静岡県沼津市出身
- 静岡県立沼津東高等学校卒業
- 早稲田大学政治経済学部経済学科卒業
- 読売新聞社記者（1987-1990年）
- 1991年静岡県議初当選（民社党）、1996年衆院初当選
- 総務副大臣（鳩山・菅内閣）、防衛副大臣（野田内閣）
- 民主党→国民民主党→立憲民主党→中道改革連合

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「公正を実感できる世の中の実現」を掲げ、行財政改革、次世代産業育成、介護社会実現など幅広い課題を設定。領土問題・拉致問題にも超党派で取り組む。ただし、独自の問題提起というより党の方針に沿った活動が中心。

### ② 解決設計力: 3/5
**評価根拠**: 総務副大臣、防衛副大臣として行政の中枢を経験。政策立案の経験は豊富だが、自ら主導した大規模な制度改革の実績は確認できない。副大臣として大臣を補佐する役割が中心。

### ③ 実行力: 4/5
**評価根拠**: 衆院10期連続当選は高い評価に値する。静岡6区で10回の選挙を勝ち抜いた実績、2024年選挙でも4000票差で勝利。政治改革特別委員長として裏金問題対応にも関与。副大臣職の実務経験も豊富。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 読売新聞記者出身で発信力はある。国会答弁や委員会質疑での経験も豊富。ただし、車検切れ問題への対応など、説明責任が問われる場面での具体的な対応は確認できず。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→国民民主党→立憲民主党→中道改革連合と党籍は変遷しているが、いずれも旧民主党系の流れであり、政策の軸（安全保障重視、行財政改革）は概ね一貫。集団的自衛権容認など党内では保守的な立場を維持。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 2025年3月に秘書が車検切れ車両使用で書類送検された問題が発覚。本人の直接的な不正ではないが、事務所管理の問題。汚職や重大スキャンダルは確認されていないが、完全にクリーンとは言えない。

## 強み・弱み

### 強み
- **豊富な経験**: 衆院10期、総務・防衛副大臣の行政経験
- **選挙区での強さ**: 静岡6区で10連勝、地盤は強固
- **安全保障の専門性**: 防衛副大臣経験、領土・拉致問題への取り組み

### 弱み（注意点）
- **党遍歴**: 民主党系内とはいえ複数党を渡り歩いた
- **事務所管理問題**: 秘書の車検切れ問題で書類送検
- **独自性の薄さ**: 大きな政策を主導した実績が見えにくい
