# 袴田サヤカ（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年頃（推定） |
| 年齢 | 49歳 |
| 選挙区 | 静岡7区（浜松市天竜区、湖西市等） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党静岡7区国政改革委員 |

## 経歴
- 愛知県豊川市出身、静岡県湖西市在住
- 小中学校の臨時教員
- 現在は消費生活センター相談員
- 参政党静岡県支部連合会所属
- 2025年12月参政党から静岡7区候補として発表

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「食と農を守る」「食料自給率100%」「外国人労働者規制の明確化」など参政党の政策に沿った課題を設定。消費生活相談員としての経験を活かした視点はあるが、独自の課題設定は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 教員、消費生活相談員としての実務経験はあるが、政策設計の実績は確認できない。参政党の政策を掲げるものの、具体的な解決策の設計は見えない。

### ③ 実行力: 1/5
**評価根拠**: 政治経験がなく、選挙経験もない完全な新人。静岡7区は城内実が7期連続当選の強固な地盤であり、参政党の新人が当選するのは極めて困難。

### ④ 説明責任: 2/5
**評価根拠**: 参政党を通じて政策発信は行っているが、個人としての詳細な政策説明は限定的。公式サイトやSNSでの情報発信は確認できない。

### ⑤ 一貫性: 3/5
**評価根拠**: 参政党の政策（食料自給、積極財政、外国人規制）を一貫して支持。政治経験がないため一貫性を評価する材料は乏しいが、党方針に沿った活動を継続。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。政治家としての活動歴がないため評価材料は限定的。

## 強み・弱み

### 強み
- **生活者視点**: 消費生活相談員として市民の生活問題に接してきた経験
- **教育経験**: 小中学校の臨時教員として教育現場を知る
- **地域在住**: 湖西市在住で選挙区内に住んでいる

### 弱み（注意点）
- **政治経験皆無**: 選挙経験も議員経験もない完全な新人
- **選挙区事情**: 静岡7区は城内実が7期連続当選で圧倒的に強い
- **党の位置づけ**: 参政党は少数政党で当選は極めて困難
- **知名度なし**: 地域での政治活動基盤がない
- **情報不足**: 公開情報が限定的
