# 北野谷富子（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年4月 |
| 年齢 | 40歳 |
| 選挙区 | 静岡7区（浜松市天竜区、湖西市等） |
| 当選回数 | 衆議院なし（新人）、浜松市議3期 |
| 主な役職 | 国民民主党静岡県第7区総支部長、浜松市議会市民クラブ会派会長 |

## 経歴
- 静岡県浜松市出身
- 浜松商業高校卒業
- 民間企業勤務
- 2015年浜松市議会議員選挙で初当選
- 2019年浜松市議再選（2期目）
- 2023年浜松市議再選（3期目）
- 2024年4月浜松市議会市民クラブ会派会長
- 2025年国民民主党入党、静岡7区総支部長就任
- 北ロータリークラブ入会

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 18歳までの医療費無償化、ガソリン暫定税率撤廃など国民民主党の政策に沿った課題を設定。市議としての経験から地域課題への理解はあるが、国政レベルでの独自の課題設定は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 浜松市議3期の経験から地方行政の実務には精通。会派会長として組織運営も担う。ただし国政レベルの政策設計能力は未知数。

### ③ 実行力: 2/5
**評価根拠**: 市議3期連続当選は評価できるが、国政は未経験。静岡7区は城内実が7期連続当選の強固な地盤であり、新人が勝つのは極めて困難。国民民主党の追い風があっても厳しい選挙区。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで政策・活動を発信。市議としての活動報告も行っている。国民民主党の政策を分かりやすく説明する姿勢は見られる。

### ⑤ 一貫性: 3/5
**評価根拠**: 2025年に国民民主党入党と政党所属は新しい。市議としては一貫して地元浜松での活動を継続。「地方から声を上げる」という姿勢は一貫している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。市議としての活動において倫理的な問題は見られない。

## 強み・弱み

### 強み
- **地方議員経験**: 浜松市議3期、地域への理解が深い
- **組織力**: 会派会長として組織運営経験
- **国民民主党の追い風**: 同党が注目される中での出馬
- **地元出身**: 浜松市生まれ育ちで地縁がある

### 弱み（注意点）
- **国政経験なし**: 衆議院議員としての経験・実績がない
- **選挙区事情**: 静岡7区は城内実が7期連続当選で圧倒的に強い
- **党歴が浅い**: 2025年に国民民主党入党、党内基盤は薄い
- **知名度**: 浜松市以外での知名度は低い
