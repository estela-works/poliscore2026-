# 城内実（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年4月19日 |
| 年齢 | 60歳 |
| 選挙区 | 静岡7区（浜松市天竜区、湖西市等） |
| 当選回数 | 衆議院7回 |
| 主な役職 | 経済財政政策担当大臣、規制改革担当大臣、経済安全保障担当大臣 |

## 経歴
- 東京大学教養学部卒業
- 外務省入省、外交官として勤務
- 在ドイツ日本大使館赴任、ドイツ語通訳（天皇陛下・総理大臣）
- 2002年自民党静岡7区支部長（公募選任）
- 2003年衆院選で初当選（無所属）
- 2005年郵政民営化法案に反対票、刺客（片山さつき）に748票差で敗北
- 2012年自民党に復党、3選、外務大臣政務官
- 外務委員長、環境副大臣、外務副大臣歴任
- 北朝鮮拉致問題特別委員会委員長
- 2024年石破内閣で経済安保担当大臣、7選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 外交官出身として国際情勢、外交安全保障に精通。経済安全保障担当大臣として経済と安全保障の交差領域を課題として取り組む。外交・安保分野での課題設定能力は高い。

### ② 解決設計力: 4/5
**評価根拠**: 外務省での実務経験、外務政務官・副大臣・委員長として政策設計に関与。経済財政政策担当大臣として現在も政策設計の中枢にいる。ドイツ語堪能な国際派として実務能力は高い。

### ③ 実行力: 5/5
**評価根拠**: 7期連続当選、2024年選挙では約6万票差の圧勝。「城内党」と呼ばれる強固な後援会を構築。2005年に郵政反対で落選するも、自力で復活し大臣まで上り詰めた実行力は特筆に値する。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。大臣として記者会見等での説明も行う。ただし2005年の郵政反対時の経緯や、その後の復党については評価が分かれる面もある。

### ⑤ 一貫性: 4/5
**評価根拠**: 2005年の郵政民営化反対は信念に基づく行動で、所属派閥（清和会）で唯一の反対票。一時離党を経験したが、外交・安保重視の政策軸は一貫。「城内党」としての独自路線を貫く。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。郵政反対で不利益を被っても信念を曲げなかった姿勢は評価できる。長年の議員活動でクリーンな政治家像を維持。

## 強み・弱み

### 強み
- **圧倒的な選挙の強さ**: 7期連続当選、「城内党」と呼ばれる強固な地盤
- **外交の専門性**: 外交官出身、ドイツ語通訳、外務副大臣経験
- **信念の政治家**: 郵政反対で落選しても自力復活、大臣に
- **閣僚経験**: 経済安保相など複数の大臣ポスト

### 弱み（注意点）
- **党との関係**: 過去に郵政反対で離党、復党の経緯あり
- **派閥色の薄さ**: 「城内党」として独自路線、党内での位置づけが独特
- **専門の偏り**: 外交・安保以外の分野での存在感は相対的に薄い
