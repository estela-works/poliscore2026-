# 源馬謙太郎（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1972年12月21日 |
| 年齢 | 51歳（2024年衆院選時） |
| 選挙区 | 静岡8区（浜松市中央区西部・浜名区） |
| 当選回数 | 衆議院3回（県議2期） |
| 主な役職 | 衆議院議員、立憲民主党国際局長、立憲民主党静岡県連代表 |

## 経歴
- 1972年12月21日：浜松市舞阪町生まれ（水産食品会社「源馬」創業家）
- 静岡大学教育学部附属浜松小学校・中学校卒業
- 国際基督教大学高等学校卒業
- 1996年3月：成蹊大学法学部政治学科卒業
- Centre College（ケンタッキー州）で国際関係学を学ぶ
- 2000年6月：アメリカン大学大学院修士課程修了（国際平和と紛争解決学）
- 2001年：紛争予防NGO勤務後、外務省より小型武器対策専門家としてカンボジア派遣
- 2003年：日本小型武器対策支援チーム（JSAC）設立、プロジェクトマネージャとして2年間で1万丁以上の武器回収
- カンボジア王国勲章3回受章
- 2005年：松下政経塾入塾（第26期生）
- 2007年：静岡県議会議員選挙で初当選（浜松市東区、28,715票で県内最多得票）
- 2011年：県議2期連続トップ当選
- 2012年、2014年：衆院選で次点落選
- 2017年10月：第48回衆院選で初当選（希望の党、比例復活）
- 2018年5月：国民民主党参加
- 2020年9月：立憲民主党参加
- 2021年：第49回衆院選で2選（小選挙区当選）
- 2024年10月：第50回衆院選で3選（小選挙区当選）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 国際平和、紛争解決、人権外交、拉致問題など国際分野で明確な課題設定。カンボジアでの小型武器回収プロジェクトの経験を活かし、具体的な課題を提起している。LGBTQの権利、選択的夫婦別姓など多様性に関する課題にも取り組む。

### ② 解決設計力: 4/5
**評価根拠**: アメリカン大学大学院で国際平和・紛争解決学の修士号取得。外務省専門家として日本初の包括的小型武器対策ODAプロジェクトを策定した実績。松下政経塾での政策立案訓練も受けている。国会での質問100回以上、法案提出29本の実績。

### ③ 実行力: 4/5
**評価根拠**: 県議2期連続トップ当選、衆院選では2度の落選を経て3期連続当選を果たした。カンボジアでは2年間で1万丁以上の武器を回収し、3度の勲章を受章。「三ツ星国会議員」（465人中17人、静岡県内唯一）に選出されるなど国会活動も高評価。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 国会質問100回近く、法案提出29本と積極的な議会活動。同性カップルの国勢調査問題など具体的なテーマで議論を展開。公式サイト、SNSでの情報発信も活発。「三ツ星国会議員」の評価は説明責任の高さを示す。

### ⑤ 一貫性: 3/5
**評価根拠**: 国際平和、人権外交への取り組みは一貫している。ただし政党は希望の党→国民民主党→立憲民主党と変遷しており、政治的立場の変化がある。野田佳彦代表選での推薦人を務めるなど現在は立憲民主党で中心的役割。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。政治とカネの問題を追及する姿勢を示している。カンボジアでの武器回収など人道的活動の実績も倫理観の高さを示す。

## 強み・弱み

### 強み
- **国際経験**：カンボジアでの武器回収プロジェクト、3度の勲章受章
- **高い学歴**：アメリカン大学大学院修士、松下政経塾26期
- **国会活動の実績**：質問100回、法案29本、三ツ星国会議員
- **選挙の強さ**：県議トップ当選、衆院選3期連続当選
- **政策の専門性**：国際平和、紛争解決、人権外交

### 弱み（注意点）
- **政党の変遷**：希望→国民民主→立憲と政党が変わっている
- **比例復活からのスタート**：初当選時は比例復活
- **野党議員**：政権与党ではないため政策実現に限界がある
