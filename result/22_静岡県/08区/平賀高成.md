# 平賀高成（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1954年5月7日 |
| 年齢 | 70歳 |
| 選挙区 | 静岡8区（浜松市中央区西部・浜名区） |
| 当選回数 | 衆議院1回、県議1回 |
| 主な役職 | 元衆議院議員、元静岡県議会議員、元運輸委員 |

## 経歴
- 1954年5月7日：静岡県生まれ
- 1976年：静岡大学工業短期大学部卒業
- 静岡大学教職員組合中央執行委員
- 1996年：第41回衆院選で静岡8区から出馬、小選挙区落選も比例東海ブロックで初当選
- 衆議院運輸委員、災害対策特別委員を歴任
- 2000年：第42回衆院選で落選
- 2003年：第43回衆院選で落選
- 2007年：参院選出馬、落選
- 2009年：参院補選出馬、落選
- 2015年：統一地方選で静岡県議会議員選挙（浜松市中区）に当選
- 2019年：県議選で落選
- 2021年：第49回衆院選で立候補予定も野党間候補者調整により取り下げ
- 2023年：県議選（浜松市中区）で落選
- 2024年10月：第50回衆院選で静岡8区から出馬、落選（得票率5.35%）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 日本共産党の政策（消費税減税、大企業課税強化、憲法護持）に沿った課題設定。衆議院運輸委員、災害対策特別委員としての経験から、交通・防災分野での課題認識を持つ。ただし独自の構造的課題提起は確認できない。

### ② 解決設計力: 3/5
**評価根拠**: 静岡大学工業短期大学部卒、教職員組合での活動経験。衆議院議員として運輸委員、災害対策特別委員を務めた実績がある。ただし国会議員在職は1期のみで、政策設計の実績は限定的。

### ③ 実行力: 2/5
**評価根拠**: 1996年に衆院議員に当選したが、2000年以降は衆院選、参院選、県議選で多くの落選を経験。2015年に県議当選も2019年で落選。2024年衆院選でも得票率5.35%で落選。長年の活動にも関わらず、選挙での結果を出せていない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 長年の政治活動で有権者への説明は継続的に行っている。衆議院議員時代の委員会活動の経験もある。ただし選挙での得票率の低さは、有権者への訴求力に課題があることを示す。

### ⑤ 一貫性: 4/5
**評価根拠**: 日本共産党で長年活動を続けており、政党・政策への一貫性は高い。1996年以来、静岡8区を中心に繰り返し立候補を続けている。多くの落選を経験しても諦めない姿勢は評価できる。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。長年の党活動を通じて党の規律に従って活動している。

## 強み・弱み

### 強み
- **国会経験**：衆議院議員1期、運輸委員・災害対策特別委員の経験
- **地方議会経験**：静岡県議会議員1期
- **一貫性**：日本共産党で長年活動、何度も立候補を続ける粘り強さ
- **地元での活動**：静岡8区で長年活動

### 弱み（注意点）
- **選挙での弱さ**：2000年以降、多くの選挙で落選
- **得票率の低さ**：2024年衆院選で5.35%
- **年齢**：70歳と高齢
- **国会議員経験の古さ**：衆院議員在職は1996-2000年の1期のみ
