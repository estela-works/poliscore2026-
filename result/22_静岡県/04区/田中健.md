# 田中健（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年7月18日 |
| 年齢 | 48歳 |
| 選挙区 | 静岡4区（富士市・富士宮市・静岡市清水区の一部） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 国民民主党政務調査会会長代理、党税制調査会副事務局長、党静岡県連代表 |

## 経歴
- 静岡県庵原郡富士川町出身
- 静岡県立富士高等学校卒業
- 青山学院大学経済学部卒業
- 第一勧業銀行入行、みずほ銀行勤務
- 2003年：大田区議会議員選挙で初当選
- 2009年：東京都議会議員選挙で当選（2期）
- 細野豪志衆議院議員の秘書を務める
- 2016年：民主党から静岡4区公認内定、故郷静岡で政治活動開始
- 2017年：希望の党から衆院選出馬、落選
- 2020年：衆議院静岡4区補欠選挙、野党統一候補として出馬するも落選
- 2021年：国民民主党から出馬、比例復活で初当選
- 2024年：第50回衆院選で深澤陽一を2,647票差で破り2選（初の小選挙区当選）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「103万円の壁」「ブラケットクリープ」など税制の構造的問題を的確に指摘。暗号資産・Web3政策、旧優生保護法被害者補償、オンラインカジノ対策など幅広い分野で具体的な課題を設定。国民民主党の看板政策「手取りを増やす」を推進。

### ② 解決設計力: 4/5
**評価根拠**: 所得税の基礎控除引き上げを「経済政策」として位置づけ、消費活性化→持続的賃上げという因果関係を論理的に設計。銀行員・都議会議員の経験を活かした政策設計能力がある。暗号資産税制では20%申告分離課税・損失繰越控除の具体的提案を行う。

### ③ 実行力: 3/5
**評価根拠**: 2024年選挙で初めて小選挙区当選を果たし、地盤固めに成功。ただし2017年、2020年、2021年と3回連続で小選挙区で敗北（うち1回落選）した過去がある。野党議員として政策実現には限界があるが、国会質疑で着実に発言を重ねている。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 本会議での財政演説質疑、所得税法改正案質疑など国会で積極的に発言。「103万円を178万円まで上げる」など政策をわかりやすく説明。Xでの情報発信も活発。選挙戦では玉木代表との連携で党の政策を浸透させた。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→希望の党→無所属→国民民主党と所属政党の変遷がある。ただし「手取りを増やす」「税制改革」という政策軸は維持。静岡4区での活動は2016年から一貫して継続しており、地域への根付きは評価できる。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。政治資金規正法改正で「裏金問題の真相究明」を求める姿勢は評価できる。野党議員として政治とカネの問題を一貫して批判している。

## 強み・弱み

### 強み
- **税制政策の専門性**：銀行員経験を活かした税制・財政政策への深い知見
- **粘り強さ**：3回の小選挙区敗北を経て初勝利を果たした精神力
- **党内での地位**：政務調査会会長代理、静岡県連代表として党の政策立案に関与

### 弱み（注意点）
- **政党変遷**：民主党→希望の党→国民民主党と所属が変わっている
- **野党の限界**：政権与党でないため政策実現力に限界
- **選挙基盤**：小選挙区での勝利は2024年が初めてで、地盤は盤石とは言えない
