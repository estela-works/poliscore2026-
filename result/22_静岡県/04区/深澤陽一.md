# 深澤陽一（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年6月21日 |
| 年齢 | 49歳 |
| 選挙区 | 静岡4区（静岡市清水区・富士宮市・富士市の一部） |
| 当選回数 | 衆議院3回 |
| 主な役職 | 自民党文部科学部会長兼副幹事長、元外務大臣政務官、元厚生労働大臣政務官 |

## 経歴
- 静岡市清水区興津出身
- 静岡県立清水東高等学校卒業
- 信州大学工学部生産システム工学科卒業
- 衆議院議員・原田昇左右秘書
- 衆議院議員・原田令嗣秘書
- 2005年：静岡市議会議員選挙で初当選（2期）
- 2011年：静岡県議会議員選挙で当選（3期）
- 2020年1月：自民党静岡4区支部長に選任（望月義夫死去に伴う公募を経て）
- 2020年4月：衆議院静岡4区補欠選挙で初当選
- 宏池会（岸田派）入会、有隣会（谷垣グループ）にも加入
- 2021年：厚生労働大臣政務官
- 2023年：外務大臣政務官
- 2024年：衆院選で田中健に敗れ比例復活（3選）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 静岡市議・県議として地方政治の経験は豊富だが、国政における独自の課題設定は確認できない。厚労・外務の政務官として省庁の方針に沿った活動が中心で、構造的な問題提起を行った形跡が乏しい。

### ② 解決設計力: 2/5
**評価根拠**: 政務官として行政執行に携わったが、独自の政策設計や制度改革の実績は確認できない。工学部出身の専門性を政策に活かした事例も見られない。

### ③ 実行力: 3/5
**評価根拠**: 静岡市議2期、県議3期、衆議院議員3期と着実に政治キャリアを積み上げた。2020年補欠選挙、2021年衆院選で連続当選。ただし2024年選挙では小選挙区で敗北し比例復活にとどまった。政務官2度就任は実績として評価できる。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2022年の旧統一教会との関係についてのアンケートで回答を拒否。その後、推薦確認書への署名を認めたが、積極的な説明とは言い難い。有権者への説明責任に課題がある。

### ⑤ 一貫性: 3/5
**評価根拠**: 原田昇左右・原田令嗣秘書から地方議員を経て国会議員へと、自民党内で一貫したキャリアを歩んでいる。政策面では選択的夫婦別姓反対の立場を維持するなど保守的姿勢は一貫。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 複数の問題が指摘されている。①旧統一教会の推薦確認書に署名、②アンケート回答拒否で説明責任を果たさず、③政務官在任中に「大臣規範」で自粛を求められている不動産取引（約1000万円）を行った。透明性に課題がある。

## 強み・弱み

### 強み
- **地方政治の経験**：市議・県議として合計約15年の地方政治経験
- **政務官経験**：厚労・外務の政務官として行政経験を積んだ
- **組織力**：自民党静岡4区の地盤を継承、後援会組織がある

### 弱み（注意点）
- **旧統一教会問題**：推薦確認書への署名、アンケート回答拒否
- **規範違反**：政務官在任中の不動産取引（大臣規範違反）
- **選挙基盤の弱体化**：2024年選挙で小選挙区敗北、比例復活にとどまる
- **政策発信の弱さ**：独自の政策提案が見えにくい
