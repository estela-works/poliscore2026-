# 中村正善（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年3月6日 |
| 年齢 | 67歳 |
| 選挙区 | 静岡県5区 |
| 当選回数 | 新人（初挑戦） |
| 主な役職 | 株式会社JINRIKI創業者・発明家 |

## 経歴
- 東京都出身
- 東京写真大学（現東京工芸大学）写真技術科中退
- 株式会社アコム（審査部・情報システム部）勤務
- 2011年 東日本大震災を契機に退職、車いすけん引装置の開発に着手
- 2012年 株式会社JINRIKI設立、代表取締役就任
- 世界初の「着脱けん引式車いす補助装置JINRIKI」を商品化
- PCT（国際特許）取得、日本発明大賞など多数受賞
- 2022年 ウクライナに2度渡航、高齢者・障害者避難支援（535台のJINRIKI寄付）
- 2024年 能登半島地震後も現地支援活動
- 2024年8月 株式会社JINRIKI事業停止（安全基準への見解の相違により）
- 2026年 中道改革連合から静岡5区に立候補

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
東日本大震災を契機に「車いす利用者の災害時避難」という見落とされがちな社会課題を独自に発見・言語化した。「災害で無くなる方を0にする」という明確なビジョンを持ち、高齢者・障害者という声を上げにくい層の課題に着目している点は、課題設定能力として高く評価できる。

### ② 解決設計力: 4/5
**評価根拠**:
JINRIKIという具体的な解決策を「てこの原理」という物理法則に基づいて設計し、特許取得・商品化まで実現した。ウクライナ支援では製品の大量生産・輸送という複雑な物流設計も行った。ただし、政策としての設計（法制度、予算など）は政治活動を始めたばかりで未知数。

### ③ 実行力: 2/5
**評価根拠**:
政治家としての実績はゼロ（新人）。ただし、起業家としてはJINRIKIの商品化、クラウドファンディングで3,800万円調達、535台のウクライナ寄付など具体的成果を上げている。しかし、2024年に事業停止となった点、政治の世界での実行力は未知数である点を考慮。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**:
クラウドファンディングでは支援者への進捗報告を丁寧に行っている。メディア取材（東洋経済、PRESIDENTなど）への対応実績もある。しかし、政治家としての説明責任（政策説明、質疑対応など）は実績がなく評価材料不足。

### ⑤ 一貫性: 4/5
**評価根拠**:
2011年の東日本大震災以降、10年以上にわたり一貫して「災害時の避難困難者支援」に取り組んできた。ウクライナ、能登半島と災害発生のたびに支援活動を継続。会社利益より安全基準を優先して事業停止を選んだ点も、理念への一貫性を示している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**:
2024年の事業停止は、総販売元との安全基準に関する見解の相違が原因。中村氏は自身が納得できない製品の販売を拒否し、事業継続より安全性を優先した。ウクライナ支援でも利益を求めず社会貢献に注力。権力や利益より社会的責任を重視する姿勢が見られる。

## 強み・弱み

### 強み
- **明確な専門性**: 防災・避難支援分野で10年以上の経験と実績
- **国際的な社会貢献**: ウクライナ支援で外務省・JICAとも連携
- **発明家としての技術力**: 特許取得、日本発明大賞受賞

### 弱み（注意点）
- **政治経験ゼロ**: 議会運営、政策立案の経験がない
- **事業の挫折経験**: 2024年にJINRIKI事業停止（理由は安全基準だが、経営面での課題も示唆）
- **知名度不足**: 発明家としては知られるが、政治家としての認知度は低い
