# 細野豪志（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1971年8月21日（53歳） |
| 選挙区 | 静岡5区 |
| 当選回数 | 衆議院9期 |
| 主な役職 | 自民党政務調査会長代理、元環境大臣、元原発事故担当大臣、元民主党幹事長 |

## 経歴
- 滋賀県出身
- 滋賀県立彦根東高校卒業
- 京都大学法学部卒業
- 三和総合研究所研究員
- 1999年 静岡県三島市で政治活動開始（落下傘候補）
- 2000年 衆院選初当選（民主党、28歳）
- 2011年 総理大臣補佐官、原発事故担当大臣
- 2012年 環境大臣（野田内閣）
- 2014年 民主党幹事長
- 2017年 希望の党結党に参加
- 2019年 自民党二階派に特別会員として入会
- 2021年 自民党入党
- 2024年10月 9選（約14万票、小選挙区全国2位）
- 2024年11月 自民党政調会長代理就任

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 原発事故対応、エネルギー政策、安全保障政策、子どもの貧困問題など、国家的課題を明確に設定。原発事故担当大臣として危機管理の最前線に立った経験。

### ② 解決設計力: 5/5
**評価根拠**: 京大法学部卒、シンクタンク出身の政策立案能力。環境大臣、党政調会長として政策設計を主導。エネルギー政策では原発・再エネ・CCUSのバランスを考慮した現実的設計。

### ③ 実行力: 5/5
**評価根拠**: 9期連続当選、2024年は14万票超で小選挙区全国2位の圧勝。環境大臣、幹事長など要職を歴任。自民党入党という困難な決断を実行し、党内で政調会長代理に就任。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 弁の立つ論客として知られ、メディア対応も巧み。ただし政党変遷の説明は「安全保障政策の違い」としているが、十分な納得を得られていない面も。

### ⑤ 一貫性: 1/5
**評価根拠**: 民主党→希望の党→無所属→自民党二階派と政党を転々。「変節漢」「カメレオン政治家」との批判。かつて二大政党制を志向し民主党幹事長まで務めながら自民党入り。政策の軸が見えにくい。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2006年フリーアナウンサーとの不倫スキャンダル（モナ騒動）。権力志向が強く、与党入りのためなら過去の主張を覆す姿勢に疑問の声。

## 強み・弱み

### 強み
- **選挙の強さ**: 9期連続当選、圧倒的得票力
- **政策専門性**: エネルギー・環境・安全保障に精通
- **実務経験**: 閣僚・党幹部として豊富な経験
- **弁論力**: メディア対応に長けた論客

### 弱み（注意点）
- **変節批判**: 民主党→自民党の転身に批判
- **モナ騒動**: 不倫スキャンダルの過去
- **信頼性**: 一貫性の欠如が信頼を損なう
- **党内基盤**: 自民党静岡県連は入党に反対していた
