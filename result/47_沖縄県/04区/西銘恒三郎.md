# 西銘恒三郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1954年8月7日（70歳） |
| 選挙区 | 沖縄4区 |
| 当選回数 | 衆議院7期 |
| 主な役職 | 自民党幹事長代理、復興大臣（第12・13代）、沖縄担当大臣、経済産業副大臣、総務副大臣 |

## 経歴
- 沖縄県島尻郡知念村（現・南城市）出身
- 父は西銘順治・元沖縄県知事（元衆議院議員）
- 沖縄県立那覇高等学校卒業
- 上智大学経済学部経営学科卒業（1979年）
- 沖縄振興開発金融公庫入社
- 父・西銘順治の秘書
- 沖縄県議会議員（4期）
- 2003年 衆院選沖縄4区で初当選
- 国土交通大臣政務官（福田改造内閣・麻生内閣）
- 総務副大臣（第2次安倍改造内閣）
- 経済産業副大臣（第3次安倍第3次改造内閣）
- 2021年 復興大臣・沖縄担当大臣（岸田内閣）
- 2023年9月 自民党幹事長代理
- 2024年1月 茂木派を退会（裏金問題を受けけじめ）
- 2024年10月 7選

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 復興大臣として東日本大震災復興、沖縄担当大臣として沖縄振興という国家的課題に取り組んだ。「アジア地域で戦争を起こさない」という平和の重要性を経済発展の前提として位置づける視点。

### ② 解決設計力: 5/5
**評価根拠**: 上智大学経済学部卒、沖縄振興開発金融公庫での実務経験。県議4期、衆議院7期の豊富な政治経験。復興大臣、沖縄担当大臣、経産副大臣、総務副大臣と多分野で政策立案に関与。

### ③ 実行力: 5/5
**評価根拠**: 7期連続当選という圧倒的な選挙力。復興大臣、沖縄担当大臣として政策を実行。自民党幹事長代理として党運営にも関与。2024年衆院選では61,289票を獲得し完勝。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 大臣経験者として国会答弁、記者会見の経験豊富。裏金問題に際し「わかりやすく、けじめを付ける」として派閥離脱を決断した姿勢は評価できる。

### ⑤ 一貫性: 4/5
**評価根拠**: 沖縄振興、復興支援という政策軸で一貫。自民党での活動を継続。ただし派閥離脱は政治的判断とも言える。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 直接的なスキャンダルの報道なし。裏金問題を受けて自ら派閥を離脱し「けじめ」をつけた姿勢。ただし世襲議員としての特権的立場は存在。

## 強み・弱み

### 強み
- **閣僚経験**: 復興大臣、沖縄担当大臣など複数の重要ポスト歴任
- **選挙の強さ**: 7期連続当選
- **地元基盤**: 父・西銘順治からの継承、沖縄での強固な支持
- **党内地位**: 幹事長代理という要職

### 弱み（注意点）
- **世襲議員**: 父からの地盤継承
- **派閥政治**: 茂木派所属→離脱の経緯
- **高齢**: 70歳で7期目
