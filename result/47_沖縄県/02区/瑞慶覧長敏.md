# 瑞慶覧長敏（社会民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年10月24日 |
| 年齢 | 67歳 |
| 選挙区 | 沖縄2区 |
| 当選回数 | 衆議院1回、南城市長1回 |
| 主な役職 | 元南城市長、元衆議院議員 |

## 経歴
- 琉球政府島尻郡大里村（現・南城市）出身
- ワシントン州立セントラル・ワシントン大学経済学部卒業
- 琉球大学法文学部経済学科卒業（編入）
- 大阪府・沖縄県で英語教育に従事
- 2009年衆院選で沖縄4区から民主党公認で初当選
- 2012年消費増税反対で民主党除名、落選
- 2018年南城市長当選（オール沖縄支援）
- 2022年南城市長選で落選
- 2026年社民党入党、沖縄2区から立候補

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 南城市長として「市民の声でまちづくり」「子育て支援」「福祉のまち」など地域課題を設定。辺野古新基地反対、SDGs推進など幅広い課題を掲げている。ただし、南城市には米軍基地がなく、基地問題の当事者としての独自の問題提起は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 南城市長として8つの基本政策を策定し、子育て支援や観光振興に取り組んだ。市長在任中の具体的な成果は限定的だが、政策立案の経験はある。衆院議員時代の議員立法実績は確認できない。

### ③ 実行力: 3/5
**評価根拠**: 2009年衆院選で自民前職を破り当選、2018年南城市長選でも65票差で現職を破る接戦を制した選挙実績あり。一方、2012年衆院選、2022年市長選と落選も経験。市長として1期4年の行政経験があるが、大きな政策成果は確認できない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 2012年の消費増税反対では党の方針に反して反対票を投じ、自らの信念を明確に示した。出馬会見でも「沖縄の声を国会に届ける人がいない」と危機感を語るなど、説明姿勢は見られる。ただし、オール沖縄勢力との調整不足で分裂選挙を招いた点は課題。

### ⑤ 一貫性: 4/5
**評価根拠**: 2012年に消費増税反対で除名されてでも信念を貫いた姿勢、辺野古新基地反対の立場を一貫して維持。オール沖縄・リベラル派の旗印を継続。党籍は民主党→無所属→社民党と変遷しているが、政策の軸は概ね一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや汚職は確認されていない。市長選で65票差の僅差で当選した際も選挙違反の指摘はなし。党の方針より自らの信念を優先した2012年の行動は、政治家としての誠実さを示している。

## 強み・弱み

### 強み
- **豊富な政治経験**: 衆議院議員1期、市長1期の経験
- **一貫した政策軸**: 辺野古反対、リベラル派として一貫
- **地元への理解**: 南城市出身、市長経験を通じた地域密着

### 弱み（注意点）
- **分裂選挙の当事者**: オール沖縄との調整不足で2区に辺野古反対派が2人立候補
- **落選経験**: 2012年衆院選、2022年市長選と直近で2度落選
- **市長実績の限定性**: 1期で再選されず、大きな成果を残せなかった
- **オール沖縄からの批判**: 「当選もしない人をぶつけるべきではない」との批判
