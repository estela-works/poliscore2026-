# 新垣邦男（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年6月19日 |
| 年齢 | 69歳 |
| 選挙区 | 沖縄2区 |
| 当選回数 | 衆議院2回、北中城村長4回 |
| 主な役職 | 元社民党副党首、元国対委員長、元政策審議会長、元沖縄県町村会長 |

## 経歴
- 那覇市出身
- 日本大学法学部卒業
- 1985年北中城村役場採用
- 2004年北中城村長に初当選（以後4期）
- 2021年衆院選で照屋寛徳の後継として社民党公認で初当選
- 2022年社民党副党首就任
- 2024年衆院選で再選
- 2025年11月社民党離党、中道改革連合に参加

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 沖縄の基地問題、PFAS（有機フッ素化合物）問題、辺野古新基地建設問題など、地域固有の課題を構造的に捉え国政で提起してきた。村長4期の経験から、安全保障と住民生活の両立という困難な課題を具体的に言語化できる。質問主意書で辺野古問題を追及するなど、課題設定力は高い。

### ② 解決設計力: 3/5
**評価根拠**: 村長として財政健全化や住民サービス改善の実績あり。基地問題では「影響評価や補償、生活再建策をセットで検討する姿勢」を示している。ただし、野党議員として法案成立に至った実績は確認できず、解決策の実装力には制約がある。

### ③ 実行力: 4/5
**評価根拠**: 村長4期16年の行政実績、衆院選2連勝、社民党の「顔」として党を支えた実績は評価できる。沖縄2区で自民候補を2度破る選挙戦での実行力も顕著。ただし国会議員としての法案成立実績は野党の構造的制約もあり限定的。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 離党会見では理由を明確に説明し、「党勢拡大に限界を感じた」「党首と意見がかみ合わなかった」と具体的に語った。ただし離党届の提出方法を誤り（郵送で無効）混乱を招いた点、党内での事前調整不足は説明責任の観点で課題。

### ⑤ 一貫性: 2/5
**評価根拠**: 村長時代から基地問題への取り組みは一貫しているが、社民党副党首でありながら離党した点は大きな方針転換。「沖縄を二度と戦場にさせない」という主張自体は継続しているものの、党を離れて中道改革連合（立民・公明系）に参加した行動は、有権者との「契約」を反故にしたとの批判も受けている。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 個人としてのスキャンダルや汚職はない。しかし、社民党の「顔」として副党首を務めながら離党し、党を衆院議席ゼロに追い込んだ行為は、組織への責任という観点で疑問が残る。社民党は「沖縄におけるそうした社民党の伝統と闘いの歴史を踏みにじる行為」と批判。

## 強み・弱み

### 強み
- **豊富な行政経験**: 村長4期16年の実績、自治体運営の知見
- **沖縄問題の専門性**: 基地問題、PFAS問題など地域課題への深い理解
- **選挙での強さ**: 自民候補を2度破った沖縄2区での実績

### 弱み（注意点）
- **政党移籍の不安定さ**: 社民党副党首から離党し中道改革連合へ、一貫性に疑問
- **党への責任放棄**: 社民党を衆院議席ゼロに追い込んだ離党は批判の対象
- **野党としての限界**: 法案成立など国政での具体的成果は限定的
