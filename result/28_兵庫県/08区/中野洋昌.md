# 中野洋昌（公明党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1978年1月4日（46歳） |
| 選挙区 | 兵庫8区 |
| 当選回数 | 衆議院5期 |
| 主な役職 | 国土交通大臣（第二次石破内閣）、元経済産業大臣政務官 |

## 経歴
- 京都市出身
- 関西創価小学校・中学校・高校卒業
- 東京大学教養学部卒業（2001年）
- 国土交通省入省
- 米国コロンビア大学国際公共政策大学院修士号（2006年）
- 新潟県中越沖地震・東日本大震災復興に従事
- 非接触型IC乗車カード企画・提案
- 冬柴鐵三国交大臣の下で緊急地震速報導入
- 2012年 衆院選初当選（兵庫8区）
- 2019年 経済産業大臣政務官
- 2024年10月 5選
- 2024年11月 国土交通大臣就任（1970年代生まれ初）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 国土交通省出身で防災・交通・復興の課題に精通。東日本大震災復興、緊急地震速報導入など具体的課題に取り組んだ実績。

### ② 解決設計力: 5/5
**評価根拠**: 東大教養学部卒、コロンビア大学院修士。国交省キャリア官僚として政策設計の専門性。IC乗車カード企画など実務的政策設計経験。

### ③ 実行力: 5/5
**評価根拠**: 5期連続当選、維新との激戦を勝ち抜く選挙力。国土交通大臣に抜擢され、1970年代生まれ初の国交大臣として入閣。政務官経験も。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 政策に精通した官僚出身で説明能力は高い。ただし大臣就任後の発信力はこれから評価される段階。

### ⑤ 一貫性: 5/5
**評価根拠**: 公明党一筋で政策も一貫。選択的夫婦別姓賛成の立場を継続。冬柴鐵三の後継として兵庫8区を守り続ける。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道なし。官僚出身のクリーンな経歴。公明党の連立離脱後も閣僚として責任を果たした。

## 強み・弱み

### 強み
- **閣僚経験**: 国土交通大臣として入閣
- **政策専門性**: 防災・交通・復興に精通
- **学歴**: 東大・コロンビア大学院
- **選挙の強さ**: 維新との激戦を5連勝
- **若さ**: 46歳で最年少閣僚

### 弱み（注意点）
- **連立政党**: 公明党の連立離脱の影響
- **地盤**: 冬柴時代からの引継ぎで独自色薄い
- **知名度**: 全国的認知度は閣僚就任まで低かった
