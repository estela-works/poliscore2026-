# 山寺崇仁（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1972年または1973年生まれ（詳細不明） |
| 年齢 | 53歳 |
| 選挙区 | 兵庫県9区 |
| 当選回数 | 新人（国政初挑戦） |
| 主な役職 | リスクコンサルタント、防災支援スペシャリスト |

## 経歴
- リスクコンサルタントとして活動
- 国内外の財物の火災・爆発リスク、地震・水害等の自然災害リスクの分析・評価
- 事業継続計画（BCP）策定支援
- 現場サーベイ実績500件以上
- 2026年1月 参政党公認で兵庫9区から衆院選出馬

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**:
リスクコンサルタントとして、災害リスク・事業継続の課題に専門的に取り組んできた経験がある。現場サーベイ500件以上という実績は、具体的な課題を把握する力を示唆する。ただし、政治的課題への取り組みは公開情報から確認できず、参政党の政策に沿った主張が中心と推測される。

### ② 解決設計力: 3/5
**評価根拠**:
BCP策定支援やリスク低減策の提案を専門としており、ビジネス・防災領域での解決策設計の経験は豊富と推測される。ただし、政治的課題に対する具体的な政策設計は公開情報から確認できない。専門分野での設計力は評価できるが、政治分野への転用は未知数。

### ③ 実行力: 2/5
**評価根拠**:
国政・地方政治ともに選挙経験・当選実績なし。ただし、リスクコンサルタントとして500件以上の現場サーベイを実施し、防災支援のスペシャリストとして活躍してきた実績は、専門分野での実行力を示す。政治的実行力は未知数。新人として構造的に低評価。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**:
公開されている政治的発言や政策説明は限られている。コンサルタント業務ではクライアントへの説明・提案を行ってきたと推測されるが、有権者向けの情報発信は確認できない。SNSでの発信やメディア露出も見当たらない。情報不足。

### ⑤ 一貫性: 2/5
**評価根拠**:
政治活動歴が短く、一貫性を評価するための材料が乏しい。リスクコンサルタントから政治家への転身の経緯や動機も公開情報から確認できない。防災・リスク管理という専門性を政治にどう活かすかの説明も不明。情報不足。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**:
個人としてのスキャンダルは確認されていない。ただし、参政党全体として代表・神谷宗幣氏の秘書急死問題・パワハラ疑惑、外国人への排外主義的発言、反ワクチン的主張への批判が存在する。所属政党の姿勢との関係は有権者の判断材料となる。個人としては評価材料不足。

## 強み・弱み

### 強み
- **専門的知見**: 防災・リスク管理の専門家として500件以上の実績
- **実務経験**: BCP策定支援等の企業支援経験
- **地元基盤**: 明石市に事務所を構え活動
- **現場主義**: 現地調査に基づくリスク評価という実践的アプローチ

### 弱み（注意点）
- **政治経験なし**: 議会・選挙経験ゼロ
- **情報発信不足**: 公開情報が少なく政策主張が不明確
- **所属政党の問題**: 参政党の反ワクチン主張・排外主義的傾向への批判
- **知名度不足**: メディア露出・SNS発信が見られない
