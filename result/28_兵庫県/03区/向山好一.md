# 向山好一（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1957年7月18日 |
| 年齢 | 67歳 |
| 選挙区 | 兵庫県3区 |
| 当選回数 | 元職（衆議院2回、2012年落選後返り咲き） |
| 主な役職 | 国民民主党兵庫県連代表、兵庫民社協会会長、元神戸市議、元兵庫県議 |

## 経歴
- 兵庫県神戸市出身
- 兵庫県立神戸高等学校卒業
- 大阪大学経済学部卒業
- 大阪ガス入社
- 大阪ガス労働組合専従役員
- 連合大阪労働政策部長
- 1993年西村眞悟衆議院議員秘書
- 1999年神戸市議会議員初当選（2期）
- 2009年衆院選初当選（兵庫2区、赤羽一嘉に勝利）
- 2012年落選（赤羽に敗北）
- 2015年兵庫県議会議員当選
- 2017年民進党→2018年国民民主党兵庫県連代表
- 2023年県議選で落選
- **2024年衆院選で約12年ぶり返り咲き（比例復活）**

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 労働組合出身として労働者の課題認識を持つ。地方議員経験から地域課題も把握。ただし国政での独自の課題設定は限定的。国民民主党の政策に沿った活動。

### ② 解決設計力: 3/5
**評価根拠**: 大阪大学経済学部卒、大阪ガス・連合大阪での実務経験。神戸市議2期、県議2期、衆議院1期の経験。政策立案の実績はあるが、国政での大きな成果は限定的。

### ③ 実行力: 3/5
**評価根拠**: 2009年に赤羽一嘉を破り初当選も2012年に落選。2023年県議選でも落選。2024年に約12年ぶりの返り咲きは評価できるが、比例復活。選挙での実行力に課題も。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 県連代表として長年党を運営。労働組合出身でコミュニケーション能力は持つ。地域での地道な活動を継続。

### ⑤ 一貫性: 4/5
**評価根拠**: 民社党系→民主党→民進党→国民民主党と中道改革路線で一貫。県連代表として党組織を維持。落選後も活動を継続し返り咲きを果たした。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。労働組合出身として庶民的な視点。クリーンなイメージを維持。

## 強み・弱み

### 強み
- **返り咲き**: 12年ぶりの国政復帰
- **地方議員経験**: 市議2期、県議2期
- **労組基盤**: 大阪ガス労組、連合出身
- **継続性**: 落選後も活動継続
- **クリーン**: スキャンダルなし

### 弱み（注意点）
- **高齢**: 67歳
- **比例復活**: 選挙区では敗北
- **2度の落選**: 2012年衆院選、2023年県議選
- **知名度**: 全国的知名度は低い
