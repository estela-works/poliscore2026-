# 松本剛明（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年4月25日 |
| 年齢 | 65歳 |
| 選挙区 | 兵庫県11区 |
| 当選回数 | 衆議院9回 |
| 主な役職 | 元総務大臣、元外務大臣 |

## 経歴
- 東京都出身、武蔵高等学校卒業
- 東京大学法学部卒業
- 日本興業銀行入行（1982年）
- 2000年民主党から初当選
- 2011年外務大臣就任（菅直人内閣）
- 2015年民主党離党、2017年自民党入党
- 2022年・2023年総務大臣就任（岸田内閣）
- 2026年引退表明
- 高祖父は伊藤博文、父は松本十郎元防衛庁長官

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 民主党時代から外交・安全保障の政策通として知られ、独自の見解を持つ。外務大臣として東日本大震災後の「復興外交」を提唱し、従来の経済外交に加えて被災地支援という新たな視点を打ち出した。竹島問題での対応など、領土問題への明確な姿勢も示している。

### ② 解決設計力: 4/5
**評価根拠**: 民主党政策調査会長として党の政策決定を主導。外務大臣として福島県産農産物の風評被害に対し、ブラジル外相との直談判で証明書添付要求の変更・撤廃を引き出すなど、具体的な成果を上げた。総務大臣としても税制・情報通信・行政改革に精通している。

### ③ 実行力: 4/5
**評価根拠**: 9期連続当選の政治的実行力。外務大臣として竹島問題で韓国大使の面会を拒否する異例の強硬姿勢を実行。東日本大震災直後のG8外相会議、日中韓外相会議など重要な外交日程をこなした。ただし2024年選挙後、体調問題で引退を決断。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 国会での居眠り問題で「目が細いので」と苦しい弁明。政治資金パーティー問題（利益率96%）への説明も不十分。大臣規範違反（パーティー自粛中の開催）を指摘されても明確な回答なし。批判への対応力に課題。

### ⑤ 一貫性: 2/5
**評価根拠**: 2000年から民主党で活動し政策調査会長まで務めたが、2015年に安保法制・共産党との連携姿勢を理由に離党、2017年に自民党入党。政策的信念はあるが、「敵だった自民党に移籍」という変節は大きく、一貫性は低い。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 政治資金パーティーで会場収容人数を大幅に超えるパー券販売、利益率96%という問題。大臣規範で自粛とされる中でのパーティー開催。安倍派が家宅捜索を受けた直後の安倍派パーティー出席。複数の倫理的問題を抱える。

## 強み・弱み

### 強み
- **外交・安全保障の専門性**: 政策通として長年活動、大臣経験も豊富
- **名門の血統**: 伊藤博文の末裔という歴史的背景

### 弱み（注意点）
- **政治資金問題**: パーティー収入の異常な利益率、大臣規範違反
- **党籍変更**: 民主党→自民党への移籍で信頼性に疑問
