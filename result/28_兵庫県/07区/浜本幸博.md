# 浜本幸博（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年頃 |
| 年齢 | 60歳 |
| 選挙区 | 兵庫7区（西宮市南部・芦屋市） |
| 当選回数 | なし（新人） |
| 主な役職 | 参政党公認候補、株式会社オー・スリー代表取締役、カジュアルフレンチレストラン経営 |

## 経歴
- 1964年頃生まれ
- カジュアルフレンチレストラン経営
- 株式会社オー・スリー代表取締役
- コロナ禍で店舗休業中、全国の農家を訪問し食材を追究
- 有機栽培等を体験しながら食の安全性に危機感を抱く
- 2023年11月、参政党兵庫県連が次期衆院選公認候補として発表
- 2024年10月、第50回衆院選で兵庫7区から出馬、落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「欧州で使用禁止の農薬が国内で普通に売られている」という食の安全問題を課題として設定。コロナ禍での農家訪問経験から実体験に基づく問題意識。

### ② 解決設計力: 2/5
**評価根拠**: 飲食店経営者として食に関する実務知識。ただし政策立案や行政経験はなく、国政での政策設計能力は未知数。

### ③ 実行力: 1/5
**評価根拠**: 選挙での当選経験がなく、政治的実行力は評価できない段階。

### ④ 説明責任: 3/5
**評価根拠**: 「食から日本の政治を変えたい」という明確なメッセージ。コロナ禍での体験から政治を志した経緯を説明。

### ⑤ 一貫性: 2/5
**評価根拠**: 政治キャリアが始まったばかりで一貫性の評価は困難。食の安全という一貫したテーマは持つ。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。経営者としての実績あり。

## 強み・弱み

### 強み
- **食の専門性**: フレンチレストラン経営者として食に精通
- **実体験**: コロナ禍での農家訪問、有機栽培体験
- **経営者**: 株式会社代表取締役として事業経験
- **明確なテーマ**: 食の安全という具体的な政策課題
- **参政党支持層**: 食の安全・保守政策を求める有権者にアピール

### 弱み（注意点）
- **政治未経験**: 議会経験なし
- **激戦区**: 自民・維新・立憲の争いに埋没
- **党勢**: 参政党の組織基盤
- **知名度不足**: 有権者への浸透が課題
- **落選**: 2024年衆院選で敗北
