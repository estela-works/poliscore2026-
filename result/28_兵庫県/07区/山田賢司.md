# 山田賢司（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年4月20日 |
| 年齢 | 58歳 |
| 選挙区 | 兵庫7区（西宮市南部・芦屋市） |
| 当選回数 | 衆議院5期 |
| 主な役職 | 経済産業副大臣兼内閣府副大臣、元外務大臣政務官 |

## 経歴
- 1966年、兵庫県西宮市生まれ
- 甲陽学院高等学校卒業
- 慶應義塾大学法学部法律学科卒業
- 住友銀行（現・三井住友銀行）入行
- 2012年、第46回衆議院議員総選挙で兵庫7区から初当選
- 2014年、2017年、2021年と連続当選（4期連続）
- 外務大臣政務官を歴任
- 2024年10月、第50回衆院選で5選（得票率36.9%）
- 2024年11月、経済産業副大臣兼内閣府副大臣に就任
- 志帥会（二階派）→麻生派に所属

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 経済産業副大臣として産業政策に携わる。外務政務官経験から外交・安全保障にも関心。ただし独自の政策課題設定は限定的。

### ② 解決設計力: 4/5
**評価根拠**: 慶應義塾大学法学部卒、住友銀行での勤務経験から経済・金融に精通。副大臣として政策立案に関与。

### ③ 実行力: 4/5
**評価根拠**: 5期連続当選で安定した選挙基盤。経産副大臣兼内閣府副大臣として行政実務を担当。得票率36.9%でトップ当選。

### ④ 説明責任: 3/5
**評価根拠**: 国会質疑での活動あり。ただし副大臣としての情報発信は限定的。

### ⑤ 一貫性: 3/5
**評価根拠**: 5期12年の議員活動で自民党の政策路線を堅持。派閥は二階派から麻生派へ移籍。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に大きなスキャンダルは報じられていない。裏金問題への関与は不明確。

## 強み・弱み

### 強み
- **5期連続当選**: 安定した選挙基盤と地元支持
- **副大臣経験**: 経産・内閣府副大臣として行政経験豊富
- **経済知識**: 住友銀行での実務経験
- **外交経験**: 外務大臣政務官経験
- **学歴**: 慶應義塾大学法学部卒

### 弱み（注意点）
- **激戦区**: 維新・立憲との三つ巴の争い
- **裏金問題の影響**: 自民党全体への逆風
- **派閥移籍**: 二階派から麻生派への移籍
- **独自色不足**: 政策面での独自性が見えにくい
