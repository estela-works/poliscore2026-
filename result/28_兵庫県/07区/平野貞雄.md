# 平野貞雄（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年3月4日（69歳） |
| 出身地 | 奈良県 |
| 選挙区 | 兵庫県7区 |
| 当選回数 | 衆議院0回（新人）/ 芦屋市議9期 |
| 主な役職 | 芦屋市議会議員、党西宮芦屋地区常任委員 |

## 経歴
- 1974年 奈良県立奈良高校卒業
- 1978年 関西学院大学法学部卒業
- 1983年 芦屋市議会議員初当選（以後通算9期）
- 市議会副議長、総務・民生・文教・建設各常任委員長、監査委員等を歴任
- 2009年 衆院選兵庫7区出馬（26,745票獲得、落選）
- 2011年 芦屋市議に復帰
- 現在、兵庫県平和委員会常任理事、芦屋原水協常任理事

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 平和・人権・社会保障を軸に活動しているが、独自の問題提起や構造分析は確認できない。共産党の政策に沿った活動が中心。地方議会レベルでの課題設定は行っているが、国政レベルの新たな問題定義は見られない。

### ② 解決設計力: 2/5
**評価根拠**: 芦屋市で子ども医療費助成制度（中学卒業まで）や中学校給食（自校方式）の実現に貢献。ただしこれらは市レベルの政策であり、国政レベルの独自の政策設計は確認できない。

### ③ 実行力: 2/5
**評価根拠**: 芦屋市議として32年超の経験があり、副議長や各常任委員長を歴任。地方議会レベルでは一定の実績がある。しかし国会議員としての実績はなく、2009年衆院選では26,745票を獲得したものの落選。野党新人として③は構造的に低くなる。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 市議会での質問実績は長年あるが、詳細な内容や説明力についての情報は限定的。ブログでの発信活動も行っている。情報不足のため低めの評価。

### ⑤ 一貫性: 5/5
**評価根拠**: 1983年の初当選以来40年以上にわたり日本共産党所属で活動。平和・人権・社会保障という政策軸も一貫。平和委員会、原水協など反核・平和運動にも長年携わっており、極めて高い一貫性がある。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道なし。平和運動・人権運動に長年取り組み、市民活動にも積極的に参加。クリーンな政治姿勢を維持している。

## 強み・弱み

### 強み
- **豊富な地方議員経験**: 市議9期32年超の経験と実績
- **抜群の一貫性**: 40年以上共産党で平和・人権を訴え続ける
- **地域密着**: 芦屋市での認知度と信頼

### 弱み（注意点）
- **国政での実績なし**: 2009年衆院選では落選
- **高齢**: 69歳での国政初挑戦
- **選挙区の厳しさ**: 保守地盤の兵庫7区で共産党候補は苦戦が予想される
