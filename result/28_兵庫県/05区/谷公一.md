# 谷公一（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1952年1月28日 |
| 年齢 | 73歳 |
| 選挙区 | 兵庫5区（豊岡市、養父市、朝来市、丹波市、篠山市等） |
| 当選回数 | 衆議院8回 |
| 主な役職 | 元国家公安委員長、元防災担当大臣、自民党東日本大震災復興加速化本部長 |

## 経歴
- 兵庫県香美町（旧村岡町）出身
- 八鹿高校卒業、明治大学政経学部政治学科卒業
- 兵庫県庁入庁、自治省・消防庁出向
- 阪神・淡路大震災時に兵庫県防災局長
- 兵庫県政策室長を経て退庁
- 2003年衆院選で初当選（父・洋一の地盤継承）
- 国土交通大臣政務官（第1次安倍改造内閣）
- 復興副大臣（第2次安倍内閣）
- 国家公安委員長・防災担当大臣（第2次岸田改造内閣、2022-2023年）
- 衆議院地域活性化・こども政策・デジタル社会形成特別委員長

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 阪神・淡路大震災時の兵庫県防災局長経験から、防災・復興を一貫した政策課題として設定。東日本大震災復興、国土強靭化など、自身の経験に基づく明確な課題認識を持つ。

### ② 解決設計力: 4/5
**評価根拠**: 兵庫県庁での行政経験、自治省・消防庁での中央官庁経験を経て、防災・復興政策の実務に精通。復興副大臣、防災担当大臣として政策設計に直接関与した実績がある。

### ③ 実行力: 4/5
**評価根拠**: 8期連続当選で兵庫5区での地盤は盤石。国家公安委員長、防災担当大臣として閣僚を務め、復興副大臣としても実務を担当。地元但馬地方での強固な支持基盤を維持。

### ④ 説明責任: 2/5
**評価根拠**: 2023年の「うなぎ発言」問題が深刻。岸田首相への爆発物投擲事件発生時に「鰻丼はしっかり食べた」と発言し、治安担当閣僚としての資質を問われた。危機管理意識や国民への説明姿勢に疑問符。

### ⑤ 一貫性: 4/5
**評価根拠**: 2003年の初当選から一貫して自民党所属。防災・復興・国土強靭化という政策軸も20年以上ブレていない。父から地盤を継承し、地元での活動を継続。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 「うなぎ発言」は国家公安委員長という治安の最高責任者として不適切。首相への爆発物投擲という重大事件の最中に「鰻丼を食べた」と語る姿勢は、危機管理意識と公人としての自覚の欠如を示す。旧二階派所属。

## 強み・弱み

### 強み
- **防災・復興の専門性**: 阪神・淡路大震災での実務経験、復興副大臣・防災担当大臣歴任
- **選挙の強さ**: 8期連続当選、但馬地方での圧倒的な地盤
- **行政経験**: 県庁・自治省での実務経験に基づく政策立案能力
- **閣僚経験**: 国家公安委員長など複数の大臣ポストを経験

### 弱み（注意点）
- **うなぎ発言問題**: 治安責任者としての資質を疑問視される失言
- **世襲議員**: 父・洋一から地盤継承
- **年齢**: 73歳であり、長期的な活動には限りがある
- **旧二階派**: 派閥政治への関与
