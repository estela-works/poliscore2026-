# 高橋ひであき（無所属連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1982年8月7日（43歳） |
| 選挙区 | 兵庫5区（豊岡市、養父市、朝来市、宍粟市、たつの市の一部、佐用町、香美町、新温泉町） |
| 当選回数 | 新人 |
| 主な役職 | 鼎立の党代表、農家、無所属連合委員 |

## 経歴
- 千葉県松戸市出身
- 早稲田大学教育学部中退
- ITベンチャー企業代表として勤務
- 2013年：兵庫県丹波市へ移住
- 丹波市で自然農法による農園を開設（黒豆、麦などを栽培）
- 2023年7月：政治団体「鼎立の党」を設立、代表に就任
- 2024年10月：第50回衆議院選挙 和歌山2区から無所属で出馬（落選）
- 2025年7月：参議院選挙 兵庫選挙区から無所属連合公認で出馬
- 2026年：第51回衆議院選挙 兵庫5区から無所属連合公認で出馬

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 食料自給率の低下、農村人口減少、小規模農家の衰退など、日本農業が抱える構造的課題を明確に指摘。「戦後半世紀にわたる失政で農村人口が減り、生産基盤が崩れた」という問題認識は具体的。ただし、農業以外の政策分野での課題設定は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 「自然回帰」「国内回帰」「脱原発」など理念は明確だが、具体的な制度設計や財源論については情報が少ない。「兼業と小規模農家の支援拡充」という方向性は示すが、詳細な政策メニューは限定的。

### ③ 実行力: 2/5
**評価根拠**: 自ら丹波市に移住し農業を実践している点は評価できる。政治団体「鼎立の党」を設立するなど行動力はあるが、選挙では2024年衆院選で落選。政治的な実行力を発揮する機会を得られていない。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトで理念や政策を発信。農業の現場経験に基づいた主張は説得力がある。「自然の声を国会に」というスローガンで一貫したメッセージを発信している。

### ⑤ 一貫性: 4/5
**評価根拠**: 2013年の移住以来、自然農法による農業を実践しながら、食料自給率向上と環境保全を一貫して訴えている。IT企業から農業への転身、都市から農村への移住という生き方自体が主張と一致しており、言行一致の姿勢が見られる。

### ⑥ 倫理観: 3/5
**評価根拠**: スキャンダルや問題行動は報じられていない。自然環境や伝統文化の保護を重視する姿勢から、持続可能性への倫理的関心が窺える。

## 強み・弱み

### 強み
- **農業の現場経験**: 自ら農家として10年以上活動し、農政を語る当事者性がある
- **一貫した理念**: 自然回帰、食料自給率向上という軸がブレない
- **地域密着**: 丹波市での移住・定住経験から但馬・丹波地域への理解がある
- **多様な経歴**: IT企業から農業へ、都市から農村へという転身経験
- **若さ**: 43歳と比較的若い

### 弱み（注意点）
- **選挙経験の浅さ**: 2024年衆院選で落選、政治経験は限定的
- **知名度不足**: 新興政治団体「鼎立の党」の認知度は低い
- **政策の幅**: 農業・環境政策に特化しており、他分野の政策が見えにくい
- **組織基盤**: 大政党のような組織的支援がない
- **選挙区の状況**: 兵庫5区は自民・谷公一の強固な地盤
