# 遠藤良太（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年12月19日 |
| 年齢 | 40歳 |
| 選挙区 | 兵庫5区（豊岡市、養父市、朝来市、丹波市、篠山市等） |
| 当選回数 | 衆議院1回（2024年落選、現在落選中） |
| 主な役職 | 日本維新の会兵庫5区支部長、元衆議院議員 |

## 経歴
- 大阪府堺市出身
- 追手門学院大学経営学部卒業
- 2007年大学在学中にバックパッカーとして世界一周
- 2009年中国山東省威海市に語学留学、日本語講師
- ハウステンボス株式会社勤務
- 2013年ロングライフホールディング（介護事業）入社
- 2015年中国駐在、有料老人ホーム立ち上げ・運営
- 2019年ロングライフグローバルコンサルタント代表取締役社長
- 2021年日本維新の会兵庫5区支部長就任
- 2021年衆院選で比例復活当選
- 国会対策副委員長、厚生労働委員会委員
- 2024年衆院選で落選（53,793票、比例復活ならず）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 介護事業での経験から高齢者福祉への問題意識を持つ。厚生労働委員会委員として社会保障分野に関与。ただし独自の課題設定というより維新の政策方針に沿った活動が中心。

### ② 解決設計力: 3/5
**評価根拠**: 介護事業会社の代表取締役として経営経験があり、ビジネス視点での問題解決能力はある。中国での老人ホーム立ち上げなど実務経験も豊富。ただし国政での政策設計実績は1期と限定的。

### ③ 実行力: 2/5
**評価根拠**: 2021年衆院選で比例復活当選したが、2024年選挙では落選（得票数も65,714票→53,793票に減少）。比例復活もならず議席喪失。兵庫5区は谷公一が圧倒的に強く、小選挙区での当選は困難。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信している。国会での委員会活動も行っていた。特に問題となる説明責任の欠如はないが、詳細な政策説明は限定的。

### ⑤ 一貫性: 3/5
**評価根拠**: 日本維新の会一筋で政治活動を継続。兵庫5区での活動も2021年から継続している。ただし大阪府堺市出身で、兵庫5区との地縁は薄い。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。1期の議員活動において倫理的な問題は見られない。クリスチャンとして信仰生活を送る。

## 強み・弱み

### 強み
- **若さ**: 40歳と若く、長期的な政治活動が可能
- **経営経験**: 介護事業会社の代表取締役経験
- **国際経験**: 世界一周、中国駐在など海外経験豊富、中国語堪能
- **国会経験**: 1期ながら厚労委員会委員、国対副委員長を経験

### 弱み（注意点）
- **選挙での敗北**: 2024年衆院選で落選、得票数も減少
- **選挙区事情**: 兵庫5区は谷公一が8期連続当選で圧倒的に強い
- **地縁の薄さ**: 大阪府堺市出身で、兵庫5区との地縁が薄い
- **比例復活頼み**: 小選挙区での勝利実績がない
