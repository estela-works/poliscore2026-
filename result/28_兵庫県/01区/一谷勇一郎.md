# 一谷勇一郎（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年1月22日 |
| 年齢 | 49歳 |
| 選挙区 | 兵庫1区 |
| 当選回数 | 1回（前職・落選） |
| 主な役職 | 元国会対策副委員長、介護事業会社代表 |

## 経歴
- 大阪市都島区出身
- 成器高等学校（現・大阪学芸高等学校）卒業
- 6年間、プロゴルファーを目指し渡米（プロテスト2回不合格、膝故障で断念）
- 関西医療学園専門学校卒業
- 2003年大阪市で整骨院を独立開業
- 2010年法人化、介護・医療事業を拡大
- 2019年東徹参議院議員の下で政治を学ぶ
- 2020年維新政治塾5期生
- 2021年衆院選初当選（比例復活、兵庫1区）
- 厚生労働委員、農林水産委員
- 日本維新の会国会対策副委員長
- 2024年衆院選落選（比例復活もならず）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 介護・医療事業経営者としての現場経験から、2019年の医療・介護報酬改定で「社会保障費の増大に危機感」を感じ政界を志した。介護現場の課題を国政に届けるという明確な問題意識を持つ。

### ② 解決設計力: 3/5
**評価根拠**: 柔道整復師として整骨院を独立開業し、介護・医療分野への事業拡大に成功。経営者としての実務能力はある。国政では厚生労働委員として政策立案に関わったが、目立った法案実績は確認できない。

### ③ 実行力: 2/5
**評価根拠**: 2021年衆院選は比例復活での当選、2024年は選挙区・比例とも及ばず落選。選挙区での勝利を果たせていない。国対副委員長として党の国会運営に携わったが、1期で落選。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 公式サイトやSNSでの発信を行っている。介護現場の経験を活かした発信は一定の説得力がある。落選後も「再び国政の場に立つべく活動を再開」と継続的な発信を行っている。

### ⑤ 一貫性: 3/5
**評価根拠**: 維新政治塾から日本維新の会で一貫して活動。介護・医療という専門分野への取り組みも継続している。ただし、大阪市出身で兵庫1区とは地縁がなく、「先祖に導かれた」という理由での出馬は説得力に欠ける面がある。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特筆すべきスキャンダルや問題行動は確認されていない。経営者として事業を展開してきた経歴はクリーン。1期で落選したため、権力の扱い方を評価する材料は限られる。

## 強み・弱み

### 強み
- **専門性**: 介護・医療現場の実務経験
- **経営者経験**: 事業拡大の実績
- **若さ**: 49歳、今後の可能性
- **クリーンな経歴**: スキャンダルなし

### 弱み（注意点）
- **選挙実績**: 選挙区勝利なし、2024年落選
- **地縁なし**: 大阪出身で兵庫との縁が薄い
- **国会実績**: 1期で落選、目立った法案実績なし
- **知名度**: 全国的な発信力は限定的
