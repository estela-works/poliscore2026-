# 井坂信彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年3月27日 |
| 年齢 | 50歳 |
| 選挙区 | 兵庫1区 |
| 当選回数 | 4回（前職） |
| 主な役職 | 立憲民主党兵庫県連代表、元政調会長代理、元幹事長代理 |

## 経歴
- 東京都世田谷区生まれ
- 東京都立戸山高等学校卒業
- 京都大学総合人間学部卒業（理論物理学専攻）
- 1999年神戸市会議員初当選（3期）
- 全国若手市議会議員の会会長
- 2010年参院選落選（みんなの党、兵庫）
- 行政書士事務所開設
- 2012年衆院選初当選（みんなの党、比例近畿）
- 2014年衆院選当選（維新の党、兵庫1区）
- 維新の党政調会長
- 2017年衆院選落選（希望の党）
- 2018年国民民主党入党
- 2021年衆院選当選（立憲民主党、兵庫1区）
- 2024年衆院選4期目当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 年金・医療・介護・子育て・働き方を専門分野とし、社会保障政策に精通。予算委員会で裏金問題や政策活動費を追及するなど、政治課題にも取り組む。国会活動量No.1「三ツ星議員」を6年連続受賞し、課題発掘能力は高い。

### ② 解決設計力: 4/5
**評価根拠**: 「非正規と介護の給料アップ法」を立案し実現させた実績がある。政調会長や幹事長代理として党の政策立案に携わった。市議3期の地方行政経験と国政の両方を活かした制度設計能力を持つ。

### ③ 実行力: 4/5
**評価根拠**: 2024年衆院選で盛山正仁・一谷勇一郎という現職2名を破り当選。三ツ星議員6年連続受賞は国会活動の量と質を示す。議員報酬3割・ボーナス5割カットを継続実施するなど、公約の実行力がある。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 「電話一本で井坂信彦」をモットーに、有権者との直接対話を重視。SNSでの発信も活発。予算委員会での質疑など、国会での発言機会を積極的に活用している。

### ⑤ 一貫性: 3/5
**評価根拠**: みんなの党→維新の党→希望の党→国民民主党→立憲民主党と政党を渡り歩いている。ただし、社会保障政策の充実という政策的な軸は一貫している。政党変遷は野党再編の流れに沿ったもの。

### ⑥ 倫理観・権力の扱い方: 5/5
**評価根拠**: 議員報酬3割・ボーナス5割カットを継続実施し、身を切る姿勢を示している。スキャンダルや問題行動は確認されていない。裏金問題を追及する立場として、自らクリーンな政治を実践している。

## 強み・弱み

### 強み
- **国会活動**: 三ツ星議員6年連続、活動量No.1
- **選挙実績**: 現職2名（自民・維新）を破り当選
- **政策専門性**: 社会保障政策に精通
- **身を切る改革**: 報酬カット継続
- **クリーンな経歴**: スキャンダルなし

### 弱み（注意点）
- **政党変遷**: 複数政党を渡り歩いた経歴
- **野党の限界**: 政権与党でないため政策実現に制約
- **1度の落選**: 2017年衆院選で落選経験
