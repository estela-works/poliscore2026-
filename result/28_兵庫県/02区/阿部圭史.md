# 阿部圭史（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1986年5月26日（38歳） |
| 年齢 | 38歳 |
| 選挙区 | 兵庫2区 |
| 当選回数 | 衆議院1回（比例復活） |
| 主な役職 | 日本維新の会国会議員団代表幹事長室室長、元WHO職員、元厚生労働省職員、医師 |

## 経歴
- 宮城県仙台市出身、岡山県で育つ
- 岡山白陵中学校・高等学校卒業
- 2011年：北海道大学医学部卒業
- 国立国際医療研究センターで初期研修医（脳神経外科専攻）
- 2013年：厚生労働省入省（感染症危機管理、医療政策、外交を担当）
- 2016年〜2018年：ジョージタウン大学外交大学院修士課程（国際政治・安全保障分野）
- 2019年〜2021年：WHO（世界保健機関）職員（新型コロナウイルス感染症対策等に従事）
- 外資系コンサルティング会社勤務
- 2023年8月：次期衆院選兵庫2区に日本維新の会から立候補表明
- 2024年10月：第50回衆院選、小選挙区落選も比例近畿ブロックで復活当選
- 2024年11月：特別国会に一番乗りで初登院（前日23時から9時間以上並ぶ）
- 2025年8月：日本維新の会国会議員団代表幹事長室室長就任
- 2025年12月：衆議院議員定数削減等法案の提出者として法案提出

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 2003年のSARS流行を契機に感染症危機管理に関心を持ち医学部へ進学、東日本大震災で行方不明の親戚を捜した経験、新型コロナ対応の経験から「危機に強い社会をつくる」という明確な課題を設定。医師・厚労省・WHOでの経験に基づく専門性の高い課題設定ができている。

### ② 解決設計力: 5/5
**評価根拠**: 北海道大学医学部卒、ジョージタウン大学外交大学院修士（国際政治・安全保障）という高い学歴。厚生労働省で感染症危機管理政策に関与、WHOで新型コロナ対策に従事。著書『感染症の国家戦略』で具体的な政策提言を行っている。衆議院議員定数削減法案の提出者として立法活動にも関与。

### ③ 実行力: 4/5
**評価根拠**: 2024年衆院選で比例復活ながら初当選を果たした。特別国会への一番乗り初登院（9時間以上並ぶ）は行動力を示す。幹事長室室長に就任し、定数削減法案提出など国会活動を展開。ただし小選挙区では公明党前職の赤羽一嘉に敗北している。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: Yahoo!ニュースエキスパート、noteで積極的に政策発信を行っている。NHK「日曜討論」出演で注目を集めた。著書で感染症危機管理政策を体系的に説明。選挙戦でも「在宅医療の現場で聞いた声なき声を国政に届ける」と明確なメッセージを発信。

### ⑤ 一貫性: 5/5
**評価根拠**: 2003年のSARSを契機に感染症危機管理への関心を持ち、医師→厚労省→WHO→政治家と一貫して危機管理分野でキャリアを構築。「危機に強い社会」という理念は終始一貫している。漫画「沈黙の艦隊」の影響で幼少期から安全保障に関心を持つなど、政治への関心も一貫。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。「政治とカネ」問題、自公政権の腐敗を批判する立場を取っている。WHOや厚労省での公務員としての経験があり、公正な権力行使の訓練を受けている。維新の会の身を切る改革の方針に沿った活動をしている。

## 強み・弱み

### 強み
- **専門性**: 医師・厚労省・WHO職員としての感染症危機管理の専門性
- **学歴**: 北海道大学医学部、ジョージタウン大学大学院
- **国際経験**: WHO職員として新型コロナ対策に従事した国際経験
- **発信力**: 著書、Yahoo!ニュース、noteなど多方面で政策発信
- **若さと行動力**: 38歳、特別国会一番乗り初登院

### 弱み（注意点）
- **小選挙区での敗北**: 兵庫2区では公明党前職に敗北（比例復活）
- **地元基盤の弱さ**: 仙台出身・岡山育ちで兵庫との縁が薄い
- **国会経験の浅さ**: 衆議院議員1期目で立法実績はこれから
- **維新の課題**: 兵庫県知事問題など維新への逆風
