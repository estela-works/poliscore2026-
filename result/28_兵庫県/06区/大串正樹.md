# 大串正樹（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年1月20日 |
| 年齢 | 59歳 |
| 選挙区 | 兵庫6区（伊丹市、宝塚市、川西市） |
| 当選回数 | 衆議院5回 |
| 主な役職 | 衆議院厚生労働委員長、経済産業副大臣、デジタル副大臣 |

## 経歴
- 兵庫県生まれ
- 報徳学園高校卒業
- 東北大学工学部卒業、同大学院資源工学専攻修了（工学修士）
- 1991年石川島播磨重工業入社
- 1996年松下政経塾入塾（17期生）
- 2003年北陸先端科学技術大学院大学博士課程修了、同大学助教
- 2008年西武文理大学准教授
- 2012年衆院選で初当選
- 経済産業大臣政務官（第3次安倍改造内閣）
- デジタル副大臣兼内閣府副大臣（第2次岸田改造内閣、2022年）
- 経済産業副大臣（第2次石破内閣）
- 2024年衆議院厚生労働委員長

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 工学博士、松下政経塾17期生として、デジタル化、産業政策、厚生労働など幅広い政策課題に取り組む。大学教員としての研究経験を活かした課題分析能力がある。

### ② 解決設計力: 4/5
**評価根拠**: 北陸先端大博士、大学准教授という学術バックグラウンドに加え、松下政経塾での政策立案訓練を受けている。デジタル副大臣、経産副大臣として実際の政策設計に携わった実績。

### ③ 実行力: 3/5
**評価根拠**: 5期当選だが、2021年・2024年とも小選挙区で敗北し比例復活。兵庫6区は立憲・維新との三つ巴で厳しい選挙区。副大臣として政府内での実務経験はあるが、選挙での強さに課題。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトで政策・活動を発信。デジタル副大臣として政策説明の機会も多かった。特に問題となる説明責任の欠如はない。

### ⑤ 一貫性: 4/5
**評価根拠**: 2012年初当選から一貫して自民党所属。デジタル・産業政策という軸もブレていない。兵庫6区での活動を継続。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。5期の議員活動において倫理的な問題は見られない。

## 強み・弱み

### 強み
- **学術的専門性**: 工学博士、大学教員経験による高い政策立案能力
- **政府経験**: デジタル副大臣、経産副大臣など複数の政府ポスト経験
- **松下政経塾**: 17期生として政策立案の訓練を受けている
- **与党議員**: 政策実現に関与できる立場

### 弱み（注意点）
- **選挙での弱さ**: 2021年・2024年と小選挙区で連敗、比例復活頼み
- **選挙区の激戦**: 立憲・維新との三つ巴で厳しい戦い
- **知名度**: 全国的な知名度は限定的
