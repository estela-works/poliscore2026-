# 谷浩一郎（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1981年10月24日（44歳） |
| 選挙区 | 兵庫6区（伊丹市、宝塚市、川西市の一部） |
| 当選回数 | 新人 |
| 主な役職 | オペラ歌手、東大阪市民オペラ芸術監督、関西歌劇団理事、大学嘱託講師 |

## 経歴
- ドイツ国立マンハイム音楽大学卒業
- 2010年：ドイツから帰国
- オペラ歌手として活動開始
- 兵庫県立芸術文化センターワンコインコンサート出演
- 2017年：同センター年間No.1アーティストに選出
- 同志社女子大学講師
- 関西歌劇団オペラ教室講師
- 神戸市民文化振興財団講師
- 東大阪市民オペラ芸術監督就任
- 関西歌劇団理事就任
- 参政党DIYワークショップファシリテーター
- 2024年10月：第50回衆議院選挙 大阪8区から参政党公認で出馬（15,272票、落選）
- 2026年：第51回衆議院選挙 兵庫6区から参政党公認で出馬

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「失われた30年」「行き過ぎたグローバリズム」など抽象的な課題認識を示すが、具体的なデータや分析に基づく課題設定とは言い難い。参政党の基本方針に沿った主張が中心で、独自の課題発掘は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 「日本人ファースト」「外国人受け入れ大幅減」などスローガン的な主張が目立ち、具体的な制度設計や政策メニューは示されていない。芸術・文化分野での専門性は高いが、政策立案の経験はない。

### ③ 実行力: 2/5
**評価根拠**: オペラ歌手として、また芸術監督・理事として一定の組織運営経験はあるが、政治的な実行力は未検証。2024年衆院選では大阪8区で15,272票を獲得したが落選。議員としての実績はない。

### ④ 説明責任: 3/5
**評価根拠**: ブログやSNSで積極的に発信を行っている。オペラ歌手としての表現力を活かし、街頭演説でも明確なメッセージを発信。参政党DIYワークショップのファシリテーターとして説明活動も行っている。

### ⑤ 一貫性: 3/5
**評価根拠**: 参政党の理念に沿った主張を一貫して展開。グローバリズム批判、伝統文化重視という軸はブレていない。2024年大阪8区から2026年兵庫6区への選挙区変更はあるが、政策的な一貫性は保たれている。

### ⑥ 倫理観: 2/5
**評価根拠**: スキャンダルや問題行動は報じられていない。ただし「自虐史観はやめよう」「外国人受け入れ大幅減」などの主張は、歴史認識や人権に関わる議論を呼ぶ可能性がある。

## 強み・弱み

### 強み
- **高い専門性**: オペラ歌手として国際的に活躍、ドイツ留学経験
- **表現力**: 芸術家としての訴求力、2017年年間No.1アーティスト選出
- **組織運営経験**: 芸術監督、理事としてのマネジメント経験
- **党組織の支援**: 参政党の組織力を活用できる
- **若さ**: 44歳と比較的若い

### 弱み（注意点）
- **政治経験なし**: 議員経験がなく、政策立案・実行の経験がない
- **選挙での落選**: 2024年衆院選で落選
- **選挙区変更**: 大阪8区から兵庫6区への選挙区移動で地盤がない
- **激戦区**: 兵庫6区は中道・桜井周、自民・大串正樹、維新・市村浩一郎と前職・経験者が揃う激戦区
- **主張の過激さ**: 一部の主張が物議を醸す可能性
