# 市村浩一郎（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1964年7月16日 |
| 年齢 | 60歳 |
| 選挙区 | 兵庫6区（伊丹市、宝塚市、川西市） |
| 当選回数 | 衆議院5回 |
| 主な役職 | 元国土交通大臣政務官、元衆議院財務金融委員長 |

## 経歴
- 福岡県福岡市生まれ（高知市で5年間過ごす）
- 福岡大学附属大濠高校卒業
- 一橋大学社会学部卒業
- 1993年松下政経塾卒塾（9期生）
- 日本新党政策研究員、新進党政策審議室
- 1995年阪神・淡路コミュニティ基金事務局長
- 2003年衆院選で初当選（民主党）
- 国土交通大臣政務官（鳩山内閣）
- 衆議院財務金融委員長
- 2017年衆院選落選後、日本維新の会に移籍
- 2021年衆院選で当選（維新）
- 2024年衆院選で比例復活当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: NPO制度改革、関西3空港一体運営、飲酒運転厳罰化など具体的な課題を設定し取り組んできた実績。阪神・淡路大震災でのボランティア経験から市民活動支援を政策課題として認識。

### ② 解決設計力: 4/5
**評価根拠**: 松下政経塾9期生として政策立案訓練を受け、NPO制度の抜本改革、国会議員年金廃止など具体的な政策を設計・実現した実績がある。国土交通政務官として行政経験も持つ。

### ③ 実行力: 3/5
**評価根拠**: 5期当選だが、2017年落選、2024年も比例復活と安定感に欠ける。2023年の兵庫維新の会代表選でも僅差で敗北。NPO制度改革など一定の政策実現実績はあるが、選挙での強さに課題。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイトやSNSで活動を発信。政策についての説明は行っている。特に問題となる説明責任の欠如はない。

### ⑤ 一貫性: 2/5
**評価根拠**: 民主党→希望の党→日本維新の会と政党を大きく変遷。特に民主党から維新への移籍は政策的な軸の変更を伴う。NPO制度改革という専門分野は一貫しているが、所属政党の変遷は評価を下げる要因。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。長年の議員活動において倫理的な問題は見られない。

## 強み・弱み

### 強み
- **政策実現実績**: NPO制度改革、国会議員年金廃止など具体的な実績
- **松下政経塾**: 9期生として政策立案の訓練を受けている
- **経験の豊富さ**: 5期当選、国交政務官、財務金融委員長経験
- **専門性**: NPO制度改革、社会基盤整備の専門家

### 弱み（注意点）
- **政党の変遷**: 民主党→維新への大きな政党移籍
- **選挙の不安定さ**: 2017年落選、2024年比例復活と安定しない
- **選挙区の激戦**: 立憲・自民との三つ巴で厳しい戦い
- **党内での地位**: 2023年代表選で敗北
