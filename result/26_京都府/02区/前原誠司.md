# 前原誠司（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1962年4月30日（62歳） |
| 年齢 | 62歳 |
| 選挙区 | 京都府2区 |
| 当選回数 | 衆議院11回 |
| 主な役職 | 日本維新の会共同代表、元外務大臣、元国土交通大臣、元民主党・民進党代表 |

## 経歴
- 京都大学法学部卒（高坂正堯ゼミ・国際政治学専攻）
- 1987年 松下政経塾第8期生
- 1991年 京都府議会議員初当選（28歳）
- 1993年 衆議院議員初当選
- 2005年 民主党代表（偽メール事件で辞任）
- 2009年 国土交通大臣、2010年 外務大臣
- 2017年 民進党代表（希望の党騒動で辞任）
- 2023年 教育無償化を実現する会設立
- 2024年 日本維新の会合流、共同代表就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 京都大学で国際政治を学び、外務大臣時代には中国脅威論を明確に提起。教育無償化、財政再建など政策課題を構造的に捉える能力あり。TPP推進では「GDP比1.5%の第一次産業保護のため他分野が犠牲になるのはおかしい」と論理的に主張。

### ② 解決設計力: 3/5
**評価根拠**: 国交大臣として関空・伊丹統合やコンセッション方式を設計・推進。教育無償化を実現する会を設立し政策実現の道筋を模索。ただし、希望の党合流など戦略的判断が裏目に出ることが多く、設計力と実行力に乖離がある。

### ③ 実行力: 2/5
**評価根拠**: 国交大臣時代の空港改革は一定の成果。しかし偽メール事件（2006年）、希望の党騒動（2017年）と大きな失敗が複数。「政局観が弱い」「ギャンブルに負け続けている」との評価。11選の実績はあるが、政権奪取や政策実現には至らず。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 偽メール事件では「すべての責任は私にある」と辞任で責任を取ったが、希望の党騒動での党分裂については十分な説明なし。国民民主党離党時は「ノーサイドにしよう」と言いながら離党し、玉木代表から「許せない」と批判される。

### ⑤ 一貫性: 2/5
**評価根拠**: 日本新党→さきがけ→民主党→民進党→希望の党→国民民主党→教育無償化を実現する会→日本維新の会と所属政党が頻繁に変遷。「渡り鳥」「壊し屋」と批判される。「非自民・反共産」という軸は主張するが、行動との整合性に疑問。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 金銭的汚職は見られない。2011年に在日外国人からの献金問題で外相辞任したが、「献金の認識はなかった」と説明。政党を渡り歩く姿勢への批判はあるが、違法行為ではなく、明確な倫理違反とは言えない。

## 強み・弱み

### 強み
- **政策立案能力**: 外交・安全保障、教育政策に関する知見が深く、課題を構造的に捉える能力がある
- **豊富な経験**: 外務大臣、国土交通大臣など閣僚経験、11回の当選実績

### 弱み（注意点）
- **政局判断の失敗**: 偽メール事件、希望の党騒動など、重要局面での判断ミスが繰り返される
- **一貫性の欠如**: 所属政党の頻繁な変更により「何を実現したいのか」が見えにくい
