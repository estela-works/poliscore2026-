# 安藤裕（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年3月28日 |
| 年齢 | 59歳 |
| 選挙区 | 京都6区（元職）→参院比例当選 |
| 当選回数 | 衆議院3回、参議院1回 |
| 主な役職 | 参政党国会議員団幹事長、元復興大臣政務官 |

## 経歴
- 神奈川県生まれ、京都府宇治市在住
- 慶應義塾普通部→高校→大学経済学部卒業
- 相模鉄道株式会社入社
- 税理士資格取得、安藤裕税理士事務所設立
- 2012年衆院選で京都6区から自民党公認で初当選（比例復活）
- 衆議院議員3期（2012〜2021年）
- 復興大臣政務官（第4次安倍内閣）
- 「日本の未来を考える勉強会」会長
- 2024年参政党入党、衆院選（京都6区）落選
- 2025年参院選（比例）当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 税理士としての専門知識を活かし、「消費税が日本経済停滞の元凶」という明確な課題設定を持つ。積極財政の必要性を「日本の未来を考える勉強会」を設立して提唱するなど、独自の問題提起ができている。

### ② 解決設計力: 4/5
**評価根拠**: 消費税廃止・減税と積極財政による経済立て直しという具体的な政策設計を持つ。コロナ禍での100兆円補正予算提言など、数字を伴う提案ができる。税理士としての専門性が政策設計に活かされている。

### ③ 実行力: 3/5
**評価根拠**: 衆議院議員3期、復興大臣政務官として政府内で実務を担当した実績がある。ただし消費税減税・積極財政という持論は自民党内で主流とならず、結果的に離党。2024年衆院選では落選し、影響力は限定的に。

### ④ 説明責任: 4/5
**評価根拠**: YouTubeチャンネルやSNSで積極的に政策を発信。消費税の問題点、インボイス制度の欺瞞など専門的な内容を分かりやすく解説。参院当選後も国会質疑で明確に主張を展開している。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党から参政党への移籍は大きな政党変更。積極財政・消費税廃止という政策軸自体は一貫しているが、自民党議員として当選しながら党の方針と異なる主張を続け、最終的に離党・移籍した経緯は一貫性に疑問符。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に問題となるスキャンダルや疑惑はない。衆議院議員3期の活動でクリーンな政治家像を維持している。

## 強み・弱み

### 強み
- **専門性**: 税理士としての知識を活かした税制・財政政策への深い理解
- **発信力**: YouTubeやSNSでの積極的な情報発信、分かりやすい政策説明
- **国会経験**: 衆議院3期、政務官経験など豊富な政治経験

### 弱み（注意点）
- **政党移籍**: 自民党から参政党への移籍は有権者からの信頼に影響
- **選挙での弱さ**: 京都6区では山井和則に勝てず、2024年も落選
- **影響力の低下**: 自民党離党後は政策実現の手段が限られる
- **参政党の位置づけ**: 少数政党からの活動で政策実現は困難
