# 浜田聡（日本自由党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年5月11日（48歳） |
| 年齢 | 48歳 |
| 選挙区 | 京都1区 |
| 当選回数 | 新人（元参議院議員1期） |
| 主な役職 | 日本自由党総裁、NHK党政調会長兼幹事長 |

## 経歴
- 京都府京都市山科区出身
- 洛南高等学校卒業
- 東京大学教育学部卒業
- 東京大学大学院教育学研究科修了
- 京都大学医学部卒業
- 放射線科専門医として臨床経験
- 2019年7月、参院選比例区にNHKから国民を守る党公認で出馬、次点
- 2019年10月、立花孝志の自動失職により繰上当選
- NHK党政調会長兼幹事長
- 2025年7月、参院選比例区で落選（335,462票獲得）
- 政治団体「日本自由党」総裁
- 2026年1月20日、衆院選京都1区への出馬を表明

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 参議院議員として国会での質問を活発に行い、様々な行政課題を取り上げた。NHK問題から始まり、行政の透明性、情報公開などの課題を設定している。ただし、課題設定の体系性には議論がある。

### ② 解決設計力: 3/5
**評価根拠**: 参議院議員として1期務め、国会質問を数多く行った。「最も働く議員」との評価もある。医師・研究者としての分析能力を政策に活かす姿勢は見られる。ただし、具体的な立法実績は限定的。

### ③ 実行力: 3/5
**評価根拠**: 参議院議員として1期務め、国会質問回数では上位の実績。2025年参院選では335,462票を獲得したが、党が議席を獲得できず落選。継続的に政治活動を行い、新党を立ち上げるなど行動力はある。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: ブログ、YouTube、SNSなど多様なメディアで積極的に発信を行っている。「シンクタンク」を運営し、政策研究の発信も行う。ネット支持層への発信力は高い評価を受けている。

### ⑤ 一貫性: 3/5
**評価根拠**: NHK問題から行政改革へと活動の軸を広げている。情報公開、透明性という観点は一貫している。ただし、NHK党との関係や新党立ち上げなど、組織的な動きには変遷がある。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2025年参院選では統一教会（世界平和統一家庭連合）および関連団体の支援を受けて立候補したことが報じられており、この点は議論がある。行政の透明性を求める姿勢とのバランスが問われる。

## 強み・弱み

### 強み
- **国会経験**: 参議院議員1期の経験、多数の国会質問
- **高学歴・専門性**: 医師、東大大学院修了
- **発信力**: ネットでの高い発信力、支持層の存在
- **地元出身**: 京都市山科区出身
- **知名度**: 一定の知名度がある

### 弱み（注意点）
- **支援団体への疑問**: 統一教会関連団体の支援を受けたことへの批判
- **政党基盤の弱さ**: 新党「日本自由党」の組織力は限定的
- **直近の選挙敗北**: 2025年参院選で落選
- **激戦区**: 8人が立候補する大混戦
