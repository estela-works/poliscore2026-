# 平竹耕三（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年1月14日 |
| 年齢 | 65歳 |
| 選挙区 | 京都府1区 |
| 当選回数 | 新人 |
| 主な役職 | 京都産業大学客員教授、元京都市文化市民局長、元ロームシアター京都初代館長 |

## 経歴
- 京都市中京区出身
- 市立竹間小学校、初音中学校
- 京都府立堀川高等学校卒業
- 京都大学法学部卒業
- 龍谷大学経済学博士
- 京都市役所勤務（文化行政に従事）
- 京都市文化市民局長、文化芸術政策監
- ロームシアター京都初代館長
- 京都市京セラ美術館オープンに尽力
- 祇園祭後祭の復興に貢献
- 大政奉還150周年記念プロジェクト
- 東アジア文化都市事業に尽力
- 2019年京都市役所定年退職
- 京都産業大学文化学部教授（現在は客員教授）
- 2026年衆院選に中道改革連合から出馬

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 京都市の文化行政を長年担当し、文化政策の課題に精通。祇園祭後祭復興、東アジア文化都市事業など文化による地域活性化を推進。京都という文化都市の課題を深く理解。

### ② 解決設計力: 4/5
**評価根拠**: 京都大学法学部卒、龍谷大学経済学博士。ロームシアター京都、京セラ美術館のオープンという大型プロジェクトを実行。文化政策の専門家。

### ③ 実行力: 3/5
**評価根拠**: 京都市の文化行政で多くの事業を実現。ただし国政経験はなく、選挙経験もない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 大学教授として教育・研究活動。文化行政の発信力はあるが、政治家としての発信は未知数。

### ⑤ 一貫性: 3/5
**評価根拠**: 文化行政一筋のキャリア。政治活動歴は短いため詳細な評価は難しい。中道改革連合からの出馬。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。長年の公務員生活でクリーンな経歴。

## 強み・弱み

### 強み
- **文化政策の専門家**: 京都市文化市民局長、ロームシアター初代館長
- **地元密着**: 京都市中京区出身、市役所勤務
- **学識**: 京大法学部、龍谷大博士、大学教授
- **実績**: 祇園祭後祭復興、文化施設オープン
- **クリーン**: スキャンダルなし

### 弱み（注意点）
- **政治経験なし**: 国政・地方議会の経験なし
- **知名度**: 政治家として低い
- **選挙経験なし**: 初の選挙戦
- **厳しい選挙区**: 7人乱立の激戦区
