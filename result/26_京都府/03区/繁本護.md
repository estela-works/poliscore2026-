# 繁本護（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1972年12月26日（53歳） |
| 出身地 | 兵庫県 |
| 選挙区 | 京都府3区 |
| 当選回数 | 衆議院1回（前職） |
| 主な役職 | 財務大臣政務官（第1次岸田内閣） |

## 経歴
- 1991年 兵庫県立姫路西高等学校卒業
- 1995年 神戸大学工学部土木工学科卒業
- 1997年 神戸大学大学院自然科学研究科建設学専攻修了、運輸省（現国土交通省）入省
- 2003年 イリノイ工科大学経営大学院修了（MBA取得）
- 2006年 北海道開発局港湾事務所長
- 2008年 釧路市港湾空港部長
- 新しい霞ヶ関を創る若手の会を設立
- 国土交通省国際調整官
- 2010年 伊東良孝衆議院議員の政策秘書
- 2017年 自民党衆議院京都府第2区支部長（公募）
- 2021年 第49回衆院選にて京都2区から出馬、当選
- 2021年11月 財務大臣政務官（第1次岸田内閣）
- 2023年5月 衆議院京都2区支部長辞任
- 2025年7月 参院選比例区出馬、落選（45,406票・31人中26位）
- 2026年1月 衆院選京都3区公認候補予定者に内定

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 国土交通省出身で、インフラ・港湾政策に関する専門知識を持つ。MBA取得など経営的視点も備える。ただし具体的な政策提言の詳細は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 国土交通省での実務経験、北海道開発局港湾事務所長、釧路市港湾空港部長などの経験から行政実務の設計能力は一定程度ある。官僚出身らしい実務能力。

### ③ 実行力: 3/5
**評価根拠**: 衆議院議員1期（2021-2023年推定）、財務大臣政務官を務めた実績がある。ただし国会議員としての在職期間は短く、目立った立法実績は確認できない。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2021年8月、衆議院厚生労働委員会でコロナ対策審議中に約20分間、衆院選向け資料をチェックしていたことが報じられた。毎日新聞の取材に対し期限までに回答しなかった。

### ⑤ 一貫性: 2/5
**評価根拠**: 京都2区支部長→参院選比例区出馬→京都3区へと選挙区を変遷。2023年に京都2区支部長を辞任し参院選を目指したが落選、その後衆院選京都3区に転身。政治的ポジション取りに一貫性を欠く印象。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2021年4月の旧統一教会系イベント『新型コロナ終息を願う京都1万人祈りの集い』で世話人として名を連ねていたことが報じられている。委員会審議中の資料チェック問題もあり、政治家としての姿勢に疑問符。

## 強み・弱み

### 強み
- **官僚経験**: 国土交通省出身でインフラ政策に詳しい
- **高学歴**: 神戸大学大学院、MBAと高い学歴
- **閣僚経験**: 財務大臣政務官の経験

### 弱み（注意点）
- **旧統一教会問題**: 教団系イベントの世話人として名を連ねた
- **委員会軽視**: 審議中に選挙資料をチェック
- **選挙区変遷**: 京都2区→参院比例→京都3区と不安定
- **参院選落選**: 2025年参院選で落選
