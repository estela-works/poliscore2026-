# 山本和嘉子（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年7月18日 |
| 年齢 | 56歳 |
| 選挙区 | 京都府5区 |
| 当選回数 | 衆議院1回 |
| 主な役職 | 立憲民主党京都府第5区総支部長、立憲民主党京都府連副会長 |

## 経歴
- 京都府立洛北高等学校卒業
- 同志社大学文学部文化学科国文学専攻卒業
- 京都信用金庫勤務
- 1993年葵祭第38代斎王代
- 福山哲郎参議院議員の公設秘書（17年間）
- 病気の父を10年間介護
- 2017年衆院選で比例北陸信越ブロックから初当選
- 2021年衆院選で落選、2024年衆院選でも落選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「暮らしの不安」「物価上昇」「子育て環境」「農業の担い手不足」など一般的な課題を挙げているが、独自の視点での問題提起は見られない。「政治とカネ」の批判は野党として当然の主張であり、構造的な分析や本質的なボトルネックの指摘には至っていない。

### ② 解決設計力: 2/5
**評価根拠**: 「まっとうな政治」「家計が第一」といったスローガンは掲げているが、具体的な制度設計や政策提案の実績が確認できない。議員立法の実績もなく、設計力を示す材料が不足している。

### ③ 実行力: 1/5
**評価根拠**: 衆議院議員1期（2017-2021）のみで、目立った政策実現や法案成立の実績がない。2021年、2024年と連続で落選しており、選挙での実行力も課題。当選1回の新人レベルとして、実行力を示す機会自体が限られていた。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 国会での答弁や記者会見に関する情報がほとんど見つからない。SNSでの発信は行っているが、説明力を評価できる材料が不足している。情報不足。

### ⑤ 一貫性: 4/5
**評価根拠**: 福山哲郎秘書として17年間勤務後、立憲民主党から一貫して出馬。落選後も京都5区総支部長として活動を継続し、2025年参院選にも挑戦予定。政党移籍や大きな方針転換はなく、野党として軸を持って活動している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 政治資金問題やスキャンダルは確認されていない。「政治とカネ」の問題を批判する立場で、クリーンなイメージを維持している。介護経験を持ち、生活者目線を重視する姿勢が見られる。

## 強み・弱み

### 強み
- **一貫した政治姿勢**: 17年の秘書経験を経て、野党として筋を通した活動
- **生活者目線**: 介護経験、地域を歩いて声を聞く姿勢

### 弱み（注意点）
- **実績の乏しさ**: 1期のみで具体的な政策実現がない
- **選挙での弱さ**: 2021年、2024年と連続落選、京都5区で勝てる見込みが立たない
