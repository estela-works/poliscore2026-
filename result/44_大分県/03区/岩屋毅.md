# 岩屋毅（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1957年8月24日 |
| 年齢 | 67歳 |
| 選挙区 | 大分3区（別府市、中津市、豊後高田市、杵築市、宇佐市、国東市など） |
| 当選回数 | 衆議院10回 |
| 主な役職 | 前外務大臣（石破内閣）、元防衛大臣、元外務副大臣 |

## 経歴
- 1957年8月24日：大分県別府市生まれ
- 早稲田大学政治経済学部卒業
- 1990年：第39回衆院選で初当選（大分3区）
- 1993年：落選
- 2000年：第42回衆院選で復帰（2期目）
- 2001年：第2次森改造内閣で防衛庁長官政務官
- 2006年：第1次安倍内閣で外務副大臣（麻生太郎外相の下）
- 2018年10月：防衛大臣就任（第1次安倍改造内閣）
- 2019年9月：防衛大臣退任
- 2024年1月：裏金問題を受け麻生派（志公会）を退会
- 2024年9月：自民党総裁選で石破茂の推薦人に
- 2024年10月1日：第1次石破内閣で外務大臣就任
- 2024年10月27日：第50回衆院選で10選（大分3区）
- 2024年11月：第2次石破内閣で外務大臣留任
- 2024年12月：訪中時に中国人向けビザ緩和を表明
- 2025年9月：石破首相退陣表明に伴い外相退任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 防衛・安全保障政策で長年取り組み、防衛大臣、外務大臣を歴任。日中関係改善、安全保障環境への対応など国家的課題を設定する立場にあった。外相として日中関係、日米同盟の両立という難題に取り組んだ。

### ② 解決設計力: 4/5
**評価根拠**: 防衛庁長官政務官、外務副大臣、防衛大臣、外務大臣と閣僚・副大臣経験が豊富。政策立案の実務経験は十分。ただし中国人ビザ緩和は党内調整を欠いたまま発表し批判を受けた。

### ③ 実行力: 4/5
**評価根拠**: 衆院選10回連続当選は大分県で圧倒的な実績。防衛大臣時代には自衛隊の海外派遣など実績を残した。ただし外相時代のビザ緩和政策は自民党内の反対で頓挫した。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 中国人ビザ緩和について事前に党側への説明がなく強い反発を招いた。議員宿舎侵入事件では「何の被害もなく、すぐお帰りいただいた」と説明も、詳細な経緯の説明が不十分で疑惑が払拭されていない。

### ⑤ 一貫性: 3/5
**評価根拠**: 安全保障政策への取り組みは一貫している。2024年に麻生派を退会し石破茂を支持したことは、派閥政治からの離脱という判断。ただし対中政策では融和的姿勢が保守層から批判されている。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: IR汚職事件で中国企業から100万円を受領（後に返金）。議員宿舎侵入事件で「ハニートラップ疑惑」が報じられた。裏金問題では麻生派所属時の対応について説明責任を果たしていない。中国人ビザ緩和で「更迭署名3万人超」を集めるなど、外相としての判断に疑問符が付いた。

## 強み・弱み

### 強み
- **豊富な閣僚経験**：防衛大臣、外務大臣、外務副大臣などを歴任
- **選挙の強さ**：大分3区で10回連続当選
- **安全保障の専門性**：防衛政策に精通
- **石破首相との近さ**：石破派中心メンバーとして信頼厚い

### 弱み（注意点）
- **IR汚職関連**：中国企業から100万円受領（返金済みだが疑惑は残る）
- **ハニートラップ疑惑**：議員宿舎侵入事件の経緯が不透明
- **対中融和批判**：ビザ緩和で保守層から強い反発
- **党内調整の欠如**：ビザ緩和を事前調整なく発表
- **派閥問題**：裏金問題に関連した麻生派からの退会
