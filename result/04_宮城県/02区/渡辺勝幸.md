# 渡辺勝幸（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年5月10日 |
| 年齢 | 49歳 |
| 選挙区 | 宮城2区（仙台市泉区・宮城野区・若林区） |
| 当選回数 | 衆議院0回（宮城県議3期） |
| 主な役職 | 自民党宮城県第2選挙区支部長、宮城県議会議員 |

## 経歴
- 仙台市若林区出身
- 宮城県仙台第一高等学校卒業
- 慶応義塾大学卒業
- 参議院議員政策秘書（大学在学中に政策担当秘書資格取得）
- 宮城県議会議員3期（2015年〜）
- 自民党宮城県連青年局長、県民運動本部長、憲法改正実現本部事務局長
- 2025年4月、自民党宮城2区支部長に選出

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 県議として地域課題に取り組んできたが、国政レベルでの独自の課題設定は見られない。憲法改正実現本部事務局長として党の方針に沿った活動が中心。構造的問題の指摘や本質的ボトルネックの言語化は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 県議としての政策経験はあるが、国政での具体的な制度設計実績はない。党の政策を踏襲する形で、独自の政策設計力を示す機会はこれから。

### ③ 実行力: 2/5
**評価根拠**: 県議3期の実績はあるが、国政での実行力は未知数。2024年衆院選には出馬していない新人候補。支部長選出では秋葉賢也元復興相を退けたが、選挙での実績はこれから。

### ④ 説明責任: 4/5
**評価根拠**: 支部長就任に際し「自民党に対して大変厳しい声をいただいている」と現状認識を率直に語り、「信頼を取り戻すために活動する」と説明責任を果たす姿勢を示している。透明性ある発信を心がけている。

### ⑤ 一貫性: 4/5
**評価根拠**: 参院議員秘書から県議へと政治の道を一貫して歩み、自民党内で活動を続けてきた。「長いものには巻かれない」姿勢を標榜し、明確な政治的軸を持っている。

### ⑥ 倫理観: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。秋葉賢也氏（統一教会問題・裏金疑惑）に代わって選出されたクリーンなイメージ。障害者スポーツ団体や各種地域団体との関わりも健全。

## 強み・弱み
### 強み
- **政策秘書資格**: 大学在学中に難関国家資格を取得した政策立案能力
- **若さと世代交代感**: 49歳で自民党の若返りをアピールできる
- **地元密着**: 仙台市若林区出身で地域団体との幅広いネットワーク

### 弱み（注意点）
- **国政経験の欠如**: 衆院選は初挑戦となり実績がない
- **知名度の低さ**: 県議としては知られるが全県・全国的認知度は低い
- **自民党への逆風**: 裏金問題等の党のマイナスイメージを背負う
