# 鎌田さゆり（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年1月8日 |
| 年齢 | 60歳 |
| 選挙区 | 宮城2区（仙台市泉区・宮城野区・若林区） |
| 当選回数 | 衆議院4回 |
| 主な役職 | 衆議院法務委員会理事、立憲民主党宮城県連副代表 |

## 経歴
- 仙台市生まれ（父は自民党宮城県連元事務局長）
- 宮城学院高等学校卒業
- 東北学院大学経済学部卒業
- 仙台市議会議員1期（1995年〜1997年、自民党会派）
- 宮城県議会議員1期
- 衆議院議員初当選（2000年、民主党）
- 2003年再選
- 2004年公選法違反事件で議員辞職
- 2021年16年10か月ぶりに国政復帰（立憲民主党）
- 2024年4期目当選
- 競技かるたA級七段、元準クイーン

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: DV被害者・子どもの権利という観点から共同親権法改正を批判し、当事者視点での課題提起を行っている。税制改革についても「年収1億円超の税負担逆転現象」という構造的問題を指摘。課題の解像度は高い。

### ② 解決設計力: 2/5
**評価根拠**: 課題提起は鋭いが、具体的な制度設計や実現可能な解決策の提示は弱い。消費税5%引き下げなど主張はあるものの、財源や実装手順が不明確。批判は得意だが建設的提案が不足。

### ③ 実行力: 4/5
**評価根拠**: 2021年に571票差で接戦を制し16年ぶりの国政復帰を果たした突破力がある。2024年も当選し継続的に議席を確保。法務委員会理事として委員会運営に関与。

### ④ 説明責任: 2/5
**評価根拠**: 2024年11月の国会での態度問題について謝罪したものの、「精神的ダメージが重なった」と言い訳を付け加え再炎上。説明に際して余計な一言が目立ち、炎上を自ら招く傾向がある。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党会派（市議時代）→民主党→立憲民主党と政党を移動。2004年の議員辞職を経て長いブランクの後に復帰。政策軸は一定あるものの、紆余曲折のある政治人生。

### ⑥ 倫理観: 1/5
**評価根拠**: 2004年に公職選挙法違反で関係者が逮捕・起訴され、最高裁で有罪確定後に議員辞職という重大な法的問題がある。さらに2024年には秘書9人が3年間で連続退職、パワハラ疑惑が週刊文春で報道された。ヤギに頭を下げさせるなどの奇行も報告されている。

## 強み・弱み
### 強み
- **選挙での粘り強さ**: 大差での敗北後も挑戦を続け、国政復帰を果たした
- **人権・福祉分野への関心**: DV被害者支援、子どもの権利など弱者視点での政策提言
- **競技かるたでの実績**: 準クイーン経験で知名度・好感度の向上に寄与

### 弱み（注意点）
- **過去の法的問題**: 公選法違反による議員辞職歴は重大な経歴
- **パワハラ疑惑**: 秘書の連続退職と週刊文春報道は深刻
- **炎上体質**: SNSや国会での言動で度々批判を招く
