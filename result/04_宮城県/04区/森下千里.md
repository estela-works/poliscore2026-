# 森下千里（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1981年9月1日 |
| 年齢 | 44歳 |
| 選挙区 | 宮城4区（石巻市・塩釜市・多賀城市・東松島市等） |
| 当選回数 | 衆議院1回（比例東北ブロック） |
| 主な役職 | 環境大臣政務官、自由民主党宮城4区支部長 |

## 経歴
- 愛知県名古屋市出身、宮城県石巻市在住
- 戸籍名は「森 千里」、旧芸名は森下ちさと
- 2001年：「レースクイーン・オブ・ザ・イヤー」受賞
- グラビアアイドル・タレントとして活動（「ロンドンハーツ」出演、「まちイチ〜千里の道も一歩から〜」MC等）
- ティーチングプロ資格取得（ゴルフ）
- 2011年3月：東日本大震災を機に被災地支援活動に従事
- 2021年3月：自民党宮城5区候補者として擁立発表
- 2021年4月：石巻市に住民票を移し移住
- 2021年10月：第49回衆議院選挙（宮城5区）出馬、61,410票獲得も落選
- 2022年11月：区割り変更により新宮城4区へ
- 2023年3月：自民党比例東北ブロックに回ることが決定
- 2024年10月：第50回衆議院選挙（比例東北ブロック単独2位）で初当選
- 2025年10月：高市早苗内閣で環境大臣政務官に就任
- 2025年12月：自民党宮城4区支部長に選任
- 2026年2月：第51回衆議院選挙（宮城4区）に立候補予定

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 環境大臣政務官として、クマ被害対策やメガソーラー問題、再生可能エネルギーの在り方について発言している。しかし、独自の構造的問題提起というより、担当分野の課題を語る段階にとどまる。2022年にYouTube番組で「食料自給率を上げたい」と発言した際、定義を正確に説明できず政策理解の浅さが批判された。

### ② 解決設計力: 2/5
**評価根拠**: 「自然と共生する再エネ」「総合的なクマ対策」など方向性は示すが、具体的な制度設計や財源への言及は限定的。政務官として政策に関与し始めたばかりで、独自の設計力を示す実績はこれから。

### ③ 実行力: 2/5
**評価根拠**: 2024年10月に比例当選し、わずか1年で環境大臣政務官に抜擢されたスピード出世。しかし、議員立法や独自の政策実現の実績はまだない。政務官としての活動は始まったばかり。比例単独2位という優遇措置での当選であり、小選挙区での勝利経験はない。

### ④ 説明責任: 2/5
**評価根拠**: 当選後初のSNS投稿で「選挙はダイエット効果もある」と発言し批判を浴びた。食料自給率の定義を説明できなかった件もあり、政策説明力には課題がある。一方、環境政務官就任時は「身引き締まる思い」と述べ、クマ被害やメガソーラー問題への取り組み姿勢を示している。

### ⑤ 一貫性: 4/5
**評価根拠**: 2021年に宮城5区から出馬して以来、落選しても宮城県に留まり活動を継続。石巻市に移住して生活拠点を完全に移した点は評価できる。震災復興・被災地支援という軸は一貫している。比例に回された後も地道に活動を続けた。

### ⑥ 倫理観: 2/5
**評価根拠**: 政治資金問題やスキャンダルは報じられていない。しかし、芸能時代の交際報道や「選挙はダイエット」発言など、政治家としての品位を問う声がある。また、非核三原則を「見直すべき」との回答や、企業・団体献金禁止に反対するスタンスは、透明性重視の観点からは疑問が残る。上沼恵美子による過去の批判的コメントも報じられている。

## 強み・弱み

### 強み
- **知名度**: タレント出身で全国的な知名度がある
- **環境大臣政務官**: 初当選1年で政務三役就任というスピード出世
- **被災地との結びつき**: 震災支援活動をきっかけに石巻市に移住
- **継続的な活動**: 2021年落選後も4年以上宮城で活動を継続
- **与党所属**: 自民党公認で組織力・政策実現の可能性

### 弱み（注意点）
- **政策理解力への疑問**: 食料自給率の定義を説明できず批判
- **小選挙区での勝利経験なし**: 2021年は落選、2024年は比例優遇での当選
- **発言の軽さ**: 「選挙はダイエット」発言で批判
- **選挙区の厳しさ**: 安住淳氏（中道改革連合）の強固な地盤に挑む構図
- **タレント議員への批判**: 能力・資質を疑問視する声
