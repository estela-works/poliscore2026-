# 津村啓介（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1971年10月27日（53歳） |
| 選挙区 | 岡山2区（比例中国復活） |
| 当選回数 | 衆議院7期 |
| 主な役職 | 内閣府大臣政務官（鳩山・菅内閣）、衆議院科学技術・イノベーション推進特別委員長 |

## 経歴
- 岡山県津山市生まれ
- 麻布中学校・高等学校卒業
- 東京大学法学部卒業
- 日本銀行入行
- 2001年 オックスフォード大学経営大学院（サイード・ビジネススクール）MBA取得
- 2003年 衆院選初当選（民主党公募、比例復活）
- 2007年 世界経済フォーラム Young Global Leaders選出
- 2009年 内閣府大臣政務官（国家戦略室、経済財政、科学技術担当）
- 2021年 落選（比例復活ならず）
- 2024年 比例復活で返り咲き、7期目当選
- 2025年 立憲民主党「幹成会」事務局長

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 日銀出身で経済・金融政策に精通。オックスフォードMBA取得、Young Global Leaders選出と国際的視野を持つ。科学技術・イノベーション政策にも取り組む。

### ② 解決設計力: 4/5
**評価根拠**: 日銀・内閣府での実務経験。MBA取得による経営学的視点。内閣府大臣政務官として政策立案に関与。

### ③ 実行力: 4/5
**評価根拠**: 7期当選の実績。2021年落選後も地道に活動を続け2024年に返り咲き。岡山2区という接戦区で山下に肉薄（約3,500票差）。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2020年にTwitterで党内決議案を無断公開し総会を混乱させ副代表解任。SNSの使い方で失態。

### ⑤ 一貫性: 4/5
**評価根拠**: 民主党→民進党→国民民主党→立憲民主党と党籍は変遷したが、野党・改革派としての立場は一貫。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2004年に参議院議長の眼鏡を破損させ公開議場戒告処分。議会秩序を乱す行為は政治家として問題。

## 強み・弱み

### 強み
- **経済金融の専門性**: 日銀出身、オックスフォードMBA
- **国際的評価**: Young Global Leaders選出
- **経験豊富**: 衆議院7期、内閣府政務官
- **粘り強さ**: 落選後も活動継続し返り咲き

### 弱み（注意点）
- **懲罰歴**: 2004年参院議長への暴力で戒告処分
- **SNS失態**: 2020年Twitter投稿で副代表解任
- **小選挙区弱さ**: 山下に5連敗
- **比例依存**: 直近当選は比例復活
