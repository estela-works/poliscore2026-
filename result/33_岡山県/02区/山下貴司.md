# 山下貴司（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年9月8日（59歳） |
| 選挙区 | 岡山2区 |
| 当選回数 | 衆議院5期 |
| 主な役職 | 法務大臣（第100代）、衆議院内閣委員長、自民党岡山県連会長 |

## 経歴
- 香川県高松市生まれ、岡山育ち
- 岡山操山高校卒業
- 東京大学法学部卒業
- 検事（東京地検特捜部など）
- 法務省国際刑事企画官
- 2010年 民主党政権に絶望し法務省辞職、弁護士登録
- 2012年 衆院選で初当選（津村啓介を破る）
- 2017年 法務大臣政務官、内閣府大臣政務官
- 2018年 法務大臣就任（当選3回で初入閣）
- 2024年 石破内閣で自民党副幹事長就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 東京地検特捜部検事として犯罪捜査に従事した経験から、法務・司法分野の課題把握能力は高い。憲法改正実現本部事務局長として憲法改正議論を主導。外国人材共生政策にも取り組む。

### ② 解決設計力: 4/5
**評価根拠**: 検事・弁護士としての法的専門性を活かし、法務大臣として入管法改正などに取り組んだ。知的財産戦略調査会長として政策設計にも関与。

### ③ 実行力: 4/5
**評価根拠**: 当選3回で法務大臣に抜擢されるスピード出世。岡山2区で5期連続当選。2024年自民党総裁選では派閥の論理より自らの判断で石破に投票する行動力。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 検事出身で論理的説明能力が高い。法務大臣として国会答弁を多数経験。岡山県連会長として地元での発信も行う。

### ⑤ 一貫性: 4/5
**評価根拠**: 石破派に所属し、2018年・2024年の総裁選で石破を支持。民主党政権への批判から政治家を志し、その姿勢は一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 検事として法の正義を追求した経歴。統一教会関連イベントへの参加報道があるが、未検証情報。裏金問題への関与は確認されていない。

## 強み・弱み

### 強み
- **法務の専門家**: 検事・弁護士・法務大臣の経験
- **スピード出世**: 当選3回で大臣
- **選挙の安定性**: 5期連続当選
- **石破人脈**: 石破派として首相と近い
- **国際経験**: 法務省で国際刑事分野担当

### 弱み（注意点）
- **派閥の逆風**: 石破内閣の支持率低下
- **自民党不信**: 裏金問題で党全体への不信感
- **接戦区**: 津村啓介との差は約3,500票
- **統一教会報道**: イベント参加の指摘あり
