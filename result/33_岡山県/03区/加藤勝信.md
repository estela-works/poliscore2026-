# 加藤勝信（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1955年11月22日 |
| 年齢 | 69歳 |
| 選挙区 | 岡山3区（津山市、真庭市、美作市、新見市等） |
| 当選回数 | 衆議院8回 |
| 主な役職 | 財務大臣、元内閣官房長官、元厚生労働大臣（4回） |

## 経歴
- 東京都出身
- 東京大学経済学部卒業
- 1979年大蔵省入省
- 国税庁倉吉税務署長
- 農林水産大臣秘書官
- 1995年大蔵省退官、加藤六月秘書
- 2003年衆議院議員初当選（岡山5区）
- 川崎医療福祉大学客員教授
- 内閣府大臣政務官、初代内閣人事局長
- 一億総活躍担当大臣、働き方改革担当大臣
- 厚生労働大臣（4回：第2次安倍改造内閣、第4次安倍改造内閣、岸田改造内閣等）
- 内閣官房長官（菅義偉内閣）
- 自民党総務会長
- 財務大臣・金融担当大臣（石破内閣）
- 2025年自民党政治制度改革本部長

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 厚生労働大臣として働き方改革を推進、一億総活躍担当大臣として少子高齢化対策を主導。財務大臣として「経済あっての財政」を掲げ、デフレ脱却と成長戦略を最優先課題に設定。内閣官房長官として政府全体の課題を俯瞰した経験も持つ。

### ② 解決設計力: 5/5
**評価根拠**: 大蔵省キャリア官僚出身で政策設計能力は極めて高い。厚労大臣として感染症対策、年金制度改革に関与。財務大臣として税制・予算編成を担当。「成長と分配の好循環」など具体的な経済政策フレームワークを提示している。

### ③ 実行力: 5/5
**評価根拠**: 厚生労働大臣4回、内閣官房長官、財務大臣と政権中枢の要職を歴任。8期連続当選で選挙基盤も盤石。2024年衆院選では現職大臣として岡山3区で圧勝（得票率73%超）。政権内での影響力は極めて大きい。

### ④ 説明責任: 2/5
**評価根拠**: 2024年に政治資金収支報告書の3850万円修正問題が発覚。「2と書くべき所を4と記載」「収入の重複計上」と説明するも、複数年にわたる事務的ミスへの批判は免れない。高利益率（85%超）の政治資金パーティーも問題視された。

### ⑤ 一貫性: 4/5
**評価根拠**: 2003年初当選以来、自民党一筋で活動。社会保障政策を軸に厚労大臣を4度務めるなど専門分野を確立。官房長官、財務大臣と要職を歴任し、党内での立ち位置も安定している。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 政治資金収支報告書の3850万円修正は「事務的ミス」との説明だが、複数年にわたる不備は管理体制の問題を示す。高利益率パーティーの常態化も政治資金の透明性への疑念を招く。自民党総裁選で裏金議員からの推薦を受けた点も批判された。

## 強み・弱み

### 強み
- **圧倒的な閣僚経験**: 厚労相4回、官房長官、財務相と政権中枢
- **政策立案能力**: 大蔵省キャリア出身の高い専門性
- **盤石な選挙基盤**: 岡山3区で8期連続当選
- **幅広い人脈**: 党内各派閥とのバランス感覚

### 弱み（注意点）
- **政治資金問題**: 3850万円の収支報告書修正
- **高利益率パーティー**: 政治資金への批判
- **裏金議員との関係**: 総裁選推薦人問題
- **高齢化**: 69歳で長期的な政治活動に懸念
