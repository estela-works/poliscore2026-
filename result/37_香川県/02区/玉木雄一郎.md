# 玉木雄一郎（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1969年5月1日 |
| 年齢 | 56歳 |
| 選挙区 | 香川2区（坂出市、丸亀市、さぬき市、東かがわ市、三豊市など） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 国民民主党代表、元財務省職員、元希望の党代表 |

## 経歴
- 1969年5月1日：香川県大川郡寒川町（現さぬき市）生まれ
- 1988年：香川県立高松高校卒業
- 1993年：東京大学法学部卒業、大蔵省入省
- 1997年：米国ハーバード大学大学院（ケネディースクール）修了
- 2005年：財務省退職、第44回衆院選に出馬、惜敗
- 2009年：第45回衆院選で初当選（民主党）
- 2017年：希望の党共同代表、その後代表
- 2018年5月：国民民主党結成、代表就任
- 2024年10月：第50回衆院選で6選、党は7議席→28議席に躍進
- 2024年11月：不倫問題が報道され謝罪、役職停止3ヶ月処分

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 「手取りを増やす」をスローガンに103万円の壁問題、ガソリン暫定税率廃止など明確な政策課題を設定。30年間動かなかった税制議論を動かした。野党として政策課題設定能力は極めて高い。

### ② 解決設計力: 4/5
**評価根拠**: 東大法学部卒、大蔵省出身、ハーバード大学院修了と政策立案の専門能力は極めて高い。103万円の壁引き上げ、ガソリン暫定税率廃止など具体的な政策を設計し与党と交渉。

### ③ 実行力: 4/5
**評価根拠**: 香川2区で6回連続当選（2009年以降負けなし）。2024年衆院選で党を7議席→28議席に躍進させた。野党候補として全国最大の得票率を記録。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: YouTubeでの発信力、テレビ出演での政策説明は上手い。ただし不倫問題発覚後の説明は「妻子ある身で心の弱さ」と述べるにとどまり、説明責任を十分に果たしたとは言い難い。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→希望の党→国民民主党と政党は変遷。「手取りを増やす」路線は一貫しているが、野党として与党批判と是々非々の姿勢のバランスに揺れがある。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2024年11月、元グラビアアイドル・小泉みゆきとの不倫が発覚。党から役職停止3ヶ月処分。不倫相手を参院選候補にしようとしていたとの報道も。党躍進直後のスキャンダルで信頼を失った。

## 強み・弱み

### 強み
- **政策発信力**：「手取りを増やす」で国民的支持獲得
- **政策専門性**：財務省・ハーバード出身の政策立案能力
- **選挙の強さ**：香川2区で6連勝、野党最大得票率
- **党の躍進**：7議席→28議席の4倍増を実現
- **103万円の壁**：30年間動かなかった税制を動かした

### 弱み（注意点）
- **不倫問題**：2024年11月に発覚、役職停止処分
- **不倫相手の政界進出計画**：批判を受ける
- **党内統治**：スキャンダル後の党内信頼に課題
- **与野党の板挟み**：是々非々路線への批判も
