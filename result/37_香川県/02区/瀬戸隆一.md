# 瀬戸隆一（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年8月2日 |
| 年齢 | 60歳 |
| 選挙区 | 香川2区（坂出市、丸亀市、さぬき市、東かがわ市、三豊市など） |
| 当選回数 | 衆議院4回 |
| 主な役職 | 復興副大臣兼内閣府副大臣、元財務大臣政務官、麻生派 |

## 経歴
- 1965年8月2日：香川県坂出市生まれ
- 坂出市立西部小学校、坂出市立坂出中学校卒業
- 1984年：香川県立丸亀高等学校卒業
- 1989年：大阪府立大学工学部経営工学科卒業
- 1991年：東京工業大学大学院理工学研究科修了
- 総務省職員として勤務
- 2012年8月：総務省退職、自民党香川2区支部長
- 2012年12月：第46回衆院選で比例復活初当選（玉木に敗北）
- 2013年：麻生派（為公会）入会
- 2014年：第47回衆院選で比例復活2選（玉木に敗北）
- 2017年：第48回衆院選で落選（比例復活もならず）
- 2021年：第49回衆院選で落選
- 2023年1月：後藤田正純辞職に伴い繰り上げ当選
- 2023年9月：財務大臣政務官（第2次岸田第2次改造内閣）
- 2024年10月：第50回衆院選で比例復活4選（玉木に敗北）
- 2025年10月：復興副大臣兼内閣府副大臣（高市内閣）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 総務省出身でIT・情報通信政策に精通。復興副大臣として復興政策を担当。「若い人たちが将来に希望の持てる日本」を掲げる。ただし独自の政策課題設定は玉木に比べ弱い。

### ② 解決設計力: 3/5
**評価根拠**: 大阪府立大学工学部→東京工業大学大学院と理系の高学歴。総務省での行政経験。財務大臣政務官として財政政策にも関与。技術系の視点を持つ。

### ③ 実行力: 2/5
**評価根拠**: 香川2区で玉木に6連敗。2017年、2021年は比例復活もできず落選。2023年は繰り上げ当選。選挙での実行力に大きな課題がある。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 「賃金を上げていく、それができるのは自公政権だけ」と明確に主張。ただし玉木の発信力に比べると存在感が薄い。

### ⑤ 一貫性: 3/5
**評価根拠**: 2012年から香川2区で一貫して自民党から出馬。麻生派に所属し党内での立場は安定。政策的な独自色は見えにくい。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段の倫理的問題は報じられていない。副大臣、政務官として政府の要職を経験。

## 強み・弱み

### 強み
- **政府経験**：復興副大臣、財務大臣政務官
- **理系専門性**：工学部・大学院の技術的視点
- **総務省経験**：IT・情報通信政策の知見
- **麻生派所属**：党内での安定した立場

### 弱み（注意点）
- **玉木に6連敗**：小選挙区で勝てない
- **比例・繰り上げ依存**：自力での当選力不足
- **2回の落選経験**：2017年、2021年
- **政策発信力の弱さ**：玉木に比べ存在感が薄い
