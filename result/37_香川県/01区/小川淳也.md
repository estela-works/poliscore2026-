# 小川淳也（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1971年4月18日 |
| 年齢 | 54歳 |
| 選挙区 | 香川県1区 |
| 当選回数 | 衆議院7回 |
| 主な役職 | 衆議院国家基本政策委員長、元立憲民主党幹事長、元総務大臣政務官 |

## 経歴
- 香川県高松市松並町出身
- 両親は美容室経営
- 高松市立香東中学校卒業
- 香川県立高松高等学校卒業
- 東京大学法学部卒業（1994年）
- 1994年自治省（現総務省）入省
- 沖縄県庁に赴任、米兵少女暴行事件の県民総決起大会に参加
- 2003年総務省退職
- 2005年衆議院議員初当選（比例復活）
- 総務大臣政務官（鳩山・菅内閣）
- 立憲民主党幹事長（第4代）兼総合選挙対策事務総長
- 立憲民主党政務調査会長
- 決算行政監視委員長
- 2026年中道改革連合に参加
- ドキュメンタリー映画「なぜ君は総理大臣になれないのか」「香川1区」の主人公

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 財政健全化、政治改革、社会保障制度改革など構造的課題に取り組む。「政治不信の解消」を一貫して訴える。統計不正問題の追及など国会での問題提起力。

### ② 解決設計力: 4/5
**評価根拠**: 東大法学部卒、総務省での官僚経験。総務大臣政務官として政策立案に関与。政調会長として党の政策立案を主導。

### ③ 実行力: 4/5
**評価根拠**: 7期連続当選。立憲民主党幹事長として党運営。小選挙区では平井卓也と激戦を繰り広げ、2021年には小選挙区で勝利。

### ④ 説明責任・コミュニケーション力: 5/5
**評価根拠**: ドキュメンタリー映画「なぜ君は総理大臣になれないのか」「香川1区」で17年間の政治活動を全て公開。政治家として異例の透明性。国会質疑でも高い評価。

### ⑤ 一貫性: 4/5
**評価根拠**: 民主党→民進党→希望の党→立憲民主党→中道改革連合と党派は変遷するが、政治改革・財政健全化という政策軸は一貫。

### ⑥ 倫理観・権力の扱い方: 5/5
**評価根拠**: スキャンダルは確認されていない。映画で私生活まで公開する透明性。沖縄での原体験から「弱者の側に立つ」姿勢を維持。権力批判を続ける姿勢。

## 強み・弱み

### 強み
- **透明性**: 映画で17年間の政治活動を全公開
- **専門性**: 東大法学部、総務省出身
- **知名度**: 映画により全国的に知名度向上
- **選挙力**: 2021年小選挙区勝利
- **党での地位**: 元幹事長、政調会長
- **非世襲**: 美容室経営の家庭から政治家に

### 弱み（注意点）
- **党派変遷**: 複数の政党を移動
- **激戦区**: 平井卓也との接戦が続く
- **野党**: 与党経験は限定的
