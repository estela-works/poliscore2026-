# 高市早苗（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年3月7日（63歳） |
| 選挙区 | 奈良2区 |
| 当選回数 | 衆議院10期 |
| 主な役職 | 内閣総理大臣（第104代）、経済安保担当大臣、総務大臣（3回） |

## 経歴
- 奈良県奈良市生まれ
- 神戸大学経営学部卒業
- 松下政経塾5期生
- 米国議会スタッフ経験
- 1993年 衆院選初当選（新進党）
- 経済産業副大臣
- 科学技術政策担当大臣（第1次安倍内閣）
- 総務大臣（2014-2017年、2022-2023年）
- 経済安全保障担当大臣（2022-2024年）
- 2024年9月 自民党総裁選出馬、決選投票で石破茂に敗北
- 2024年10月 10選（128,554票）
- 2025年11月 内閣総理大臣就任（日本初の女性首相）

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 経済安全保障という新しい政策分野を担当大臣として確立。重要経済安保情報保護法を成立させた。科学技術・宇宙政策にも精通。

### ② 解決設計力: 5/5
**評価根拠**: 松下政経塾出身、米国議会での実務経験。総務大臣として通信・放送政策、経済安保大臣として法整備を主導。政策設計能力は極めて高い。

### ③ 実行力: 5/5
**評価根拠**: 10期連続当選、日本初の女性首相就任。総裁選で決選投票まで進む党内支持を獲得。128,554票で圧勝の選挙力。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 2016年総務相時代の「電波停止」発言で報道の萎縮懸念を招いた。発言が波紋を呼ぶことが多く、丁寧な説明に課題。

### ⑤ 一貫性: 4/5
**評価根拠**: 保守派として一貫した主張。ただし新進党→自民党への移籍、離婚・再婚の私生活など変遷もある。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2024年総裁選で政治団体が8000万円超の宣伝費を支出。「電波停止」発言は権力の行使に関する姿勢に疑問を呈された。

## 強み・弱み

### 強み
- **初の女性首相**: 歴史的な地位
- **政策専門性**: 経済安保、科学技術、通信
- **選挙の強さ**: 10期連続当選
- **党内支持**: 総裁選決選投票進出
- **実務経験**: 大臣複数回歴任

### 弱み（注意点）
- **電波停止発言**: 報道の萎縮懸念を招いた
- **総裁選資金**: 8000万円超の宣伝費
- **発言の波紋**: 批判を呼ぶことが多い
- **強硬姿勢**: 批判への対応が硬い
