# 川戸康嗣（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年頃（49歳） |
| 選挙区 | 奈良3区 |
| 当選回数 | なし（新人・落選） |
| 主な役職 | 立憲民主党奈良3区総支部長、教育コンサルティング会社経営 |

## 経歴
- 兵庫県三田市出身
- 臨床工学技士として働きながら奈良県立大学商学コース（夜間）卒業
- 医療機器メーカー勤務
- 教育コンサルティング会社経営（大阪府）
- 青果市場、倉庫運搬、医療・介護、ワクチン接種会場誘導など多様な職場経験
- 2021年10月 衆院選大阪18区出馬（3位落選、得票率11%）
- 2024年10月 衆院選奈良3区出馬（落選）※国政3度目の挑戦

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「政治とカネの問題が政治不信を招いている」と明確に問題を指摘。「今の少子化に歯止めをかけるうえで高等教育の無償化は欠かせない」と具体的な課題設定。

### ② 解決設計力: 4/5
**評価根拠**: 臨床工学技士として医療現場の知識、教育コンサルティング会社経営の実務経験。高等教育無償化という具体的な政策提案。夜間大学で学びながら働いた経験から現実的な視点を持つ。

### ③ 実行力: 2/5
**評価根拠**: 国政選挙3度目の挑戦だが、いずれも落選。2021年大阪18区では得票率11%と苦戦。選挙での結果を出せていない。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 「子どもたちの笑顔と未来のために」と分かりやすいメッセージ。多様な職場経験（青果市場、医療介護、ワクチン会場など）から庶民目線での訴えができる。

### ⑤ 一貫性: 4/5
**評価根拠**: 立憲民主党からの出馬を続け、高等教育無償化・減税という主張で一貫。国政挑戦を諦めずに3度挑戦する姿勢。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道なし。臨床工学技士として医療現場で働いた経験、様々な職場を経験したクリーンな経歴。

## 強み・弱み

### 強み
- **多様な職場経験**: 医療、教育、物流など幅広い現場を知る
- **庶民目線**: 働きながら夜間大学卒業、多様な仕事経験
- **教育への関心**: 高等教育無償化を一貫して訴え
- **諦めない姿勢**: 3度の国政挑戦

### 弱み（注意点）
- **選挙での苦戦**: 3度挑戦していずれも落選
- **国政経験なし**: 政策実現の実績がない
- **知名度不足**: 全国的な認知度がない
- **選挙区変更**: 大阪18区から奈良3区へ移動
