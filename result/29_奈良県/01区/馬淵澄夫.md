# 馬淵澄夫（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年8月23日 |
| 年齢 | 64歳 |
| 選挙区 | 奈良県1区 |
| 当選回数 | 衆議院8回 |
| 主な役職 | 元国土交通大臣、元内閣総理大臣補佐官、元首相補佐官（東日本大震災対応）、立憲民主党国会対策委員長代理 |

## 経歴
- 奈良県奈良市出身
- 横浜国立大学工学部土木工学科卒業
- 三井建設勤務
- 1999年奈良県議会議員当選
- 2003年衆議院議員初当選（民主党）
- 国土交通副大臣（鳩山内閣）
- 2010年国土交通大臣（菅内閣）
- 内閣総理大臣補佐官（東日本大震災対応）
- 耐震偽装問題追及で注目
- 2024年衆院選で落選、比例復活も逃す
- 2026年衆院選に中道改革連合から立候補予定

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 耐震偽装問題の追及、八ッ場ダム問題、東日本大震災対応など重要課題に取り組んできた。土木工学の専門知識を活かしたインフラ政策に強み。「コンクリートから人へ」政策の推進。

### ② 解決設計力: 4/5
**評価根拠**: 横浜国立大学工学部卒、三井建設での実務経験。技術者としての専門知識を政策に活かす。国交大臣として具体的な政策立案を経験。

### ③ 実行力: 4/5
**評価根拠**: 8期の国政経験。国土交通大臣、首相補佐官として重要ポストを歴任。東日本大震災では原発対応の補佐官として現場対応。2024年落選も再挑戦。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 耐震偽装問題での国会質疑が高く評価された。テレビ討論での論戦力に定評。技術的な問題を分かりやすく説明する能力。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→民進党→希望の党→無所属→立憲民主党→中道改革連合と党派を変遷。ただし政策的には一貫してリベラル寄りの立場を維持。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 大きなスキャンダルは確認されていない。耐震偽装問題など権力の監視・追及に積極的。権力側にいた時期も透明性を重視した姿勢。

## 強み・弱み

### 強み
- **専門性**: 土木工学の知識、建設業界での実務経験
- **豊富な経験**: 8期の国政経験、閣僚経験
- **追及力**: 耐震偽装問題での国会質疑
- **危機管理**: 東日本大震災対応の実績
- **知名度**: 地元での長年の活動

### 弱み（注意点）
- **2024年落選**: 前回選挙で小選挙区・比例とも落選
- **党派変遷**: 複数の政党を渡り歩いた経緯
- **高齢化**: 64歳、若さでの訴求は難しい
- **与党経験の古さ**: 閣僚経験は2010-2011年
