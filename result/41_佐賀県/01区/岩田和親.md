# 岩田和親（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1973年9月20日 |
| 年齢 | 52歳 |
| 選挙区 | 佐賀県1区 |
| 当選回数 | 衆議院5回（うち4回比例復活） |
| 主な役職 | 内閣府副大臣、元経済産業副大臣、元防衛大臣政務官 |

## 経歴
- 佐賀県佐賀市出身
- 佐賀県立佐賀西高等学校卒業
- 九州大学法学部卒業
- 大前研一の秘書を務める
- 1999年佐賀県議会議員初当選（3期）
- 2010年自民党佐賀県第1選挙区支部長就任
- 2012年衆議院議員初当選（原口一博に勝利）
- 2014年・2017年・2021年衆院選で原口に敗北、比例復活
- 2019年防衛大臣政務官
- 2021年経済産業大臣政務官兼内閣府大臣政務官
- 2023年経済産業副大臣兼内閣府副大臣
- 2024年衆院選で比例復活5選
- **旧統一教会関連団体との関係が指摘**

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 経済産業政策、国防政策を中心に活動。経済産業部会長、国防部会長など党内役職を歴任。党の方針に沿った政策活動。

### ② 解決設計力: 3/5
**評価根拠**: 九州大学法学部卒、大前研一秘書という経歴。県議3期、衆議院5期の経験。経済産業副大臣として政策立案に携わった実績あり。

### ③ 実行力: 3/5
**評価根拠**: 5期連続当選も、選挙区での勝利は2012年の1回のみ。4回は比例復活。原口一博との直接対決では劣勢が続く。副大臣級の役職は歴任。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 統一教会との関係についてアンケートには「日韓トンネル実現九州連絡協議会に祝電を出した」と回答。一定の説明は行っている。

### ⑤ 一貫性: 4/5
**評価根拠**: 県議時代から一貫して自民党所属。経済産業・防衛分野での政策活動も継続的。地元佐賀で着実に活動を続けている。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: **旧統一教会関連団体との複数の接点**。2021年衆院選で久留米家庭教会から選挙支援。2022年に統一教会関連「天宙平和連合」のウェビナーにオンライン参加。「日韓トンネル実現九州連絡協議会」に祝電。

## 強み・弱み

### 強み
- **副大臣経験**: 経済産業副大臣、内閣府副大臣
- **地方議員経験**: 県議3期
- **専門性**: 経済産業・防衛分野
- **継続性**: 5期連続当選
- **党内基盤**: 自民党で一貫した活動

### 弱み（注意点）
- **比例復活常連**: 選挙区での勝利は1回のみ
- **原口に劣勢**: 直接対決で4連敗
- **統一教会関係**: 選挙支援、イベント参加、祝電
