# 眞野哲（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年6月14日 |
| 年齢 | 63歳 |
| 選挙区 | 岐阜5区（多治見市、土岐市、瑞浪市、恵那市、中津川市） |
| 当選回数 | 衆議院1回（比例復活） |
| 主な役職 | 全国悪質運転ZEROの会代表、日本福祉大学実務家教員 |

## 経歴
- 愛知県名古屋市生まれ
- 名古屋第一高等学校卒業
- 日本ビジネススクール名古屋校経営実務家卒業
- 日本福祉大学大学院修了
- 調理師免許、福祉用具専門相談員資格保有
- 名古屋市で介護事業所・鍼灸治療院を経営
- 2011年長男をひき逃げ事故で失う
- 「全国悪質運転ZEROの会」代表として活動
- 2023年立憲民主党から岐阜5区擁立発表
- 2024年衆院選で比例復活当選（初当選）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 長男をひき逃げで失った経験から「悪質運転撲滅」という明確な社会課題を設定し、約4万9千人の署名を集めて危険運転致死罪への訴因変更を求めた。介護事業経営者として高齢者福祉の課題にも精通。個人的体験を社会問題へ昇華させる能力がある。

### ② 解決設計力: 3/5
**評価根拠**: 「全国悪質運転ZEROの会」代表として交通被害者支援活動を組織化。介護事業経営の実務経験から福祉政策への理解も深い。ただし国会議員1期目であり、国政レベルでの政策設計実績はこれから積み上げる段階。

### ③ 実行力: 3/5
**評価根拠**: 選挙区では古屋圭司に敗れたものの、比例復活で初当選を果たした。12期のベテラン相手に一定の票を獲得。4万9千人の署名活動など市民運動での実行力は実証済み。国会での実績はこれから。

### ④ 説明責任: 4/5
**評価根拠**: 長男を失った経験と政治を志した理由を明確に公開している。公式サイトやSNSで活動を発信し、有権者への説明姿勢は積極的。介護事業者としての専門性も説得力を持って伝えている。

### ⑤ 一貫性: 4/5
**評価根拠**: 悪質運転撲滅という軸で10年以上活動を継続。立憲民主党から出馬し一貫した野党としての立場。介護事業運営も長年継続しており、言動と行動の一貫性が高い。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 交通被害者遺族として法の不備に立ち向かった経緯は、正義感と倫理観の高さを示す。特にスキャンダルや疑惑はない。介護・福祉分野での活動は社会的弱者支援への姿勢を表す。

## 強み・弱み

### 強み
- **明確な原点**: 長男の死という強烈な動機と説得力
- **専門性**: 介護事業経営者、日本福祉大学教員としての福祉政策への知見
- **市民運動経験**: 署名活動など草の根活動の実績
- **当事者性**: 交通被害者遺族としての発言力

### 弱み（注意点）
- **国政経験が浅い**: 1期目で実績を積む段階
- **選挙区での劣勢**: 古屋圭司に比べ知名度・組織力で劣る
- **比例復活**: 選挙区では勝利できていない
- **政策の幅**: 悪質運転・福祉以外の政策分野での発信は限定的
