# 山田良司（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1960年頃（推定） |
| 年齢 | 64歳 |
| 選挙区 | 岐阜5区（多治見市、土岐市、瑞浪市、恵那市、中津川市） |
| 当選回数 | 衆議院1回（2009年比例）、下呂市長1期、下呂町議1期 |
| 主な役職 | 日本維新の会岐阜5区支部長、岐阜県総支部副幹事長 |

## 経歴
- 1991-1995年下呂町議会議員
- 2000年衆院選（岐阜4区）民主党公認で出馬、落選
- 2003年衆院選（岐阜4区）民主党公認で出馬、落選
- 2004-2008年初代下呂市長
- 2007年市議会で辞職勧告決議可決
- 2008年下呂市長選で落選
- 2009年衆院選で民主党比例当選（1期）
- 2021年衆院選（岐阜5区）維新公認で出馬、落選（得票率5.86%）
- 2022年日本維新の会岐阜県総支部幹事長
- 2024年衆院選（岐阜5区）落選（得票率13.09%）
- 2025年岐阜県総支部副幹事長

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「最高裁移転」を掲げたイベントを主催し鳩山由紀夫元首相を招くなど独自の活動はあるが、主流の政策議論とは離れている。維新の政策に沿った活動が中心で、独自の課題設定は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 下呂市長として行政経験はあるが、市議会から辞職勧告を受けるなど運営に問題があった。国会議員1期（2009-2012年）の実績も限定的で、具体的な政策立案の成果は確認しにくい。

### ③ 実行力: 2/5
**評価根拠**: 2021年衆院選は得票率5.86%で供託金没収、2024年も13.09%で落選と、選挙での結果を出せていない。下呂市長再選も果たせず、2008年以降は落選が続いている。粘り強く挑戦を続ける姿勢はあるが成果に結びついていない。

### ④ 説明責任: 2/5
**評価根拠**: 維新の会を通じた活動はあるが、個人としての政策発信や活動報告は限定的。2007年の辞職勧告決議に至った経緯についての説明も十分とは言えない。

### ⑤ 一貫性: 2/5
**評価根拠**: 民主党→日本維新の会と政党を変更。2009年の政権交代時は民主党から出馬し、その後維新に移籍。政策的な軸が明確でなく、時流に合わせた動きにも見える。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 下呂市長時代に県議選で現職を支持せず新人を応援したことで市議会から辞職勧告決議を受けた。政治判断の妥当性はともかく、議会との関係悪化は権力運営に問題があったことを示す。

## 強み・弱み

### 強み
- **地方行政経験**: 初代下呂市長として行政トップの経験
- **国会経験**: 2009年当選で衆議院議員経験あり
- **粘り強さ**: 複数回の落選にも関わらず挑戦を継続
- **維新の組織力**: 岐阜県総支部幹事長を務め党内での位置づけ

### 弱み（注意点）
- **選挙での連敗**: 2008年以降、当選できていない
- **辞職勧告の過去**: 市長時代の議会との対立
- **得票率低迷**: 2021年は供託金没収ラインの惨敗
- **政党移動**: 民主党から維新への移籍で一貫性に疑問
- **知名度不足**: 古屋圭司の壁は厚く浸透に苦戦
