# 仙田晃宏（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1982年10月11日 |
| 年齢 | 42歳 |
| 選挙区 | 岐阜3区（各務原市・関市・美濃市・羽島市など） |
| 当選回数 | 衆議院1回 |
| 主な役職 | 国民民主党国会対策副委員長 |

## 経歴
- 愛知県丹羽郡大口町出身
- 名古屋中学校・高等学校卒業
- 2005年：明治大学政治経済学部卒業
- 卒業後：NTTデータグループ入社
- 2023年：国会議員秘書として勤務
- 2024年3月：国民民主党岐阜県第3区総支部長に就任
- 2024年10月：第50回衆院選で岐阜3区から出馬、比例復活で初当選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 祖父が町議であったことから大学時代に政治家を志した。NTTデータでのIT業界経験を持つが、独自の課題設定は確認できない。国民民主党の政策（消費税減税、手取り増加など）に沿った主張が中心。

### ② 解決設計力: 2/5
**評価根拠**: IT企業勤務の経験はあるが、政策設計の具体的実績は乏しい。候補者アンケートでは「財政出動優先」「消費税下げるべき」「教育無償化」など方向性を示すが、詳細な制度設計は党の方針に依存。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で比例復活当選したことは評価できる。ただし小選挙区では落選しており、選挙での浸透力に課題。当選後、国会対策副委員長に就任したが、実績を評価するには時期尚早。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 新人議員として活動を開始したばかりで、説明責任を評価する材料が乏しい。候補者アンケートには回答しているが、独自の発信力は確認できない。

### ⑤ 一貫性: 3/5
**評価根拠**: 大学時代から政治家を志し、国会議員秘書を経て国政に挑戦した経緯は一貫している。国民民主党の政策路線（現実的な野党、手取り増加）に賛同して活動。安全保障では「反撃能力」賛成、「防衛費GDP比2%」支持と現実主義的立場。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。民間企業経験を経て政界入りしており、クリーンなイメージを維持。

## 強み・弱み

### 強み
- **若手・新人**：42歳と比較的若く、フレッシュな視点
- **IT業界経験**：NTTデータグループでの勤務経験、デジタル政策への知見
- **現実主義的立場**：防衛政策で現実路線、国民民主党の中道路線に合致

### 弱み（注意点）
- **政治経験の浅さ**：国会議員秘書経験も1年程度と短い
- **選挙基盤**：小選挙区では敗北、比例復活での当選
- **知名度不足**：全国的・地元でも知名度が低い
- **独自性の欠如**：政策が党の方針に依存し個人としての特徴が見えにくい
