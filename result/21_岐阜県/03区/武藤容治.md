# 武藤容治（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1955年10月18日 |
| 年齢 | 69歳 |
| 選挙区 | 岐阜3区（各務原市・関市・美濃市など） |
| 当選回数 | 衆議院6回 |
| 主な役職 | 経済産業大臣（第31・32代）、GX実行推進担当大臣、原子力経済被害担当大臣 |

## 経歴
- 岐阜県生まれ
- 慶應義塾大学商学部卒業
- 1978年：富士写真フイルム（現・富士フイルム）入社
- 1988年：武藤嘉商事入社
- 1990年：通商産業大臣秘書
- 1996年：武藤嘉商事代表取締役社長
- 2003年：岐阜県肥料商業協同組合代表理事
- 2004年：菊川株式会社代表取締役社長
- 2005年：第44回衆院選で初当選（父・武藤嘉文の地盤を継承）
- 2008年：麻生派（為公会）入り
- 2009年：衆院選で落選
- 2012年：衆院選で国政復帰
- 2014年：総務大臣政務官
- 2016年：外務副大臣
- 2017年：経済産業副大臣
- 2024年10月：経済産業大臣に就任（石破内閣）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 経済産業大臣として「賃上げと投資が牽引する成長型経済」「エネルギー安全保障」「半導体産業振興」など重要課題を担当。ただしこれらは政府方針に沿った課題であり、独自の問題提起というよりは担当閣僚としての職務遂行。

### ② 解決設計力: 3/5
**評価根拠**: 経産副大臣、経産大臣として政策立案に携わってきた。半導体支援（ラピダスへの9,200億円支援）、エネルギー政策、中小企業の取引適正化など具体的な政策を推進。ただし官僚組織の支援を受けての実施であり、独自の設計力は判断が難しい。

### ③ 実行力: 4/5
**評価根拠**: 衆議院6期当選、総務政務官→外務副大臣→経産副大臣→経産大臣と着実にキャリアを積み上げ、父親と同じ経産大臣（旧通産大臣）に就任。麻生派の重鎮として党内基盤も固い。2009年の落選から復活した粘り強さも評価できる。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 経産大臣として定例記者会見で政策説明を行っている。エネルギー政策、物価高対策などについて丁寧な説明を心がけている様子。ただし特段の発信力や説明の巧みさを示す材料は限られる。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党内で一貫して活動し、麻生派に所属。経済産業分野を中心にキャリアを積み、経産副大臣から経産大臣へと専門性を深めてきた。「ものづくり産業の振興」という軸は維持している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特段のスキャンダルや問題発言は確認されていない。世襲議員であることへの批判はあり得るが、それ自体は倫理的問題ではない。旧統一教会との関係についても報道は確認できなかった。

## 強み・弱み

### 強み
- **閣僚経験**：経済産業大臣として政権中枢で活動
- **専門性**：経済産業分野でのキャリアを積み上げた専門知識
- **地盤の強さ**：父から継承した岐阜3区での盤石な支持基盤
- **党内基盤**：麻生派の重鎮として党内での影響力

### 弱み（注意点）
- **世襲議員**：父・祖父・曾祖父と4代続く政治家一家
- **独自色の薄さ**：政府・党の方針に沿った活動が中心で独自の政策提案が見えにくい
- **選挙での危うさ**：2009年に一度落選している
