# 田中克和（減税日本・ゆうこく連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年3月23日（40歳） |
| 選挙区 | 愛知県5区（名古屋市中村区・中川区・清須市） |
| 当選回数 | なし（新人） |
| 主な役職 | 元名古屋市長特別秘書、河村たかし衆院議員公設秘書 |

## 経歴
- 愛知県一宮市出身
- 三重大学教育学部卒業
- ソフトバンク社員
- 2018年6月 名古屋市長特別秘書就任（河村たかし市長の側近）
- 2024年10月 名古屋市長特別秘書を辞職、衆院選出馬
- 2024年10月 衆院選愛知5区から日本保守党公認で出馬（落選）
- 2024年11月 河村たかし衆院議員の公設第一秘書就任
- 2025年7月 参院選愛知選挙区出馬予定（減税日本公認、日本保守党推薦）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「政治とカネの問題」を選挙の一番のテーマに掲げた。河村たかし市長の下で市政改革に携わった経験から行政課題への認識はあるが、独自の政策課題設定は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 情報不足。市長特別秘書として6年半、政策に関する情報収集や分析、関係機関との調整を担当した経験はあるが、自身の政策設計実績は確認できない。

### ③ 実行力: 1/5
**評価根拠**: 2024年衆院選が初の選挙挑戦で落選。中村区9.2%、中川区8.1%、清須市6.6%の得票率で5人中下位。政策を実行する場を得られていない。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 選挙戦では「庶民の立場に立ち、幸せな社会つくる」「カネにクリーンな政治」を訴えた。中日新聞のインタビューで政策を発信しているが、全国的な知名度は低い。

### ⑤ 一貫性: 4/5
**評価根拠**: 河村たかし氏の側近として6年半活動し、その方針に沿って政治活動を継続。衆院選落選後も河村氏の公設秘書として活動を続け、参院選にも挑戦予定。政治的立ち位置は一貫している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道はない。「カネにクリーンな政治」を掲げており、裏金問題への批判姿勢を明確にしている。若手として清廉な政治活動を続けている印象。

## 強み・弱み

### 強み
- **若さ**: 40歳の若手候補
- **行政経験**: 市長特別秘書として6年半の経験
- **河村たかしとの関係**: 有力政治家の最側近
- **挑戦の継続**: 衆院選落選後も参院選に挑戦予定
- **クリーンな印象**: スキャンダルなし、「カネにクリーンな政治」を標榜

### 弱み（注意点）
- **選挙経験の浅さ**: 2024年衆院選が初挑戦で落選
- **知名度の低さ**: 全国的にも地元でも認知度が限定的
- **党の調整**: 参院選では保守党公認を得られず推薦に（減税日本公認）
- **政策の独自性**: 河村氏の方針に沿った活動が中心
