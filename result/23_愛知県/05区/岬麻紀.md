# 岬麻紀（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年12月26日（55歳） |
| 選挙区 | 愛知県5区（名古屋市中村区・中川区・清須市） |
| 当選回数 | 衆議院1回（現在落選中） |
| 主な役職 | 日本維新の会愛知5区支部長、元衆議院厚生労働委員 |

## 経歴
- 愛知県名古屋市千種区生まれ、名東区育ち
- 名古屋市立高針小学校、高針台中学校卒業
- 愛知県立東郷高等学校卒業
- 愛知大学法学部2部法律学科中退（出産のため）
- 名古屋市千種区の歯科医院助手
- 名古屋大学医学部付属病院事務補佐員（パート職員）
- 1994-95年 博覧会パビリオンコンパニオン
- 株式会社ポリッシュ設立
- 広告代理店設立に携わる
- 2011年 「国民的美魔女コンテスト」ファイナリスト
- 2012年 維新政治塾1期生
- 2012年12月 衆院選水戸市落選（みんなの党）
- 2013年 都議選世田谷区落選
- 2017年 都議選町田市落選
- 2019年 大阪市議会議員選挙東成区初当選（4票差）
- 2019年7月 参院選愛知県選挙区落選
- 2021年 衆院選愛知5区で比例復活当選
- 2024年10月 衆院選愛知5区で落選（比例復活ならず）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 情報不足。厚生労働委員会に所属し、介護経験（祖父を20年間介護）から社会保障への問題意識を持つ。ただし独自の課題設定についての具体的な情報は確認できない。

### ② 解決設計力: 2/5
**評価根拠**: 情報不足。衆議院議員1期（約3年）、大阪市議1期の経験があるが、具体的な政策設計や条例提案の実績は確認できない。維新の政策に沿った活動が中心と推測。

### ③ 実行力: 2/5
**評価根拠**: 2012年から政治活動を開始し、複数回の落選を経て2019年大阪市議選で初当選（4票差）、2021年衆院選で比例復活当選。しかし2024年は惜敗率61.2%で比例復活できず落選。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 「国民的美魔女コンテスト」ファイナリストとしての知名度、SNS発信を行っている。一方で大阪市議時代に「居住実態」を疑問視する報道があり、説明を求められた経緯がある。

### ⑤ 一貫性: 4/5
**評価根拠**: 2012年の維新政治塾入塾以来、一貫して維新系で活動を続けている。複数回の落選にもめげず挑戦を継続する姿勢は評価できる。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 大阪市議時代に「居住実態疑惑」（大阪に住んでいないのでは）を報道された。本人は反論しているが、疑念を払拭できていない。選挙区を水戸→東京→大阪→愛知と変えていることも指摘される。

## 強み・弱み

### 強み
- **知名度**: 美魔女コンテストファイナリスト、参院選で注目
- **粘り強さ**: 複数回の落選を経て当選を勝ち取った
- **介護経験**: 祖父を20年間介護した実体験
- **維新一筋**: 2012年から維新系で活動継続
- **華道・書道**: 師範・準師範の資格

### 弱み（注意点）
- **居住実態疑惑**: 大阪市議時代の報道
- **選挙区変更**: 水戸→東京→大阪→愛知と移動
- **比例復活できず**: 2024年は惜敗率61.2%で落選
- **政策発信の弱さ**: 具体的な政策実績が見えにくい
