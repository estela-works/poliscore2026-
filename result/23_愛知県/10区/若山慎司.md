# 若山慎司（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1973年7月15日 |
| 年齢 | 52歳（2026年1月報道） |
| 選挙区 | 愛知10区 |
| 当選回数 | 衆院1回 |
| 主な役職 | 内閣府大臣政務官 |

## 経歴
- 岐阜県出身。江崎鉄磨議員の政策秘書として約30年活動し、2024年に後継候補として擁立。第50回衆院選で愛知10区は落選したが比例復活で初当選。名岐道路・一宮PAスマートIC整備、教育無償化、防災減災、経済と外交安全保障の強化などを訴える。

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 物価高への経済対策、子育て・教育、防災、地域インフラなど幅広い課題を提示しているが、構造的な原因分析や優先順位の整理は公開情報では限定的。

### ② 解決設計力: 2/5
**評価根拠**: インフラ整備や教育無償化など方向性は示すものの、財源や制度設計まで踏み込んだ具体像は確認できない。

### ③ 実行力: 1/5
**評価根拠**: 2024年初当選の新人で、議員立法や政策実現の実績は確認できない。政務官就任はあるが、成果を検証できる材料はまだ少ない。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 公式サイトで政策の方向性を示しているが、争点への具体説明や反対意見への応答は十分に確認できない。

### ⑤ 一貫性: 2/5
**評価根拠**: 国政での活動期間が短く、長期的な主張の変遷を十分に追える資料は限られる。秘書時代から掲げる地域インフラや教育・防災の訴えは一貫しているが、検証材料が少ないため低めに評価する。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 重大な不祥事や公的な批判報道は確認できない一方、資金の透明性や説明責任の具体的な検証材料は限られる。慎重に低めの評価とする。

## 強み・弱み

### 強み
- 代議士秘書としての長い実務経験があり、地元課題の把握と調整能力が見込まれる。
- インフラ、防災、教育など生活に近い政策分野を明確に掲げている。

### 弱み（注意点）
- 国政での実績がまだ少なく、政策実現の具体的な成果が見えにくい。
- 政策の実装設計や財源の説明が限定的。
