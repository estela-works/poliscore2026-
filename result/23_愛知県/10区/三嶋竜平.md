# 三嶋竜平（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1978年頃（47歳） |
| 年齢 | 47歳 |
| 選挙区 | 愛知10区（一宮市・岩倉市） |
| 当選回数 | 新人（初挑戦） |
| 主な役職 | 国民民主党愛知県連政策委員 |

## 経歴
- 愛知県豊田市出身
- 2020年4月より約6年間、広島県東広島市にて中小企業・個人事業主支援の拠点でセンター長を務める
- 国民民主党愛知県連で政策委員として活動

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**:
「中小企業を元気にする」「手取りを増やす」という主張は、国民民主党の党是をそのまま述べたもの。一宮市・岩倉市の中小企業が多いという地域特性は認識しているが、独自の課題分析や構造的な問題提起は確認できず。表面的な課題認識に留まる。

### ② 解決設計力: 2/5
**評価根拠**:
中小企業支援センターでの実務経験はあるが、政策としての解決設計を公に示した実績は確認できない。「国民民主党の政策が必要」と述べるのみで、具体的な制度設計や財源論への言及なし。情報不足。

### ③ 実行力: 1/5
**評価根拠**:
政治家としての実行実績がない新人候補。センター長としての民間経験は評価できるが、政策を実現した実績、議員立法、政治的成果はゼロ。新人として構造的に低評価となる。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**:
第一声での発言は確認できるが、詳細な政策説明や記者会見での質疑応答は情報不足。支持者向けの発信が中心で、批判への対応力は未知数。

### ⑤ 一貫性: 2/5
**評価根拠**:
国民民主党の政策（手取りを増やす、中小企業支援）に沿った主張は一貫しているが、長期的な政治活動の軸や過去の発言履歴が確認できず、判断材料が限られる。情報不足。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**:
スキャンダルや問題行動の報道は確認されない。政治資金問題等の疑惑もなし。ただし政治家としての実績がないため、権力の扱い方を評価する材料がなく、中立的評価。

## 強み・弱み

### 強み
- **中小企業支援の実務経験**: 約6年間のセンター長経験で、中小企業・個人事業主の実情を把握
- **地域密着**: 豊田市出身で愛知県の中小企業事情に通じている

### 弱み（注意点）
- **政治経験ゼロ**: 議員活動、政策立案の実績がなく実行力は未知数
- **情報発信不足**: 独自の政策提言や詳細な主張が確認できない
