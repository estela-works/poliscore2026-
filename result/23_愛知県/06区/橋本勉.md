# 橋本勉（減税日本・ゆうこく連合）

## 基本情報
| 項目 | 内容 |

| 年齢 | 72歳 |
| 選挙区 | 愛知県6区 |

| 主な役職 | 元衆議院議員 |

## 経歴
- 岐阜工業高等専門学校卒業
- 東京外国語大学外国語学部印パ学科卒業
- 筑波大学大学院経営・政策科学研究科修了
- 職業：税理士
- 2009年衆院選で民主党から岐阜2区出馬、比例復活で初当選
- 2012年消費増税法案に反対票を投じ、民主党を離党
- 減税日本、日本未来の党、新党大地、国民怒りの声などを経て
- 2026年衆院選で減税日本・ゆうこく連合から愛知6区で出馬
