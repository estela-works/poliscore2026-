# 原田学（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1950年頃（推定） |
| 年齢 | 74歳 |
| 選挙区 | 愛知6区（春日井市・瀬戸市） |
| 当選回数 | なし（新人） |
| 主な役職 | 元瀬戸市議会議員 |

## 経歴
- 瀬戸市議会議員（複数期）
- 2023年瀬戸市議選で引退
- 2024年衆院選（愛知6区）から出馬、落選

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 「30年以上の自民党の誤った経済政策」を批判し、労働者の処遇改善を訴えた。軍事費増大への批判、地球温暖化対策など具体的な課題を設定。ただし政策の具体性は党の方針に沿ったもの。

### ② 解決設計力: 2/5
**評価根拠**: 市議としての地方政治経験はあるが、国政レベルの政策設計能力は未知数。共産党の政策に依存する傾向が見られる。

### ③ 実行力: 2/5
**評価根拠**: 2024年衆院選で26,425票（14.8%）を獲得したが落選。愛知6区は現職の丹羽が圧倒的に強く、共産党新人の当選は困難。一度引退後の公示直前出馬という経緯も影響。

### ④ 説明責任: 3/5
**評価根拠**: 労働政策、軍事費批判、気候変動対策について具体的に説明。「軍事対軍事か外交か」という選択を明確に提示した。

### ⑤ 一貫性: 4/5
**評価根拠**: 共産党で一貫して活動。労働者・生活者の立場からの主張を継続。引退後も党の要請で出馬する姿勢は党への忠誠を示す。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。市議として長年活動し、企業献金を受けない共産党で清廉な政治姿勢を維持。

## 強み・弱み

### 強み
- **市議経験**: 瀬戸市議として地方政治経験
- **労働政策**: 労働者の処遇改善への明確な主張
- **平和政策**: 軍事費批判、外交重視
- **一貫性**: 共産党での長年の活動
- **地元への理解**: 瀬戸市での活動実績

### 弱み（注意点）
- **高齢**: 74歳での国政初挑戦
- **引退後の出馬**: 一度引退していた
- **党勢の限界**: 共産党の組織基盤
- **対立候補の強さ**: 現職7期に対抗困難
- **国政経験なし**: 市議経験のみ
