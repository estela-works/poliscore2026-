# 河村たかし（日本保守党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1948年11月3日 |
| 年齢 | 76歳 |
| 選挙区 | 愛知1区（名古屋市東区・北区・西区・中区） |
| 当選回数 | 衆議院5期（通算） |
| 主な役職 | 日本保守党共同代表、元名古屋市長（4期15年） |

## 経歴
- 愛知県名古屋市出身
- 愛知県立旭丘高等学校卒業
- 一橋大学商学部卒業
- 家業の古紙回収業に従事
- 愛知県議会議員（1期）
- 1993年衆院選で初当選（日本新党）
- 衆議院議員（5期、1993-2009年）
- 2009年名古屋市長選で初当選
- 名古屋市長（4期、2009-2024年）
- 減税日本代表
- 2023年日本保守党共同代表就任（百田尚樹が代表）
- 2024年衆院選で15年ぶりに国政復帰（愛知1区当選）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「減税」を政治活動の核心に据え、市民税10%減税を実現。地方議会改革、議員報酬削減なども主張。庶民感覚での課題設定に定評がある。ただし課題の範囲が限定的との批判も。

### ② 解決設計力: 3/5
**評価根拠**: 一橋大学商学部卒。市長として減税条例、議会改革などを実現したが、政策の体系性よりも話題性重視の傾向。日本保守党の政策は保守的イデオロギー色が強い。

### ③ 実行力: 5/5
**評価根拠**: 名古屋市長4期15年、衆議院議員5期と長期にわたり当選。2024年衆院選では愛知1区で95,613票（46.6%）を獲得し圧勝。日本保守党を国政政党に押し上げた。リコール運動など型破りな行動力は突出している。

### ④ 説明責任: 3/5
**評価根拠**: 名古屋弁を駆使した独特の話術で有権者への発信力は高い。ただし政策の詳細な説明よりもキャッチフレーズ重視の傾向。愛知県知事リコール運動での署名偽造問題では説明責任が問われた。

### ⑤ 一貫性: 4/5
**評価根拠**: 「減税」を30年以上にわたり一貫して主張。市長時代も減税路線を堅持。ただし日本新党→新進党→自由党→民主党→無所属→日本保守党と政党は多く渡り歩いている。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 愛知県知事リコール運動での署名偽造問題に関与が疑われた（本人は否定）。東京五輪で選手のメダルを噛んだ行為が批判を受けた。権力の行使において物議を醸すことが多い。

## 強み・弱み

### 強み
- **圧倒的な知名度**: 名古屋市長15年の実績
- **選挙の強さ**: 愛知1区で圧勝
- **減税の旗手**: 一貫した減税主張
- **発信力**: 独特の話術で有権者を引きつける
- **党の顔**: 日本保守党を国政政党化

### 弱み（注意点）
- **署名偽造問題**: リコール運動での疑惑
- **物議を醸す言動**: メダル噛み事件など
- **高齢**: 76歳で体力面の懸念
- **保守党の限界**: 党勢拡大の持続性
- **政策の偏り**: 減税以外の政策が見えにくい
