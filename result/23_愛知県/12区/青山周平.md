# 青山周平（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1977年4月28日 |
| 年齢 | 47歳（2024年衆院選時） |
| 選挙区 | 愛知12区 |
| 当選回数 | 衆院4回 |
| 主な役職 | 文部科学副大臣、文部科学大臣政務官、内閣府政務官 |

## 経歴
- 法政大学社会学部卒。民間企業勤務と学校法人での園長・事務長経験を経て、2012年に衆院初当選。文科政務官や文科副大臣を歴任し、教育分野の政策に関与してきた。

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 経済・エネルギー・安全保障・教育など広い分野の課題を提示するが、課題の優先順位や構造分析の説明は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 積極投資、原発再稼働、教育改革など方向性は示すが、財源や実施手順まで踏み込んだ具体設計は十分に示されていない。

### ③ 実行力: 3/5
**評価根拠**: 文科副大臣など政府ポストの経験があり、教育機会確保法の成立に関与した実績も示している。大規模改革の主導実績は確認できないが一定の行政経験はある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 公式サイトで政策を提示しているが、争点ごとの詳細説明や反対意見への整理された反論は公開情報では限定的。

### ⑤ 一貫性: 情報不足
**評価根拠**: 長期の政治活動はあるが、政策主張の変更や理由を体系的に検証できる資料が不足。

### ⑥ 倫理観・権力の扱い方: 情報不足
**評価根拠**: 重大な不祥事や説明責任を問う報道の有無を判断できる情報が不足。

## 強み・弱み

### 強み
- 教育分野での行政経験があり、政策実行に携わった実績がある。
- 経済・防災・農業など幅広い政策領域を掲げている。

### 弱み（注意点）
- 政策の具体設計や財源の説明が限定的で、実現手段が見えにくい。
- 近年の政策重点の整理が外部から把握しづらい。
