# 重徳和彦（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年12月 |
| 年齢 | 53歳（2024年衆院選時） |
| 選挙区 | 愛知12区 |
| 当選回数 | 衆院5回 |
| 主な役職 | 立憲民主党 政務調査会長、愛知県連代表、衆院安保委員など |

## 経歴
- 総務省（旧自治省）に入省し各県庁で勤務。新潟県中越地震の救助活動に携わった経験を持つ。2012年に初当選後、衆院で複数期当選し、政務調査会長として物価高や災害復興、社会保障などの政策提案を主導。

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 物価高、災害復興、社会保障やデジタル化など複数の構造課題を示し、2025年の代表質問でも長期停滞の原因を問い直している。

### ② 解決設計力: 3/5
**評価根拠**: 能登復興と物価高対策の総合パッケージを示し、保険証廃止延期や就労支援給付など具体法案の再提出を主導している。一方、制度の実装や財源の全体像はなお詳細が限定的。

### ③ 実行力: 3/5
**評価根拠**: 複数期の国会在職に加え、議員立法の提出実績がある。政府側での大規模政策実現は確認できないが、立法・政策調整の実績は一定程度確認できる。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 政務調査会長として会見や代表質問で論点を説明しており、公的説明の機会は多い。ただし政策の優先順位やトレードオフの丁寧な説明は資料からは限定的。

### ⑤ 一貫性: 2/5
**評価根拠**: 過去に所属政党の変更が複数あり、路線変更の背景や説明が十分に検証できる情報が少ない。

### ⑥ 倫理観・権力の扱い方: 情報不足
**評価根拠**: 重大な不祥事や説明責任を問う報道の有無を判断できる情報が不足。

## 強み・弱み

### 強み
- 政策責任者として立法提案や党内調整を担い、政策形成の実務経験がある。
- 災害復興や物価高対策など、時事性の高い課題を継続的に取り上げている。

### 弱み（注意点）
- 党派変遷があり、長期的な政策一貫性の評価は難しい。
- 具体的な制度設計や実施の成果については検証材料が限られる。
