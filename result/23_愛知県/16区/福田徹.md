# 福田徹（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1982年6月2日（42歳） |
| 選挙区 | 愛知県16区 |
| 当選回数 | 衆議院1回 |
| 主な役職 | 衆議院議員、救急科専門医、MBA |

## 経歴
- 岐阜県可児市出身
- 私立滝中学校・高等学校卒業
- 2007年 三重大学医学部医学科卒業
- 2007年〜 日本赤十字社愛知医療センター名古屋第二病院 初期研修医
- 2009年〜 同院 救急科医師
- 2017年〜 同院 救急科医長
- グロービス経営大学院 経営研究科修了（MBA取得）
- 2022年10月 病院退職、国民民主党愛知県第16区総支部長就任
- 2024年10月 衆院選愛知16区で初当選（65,923票）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 救急医としての現場経験から「医療費の急増が国民生活や経済成長を圧迫」という具体的な課題を設定。科学的根拠に基づく医療政策を提唱し、専門性を活かした独自の課題設定ができている。

### ② 解決設計力: 4/5
**評価根拠**: 「価値の大きい医療と小さい医療を見分け、価値の大きい医療に集中投資する」という具体的なソリューションを提案。医師としての専門知識とMBAでの経営学習を組み合わせた政策設計能力がある。

### ③ 実行力: 4/5
**評価根拠**: 初挑戦の衆院選で新設の愛知16区を制した。保守地盤とされる地域で、裏金問題への逆風を追い風に変え、公明党・立憲民主党の候補を破って当選。2年間の地道な活動が実を結んだ。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 「救急医 福田とおるが、人に優しい医療改革で、命と豊かさを守ります」という明確なメッセージを発信。医療政策の専門性を分かりやすく有権者に伝え、支持を集めた。

### ⑤ 一貫性: 4/5
**評価根拠**: 2022年の政治家転身以来、国民民主党で一貫して活動。医療改革という政策軸もブレていない。ただし政治家としてのキャリアは2年と短く、長期的な一貫性は今後の評価。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道なし。救急医として15年間地域医療に従事してきた実績からクリーンな印象。医師という専門職の倫理観を持っている。

## 強み・弱み

### 強み
- **専門性**: 救急科専門医としての高度な医療知識
- **経営能力**: グロービスMBA、病院経営への知見
- **若さ**: 42歳で今後の成長余地
- **選挙実績**: 初挑戦で新設選挙区を制覇
- **政策の明確さ**: 医療改革という軸が明確

### 弱み（注意点）
- **政治経験不足**: 国政1期目、地方議会経験もなし
- **党の規模**: 国民民主党の国会での影響力は限定的
- **実績未知数**: 国会での実績はこれから
- **専門分野偏重**: 医療以外の政策領域は未知数
- **知名度**: 全国的な認知度は低い
