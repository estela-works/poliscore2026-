# 石井拓（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年4月15日 |
| 年齢 | 59歳（2024年衆院選時） |
| 選挙区 | 愛知13区 |
| 当選回数 | 衆院4回 |
| 主な役職 | 内閣府副大臣、財務大臣政務官、衆院財務金融委員会委員長 |

## 経歴
- 慶應義塾大学卒。野村證券勤務を経て2009年衆院初当選。財務金融委員長や内閣府副大臣などを歴任し、経済・金融分野の政策に携わる。

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 物価高や中小企業支援などを課題に掲げるが、課題の構造や優先順位の説明は公開情報では限定的。

### ② 解決設計力: 3/5
**評価根拠**: 経済・金融分野の政府役職経験があり、金融政策や中小企業支援の具体策を提示している。制度設計や財源の詳細は限定的だが、政策実務の関与は確認できる。

### ③ 実行力: 3/5
**評価根拠**: 衆院4期で副大臣などを歴任し、政策実装に関与している。大規模改革の主導実績は確認できないが行政実務経験はある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 公式サイトでの政策発信はあるが、争点ごとの詳細説明や反対意見への整理された反論は限定的。

### ⑤ 一貫性: 情報不足
**評価根拠**: 長期の政治活動は確認できるが、政策主張の変更やその理由を検証できる資料が不足。

### ⑥ 倫理観・権力の扱い方: 情報不足
**評価根拠**: 重大な不祥事や説明責任を問う報道の有無を判断できる情報が不足。

## 強み・弱み

### 強み
- 財務金融分野での政府・国会役職経験があり、政策実務の知見を持つ。
- 経済・金融政策を中心に具体的な施策を掲げている。

### 弱み（注意点）
- 政策設計や財源の説明が限定的で、実現手順が見えにくい。
- 対外的な説明や論点整理の発信が少ない。
