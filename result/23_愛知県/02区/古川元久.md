# 古川元久（国民民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年12月6日 |
| 年齢 | 58歳 |
| 選挙区 | 愛知県2区 |
| 当選回数 | 衆議院10回 |
| 主な役職 | 国民民主党代表代行兼国会対策委員長、元国家戦略担当大臣、元内閣官房副長官 |

## 経歴
- 愛知県名古屋市出身
- 愛知県立旭丘高等学校卒業
- 1986年司法試験合格（20歳、東大在学中）
- 1988年東京大学法学部卒業、大蔵省入省
- 1993年コロンビア大学大学院留学（国際関係論）
- 1994年大蔵省退官、政治家へ
- 1996年初当選（旧民主党）
- 2005年世界経済フォーラム Young Global Leaders選出
- 2007年NPO「TABLE FOR TWO International」設立
- 2009年内閣府副大臣、初代国家戦略室長
- 2010年内閣官房副長官（菅内閣）
- 2011年国家戦略担当大臣、経済財政政策担当大臣（野田内閣）
- 社会保障と税一体改革担当大臣
- 2018年国民民主党幹事長→代表代行
- 2020年新・国民民主党国会対策委員長
- 2024年10期目当選、代表代行兼国対委員長

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 国家戦略担当大臣として国の長期課題を設定。社会保障と税の一体改革を主導。「TABLE FOR TWO」設立で国際的な飢餓問題にも取り組む。経済財政・科学技術・宇宙開発と幅広い課題に対応。

### ② 解決設計力: 5/5
**評価根拠**: 東大法学部、司法試験合格（20歳）、大蔵省キャリア、コロンビア大学院という極めて高い知的基盤。国家戦略室長として政策立案を主導。世界経済フォーラムのYoung Global Leadersに選出される国際的評価。

### ③ 実行力: 4/5
**評価根拠**: 10期連続当選という抜群の安定感。国家戦略担当大臣、内閣官房副長官など要職を歴任。ただし野党時代が長く、政策実現には限界があった。玉木代表不在時の代表代行として党運営を担う。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 国会対策委員長として与野党協議を担当。財務省悪玉論への冷静な反論など、論理的な説明能力。ブログやSNSでの発信も継続。

### ⑤ 一貫性: 4/5
**評価根拠**: 1996年の旧民主党結党から参加。民主党→民進党→国民民主党と党の変遷はあるが、「対決より解決」という中道改革路線は一貫。立憲との合流に参加せず、国民民主党で活動を継続。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動は確認されていない。玉木代表の不倫問題では冷静に代表代行として対応。「TABLE FOR TWO」など社会貢献活動も行う。クリーンなイメージ。

## 強み・弱み

### 強み
- **圧倒的な知性**: 東大法学部、司法試験20歳合格、大蔵省
- **政策経験**: 国家戦略担当大臣、経済財政政策担当大臣
- **安定感**: 10期連続当選、愛知2区で盤石
- **国際経験**: ダボス会議Young Global Leaders
- **クリーン**: スキャンダルなし

### 弱み（注意点）
- **野党時代の長さ**: 民主党政権以降は野党
- **党の規模**: 国民民主党は中規模政党
- **高齢化**: 58歳で党の世代交代課題
