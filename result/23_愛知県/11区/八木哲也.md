# 八木哲也（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1947年8月10日 |
| 年齢 | 77歳（2024年衆院選時） |
| 選挙区 | 愛知11区 |
| 当選回数 | 衆院4回 |
| 主な役職 | 環境副大臣、環境大臣政務官 |

## 経歴
- 中央大学理工学部卒。小島プレス工業勤務を経て豊田市議を4期務め、市議会議長や監査委員を歴任。2012年衆院初当選以降、環境大臣政務官・環境副大臣など政府役職を経験。

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 選挙アンケートでは景気対策を重視する姿勢が示されるが、課題の構造や優先順位の説明は公開情報では限定的。

### ② 解決設計力: 2/5
**評価根拠**: 経済や環境分野の役職経験はあるが、制度設計・財源・実施手順まで踏み込んだ具体策の発信は確認できる範囲で多くない。

### ③ 実行力: 3/5
**評価根拠**: 衆院4期の実績があり、環境副大臣など政府ポストを経験。大規模改革の主導実績は不明だが、行政実務の遂行経験は評価できる。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 環境省での就任会見や政務官会見など公的な説明の場に立っており、一定の説明機会はある。ただし政策論点の詳細な説明は限定的。

### ⑤ 一貫性: 2/5
**評価根拠**: 長期の政治活動は確認できるが、政策主張の変遷や変更理由を体系的に追える資料は限られる。大きな矛盾は見当たらない一方、裏付けが薄いため低めに評価する。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 重大な不祥事や公的な批判報道は確認できないが、資金の透明性や説明責任の具体的な検証材料は限られる。慎重に低めの評価とする。

## 強み・弱み

### 強み
- 市議から国政までの行政・議会経験が長く、実務に基づく対応力がある。
- 環境副大臣など政府ポスト経験があり、政策執行に関わった実績がある。

### 弱み（注意点）
- 政策の具体設計や成果の可視化が乏しく、実績の中身が見えにくい。
- 近年の重点政策や方向性の整理が外部からは把握しにくい。
