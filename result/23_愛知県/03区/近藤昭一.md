# 近藤昭一（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1958年5月26日 |
| 年齢 | 66歳 |
| 選挙区 | 愛知3区（名古屋市昭和区・緑区・天白区） |
| 当選回数 | 衆議院10期 |
| 主な役職 | 環境委員長、元環境副大臣 |

## 経歴
- 愛知県立昭和高等学校卒業
- 上智大学法学部卒業
- 中日新聞社記者（政治部など）
- 1996年衆院選で初当選（旧民主党）
- 環境副大臣（菅内閣）
- 立憲民主党副代表
- 衆議院環境委員長
- 2024年衆院選で10期目当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 環境問題をライフワークとし、気候変動対策、再生可能エネルギー推進、脱原発を一貫して主張。記者時代の経験を活かし、政策課題の掘り下げに優れる。地元名古屋の都市問題にも精通。

### ② 解決設計力: 4/5
**評価根拠**: 環境副大臣として実務経験があり、政策立案能力は高い。環境委員長として委員会運営を主導。10期の経験から立法プロセスへの理解も深い。

### ③ 実行力: 4/5
**評価根拠**: 10回連続当選は愛知3区での強固な支持基盤を示す。2024年衆院選では池田佳隆元議員の裏金事件で自民空白となった選挙区で圧勝（約8万票）。環境政策では超党派での連携実績もある。

### ④ 説明責任: 4/5
**評価根拠**: 元中日新聞記者として情報発信に長けている。地元での活動報告、政策説明会を継続的に実施。環境問題について分かりやすく発信する姿勢は評価できる。

### ⑤ 一貫性: 4/5
**評価根拠**: 1996年の初当選以来、環境・平和・人権を軸に活動を継続。旧民主党から立憲民主党（中道改革連合）まで、中道リベラルの立場を一貫して維持している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルや疑惑は報じられていない。政治資金についても透明性を維持。ベテラン議員として権力の抑制的な行使を心がけている。

## 強み・弱み

### 強み
- **圧倒的な経験**: 衆議院10期の豊富な国政経験
- **専門性**: 環境政策のエキスパート
- **地盤の強さ**: 愛知3区で安定した支持基盤
- **発信力**: 記者出身で説明能力が高い
- **委員長経験**: 環境委員長として委員会運営を主導

### 弱み（注意点）
- **野党の限界**: 与党経験は短く、政策実現力に制約
- **年齢要因**: 66歳でベテランの域
- **党勢の課題**: 立憲民主党の支持率低迷
- **後継者問題**: 長期在職による世代交代の課題
- **政権担当能力**: 野党として政権運営経験が乏しい
