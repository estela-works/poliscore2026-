# 伴野豊（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1961年1月1日 |
| 年齢 | 63歳 |
| 選挙区 | 愛知8区（半田市・常滑市・東海市・知多市・阿久比町・東浦町・南知多町・美浜町・武豊町） |
| 当選回数 | 衆議院7期 |
| 主な役職 | 立憲民主党愛知8区総支部長、元外務副大臣、元国土交通副大臣 |

## 経歴
- 愛知県東海市出身（父は警察官）
- 愛知県立瑞陵高等学校卒業
- 名古屋工業大学工学部卒業
- 名古屋工業大学大学院修士課程修了
- 1985年、日本国有鉄道入社
- 1987年、JR東海入社（国鉄分割民営化）
- 1994年、退社して河村たかし議員の政策担当秘書
- 2000年、衆議院議員初当選
- 民主党副幹事長
- 民主党国際局長
- 外務副大臣（菅直人第1次・第2次改造内閣）
- 国土交通副大臣（野田第3次改造内閣）
- 2020年、新「立憲民主党」に参加
- 2024年、第50回衆院選愛知8区で7期目の当選

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 外務副大臣、国交副大臣として外交・交通政策の課題に取り組む。JR東海出身で交通インフラに精通。地元知多半島の課題にも継続的に対応。

### ② 解決設計力: 4/5
**評価根拠**: 名工大大学院修了の理系学識。JRでの実務経験と国会議員としての政策立案経験を持つ。外交・国交の両副大臣として政策設計に携わる。

### ③ 実行力: 4/5
**評価根拠**: 7期連続当選で選挙での実行力は高い。現職の復興大臣を小選挙区で破る実績。無党派層の票を着実に積み上げる選挙戦略。

### ④ 説明責任: 4/5
**評価根拠**: 地元での活動を継続し、有権者への説明を行う。国会での質問、委員会活動など議員活動を説明。UAゼンセン所属で労組とも連携。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→民進党→国民民主党→立憲民主党と政党の変遷があるが、野党・改革路線は一貫。JR出身・交通政策という軸は継続。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 特に問題となるスキャンダルは報じられていない。JR出身で地道に活動を継続。清廉な政治姿勢を維持。

## 強み・弱み

### 強み
- **選挙強さ**: 7期連続当選、現職大臣を破る
- **政府経験**: 外務・国交の両副大臣経験
- **専門性**: JR出身で交通インフラに精通
- **学歴**: 名工大大学院修了
- **地元基盤**: 知多半島で強固な支持

### 弱み（注意点）
- **政党変遷**: 複数の政党を経験
- **野党**: 政策実現の機会が限定的
- **高齢化**: 63歳で今後の活動期間
- **党内基盤**: 立憲民主党内での影響力
- **政権経験**: 民主党政権時の評価
