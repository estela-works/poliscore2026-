# 宮内史織（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1992年頃（推定） |
| 年齢 | 32歳 |
| 選挙区 | 北海道2区（札幌市東区・北区） |
| 当選回数 | 衆議院0回（2024年落選） |
| 主な役職 | 日本共産党北海道委員会常任委員、道平和運動部長 |

## 経歴
- 日本共産党北海道委員会で活動
- 党北海道常任委員に就任
- 道平和運動部長として活動
- 2024年3月：日本共産党が北海道2区の候補者として擁立を発表
- 2024年10月：衆議院選挙に出馬、31,855票（得票率13.5%）を獲得するも落選
- 2024年12月：日本共産党が2025年参院選北海道選挙区への擁立を発表

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 物価高対策として「大学までの教育費無償化」「消費税減税」「賃金アップの実現」を訴えるが、これらは日本共産党の基本方針であり、独自の課題設定とは言い難い。平和運動部長として平和問題に取り組んでいるが、具体的な課題提起の記録は公開情報では限定的。

### ② 解決設計力: 2/5
**評価根拠**: 党の政策を訴える形が中心で、独自の制度設計や具体的な政策パッケージの提示は確認できない。若手候補として今後の成長に期待する面はあるが、現時点では政策設計能力は未検証。

### ③ 実行力: 1/5
**評価根拠**: 国政経験がなく、地方議員の経験もない。党の常任委員・平和運動部長として組織内での活動実績はあると推察されるが、政治家としての実行力を示す具体的な成果は確認できない。衆院選での落選により、国政での実行機会を得られていない。

### ④ 説明責任: 2/5
**評価根拠**: 選挙期間中の街頭演説やメディア対応は行っているが、詳細な政策説明や有権者との対話の記録は限定的。日本共産党の基本方針を説明する形が中心で、独自の見解を深く説明する機会は少なかった。

### ⑤ 一貫性: 4/5
**評価根拠**: 日本共産党に所属し、党の方針に沿った活動を継続している。政党間の移籍歴はなく、平和運動への取り組みも一貫している。2025年参院選への出馬表明も、継続的な政治参加の姿勢を示している。

### ⑥ 倫理観: 4/5
**評価根拠**: 政治資金問題やスキャンダルは報じられていない。日本共産党の党員として、党の倫理規定に沿った活動を行っているものと推察される。若手女性候補として、クリーンなイメージを維持している。

## 強み・弱み

### 強み
- **若さ**: 32歳と若く、長期的な政治活動が可能
- **党組織での経験**: 常任委員、平和運動部長として組織運営に関与
- **継続的な挑戦**: 2025年参院選への出馬を表明し、継続して国政を目指す姿勢
- **クリーンなイメージ**: スキャンダルがなく、誠実な印象

### 弱み（注意点）
- **政治経験の欠如**: 地方議員経験もなく、選挙での当選実績がない
- **政策の独自性不足**: 党の方針に沿った主張が多く、独自の政策提案が見えない
- **知名度の低さ**: 得票率13.5%にとどまり、選挙区での浸透度に課題
- **共産党の支持基盤縮小**: 全国的に共産党の支持率が低下する中での活動
