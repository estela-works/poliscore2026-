# 道下大樹（立憲民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年12月24日 |
| 年齢 | 48歳 |
| 選挙区 | 北海道1区（札幌市中央区・南区・西区・北区の一部） |
| 当選回数 | 衆議院3回 |
| 主な役職 | 立憲民主党北海道総支部連合会副代表、元北海道議会議員（3期） |

## 経歴
- 北海道新得町生まれ、苫小牧市・八雲町・留萌市・旭川市で育つ
- 北海道旭川東高等学校卒業
- 中央大学法学部法律学科卒業
- 衆議院議員・横路孝弘の公設秘書を務める
- 2007年：北海道議会議員初当選（3期）
- 2017年：立憲民主党から衆議院議員総選挙に出馬し初当選
- 2021年：衆議院議員2期目当選
- 2024年10月：衆議院議員3期目当選（5人が立候補する激戦を制し圧勝）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 子どもの貧困、介護・福祉従事者の待遇、外国人労働者問題など、社会構造的な課題を的確に捉えている。2022年「子ども総合基本法案」、2024年「訪問介護緊急支援法案」「外国人労働者安心就労法案」など、課題を具体的な法案として言語化できる能力は高い。手話言語法案の提出など、マイノリティの課題にも目を向けている点は評価できる。

### ② 解決設計力: 4/5
**評価根拠**: 議員立法を多数提出しており、具体的な制度設計能力を示している。消費税還付法案（給付付き税額控除）、不払養育費の立替・取立制度など、既存制度の欠陥を補う設計を行っている。2025年の高額療養費自己負担引上げ凍結法案など、タイムリーな政策提案もできている。

### ③ 実行力: 4/5
**評価根拠**: 野党議員でありながら、少数与党状況を活かし「ガソリン軽油暫定税率廃止」「子ども2万円給付」「医療・介護・障がい福祉従事者の待遇改善」などの政策実現に貢献。道議時代から通算16年の議員経験があり、政策の実現までのプロセスを理解し、着実に結果を出している。

### ④ 説明責任: 4/5
**評価根拠**: 財務金融委員会、沖縄北方特別委員会、総務委員会など複数の委員会で積極的に質問を行い、政府への追及姿勢を示している。SNSや地元活動を通じた情報発信も活発で、有権者への説明責任を果たそうとする姿勢が見られる。

### ⑤ 一貫性: 4/5
**評価根拠**: 2017年、民進党が希望の党への合流を決定した際、不参加を表明し立憲民主党から出馬。政治的圧力の中でも自らの立場を貫いた。憲法9条改正反対、原発将来廃止、選択的夫婦別姓賛成など、一貫した政策スタンスを維持している。横路孝弘秘書時代からの政治姿勢を継承。

### ⑥ 倫理観: 4/5
**評価根拠**: 大きなスキャンダルや政治資金問題は報じられていない。弱者支援・人権擁護を政策の軸に据え、手話言語法案など社会的マイノリティへの配慮を示している。政治活動においてクリーンなイメージを維持。

## 強み・弱み

### 強み
- **議員立法の実績**: 多くの法案を提出し、具体的な政策立案能力を示している
- **選挙基盤の安定**: 3期連続当選、2024年は2位に3万票差の圧勝で、地元での信頼が厚い
- **横路孝弘からの政治的継承**: 革新道政の流れを汲み、明確な政治的軸を持つ
- **少数与党下での政策実現**: 野党でありながら具体的成果を上げている

### 弱み（注意点）
- **野党議員の限界**: 与党でない限り、大きな政策変更を主導することは困難
- **全国的知名度の低さ**: 北海道外での認知度は限定的
- **専門分野の偏り**: 福祉・労働分野に強みがある一方、経済・産業政策での発言は少ない
