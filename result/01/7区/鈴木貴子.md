# 鈴木貴子（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1986年1月5日 |
| 年齢 | 39歳 |
| 選挙区 | 北海道7区（釧路市、根室市、釧路総合振興局管内、根室振興局管内） |
| 当選回数 | 衆議院5回 |
| 主な役職 | 自民党広報本部長、元外務副大臣、元防衛大臣政務官 |

## 経歴
- 父は衆議院議員・鈴木宗男
- カナダ・トレント大学卒業
- NHK勤務
- 2012年：NHK退職、衆院選北海道7区に新党大地公認で出馬、落選
- 2013年：比例北海道ブロック繰り上げ当選
- 2016年：民主党を離党（共産党との共同歩調を批判）、除籍処分
- 2017年：自由民主党に入党、平成研究会（茂木派）入会
- 2018年：防衛大臣政務官就任（第4次安倍改造内閣）
- 2020年〜2021年：外務副大臣
- 2024年10月：衆院選北海道7区で初の選挙区当選（5期目）
- 2025年10月：自民党広報本部長就任

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**:
千島海溝沖地震対策、持続可能な一次産業、包括的性教育、こども若者政策、外交安全保障、孤独孤立政策、冤罪対策・再審法改正など、多岐にわたる政策課題を明確に設定している。特に北方領土問題や日ロ外交については父・宗男譲りの深い問題意識があり、地域の漁業や安全保障を構造的に捉えている。

### ② 解決設計力: 2/5
**評価根拠**:
外務副大臣、防衛大臣政務官を歴任したが、在任中に具体的な政策成果として目に見える形で実現したものは限定的。課題設定は幅広いが、それぞれの課題に対する具体的な解決策の設計は発展途上。特に日ロ外交については、北方領土問題の進展につながる具体的成果は出せていない。

### ③ 実行力: 4/5
**評価根拠**:
衆院議員5期、外務副大臣、防衛大臣政務官、自民党広報本部長と、政府・党の要職を歴任。2024年には初めて選挙区で勝利し、組織を動かす力を証明した。4児の母でありながら政治活動を継続する実行力も評価できる。ただし、具体的な政策成果としては明確なものが見えにくい。

### ④ 説明責任: 2/5
**評価根拠**:
広報本部長として党の発信を担う立場だが、自身の政策や立場変更（民主党離党、自民党入党）について十分な説明を行ってきたかは疑問が残る。父・宗男の影響を受けた政治活動について、独自の政治信条との関係性の説明も不明確。

### ⑤ 一貫性: 2/5
**評価根拠**:
新党大地→民主党→自民党と政党を移動。2016年には民主党の共産党との共同歩調を批判して離党したが、翌年には自民党に入党。政党の移動に一定の説明はあるものの、政治信条の軸が見えにくい。父・宗男も維新→自民と移動しており、親子で政党を渡り歩く姿は一貫性への疑念を生む。

### ⑥ 倫理観: 2/5
**評価根拠**:
本人に直接的なスキャンダルは確認されていないが、父・鈴木宗男の汚職事件（収賄、あっせん収賄等で有罪確定）の影響を受けた政治キャリアである点は留意が必要。父の地盤・看板を引き継ぐ形での活動は、政治の世襲や利益誘導の構造との関連で注視される。また、政党移動の経緯は機会主義的と見られる面もある。

## 強み・弱み

### 強み
- **政府・党での経験**：外務副大臣、防衛大臣政務官、広報本部長など要職を歴任
- **外交・安全保障への知見**：北方領土問題、日ロ外交に関する深い問題意識
- **女性政治家としての存在感**：4児の母として、女性の政治参画を体現
- **若さ**：39歳で衆院5期、今後の成長余地がある

### 弱み（注意点）
- **政党の移動**：新党大地→民主党→自民党の変遷は一貫性への疑念を生む
- **世襲批判**：父・宗男の地盤を引き継いだ経緯は、自身の能力評価を複雑にする
- **父の過去**：宗男の汚職事件は完全には払拭されておらず、イメージに影響
- **具体的成果**：要職を歴任しているが、目に見える政策成果は限定的
