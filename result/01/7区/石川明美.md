# 石川明美（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1951年5月16日 |
| 年齢 | 73歳 |
| 選挙区 | 北海道7区（釧路市、根室市、釧路総合振興局管内、根室振興局管内） |
| 当選回数 | 衆議院0回（新人）、釧路市議4期 |
| 主な役職 | 党北海道7区政策委員長、釧根地区副委員長 |

## 経歴
- 1951年：埼玉県坂戸市出身
- 1964年：坂戸町立坂戸小学校卒業
- 1967年：坂戸町立坂戸中学校卒業
- 1970年：埼玉県立川越工業高等学校卒業
- 1970年：日本電気エンジニアリング株式会社入社
- 1971年：日本共産党入党
- 1978年：日本民主青年同盟釧路地区委員長
- 1980年：日本共産党釧路地区委員会専従
- 1996年：衆院選北海道13区に立候補、落選
- 1997年：釧路市議選に立候補、当選（以後4期）
- 2014年：衆院選北海道7区に立候補、落選
- 2015年：道議選に立候補、落選
- 2016年：釧路市長選に立候補、落選
- 2017年：衆院選北海道7区に立候補、落選（47,740票）
- 2019年：道議選に立候補、落選
- 2021年：衆院選北海道7区に立候補、落選（12,913票）
- 2024年：衆院選北海道7区に立候補、落選（得票率9.2%）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**:
「コロナ禍で格差が拡大、社会の脆弱性を生んだのは安倍政権」といった課題認識は示しているが、構造的な分析というよりは政権批判の域を出ていない。釧路市議4期の経験から地域課題への理解はあるが、国政レベルでの独自の課題設定は見られない。共産党の一般的な政策課題を踏襲している。

### ② 解決設計力: 2/5
**評価根拠**:
釧路市議として4期の経験があり、地方議会での活動実績はある。しかし、国政レベルでの具体的な政策設計は確認できない。共産党の政策パッケージを訴える立場であり、独自の解決策の設計は限定的。

### ③ 実行力: 2/5
**評価根拠**:
釧路市議4期の当選実績は評価できる。しかし、国政選挙では1996年以来、繰り返し落選。2017年の47,740票から2021年の12,913票へと得票数も減少傾向にある。釧路市長選でも落選しており、大きな組織を動かす力は示せていない。50年以上の党活動の継続は粘り強さの表れだが、国政での実行力は未検証。

### ④ 説明責任: 2/5
**評価根拠**:
共産党の政策を説明する活動は継続しているが、独自の主張や深い説明は確認できない。選挙での得票数が減少傾向にあることは、有権者への説明・説得力に課題があることを示唆している。

### ⑤ 一貫性: 5/5
**評価根拠**:
1971年の入党以来、50年以上にわたり日本共産党一筋で活動。政党の移動は一切なく、釧路・根室地域を拠点とした活動も継続。格差是正、弱者支援という主張も一貫している。繰り返し落選しながらも活動を続ける姿勢は、一貫性の観点から最高評価に値する。

### ⑥ 倫理観: 4/5
**評価根拠**:
スキャンダルや問題行動の報道は確認されていない。50年以上の政治活動において、金銭問題や不祥事がないことは評価できる。労働組合活動、市議としての活動を通じて、一貫して弱者の立場に立った姿勢を示してきた。

## 強み・弱み

### 強み
- **一貫性**：50年以上の党活動、政党移動なし
- **地方議会経験**：釧路市議4期の実績
- **継続力**：繰り返し落選しながらも活動を継続する粘り強さ
- **クリーンさ**：長年の活動で不祥事なし

### 弱み（注意点）
- **年齢**：73歳という年齢は体力面での懸念
- **得票数の減少**：2017年→2021年で得票数が大幅減少
- **国政実績なし**：複数回挑戦しているが国政での実績は皆無
- **政策の独自性**：共産党の政策を踏襲しており、独自色が弱い
