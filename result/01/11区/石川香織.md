# 石川香織（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年5月10日 |
| 年齢 | 40歳 |
| 選挙区 | 北海道11区（十勝管内） |
| 当選回数 | 衆議院3回 |
| 主な役職 | 立憲民主党国民運動局長、厚生労働委員会委員 |

## 経歴
- 神奈川県横浜市出身、東京都大田区で育つ
- 聖心女子学院中学校・高等学校卒業
- 聖心女子大学文学部哲学科卒業
- 2007年日本BS放送（BS11）入社、アナウンサーとして活動
- 2011年元衆議院議員・石川知裕と結婚、帯広市に移住
- 2017年衆院選で北海道11区から立憲民主党公認で初当選
- 2021年衆院選で再選
- 2024年衆院選で3選（共産党候補擁立下での勝利）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 十勝の基幹産業である農業の課題を的確に捉え、農家への所得保障を中心的テーマに据えている。女性の健康問題（緊急避妊薬へのアクセス改善）、ジェンダー平等など、構造的な課題を言語化できている。アナウンサー経験を活かした発信力で課題を可視化する能力が高い。

### ② 解決設計力: 4/5
**評価根拠**: 農家への直接支払制度導入など、具体的な政策設計を提示。厚生労働委員会での活動を通じて、制度設計に関与している。緊急避妊薬の入手しやすさ向上、同性婚法制化など、法整備を含めた解決策を提案。

### ③ 実行力: 4/5
**評価根拠**: 3期連続当選、うち3回とも自民党・中川郁子との一騎打ち（または三つ巴）を制している。2024年は共産党が候補を擁立する中での勝利であり、厳しい選挙戦を勝ち抜く実行力がある。野党議員として委員会質疑でも積極的に発言。

### ④ 説明責任: 4/5
**評価根拠**: アナウンサー出身の強みを活かし、政策をわかりやすく説明する能力が高い。農業票を意識した丁寧な地元活動、自民党不信の保守層への働きかけなど、多様な層への説明を心がけている。SNSでの情報発信も活発。

### ⑤ 一貫性: 4/5
**評価根拠**: 2017年の立憲民主党結党時から一貫して所属。農業振興、女性政策、平和主義という軸がブレていない。夫・石川知裕の地盤を引き継ぐ形だが、独自の政策領域（ジェンダー、医療アクセス）を確立している。

### ⑥ 倫理観: 4/5
**評価根拠**: 政治資金問題やスキャンダルは報じられていない。対立候補（中川郁子）陣営の公職選挙法違反が発覚する中、自身はクリーンな選挙活動を維持。ルールを尊重した政治活動が確認できる。

## 強み・弱み

### 強み
- **発信力**: アナウンサー経験を活かした政策説明能力
- **選挙の強さ**: 自民党候補に3連勝、十勝での地盤を確立
- **政策の明確さ**: 農業、ジェンダー、医療アクセスで具体的な提案

### 弱み（注意点）
- **野党経験のみ**: 政権与党での政策実現経験がない
- **世襲的要素**: 夫の地盤を引き継いだ経緯がある
