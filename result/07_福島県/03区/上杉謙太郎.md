# 上杉謙太郎（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年4月1日 |
| 年齢 | 49歳（2024年選挙時） |
| 選挙区 | 福島3区 |
| 当選回数 | 衆議院2回（2024年落選） |
| 主な役職 | 元自民党福島県第3区支部長 |

## 経歴
- 福島県二本松市出身
- 安達高等学校卒業
- 慶應義塾大学商学部卒業
- 株式会社電通入社
- 2007年：福島県議会議員に初当選
- 2011年：福島県議会議員に再選
- 2017年：第48回衆議院議員総選挙で初当選（自民党・福島4区）
- 2021年：第49回衆議院議員総選挙で再選（自民党・福島4区、比例復活）
- 2024年2月：裏金問題発覚（安倍派から5年間で計309万円）
- 2024年4月：自民党から幹事長注意処分
- 2024年10月11日：衆院選への出馬辞退を表明
- 2024年10月13日：一転して福島3区から無所属で出馬を表明
- 2024年10月27日：福島3区で落選
- 2025年4月：自民党福島県連、福島3区支部長に再選任

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**:
県議・衆議院議員として地域課題に取り組んできたが、独自の課題設定や構造的な問題提起は見えにくい。電通出身という経歴を活かした発信力はあるが、政策面での課題設定は既存の党方針を踏襲する傾向。震災復興に関わる立場にあったが、独自の視点は確認されていない。

### ② 解決設計力: 2/5
**評価根拠**:
衆議院議員2期を務めたが、自ら設計した政策や制度改革の実績は限定的。県議時代からの活動も含め、具体的な政策設計を主導した形跡は見えにくい。党の方針に沿った活動が中心であり、独自の解決策の提示には至っていない。

### ③ 実行力: 2/5
**評価根拠**:
衆議院2期当選の実績はあるが、2024年選挙で落選。出馬辞退を表明してから2日後に一転出馬という迷走は、実行力の欠如を示す。裏金問題で非公認となり、無所属での出馬を余儀なくされた点も、自らの行動の結果として実行力に影響を与えた。

### ④ 説明責任: 1/5
**評価根拠**:
裏金問題について「責任を取り出馬辞退」と表明しながら、わずか2日後に「県連の要請」で一転出馬。この急転換について十分な説明がなされていない。衆議院政治倫理審査会への出席も自民党議員として拒否していた。309万円のキックバックについての説明も不十分。論点をずらす姿勢が顕著。

### ⑤ 一貫性: 1/5
**評価根拠**:
2024年10月の行動だけを見ても、出馬辞退（11日）→無所属出馬表明（13日）という急転換があり、一貫性に大きな疑問がある。「責任を取る」という発言と実際の行動が矛盾している。自民党として活動しながら、処分を受けて無所属で出馬するという経緯も、政治姿勢の一貫性を欠く。

### ⑥ 倫理観: 1/5
**評価根拠**:
2018年〜2022年の5年間で計309万円の裏金（安倍派からのキックバック）が発覚。政治資金収支報告書への不記載という法令違反。「責任を取り辞退」と発言しながら2日後に出馬という行動は、責任感の欠如を示す。自民党福島県連が「けじめはまだついていない。説明責任を果たす必要がある」と指摘するほど、問題は解決されていない。

## 強み・弱み
### 強み
- **発信力**：電通出身で、メディア対応・広報に強み
- **地元での知名度**：県議2期、衆議院2期で一定の認知度
- **組織の支援**：自民党県連が再び支部長に選任（2025年）

### 弱み（注意点）
- **深刻な裏金問題**：309万円の不記載、説明責任未了
- **一貫性の欠如**：出馬辞退→出馬という急転換で信頼失墜
- **選挙での敗北**：2024年選挙で落選、政治的影響力低下
