# 樽床伸二（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年8月6日（66歳） |
| 選挙区 | 大阪12区（寝屋川市、大東市、四條畷市） |
| 当選回数 | 衆議院6期（元職） |
| 主な役職 | 元総務大臣、元内閣府特命担当大臣、元希望の党代表代行 |

## 経歴
- 島根県出身
- 大阪大学卒業
- 松下政経塾出身（1期生）
- 1990年：衆院選出馬、落選
- 1993年：日本新党公認で衆院選初当選
- 1998年：民政党を経て民主党に合流
- 2000年・2003年：大阪12区で当選
- 2005年：大阪12区で落選
- 2009年：大阪12区で約4万票差で当選、国政復帰
- 2010年：民主党代表選出馬（菅直人に敗れる）
- 2011年：野田政権で民主党幹事長代行
- 2012年：野田改造内閣で総務大臣、内閣府特命担当大臣（沖縄及び北方対策、地域主権推進、地域活性化）
- 2012年・2014年：衆院選落選
- 2016年：民進党離党
- 2017年：希望の党代表代行
- 2019年：大阪12区補選出馬、落選
- 2026年：第51回衆議院選挙 大阪12区から中道改革連合公認で立候補

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 総務大臣、内閣府特命担当大臣として地域主権推進、沖縄北方問題など国政課題に取り組んだ経験がある。ただし、近年の具体的な政策提言は限定的。

### ② 解決設計力: 4/5
**評価根拠**: 松下政経塾出身で政策立案能力には定評がある。総務大臣として地域主権改革を推進した実績がある。民主党代表選では「世代交代」「党内結束」を掲げ政策を提示。

### ③ 実行力: 3/5
**評価根拠**: 衆議院6期当選、総務大臣、幹事長代行など重要ポストを歴任。ただし、2012年以降の衆院選では連敗が続いており、近年の選挙での実績には課題がある。

### ④ 説明責任: 3/5
**評価根拠**: 民進党離党時に「政権運営の失敗」「国民の信頼喪失」と明確に理由を説明。政治家として説明責任を果たす姿勢は見られる。

### ⑤ 一貫性: 2/5
**評価根拠**: 日本新党→新進党→民政党→民主党→民進党（離党）→無所属→希望の党→中道改革連合と所属政党が多く変遷。政策的な軸は持ちつつも、党を渡り歩いてきた印象がある。

### ⑥ 倫理観: 3/5
**評価根拠**: 大きなスキャンダルは報じられていない。長年の政治活動を通じて一定の信頼を維持している。

## 強み・弱み

### 強み
- **豊富な政治経験**: 衆議院6期、総務大臣、幹事長代行など重要ポスト歴任
- **政策立案能力**: 松下政経塾出身で政策立案のプロ
- **地域での知名度**: 大阪12区で長年活動し、一定の支持基盤がある
- **閣僚経験**: 総務大臣、内閣府特命担当大臣の経験
- **党組織の支援**: 中道改革連合の組織力を活用できる

### 弱み（注意点）
- **近年の選挙連敗**: 2012年、2014年、2019年と衆院選・補選で敗北が続く
- **党の変遷**: 所属政党が多く変わっており、一貫性に欠ける印象
- **高齢**: 66歳での再挑戦
- **激戦区**: 大阪12区は維新が強い選挙区
