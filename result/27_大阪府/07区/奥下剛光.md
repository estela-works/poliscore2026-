# 奥下剛光（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年10月4日 |
| 年齢 | 49歳 |
| 選挙区 | 大阪府7区 |
| 当選回数 | 衆議院2回 |
| 主な役職 | 国土交通委員会理事、元橋下徹大阪市長特別秘書 |

## 経歴
- 大阪府茨木市出身
- 履正社高等学校卒業
- 専修大学卒業
- 宮澤喜一元首相秘書
- 逢沢一郎元外務副大臣秘書
- 2007年 大阪府議選茨木市選挙区（無所属）落選
- 橋下徹大阪府知事私設秘書
- 橋下徹大阪市長特別秘書
- 家業の建材会社役員
- 2016年 日本維新の会大阪7区支部長
- 2017年 第48回衆院選で落選（渡嘉敷奈緒美に敗北）
- 2021年10月 第49回衆院選で初当選（渡嘉敷を破る）
- 2024年10月 第50回衆院選で2期目当選
- 家族：妻と一人息子

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 国土交通委員会理事として交通インフラや災害対策に取り組む。地方創生、子ども政策、デジタル社会推進に関与。ただし、独自の際立った課題設定は確認できない。

### ② 解決設計力: 3/5
**評価根拠**: 橋下徹市長の特別秘書として行政経験がある。国土交通委員会で質問を積極的に行っている。ただし、具体的な政策立案の実績は限定的。

### ③ 実行力: 3/5
**評価根拠**: 2017年落選後も活動を継続し、2021年に初当選。維新の強い大阪7区とはいえ、自民党現職を破った実績がある。2期連続当選で地盤を固めつつある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: キャバクラ政治資金問題での釈明は21分に及んだが、「ポケットマネーでやるには限界がある」という説明は説得力に欠けた。「返金する」との対応も後手に回った印象。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党系秘書→無所属府議選→橋下徹秘書→維新という経歴で、政治的立場は変遷がある。ただし、維新所属後は7区支部長として一貫して活動。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 政治資金でキャバクラ・ラウンジに約9万円を支出した問題が発覚。「身を切る改革」を掲げる維新所属議員としては致命的な矛盾。専門家からも「クラブで仕事をするという理屈は成立し難い」と厳しく批判されている。

## 強み・弱み

### 強み
- **行政経験**: 橋下市長の特別秘書として行政運営を経験
- **地盤の強さ**: 大阪7区で2期連続当選
- **維新の組織力**: 維新の強い大阪での活動基盤
- **秘書経験**: 首相経験者を含む複数議員の秘書経験

### 弱み（注意点）
- **キャバクラ政治資金問題**: 約9万円のキャバクラ等への支出が発覚
- **維新の理念との矛盾**: 「身を切る改革」を掲げながらの不適切支出
- **説明責任の不足**: 問題発覚後の対応が後手に
- **学歴**: 高卒（専修大学卒業）

## 主な政策主張
- 交通インフラの整備
- 災害対策の強化
- 地方創生
- 子ども政策
- デジタル社会の推進
