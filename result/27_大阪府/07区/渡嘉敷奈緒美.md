# 渡嘉敷奈緒美（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1962年7月16日 |
| 年齢 | 62歳 |
| 選挙区 | 大阪府7区 |
| 当選回数 | 衆議院4回（2021年落選、元職） |
| 主な役職 | 元環境副大臣、元厚生労働副大臣、元衆議院厚生労働委員長、自民党大阪府連会長 |

## 経歴
- 京都府京都市出身
- 祖父母は沖縄県在住（代々琉球王朝の御典医の家系）
- 聖心女子学院中等科・高等科卒業
- 昭和大学薬学部卒業（薬剤師免許取得）
- 株式会社資生堂勤務（本社広報部、新規事業部、15年間）
- 早稲田大学ビジネススクール卒業（休職中に取得）
- 1999年 杉並区議会議員初当選
- 2003年 杉並区議2期目当選
- 2005年 第44回衆院選大阪7区で初当選
- 2009年 落選（藤村修に敗北）
- 2012年 第46回衆院選で再選、厚生労働大臣政務官就任
- 2015年 厚生労働副大臣
- 2017年 環境副大臣
- 2019年5月 自民党大阪府連会長就任
- 2020年10月 衆議院厚生労働委員長
- 2021年10月 第49回衆院選で落選（奥下剛光に敗北）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 薬剤師資格を持ち、厚生労働分野の課題に精通。医療、介護、子育て支援、環境政策と幅広い分野で課題を設定。コロナ禍では厚生労働委員長として医療体制の課題に取り組んだ。

### ② 解決設計力: 4/5
**評価根拠**: 厚生労働大臣政務官、厚生労働副大臣、環境副大臣として政策立案に直接関与。資生堂での15年間の企業経験とMBAも政策設計に活かされている。専門性の高い政策分野で実績がある。

### ③ 実行力: 3/5
**評価根拠**: 衆院議員4期当選、副大臣2ポスト経験という豊富なキャリア。ただし、2021年衆院選で維新・奥下に敗北し落選。維新の強い大阪での選挙戦に苦戦している。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 資生堂広報部での経験から、対外的な説明能力は高い。厚生労働委員長として国会運営にも携わった。ただし、維新との対立構図の中で有権者への訴求力に課題。

### ⑤ 一貫性: 4/5
**評価根拠**: 自民党一筋で政党移動なし。杉並区議→衆院議員→副大臣と着実にキャリアを積み上げてきた。厚生労働・環境分野を一貫して担当。落選後も大阪7区での活動を継続。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 本人に関するスキャンダルは確認されていない。ただし、自民党大阪府連会長として、党全体の政治とカネ問題への対応は問われる立場。

## 強み・弱み

### 強み
- **専門性**: 薬剤師資格、厚生労働・環境分野での豊富な経験
- **行政経験**: 副大臣2ポスト、委員長経験
- **企業経験**: 資生堂15年、MBA取得
- **組織力**: 自民党大阪府連会長として組織を統括
- **クリーンなイメージ**: 大きなスキャンダルなし

### 弱み（注意点）
- **選挙での苦戦**: 2021年落選、維新の強い大阪での戦い
- **党の逆風**: 自民党全体への批判の影響
- **地盤の弱さ**: 京都出身で大阪との縁は選挙区のみ
- **落下傘候補**: 元々は杉並区議からのスタート

## 主な政策主張
- 医療・介護政策の充実
- 子育て支援
- 環境政策の推進
- 薬剤師としての専門知識を活かした医療政策
