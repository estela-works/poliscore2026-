# 梅村聡（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年2月13日 |
| 年齢 | 49歳 |
| 選挙区 | 大阪府5区 |
| 当選回数 | 衆議院1回、参議院2回 |
| 主な役職 | 日本維新の会国会議員団幹事長代理、元厚生労働大臣政務官、医師 |

## 経歴
- 大阪府堺市生まれ、大阪市淀川区十三で育つ
- 大阪教育大学附属池田中学校・高等学校卒業
- 2001年 大阪大学医学部卒業
- 大阪大学第2内科入局、大阪大学医学部附属病院研修医
- 箕面市立病院、大阪大学医学部附属病院内分泌・代謝内科勤務
- 日本内科学会認定内科医
- 2007年7月 参議院議員初当選（民主党・大阪府選挙区、過去最多1,281,502票）
- 2012年10月 厚生労働大臣政務官（野田第3次改造内閣）
- 2013年7月 参院選落選
- 2013年9月 民主党離党
- 2015年 よどがわ内科クリニック理事長
- 2019年7月 参議院議員再選（日本維新の会・比例区）
- 2024年10月 参議院議員自動失職、衆院選大阪5区で初当選
- 2025年8月 日本維新の会国会議員団幹事長代理

## 評価詳細

### ① 課題設定能力: 5/5
**評価根拠**: 医師として医療現場の課題を深く理解。医師法20条の誤った解釈問題を国会で指摘し、63年ぶりの解釈通知につなげた。東日本大震災での透析患者転院問題など、現場の課題を的確に把握する能力が高い。

### ② 解決設計力: 4/5
**評価根拠**: 厚生労働大臣政務官として生活保護制度改革に取り組んだ。維新の医療制度改革タスクフォース長として高齢者医療負担の見直しを提起。医師としての専門知識を活かした政策設計能力がある。

### ③ 実行力: 4/5
**評価根拠**: 東日本大震災で約740人の透析患者転院を自身の判断で実現した実績。参議院議員2期、衆議院議員1期の政治キャリア。公明党現職を破り大阪5区で初当選。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 医師として患者への説明経験が豊富。国会での質疑も専門的かつ分かりやすい。ただし、民主党から維新への移籍の経緯についての詳細な説明は限定的。

### ⑤ 一貫性: 3/5
**評価根拠**: 民主党→民主党離党→日本維新の会と政党移動あり。ただし、医療政策への取り組みは一貫しており、医師としての専門性を活かした活動を継続。政策的な軸は維持している。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 震災時の透析患者救済は、官僚の許可を待たず人命を優先した判断として評価できる。スキャンダルは確認されていない。医師としての倫理観は高いと推測される。

## 強み・弱み

### 強み
- **医師としての専門性**: 現役医師、内科認定医
- **医療政策の実績**: 医師法解釈変更、震災時の患者救済
- **政府経験**: 厚生労働大臣政務官
- **選挙力**: 参院選で過去最多得票、公明党現職を破り当選
- **党内地位**: 国会議員団幹事長代理

### 弱み（注意点）
- **政党移動**: 民主党から維新への移籍
- **落選経験**: 2013年参院選で落選
- **衆院議員としては新人**: 衆議院では1期目

## 主な政策主張
- 医療制度改革（高齢者医療負担の見直し）
- 社会保障制度の持続可能性確保
- 医療現場の課題解決
- 維新の改革路線
