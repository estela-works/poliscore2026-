# 宮本岳志（日本共産党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1959年12月25日 |
| 年齢 | 65歳 |
| 選挙区 | 大阪府5区 |
| 当選回数 | 衆議院5回、参議院1回 |
| 主な役職 | 日本共産党中央委員、元衆議院議員 |

## 経歴
- 和歌山県和歌山市生まれ
- 大阪府立岸和田高等学校卒業
- 和歌山大学教育学部入学（後に除籍）
- 1975年 日本民主青年同盟加盟
- 1978年 日本共産党入党（18歳）
- 1998年7月 参議院議員初当選（大阪府選挙区）
- 2009年8月 衆議院議員初当選（比例近畿ブロック）
- 2012年、2014年、2017年 衆院選当選
- 2019年4月 大阪12区補選で落選（得票率8.9%、供託金没収）
- 2021年10月 衆院選比例復活当選
- 2024年10月 衆院選落選（大阪5区、比例名簿登載なし）
- 自宅：大阪府岸和田市
- 趣味：ギター演奏、落語鑑賞

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 森友学園問題を国会で最初に取り上げ、土壌汚染除去費用の総額を聞き出すなど、社会問題の発掘能力が高い。サラ金と警察の癒着問題、大学改革の問題点など、多角的な課題設定を行っている。

### ② 解決設計力: 3/5
**評価根拠**: GID特例法（性同一性障害の方の性別変更を認める法律）の制定に超党派で携わった実績がある。グレーゾーン金利問題では貸金業法改正に貢献。ただし、党の方針に沿った政策提案が中心。

### ③ 実行力: 4/5
**評価根拠**: 衆議院5期、参議院1期の豊富な国会経験。森友問題追及、国際人権規約の学費無償条項の保留撤回を文科省に明言させるなど、具体的な成果を上げている。ただし、2024年衆院選で落選（比例名簿非登載）。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 国会質疑での追及力は高い。ただし、メディア露出は限定的で、一般有権者への訴求力には課題がある。2019年補選では供託金没収となるほど得票が低迷した。

### ⑤ 一貫性: 5/5
**評価根拠**: 18歳で共産党入党以来、約45年間一貫して党で活動。政党移動なし。教育、福祉、労働問題への取り組みも一貫している。政策的な軸がぶれない。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: 森友問題追及など権力監視の姿勢は明確。スキャンダルや問題行動は確認されていない。サラ金と警察の癒着追及など、権力の不正に対する批判的姿勢を維持。

## 強み・弱み

### 強み
- **国会経験**: 衆参合わせて6期の豊富な経験
- **追及力**: 森友問題を最初に国会で取り上げた実績
- **法案成立への貢献**: GID特例法、貸金業法改正
- **一貫性**: 45年以上の共産党での活動
- **専門性**: 教育、福祉、放送政策など幅広い分野

### 弱み（注意点）
- **選挙での苦戦**: 2019年補選で供託金没収、2024年落選
- **比例名簿非登載**: 2024年選挙で党から比例名簿に載せられず
- **年齢**: 65歳、世代交代の議論
- **知名度**: 一般有権者への浸透が弱い
- **大学除籍**: 和歌山大学を除籍

## 主な政策主張
- 自民党裏金問題の追及
- 企業・団体献金の禁止
- 消費税減税
- 大学学費の無償化
- 放送の多様性確保
- 地方議員の請負禁止維持
