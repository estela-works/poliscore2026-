# 西田薫（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1967年4月30日 |
| 年齢 | 57歳 |
| 選挙区 | 大阪府6区 |
| 当選回数 | 衆議院1回（大阪府議5期） |
| 主な役職 | 元大阪府議会警察常任委員会委員長 |

## 経歴
- 大阪府立守口北高校（現・芦間高等学校）卒業
- 大阪ビジネスカレッジ専門学校卒業
- 1988年 三井物産関連代理店勤務
- 1991年 オーストラリアの免税店勤務
- 1992年 吉村剛太郎参議院議員の秘書（10年間）
- 2003年 守口市議会議員当選（1期）
- 2007年 大阪府議会議員当選（5期連続）
- 2024年 衆議院議員初当選（公明党・伊佐進一を破る）

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: インテリジェンス改革（国家報局・対外情報庁の創設、スパイ防止法）に取り組むと掲げるが、独自の問題設定は見えない。母子家庭で育った経験から「光の当たらないところに光を当てる」という原点を持つが、具体的な課題定義に結びついていない。

### ② 解決設計力: 2/5
**評価根拠**: 維新の政策（身を切る改革等）を踏襲するが、自身の政策設計の実績は確認できず。府議会では委員長職を歴任したが、具体的な条例制定や制度設計の実績は不明。情報不足。

### ③ 実行力: 3/5
**評価根拠**: 大阪府議5期の経験があり、警察常任委員会委員長、教育常任委員会副委員長等を歴任。地方議員としての実務経験は豊富。2024年衆院選で公明党現職を破り初当選した点も評価できる。ただし国政での実績はまだない。

### ④ 説明責任・コミュニケーション力: 1/5
**評価根拠**: 選挙はがき問題（他人の後援会名・故人を推薦人に無断記載）で謝罪会見を開くも、収支報告書虚偽記載問題では「出納責任者のミス」と説明し、当該責任者から「虚偽で名誉毀損」と刑事告訴される事態に。責任転嫁と批判されており、説明責任を果たせていない。

### ⑤ 一貫性: 2/5
**評価根拠**: 維新一筋で政党移動はないが、問題への対応で説明が二転三転。守口市長との対立（2020年ハラスメント問題）、選挙はがき問題、収支報告書問題と、次々と問題が発生しており、行動の一貫性に疑問符。

### ⑥ 倫理観・権力の扱い方: 1/5
**評価根拠**: 複数の深刻な問題が発覚。①選挙はがきの虚偽記載（公職選挙法違反の可能性）、②収支報告書の虚偽記載（実際には支払っていない79万円を記載）、③守口市職員へのハラスメント（百条委員会で「ハラスメントに該当」と認定）。党から戒告処分を受けている。

## 強み・弱み

### 強み
- **地方議員経験**：府議5期の実績、地域事情への理解がある
- **選挙の強さ**：公明党現職を破り初当選、維新の大阪での組織力を活用

### 弱み（注意点）
- **倫理的問題の連発**：選挙違反疑惑、虚偽記載、ハラスメント認定と問題が多すぎる
- **説明能力の欠如**：問題発覚時の対応が「責任転嫁」と批判され、刑事告訴に発展
