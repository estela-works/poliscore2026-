# 宇都宮ゆうこ（中道改革連合）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1976年（49歳） |
| 選挙区 | 大阪3区（大阪市大正区、住之江区、住吉区、西成区） |
| 当選回数 | 新人 |
| 主な役職 | 元立憲民主党大阪12区支部長、作曲家、政治団体「自由の会」代表 |

## 経歴
- 大阪府出身
- 大阪府立寝屋川高等学校卒業（在学中に定時制課程に編入）
- 作曲家「真白リョウ」として活動開始
- 2006年：EXILEのシングル『SUMMER TIME LOVE』に楽曲提供
- 小沢一郎政治塾に入塾、2年間の勉強を経て卒塾
- 政治団体「自由の会」代表就任
- 2020年12月：立憲民主党大阪12区支部長就任
- 2021年10月：第49回衆議院選挙 大阪12区から出馬（落選）
- 2026年：第51回衆議院選挙 大阪3区から中道改革連合公認で出馬

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 「夢と希望ある日本」「この日本を変えたい」という理念は示すが、具体的な課題設定についての情報が限られている。小沢政治塾での学びはあるが、独自の課題提起は確認できない。

### ② 解決設計力: 2/5
**評価根拠**: 政策立案の経験や具体的な制度設計についての情報が少ない。作曲家としてのクリエイティブな能力はあるが、政策分野での解決設計の実績は確認できない。

### ③ 実行力: 2/5
**評価根拠**: 2021年衆院選で落選し、議員としての実績はない。作曲家としてはEXILEへの楽曲提供など一定の成果を上げている。政治活動は継続しているが、目に見える成果は限定的。

### ④ 説明責任: 3/5
**評価根拠**: 公式サイト、SNS（X、ブログ、Facebook）で継続的に情報発信を行っている。有権者とのコミュニケーションを重視する姿勢が見られる。

### ⑤ 一貫性: 3/5
**評価根拠**: 政治家を志してから一貫して活動を継続。立憲民主党から中道改革連合への移行、大阪12区から大阪3区への選挙区変更はあるが、政治に対する情熱は維持している。

### ⑥ 倫理観: 3/5
**評価根拠**: スキャンダルや問題行動は報じられていない。芸術家として、また政治活動家として誠実に活動を続けている様子が窺える。

## 強み・弱み

### 強み
- **芸術家としての発信力**: 作曲家としての経験から、感性に訴える発信ができる
- **小沢政治塾での学び**: 政治・経済・憲法・ディベートなどを体系的に学んだ
- **継続的な活動**: 落選後も政治活動を続ける粘り強さ
- **党組織の支援**: 中道改革連合の組織力を活用できる

### 弱み（注意点）
- **政治経験なし**: 議員経験がなく、政策実現の実績がない
- **選挙区変更**: 大阪12区から大阪3区への移動で地盤がない
- **激戦区**: 大阪3区は維新・東徹、自民・柳本あきらなど有力候補がひしめく
- **知名度不足**: 大阪3区での認知度は限定的
