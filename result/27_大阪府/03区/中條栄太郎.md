# 中條栄太郎（無所属）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1970年頃（推定） |
| 年齢 | 55歳 |
| 選挙区 | 大阪3区（住之江区、住吉区、西成区） |
| 当選回数 | なし（新人） |
| 主な役職 | 株式会社共栄不動産代表取締役 |

## 経歴
- 大成建設グループで賃貸事業
- 住友林業で注文住宅事業
- 父より奈良建設（住之江区）を継承
- 株式会社共栄不動産代表取締役
- 2017年衆院選（大阪3区）無所属で出馬、落選
- 自民党一般党員だったが除名処分
- 介護職員初任者研修資格取得
- 母の在宅介護・看取りを経験
- 西成子供食堂でボランティア活動

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 不動産事業者として地域経済への理解はある。介護経験から福祉への関心、子供食堂ボランティアから貧困問題への意識も持つ。ただし政策としての課題設定は体系化されておらず、独自の政策提言は限定的。

### ② 解決設計力: 2/5
**評価根拠**: 不動産会社経営者としての実務経験はあるが、政策立案の実績はない。政治経験がなく、国政レベルでの政策設計能力は未知数。

### ③ 実行力: 1/5
**評価根拠**: 2017年衆院選で無所属から出馬し16,231票を獲得（無所属では全国2位）するも落選。その後は政治活動よりも介護・ボランティアに時間を費やしており、選挙での当選は困難。

### ④ 説明責任: 2/5
**評価根拠**: 公式サイトで経歴や活動を公開。自民党除名の経緯（党公認なく出馬したため）も説明している。ただし政策の詳細な説明は限定的。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党一般党員として政治活動を始めるも、党の意向に反して無所属出馬し除名。現在も無所属で活動を継続するが、政党との関係が不安定。地元住之江区での活動という軸は維持している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 母の介護や子供食堂ボランティアなど、社会貢献活動への姿勢は評価できる。自民党除名は党規違反によるものだが、公明党批判を展開した独自の政治信念に基づく行動でもあった。

## 強み・弱み

### 強み
- **経営者経験**: 不動産会社代表として実業界での実績
- **介護経験**: 親の介護・看取りを経験した当事者性
- **ボランティア活動**: 子供食堂での社会貢献
- **地元密着**: 住之江区で地域に根差した活動
- **独立心**: 党の方針に縛られない独自の判断力

### 弱み（注意点）
- **政治経験なし**: 当選経験がない
- **組織なし**: 政党の支援がない無所属
- **自民党除名**: 党規違反による除名処分
- **選挙区事情**: 大阪3区は維新・公明が強い
- **知名度不足**: 主要候補と比べ知名度が低い
