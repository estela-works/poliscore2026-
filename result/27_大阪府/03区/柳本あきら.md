# 柳本あきら（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1974年1月29日（52歳） |
| 選挙区 | 大阪3区（大阪市大正区、住之江区、住吉区、西成区） |
| 当選回数 | 元職（衆議院1期）、大阪市会議員5期 |
| 主な役職 | 元環境大臣政務官、元内閣府大臣政務官、元自民党大阪市会議員団幹事長 |

## 経歴
- 大阪府大阪市西成区山王出身
- 大阪教育大学附属高等学校平野校舎卒業
- 京都大学法学部卒業
- 1997年：関西電力入社
- 1999年：父・柳本豊市議の死去に伴う補欠選挙で大阪市議に初当選（25歳、最年少）
- 大阪市会議員5期務める
- 自民党大阪市会議員団幹事長歴任
- 2015年：大阪市長選挙出馬（落選）
- 2019年：大阪市長選挙出馬（落選）
- 2021年：第49回衆議院選挙 比例近畿ブロック単独2位で当選
- 2022年：環境大臣政務官兼内閣府大臣政務官就任（第2次岸田第1次改造内閣）
- 2024年：第50回衆議院選挙 立候補見送り（自民比例名簿に掲載されず）
- 2025年7月：参議院選挙 大阪選挙区出馬（5位落選）
- 2026年：第51回衆議院選挙 大阪3区から自民党公認で出馬（30年ぶりの大阪3区自民候補）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 環境大臣政務官として環境政策に携わった経験があり、政策課題への理解がある。大阪市議として長年地域課題に取り組んできた。ただし国政での独自の課題提起は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 大阪市議5期の経験から地方行政の制度設計に精通。環境大臣政務官として政府の政策立案にも関与。市長選2度の経験から行政運営のビジョンも持つ。

### ③ 実行力: 3/5
**評価根拠**: 市議5期、衆議院1期と着実に経験を積んできた。一方で市長選2度、2025年参院選と重要な選挙で敗北。政務官としての実績はあるが、トップとしての実行力は未検証。

### ④ 説明責任: 3/5
**評価根拠**: SNSや公式サイトで活動を発信。環境政務官時代には政府の立場で説明責任を果たす経験も。落選後も活動を継続し、有権者とのコミュニケーションを維持。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党員として一貫した活動を続けているが、選挙区については状況に応じて変化（比例→参院大阪→衆院大阪3区）。大阪での自民党勢力拡大という目標は一貫。

### ⑥ 倫理観: 3/5
**評価根拠**: スキャンダルや問題行動は報じられていない。政治家一族として公人としての振る舞いを身につけている。

## 強み・弱み

### 強み
- **豊富な政治経験**: 市議5期、衆議院1期、政務官経験
- **政治家一族**: 叔父・父が政治家で、組織・人脈を継承
- **行政経験**: 環境大臣政務官として政府の政策立案に関与
- **知名度**: 市長選2度出馬で大阪での知名度がある
- **党組織の支援**: 自民党の組織力を活用できる

### 弱み（注意点）
- **大阪での選挙弱さ**: 市長選2度、2025年参院選で落選
- **維新との対決**: 大阪3区は維新・東徹が強く、厳しい選挙区
- **30年ぶりの自民候補**: 大阪3区での自民の地盤は脆弱
- **自公連携の解消**: 旧公明支持層の取り込みが課題
