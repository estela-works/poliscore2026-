# 高見亮（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1978年5月20日（47歳） |
| 選挙区 | 大阪府2区 |
| 当選回数 | 衆議院0回（新人）※大阪市会議員3期 |
| 主な役職 | 大阪市会議員（東住吉区・3期）、大阪維新の会政調会長、公認会計士・税理士 |

## 経歴
- 生駒市立上中学校卒業
- 大阪大学卒業
- 監査法人トーマツ入所（上場企業監査、内部統制コンサルティング）
- 公認会計士・税理士資格取得、高見会計事務所設立
- 2013年 維新政治塾卒業
- 2014年 大阪市会議員・坂井良和氏の秘書
- 2015年 大阪市会議員（東住吉区）初当選
- 2019年 大阪市会議員再選（2期目）
- 2020年 大阪維新の会副政調会長就任
- 2021年 大阪維新の会市議団政調会長就任
- 2023年 大阪市会議員3期目当選
- 2026年 衆院選大阪2区に維新公認で出馬

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: 公認会計士としての財政・税制の専門知識はあるが、独自の政策課題の発見・定義に関する具体的な発信は確認できない。政調会長として党の政策全般に関与する立場だが、個人としてどのような問題提起を行ったかは不明。専門性と課題設定力は別物。

### ② 解決設計力: 2/5
**評価根拠**: 大阪維新の会政調会長として政策立案プロセスに関与する立場にあるが、個人として具体的にどのような制度設計に貢献したかは確認できない。公認会計士として制度設計の素養はあると推測されるが、具体例が不足。

### ③ 実行力: 4/5
**評価根拠**: 大阪市会議員3期連続当選（10年以上）。維新の会の市議団政調会長として、大阪維新が推進してきた行財政改革の政策推進に組織的に関与。市会運営委員・建設港湾委員・大都市税財政制度特別委員と複数の委員会で活動。維新政治塾から秘書を経て議員、そして政調会長へと着実にキャリアを積み上げた実行力がある。

### ④ 説明責任・コミュニケーション力: 2/5
**評価根拠**: 情報不足。「どんな小さなお声も真摯に聞く」をモットーとし、複数委員会で活動しているが、国会答弁や記者会見などの公の場での説明力を評価する具体的な材料がない。

### ⑤ 一貫性: 4/5
**評価根拠**: 2013年の維新政治塾以来、一貫して維新の改革路線で活動。秘書→市議→市議3期→政調会長→衆院選と維新の中で着実にステップアップ。党移籍や方針転換はなく、大阪維新の行政改革という軸がブレていない。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルや問題行動の報道なし。公認会計士・税理士という職業的倫理観の高い背景を持つ。透明性・法令遵守が求められる専門職からの転身であり、権力の私物化等の懸念は見られない。

## 強み・弱み

### 強み
- **専門性**: 公認会計士・税理士としての財政・税制の専門知識
- **地方議員経験**: 大阪市会議員3期10年超の実績
- **党内の信頼**: 政調会長を務める党内での信任
- **組織力**: 維新という強固な組織基盤

### 弱み（注意点）
- **国政未経験**: 地方議会の実績が国政に直結するとは限らない
- **独自色の薄さ**: 個人の政策提案より党の方針に沿った活動が中心
- **全国的知名度なし**: 大阪市東住吉区以外での認知度は低い
- **守島正との対立**: 同じ維新出身の守島正（現職・無所属）との対決構図
