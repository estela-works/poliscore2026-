# 池上和日子（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1984年5月25日 |
| 年齢 | 41歳 |
| 選挙区 | 大阪府16区 |
| 当選回数 | 新人（国政初挑戦） |
| 主な役職 | 参政党公認候補、二児の母 |

## 経歴
- 1984年5月25日 大阪府河内長野市出身
- 堺市で生まれ育ち、現在も堺市在住
- 米オクラホマ州立大学 中退
- 建設会社勤務
- 共産思想の家庭で育つ
- 留学を通じて日本の教育に疑問を感じ、歴史や地政学を学び直す
- 身近な人の病気をきっかけに医療制度に疑問、食の大切さを認識
- 2022年 参政党に入党
- タウンミーティングのファシリテーターとして活動
- 2024年9月 参政党公認（第1次）として発表
- 2024年10月 大阪16区からの出馬が決定
- 2026年1月 参政党公認で大阪16区から衆院選出馬

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**:
「明るく強く、子供たちに誇りを持って渡せる日本を実現」という課題認識を明確に示している。自身の経験（留学による教育への疑問、身近な人の病気による医療・食への関心）から課題意識を形成しており、当事者性がある。ただし、具体的な政策課題の言語化は限定的。

### ② 解決設計力: 2/5
**評価根拠**:
参政党の政策（教育改革、食と健康）に沿った方向性を示しているが、具体的な解決策の設計は公開情報から確認できない。建設会社での実務経験はあるが、政策設計に活かされているかは不明。情報不足。

### ③ 実行力: 2/5
**評価根拠**:
国政・地方政治での選挙経験・当選実績なし。ただし、タウンミーティングのファシリテーターとして党の活動に携わり、地道な普及活動を行っている。二児の母として子育ての実践経験も。政治的実行力は未知数。新人として構造的に低評価。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**:
第一声で「新しい選択肢としての参政党」を訴え、自身の立候補の意義を説明している。タウンミーティングのファシリテーター経験は対話力を示唆。堺で生まれ育った地元への愛着も訴求している。

### ⑤ 一貫性: 2/5
**評価根拠**:
2022年参政党入党以来、一貫して党の活動に参加。共産思想の家庭から参政党への転向という経歴は価値観の変化を示すが、それ自体が問題とは言えない。政治活動歴は短く、長期的な一貫性を評価する材料が乏しい。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**:
個人としてのスキャンダルは確認されていない。ただし、参政党全体として代表・神谷宗幣氏の秘書急死問題・パワハラ疑惑、外国人への排外主義的発言、反ワクチン的主張への批判が存在する。所属政党の姿勢との関係は有権者の判断材料となる。

## 強み・弱み

### 強み
- **地元出身**: 堺で生まれ育ち、現在も堺市在住の当事者
- **子育て世代**: 二児の母として子育て当事者の視点
- **グラスルーツ活動**: タウンミーティングでの地道な活動経験
- **海外経験**: 米国留学経験から得た視点

### 弱み（注意点）
- **政治経験なし**: 議会・選挙経験ゼロ
- **所属政党の問題**: 参政党の反ワクチン主張・排外主義的傾向への批判
- **政策の具体性**: 抽象的なスローガンが中心
- **大学中退**: 学歴面での弱み
