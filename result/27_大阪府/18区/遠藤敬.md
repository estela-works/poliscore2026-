# 遠藤敬（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1968年6月6日 |
| 年齢 | 56歳 |
| 選挙区 | 大阪18区（岸和田市・泉大津市・和泉市・高石市・泉北郡忠岡町） |
| 当選回数 | 衆議院5期 |
| 主な役職 | 内閣総理大臣補佐官、維新の会国対委員長、衆議院安全保障委員長 |

## 経歴
- 大阪府高石市出身
- 大阪産業大学附属高等学校卒業
- 飲食店経営
- 高石青年会議所理事長
- 日本青年会議所大阪ブロック協議会長
- 日本教育再生機構大阪会長
- 中山太郎後援会青年局長
- 2011年自民党大阪18区支部長就任
- 2012年自民党離党、日本維新の会入党
- 2012年衆院選で初当選
- 維新の会国会対策委員長（2015年～）
- 衆議院安全保障委員長（2024年）
- 内閣総理大臣補佐官（2025年～）

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 国対委員長として国会運営を担当。教育再生機構会長として教育問題にも関与。ただし政策面より国会運営・党務が中心で、独自の政策課題設定は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 高卒後、飲食店経営から政界入りした叩き上げ。学歴・専門性より実務能力で勝負するタイプ。国対委員長として与野党折衝の手腕は評価されている。

### ③ 実行力: 5/5
**評価根拠**: 5期連続当選で大阪18区の地盤は盤石。国対委員長として維新の国会戦略を主導。内閣総理大臣補佐官に就任し、連立政権のキーマンとなった。維新を野党第一党級に押し上げた功労者の一人。

### ④ 説明責任: 4/5
**評価根拠**: 国対委員長として国会対応の説明を担当。与野党の橋渡し役として発信力がある。マスコミへの露出も多く、党の顔として説明責任を果たしている。

### ⑤ 一貫性: 2/5
**評価根拠**: 自民党から維新への移籍歴あり。2011年に自民党支部長に就任しながら、翌年には維新に移った。ただし維新移籍後は12年以上一貫して活動している。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に重大なスキャンダルは報じられていないが、愛煙家議連「もくもく会」副会長として喫煙擁護の立場。秋田犬保存会会長としてザギトワに犬を贈呈するなど、話題作りの面も。

## 強み・弱み

### 強み
- **国対委員長**: 国会運営のキーマン
- **総理補佐官**: 連立政権で重要ポスト
- **安定した地盤**: 5期連続当選
- **人脈**: 与野党に幅広い人脈
- **松井一郎の側近**: 党内での影響力

### 弱み（注意点）
- **政党移籍歴**: 自民党から維新へ
- **政策より党務**: 独自政策が見えにくい
- **維新依存**: 党勢と連動した影響力
- **学歴**: 高卒（ただし実務能力で補完）
- **連立の難しさ**: 政権運営の責任増大
