# 中司宏（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1956年3月11日 |
| 年齢 | 68歳 |
| 選挙区 | 大阪11区（枚方市・交野市） |
| 当選回数 | 衆議院2回 |
| 主な役職 | 日本維新の会幹事長（第5代）、元枚方市長（4期） |

## 経歴
- 大阪府枚方市生まれ
- 枚方市立殿山第二小学校、枚方市立第三中学校卒業
- 大阪府立寝屋川高等学校卒業
- 早稲田大学第一文学部卒業
- 産経新聞社入社、京都支局を経て東京本社政治部記者
- 政治部記者として中曽根康弘首相、自民党（田中派・竹下派）を担当
- 大阪府議会議員（4期、1987年〜1995年）
- 枚方市長（第14・15・16・17代、1995年〜2011年、4期16年）
- 大阪府市長会会長（2期）
- 全国青年市長会会長
- 道路整備促進期成同盟会全国協議会（道全協）会長
- 大阪府議会議員（2015年〜2021年）
- 第49回衆議院議員総選挙で初当選（2021年）
- 第50回衆議院議員総選挙で再選（2024年）
- 日本維新の会幹事長就任（2025年8月）

## 評価詳細

### ① 課題設定能力：4/5
枚方市長として4期16年、地方自治の課題に取り組んだ経験がある。産経新聞政治部記者として中央政界を取材した経験から、国政の課題把握能力も高い。維新の会幹事長として党の政策課題を統括する立場にある。

### ② 解決設計力：4/5
早稲田大学卒業後、新聞記者、府議、市長、再び府議、そして国会議員と多様な立場を経験。枚方市長時代には行財政改革を推進し、道路整備促進期成同盟会全国協議会会長として政策提言を行った実績がある。

### ③ 実行力：5/5
枚方市長4期連続当選、2021年衆院選では自民党現職・佐藤ゆかりと立憲現職・平野博文を破り初当選、2024年も再選。選挙での強さは際立つ。維新の会幹事長への抜擢は党内での実行力評価の表れ。

### ④ 説明責任：3/5
地元枚方市での長年の活動で有権者との信頼関係を構築。公式サイトでの情報発信も行っている。ただし、全国的な発信力や説明責任という点では、特筆すべき実績は確認できない。

### ⑤ 一貫性：3/5
自民党から政治キャリアをスタートし、維新の会に移籍した経歴がある。ただし、地方分権・行政改革という政策の軸は一貫している。維新の会での活動は2015年からであり、比較的長い。

### ⑥ 倫理観：3/5
特筆すべきスキャンダルや問題行動は確認されていない。市長4期、府議4期と長年にわたり公職を務め、有権者の信任を得続けている。ただし、積極的に倫理面でリーダーシップを発揮した事例は確認できない。

## 強み・弱み
### 強み
- **豊富な行政経験**：市長4期16年、府議4期の実績
- **選挙の強さ**：現職閣僚・野党幹部を破り当選
- **党幹部としての地位**：維新の会幹事長
- **地元での信頼**：枚方市で長年活動

### 弱み
- **高齢**：68歳で今後の活動期間に制約
- **政党変遷**：自民党から維新への移籍歴
- **全国的知名度**：地元以外での認知度は限定的
