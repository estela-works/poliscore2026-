# 岡下昌平（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年3月8日 |
| 年齢 | 49歳 |
| 選挙区 | 大阪17区（堺市西区・中区・南区） |
| 当選回数 | 衆議院2期（元職）、大阪府議1期 |
| 主な役職 | 元デジタル大臣政務官兼内閣府大臣政務官、元大阪府議会議員 |

## 経歴
- 1975年3月8日、東京都目黒区生まれ
- 暁星国際高等学校卒業
- 日本大学経済学部経済学科卒業
- 父・岡下昌浩の秘書
- 2000年〜、母・岡下信子（元衆議院議員）の秘書
- 2011年、大阪府議会議員選挙で初当選（1期）
- 2012年、母・岡下信子が馬場伸幸に敗れ落選
- 2014年、第47回衆院選で大阪17区から出馬、馬場に敗北も比例復活で初当選
- 2017年、第48回衆院選で比例復活当選（2期目）
- デジタル大臣政務官兼内閣府大臣政務官
- 二階派所属
- 2021年、第49回衆院選で落選
- 2023年、自民党大阪17区支部長に再任
- 2024年、第50回衆院選で馬場に敗れ落選

## 評価詳細

### ① 課題設定能力: 2/5
**評価根拠**: デジタル政務官としてデジタル化推進に関与。ただし独自の政策課題設定は限定的。

### ② 解決設計力: 3/5
**評価根拠**: 日本大学経済学部卒。府議・国会議員としての行政経験。デジタル政務官として政策実務を担当。

### ③ 実行力: 2/5
**評価根拠**: 衆院選で4回連続馬場伸幸に敗北。小選挙区で1度も勝てず、比例復活2回、落選2回。

### ④ 説明責任: 2/5
**評価根拠**: 国会での活動は限定的。有権者への情報発信も標準以下。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党一筋で活動継続。二階派所属。母の地盤を継ぐ世襲候補。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 特に大きなスキャンダルは報じられていない。

## 強み・弱み

### 強み
- **政務官経験**: デジタル・内閣府大臣政務官
- **自民党基盤**: 公明党推薦を受ける
- **世襲**: 母・岡下信子の地盤継承
- **若さ**: 49歳で今後に期待
- **粘り強さ**: 4回落選しても挑戦継続

### 弱み（注意点）
- **馬場に勝てない**: 4回連続敗北
- **小選挙区未勝利**: 比例復活のみ
- **落選中**: 2021年・2024年と連続落選
- **大阪での自民弱体**: 維新の牙城
- **知名度**: 全国的認知度は低い
