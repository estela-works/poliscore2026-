# 馬場伸幸（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1965年1月27日 |
| 年齢 | 59歳 |
| 選挙区 | 大阪17区（堺市西区・中区・南区） |
| 当選回数 | 衆議院5期、堺市議6期 |
| 主な役職 | 日本維新の会顧問（元代表）、元堺市議会議長 |

## 経歴
- 1965年1月27日生まれ
- 大阪府立鳳高等学校卒業
- オージー・ロイヤル（現ロイヤルフードサービス）入社、コックとして勤務
- 調理師免許取得
- 1986年、中山太郎参議院議員秘書
- 1993年、堺市議会議員初当選（6期19年）
- 2006年〜2007年、堺市議会副議長
- 2011年〜2012年、堺市議会議長
- 市議時代は自民党所属、2010年離党し大阪維新の会結成に参加
- 2012年、第46回衆院選で大阪17区から初当選
- 日本維新の会幹事長、共同代表を歴任
- 2022年8月、日本維新の会代表就任（得票率78.8%）
- 2024年10月、第50回衆院選で5選
- 2024年11月、代表選不出馬を表明
- 2025年8月、日本維新の会顧問就任

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 憲法改正、首相公選制、地方分権強化など維新の政策課題を設定。ただし独自色は限定的で党方針に沿った活動。

### ② 解決設計力: 3/5
**評価根拠**: 高卒、調理師から政治家への転身。堺市議19年の地方行政経験。国会議員としての政策立案は標準的。

### ③ 実行力: 4/5
**評価根拠**: 5期連続当選の安定した選挙基盤。代表として党を率いるも、2024年衆院選で維新6議席減の責任を問われる。大阪17区では圧勝。

### ④ 説明責任: 3/5
**評価根拠**: 党代表として記者会見等で発言。ただし選挙敗北後の責任の取り方は党内から批判。

### ⑤ 一貫性: 3/5
**評価根拠**: 自民党から維新へ移籍。維新内では幹事長・共同代表・代表と昇格。代表辞任後は顧問に。

### ⑥ 倫理観・権力の扱い方: 2/5
**評価根拠**: 2023年、野党第一党の立憲民主党を「日本の利益よりも他国の利益のために動く政党」と批判し波紋。他党への攻撃的な発言が問題視されることも。

## 強み・弱み

### 強み
- **大阪での強さ**: 大阪17区で5期連続当選
- **党内基盤**: 代表・幹事長を歴任
- **地方経験**: 堺市議6期19年の経験
- **たたき上げ**: コックから政治家への異色経歴
- **実務能力**: 市議会議長経験

### 弱み（注意点）
- **代表辞任**: 2024年衆院選敗北で責任論
- **攻撃的発言**: 野党批判での失言
- **学歴**: 高卒（大学受験失敗）
- **党内対立**: 浅田参院会長らとの確執
- **維新低迷**: 党勢縮小の責任
