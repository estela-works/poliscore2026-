# 吉田知代（日本維新の会）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1975年2月23日 |
| 年齢 | 50歳 |
| 選挙区 | 徳島県1区 |
| 当選回数 | 衆議院1回（比例復活）、元丹波篠山市議2期 |
| 主な役職 | 日本維新の会徳島1区支部長、徳島維新の会代表 |

## 経歴
- 祖父母が徳島県出身
- 松蔭女子中学・高等学校（神戸市）卒業
- 神戸松蔭女子学院短期大学卒業（1995年）
- トヨタグループ輸送機器メーカー勤務
- 損害保険会社勤務
- 丹波篠山市議会議員（2期）
- 2021年衆院選徳島1区から維新公認で出馬、比例復活で初当選
- 2022年「徳島維新の会」設立、代表就任
- 2024年衆院選で落選（小選挙区・比例とも）
- シングルマザー

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 維新の会の政策（身を切る改革、規制改革）を訴える。シングルマザーとして子育て政策への関心。地方創生への意欲。

### ② 解決設計力: 2/5
**評価根拠**: 短期大学卒、民間企業での勤務経験。市議2期の経験はあるが、国政レベルの政策立案経験は限定的。

### ③ 実行力: 2/5
**評価根拠**: 2021年比例復活当選も、2024年は落選。地元との軋轢で県支部が一度解散する事態も経験。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 「国政で戦うシングルマザー」として発信。ただし地元との対立も報じられた。

### ⑤ 一貫性: 3/5
**評価根拠**: 維新から一貫して活動。ただし地元との関係構築に課題があった時期も。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。クリーンなイメージ。

## 強み・弱み

### 強み
- **国政経験**: 衆議院議員1期の経験
- **当事者性**: シングルマザーとしての視点
- **党の支援**: 維新の会の組織力
- **地方議会経験**: 丹波篠山市議2期
- **再挑戦**: 落選から再起を図る姿勢

### 弱み（注意点）
- **2024年落選**: 前回選挙で議席を失う
- **地元との軋轢**: 県支部解散の経緯
- **地縁の弱さ**: 祖父母が徳島出身だが本人は兵庫育ち
- **選挙の弱さ**: 小選挙区で勝利したことがない
