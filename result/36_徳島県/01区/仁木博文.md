# 仁木博文（自由民主党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1966年5月23日 |
| 年齢 | 58歳 |
| 選挙区 | 徳島県1区 |
| 当選回数 | 衆議院3回 |
| 主な役職 | 厚生労働副大臣、医師、博士（医学） |

## 経歴
- 徳島県阿南市出身
- 阿南市立宝田小学校卒業
- 徳島大学教育学部附属中学校卒業
- 徳島市立高等学校理数科卒業
- 東京大学教養学部基礎科学科卒業（1991年）
- 徳島大学医学部卒業（1997年）、医師免許取得
- 高知市立市民病院、国立高知病院、宝塚市立病院勤務
- 徳島大学大学院博士（医学）取得（2014年）
- 2003年、2005年衆院選徳島3区から民主党公認で出馬、落選
- 2009年衆院選徳島3区から出馬、比例復活で初当選（民主党）
- 2014年衆院選徳島1区から民主党公認で出馬、落選
- 2017年衆院選希望の党公認で出馬、落選
- 2021年衆院選無所属で出馬、小選挙区当選（後藤田正純らに勝利）
- 2021年「有志の会」結成
- 2023年自民党徳島1区支部長に就任（後藤田の知事選出馬に伴う）
- 2023年麻生派入会
- 2024年衆院選自民党公認で3選
- 厚生労働副大臣（第2次石破内閣）

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 医師として医療現場を知る。厚生労働副大臣として社会保障・医療政策の課題に取り組む。博士号を持つ専門家。

### ② 解決設計力: 4/5
**評価根拠**: 東京大学・徳島大学医学部卒、博士（医学）。医師としての専門性と政策立案能力。厚労副大臣として政策に関与。

### ③ 実行力: 4/5
**評価根拠**: 3期当選。2021年には無所属で後藤田正純に勝利という実績。厚労副大臣として政策実行。粘り強く落選から復活。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 自民党入党時「与党でなければできないことがある。苦渋の決断だった」と説明。ただし党派変遷の説明には批判も。

### ⑤ 一貫性: 2/5
**評価根拠**: 民主党→希望の党→無所属→自民党と大きく党派を変遷。政策的に一貫しているかは疑問が残る。

### ⑥ 倫理観・権力の扱い方: 3/5
**評価根拠**: 大きなスキャンダルは確認されていない。ただし野党→与党への転身は「権力への接近」と批判される可能性。

## 強み・弱み

### 強み
- **専門性**: 医師・博士（医学）、医療政策の専門家
- **学歴**: 東大・徳島大医学部卒
- **政府経験**: 厚生労働副大臣
- **選挙力**: 2021年に後藤田正純に勝利
- **粘り強さ**: 複数回の落選から復活

### 弱み（注意点）
- **党派変遷**: 民主党→希望→無所属→自民と変遷
- **転向批判**: 野党から与党への転身
- **派閥政治**: 麻生派所属
- **説明責任**: 党派変遷の理由に対する疑問
