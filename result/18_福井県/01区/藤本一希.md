# 藤本一希（参政党）

## 基本情報
| 項目 | 内容 |
|------|------|
| 生年月日 | 1996年5月9日 |
| 年齢 | 29歳 |
| 選挙区 | 福井県1区 |
| 当選回数 | 新人（国政）、福井県議1期 |
| 主な役職 | 参政党青年局長、福井県議会議員、株式会社REGIE代表取締役 |

## 経歴
- 福井県鯖江市生まれ
- 福井県立藤島高等学校卒業
- 福井大学医学部医学科入学（後に中退）
- 医学部休学中に医療ITサービス会社社員、医療法人社団理事を経験
- 2020年大学中退、医療系キャリア支援会社を起業
- 株式会社REGIE代表取締役
- 2021年ふくい若者フォーラム会長
- 2022年一般社団法人KISA2隊理事、株式会社介護こねくてぃぶ取締役
- 2023年福井県議選で福井市選挙区から参政党公認で初当選
- 参政党ボードメンバー、青年局長
- 2026年衆院選福井1区に出馬表明

## 評価詳細

### ① 課題設定能力: 3/5
**評価根拠**: 減税・積極財政、外国人問題、少子化問題を課題として挙げる。「このままでは日本は危ない」という危機感を持つ。参政党の政策に沿った課題設定。

### ② 解決設計力: 3/5
**評価根拠**: 福井県立藤島高校卒、福井大学医学部中退。起業家として会社経営経験。医療系の知識と経営経験を持つが、国政レベルの政策立案経験はこれから。

### ③ 実行力: 3/5
**評価根拠**: 県議1期当選。起業家として複数の役職を務める。参政党青年局長として党活動。国政での実績はまだない。

### ④ 説明責任・コミュニケーション力: 3/5
**評価根拠**: 若者フォーラム会長として若者への発信。記者会見で政策を説明。SNSでの発信活動。

### ⑤ 一貫性: 3/5
**評価根拠**: 参政党で活動。医学部中退→起業というキャリアの転換あり。政治活動歴は短い。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。若い挑戦者として地道な活動。

## 強み・弱み

### 強み
- **若さ**: 29歳、最年少候補
- **起業経験**: 会社経営者としての実績
- **医療知識**: 医学部での学習経験
- **地元出身**: 福井県鯖江市出身
- **党内地位**: 参政党青年局長
- **県議経験**: 地方議会での1期の経験

### 弱み（注意点）
- **学歴**: 医学部中退
- **政治経験浅い**: 県議1期のみ
- **知名度**: 稲田朋美に比べ低い
- **参政党**: 小政党で組織力に限界
