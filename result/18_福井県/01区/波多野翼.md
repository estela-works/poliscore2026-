# 波多野翼（中道改革連合）

## 基本情報
| 項目 | 内容 |

| 生年月日 | 1984年11月6日 |
| 年齢 | 41歳 |
| 選挙区 | 福井県1区 |
| 当選回数 | 衆議院1回 |
| 主な役職 | 立憲民主党衆議院議員 |

## 経歴
- 福井県武生市（現・越前市）出身
- 福井県立鯖江高等学校卒業
- 仁愛大学人間学部コミュニケーション学科卒業
- ビーエムサプライ株式会社従業員
- 2009年越前市役所入庁
- 2010年日本ボルガラー協会を結成、会長就任
- ご当地グルメ「ボルガライス」の普及を通じたまちづくりの立役者
- 2019年「地方公務員が本当にすごい!と思う地方公務員アワード」受賞
- 2019年より絵本作家としても活動開始
- 2024年10月越前市役所退職
- 2024年衆院選立憲民主党公認で出馬、比例復活で初当選
- 3姉妹の父

## 評価詳細

### ① 課題設定能力: 4/5
**評価根拠**: 「福井の可能性を引き出す」「子育てしたい環境を作り出す」「十分な食料を作り出す」という3つの政策を明確に掲げる。地方公務員としての経験から市民生活の課題を把握。

### ② 解決設計力: 3/5
**評価根拠**: 仁愛大学卒、地方公務員としての15年の経験。ボルガライス普及による地域活性化の実績。国政での経験はまだ浅い。

### ③ 実行力: 3/5
**評価根拠**: 1期目で比例復活当選。地方公務員アワード受賞の実績。ボルガライス協会の活動で地域活性化を実現。

### ④ 説明責任・コミュニケーション力: 4/5
**評価根拠**: 「若者を含めて政治に関心を持つ必要がある」と発信。絵本作家として子どもへのコミュニケーション能力。市民目線での政治活動。

### ⑤ 一貫性: 4/5
**評価根拠**: 地方公務員として15年間市民生活に関わり、その延長線上で国政に挑戦。地域活性化への一貫した取り組み。

### ⑥ 倫理観・権力の扱い方: 4/5
**評価根拠**: スキャンダルは確認されていない。地方公務員として市民に寄り添う姿勢。クリーンなイメージ。

## 強み・弱み

### 強み
- **地方行政経験**: 15年間の市役所勤務
- **地域活性化実績**: ボルガライス普及、公務員アワード受賞
- **市民目線**: 生活者視点での政策
- **若さ**: 41歳、子育て世代の当事者
- **クリーン**: スキャンダルなし
- **多彩な活動**: 絵本作家としても活動

### 弱み（注意点）
- **国政経験浅い**: 1期目、比例復活
- **知名度**: 稲田朋美に比べ低い
- **選挙区敗北**: 2024年小選挙区では敗れた
